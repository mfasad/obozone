Где найти отчёт о реализации на Ozon: 5 проверенных способов для продавцов

Отчёт о реализации на Ozon — это ключевой документ для продавцов, который содержит детализированную информацию о продажах, возвратах, комиссиях и финансовых операциях. Без него невозможно вести точный учёт доходов, планировать закупки или анализировать эффективность работы. Однако многие новички сталкиваются с проблемой: где именно искать этот отчёт в личном кабинете, как его скачать в удобном формате и что делать, если данные не совпадают с реальными продажами.

В этой статье мы разберём все актуальные способы получения отчёта о реализации на Ozon в 2026 году — от стандартного экспорта через личный кабинет до автоматизированных решений через API. Также вы узнаете, как читать отчёт, какие ошибки чаще всего допускают продавцы и как избежать расхождений с бухгалтерией. Если вы продаёте на Ozon по модели FBS или FBO, эта инструкция поможет систематизировать финансовые данные и сэкономить время на ручной обработке.

1. Что такое отчёт о реализации на Ozon и зачем он нужен

Отчёт о реализации (или отчёт по продажам) — это документ, который формирует Ozon для каждого продавца. Он содержит информацию о:

  • 📦 Проданных товарах — количество, артикулы, цены, скидки.
  • 💰 Финансовых операциях — суммы заказов, комиссии маркетплейса, вознаграждения за логистику.
  • 🔄 Возвратах и отменах — причины, суммы удержаний, статус возмещения.
  • 📊 Итоговых выплатах — сколько денег зачислено на счёт продавца после всех удержаний.

Этот отчёт нужен для:

  • 📈 Аналитики продаж — понимания, какие товары приносят прибыль, а какие уходят в убыток.
  • 📑 Бухгалтерского учёта — подтверждения доходов для налоговой (особенно важно для ИП и ООО).
  • 🔍 Контроля комиссий — проверки, правильно ли Ozon удерживает проценты за продажи и логистику.
  • 📉 Планирования закупок — прогнозирования спроса на основе исторических данных.

Без этого отчёта продавец работает "вслепую": не видит реальную рентабельность, не может отследить ошибки в выплатах и рискует получить штрафы от налоговой за несоответствие деклараций. Например, если в отчёте указано, что вам выплатили 100 000 ₽, а по факту на счёт поступило 95 000 ₽, это повод разобраться, куда делись 5 000 ₽ — возможно, Ozon удержал комиссию за хранение или штраф за нарушение правил.

💡

Сверяйте данные из отчёта о реализации с выписками из банка не реже 1 раза в неделю. Это поможет оперативно выявить ошибки в выплатах или неучтённые комиссии.

2. Где искать отчёт о реализации в личном кабинете Ozon Seller

Основной способ получения отчёта — через личный кабинет Ozon Seller. Инструкция актуальна для всех моделей работы (FBS, FBO, DBS):

  1. Авторизуйтесь на сайте seller.ozon.ru под своим логином и паролем.

  2. В верхнем меню выберите раздел Финансы → Отчёты.

  3. В блоке Отчёты по продажам найдите вкладку Отчёт о реализации (иногда называется Детализация продаж).

  4. Укажите период, за который нужен отчёт (по умолчанию — текущий месяц).

  5. Нажмите Сформировать и дождитесь генерации файла (может занять до 5 минут).

  6. Скачайте отчёт в формате Excel (.xlsx) или CSV.

Если вкладка Отчёт о реализации отсутствует, проверьте:

  • 🔹 Наличие активных заказов за выбранный период (если продаж не было, отчёт не формируется).
  • 🔹 Права доступа вашего аккаунта (возможно, у вас роль "Менеджер" без прав на финансовые отчёты).
  • 🔹 Регион продаж (отчёты по Ozon Global формируются отдельно).
Что делать, если отчёт не формируется?

Если после нажатия Сформировать появляется ошибка или отчёт пустой, попробуйте:

1. Сменить браузер (рекомендуется Google Chrome или Mozilla Firefox).

2. Очистить кэш и cookies.

3. Уменьшить период (например, вместо месяца запросить данные за неделю).

4. Обратиться в поддержку Ozon через чат в личном кабинете с указанием ID вашего аккаунта и скриншотом ошибки.

3. Как скачать отчёт о реализации через API Ozon

Если вам нужно автоматизировать получение отчётов (например, для интеграции с 1С, МойСклад или другими системами учёта), используйте API Ozon. Это актуально для продавцов с большим объёмом продаж (100+ заказов в день), когда ручной экспорт занимает слишком много времени.

Для работы с API потребуется:

  • 🔑 Client-ID и API-ключ (генерируются в личном кабинете в разделе Настройки → API).
  • 💻 Знания основ программирования (или помощь разработчика) для отправки запросов.
  • 📡 Доступ к серверу (можно использовать Postman для тестовых запросов).

Пример запроса для получения отчёта о реализации (метод POST /v2/finance/transaction/total):

{

"date": {


"from": "2026-01-01T00:00:00Z",


"to": "2026-01-31T23:59:59Z"


},


"page": 1,


"page_size": 1000


}

В ответ Ozon вернёт JSON с данными о транзакциях, которые можно преобразовать в удобный формат. Официальная документация API доступна по ссылке: docs.ozon.ru/api/seller.

Преимущества API:

  • ⚡ Автоматизация — отчёты формируются без ручного вмешательства.
  • 🔄 Актуальность данных — можно получать информацию в реальном времени.
  • 📊 Интеграция с другими системами — например, автоматически загружать продажи в 1С.

Недостатки:

  • 🛠️ Требуются технические навыки для настройки.
  • 🔒 Ограничения по количеству запросов (например, не более 1000 запросов в час).
  • 📈 Сложность обработки больших объёмов данных (нужно оптимизировать скрипты).
💡

API Ozon позволяет получать данные не только по реализации, но и по остаткам, заказам, возвратам. Это полноценный инструмент для автоматизации бизнеса на маркетплейсе.

4. Альтернативные способы получения отчёта о продажах

Если стандартный экспорт через личный кабинет или API не подходит, есть ещё несколько способов получить данные о реализации:

4.1. Отчёт через Ozon Business (для юридических лиц)

Крупные продавцы, работающие через ООО, могут запрашивать расширенные финансовые отчёты в разделе Ozon Business. Здесь доступны:

  • 📄 Акты выполненных работ — для бухгалтерского учёта.
  • 💳 Счета-фактуры — если вы платите НДС.
  • 📈 Аналитика по категориям — детализация продаж по группам товаров.

4.2. Экспорт через Ozon Partner (для франчайзи)

Если вы работаете по франшизе Ozon (например, открыли ПВЗ), отчёты о продажах доступны в личном кабинете Ozon Partner. Здесь данные разделены по:

  • 🏬 Точкам выдачи — если у вас несколько ПВЗ.
  • 📦 Типам заказов — FBS, FBO, DBS.
  • 💰 Видам дохода — комиссии за выдачу, бонусы за объём.

4.3. Отчёт от бухгалтерии Ozon (по запросу)

В редких случаях (например, при расхождениях в данных) можно запросить отчёт напрямую у бухгалтерии Ozon. Для этого:

  1. Напишите в поддержку через чат в личном кабинете.
  2. Укажите период, за который нужен отчёт.
  3. Приложите скриншоты расхождений (если они есть).
  4. Дождитесь ответа (обычно до 3 рабочих дней).

Этот способ самый долгий, но иногда необходим, если другие методы не дают корректных данных.

💡

Если вы ведёте учёт в Excel, настройте сводные таблицы (PivotTable) для автоматического анализа отчётов. Это сэкономит часы ручной работы.

5. Распространённые ошибки при работе с отчётом о реализации

Даже опытные продавцы иногда сталкиваются с проблемами при анализе отчётов. Вот самые частые ошибки и как их избежать:

5.1. Несовпадение сумм с банковскими выписками

Если в отчёте указано, что вам выплатили 50 000 ₽, а на счёт пришло 48 000 ₽, проверьте:

  • 🔹 Комиссии за логистику (FBS или FBO).
  • 🔹 Штрафы за нарушение правил (например, за просроченную отправку).
  • 🔹 Удержания по возвратам (если товар вернулся после выплаты).
  • 🔹 Налоги (если вы на НПД или УСН, Ozon может удерживать налоговый агент).

5.2. Отсутствие данных по некоторым заказам

Если в отчёте не отображаются отдельные продажи, причины могут быть следующими:

  • 🔹 Заказ ещё не оплачен покупателем (статус Не оплачен).
  • 🔹 Заказ отменён до оплаты.
  • 🔹 Ошибка в фильтрах при формировании отчёта (проверьте даты).
  • 🔹 Технический сбой на стороне Ozon (решается через поддержку).

5.3. Неправильный учёт возвратов

Многие продавцы забывают, что возвращённый товар может:

  • 🔹 Вернуться на склад (тогда его можно продать снова).
  • 🔹 Быть списан (если товар повреждён).
  • 🔹 Потребовать возмещения (если вина Ozon или перевозчика).

В отчёте возвраты отображаются со знаком минус, но не всегда понятно, какой именно товар вернулся. Чтобы избежать путаницы, ведите отдельную таблицу возвратов с указанием причины и статуса.

💡

Всегда сверяйте данные отчёта о реализации с отчётом по остаткам (Финансы → Остатки). Это поможет выявить неучтённые возвраты или списания.

5.4. Ошибки в валюте (для Ozon Global)

Если вы продаёте за рубеж, в отчёте могут быть суммы в долларах, евро или тенге. Обратите внимание на:

  • 🔹 Курс конвертации (Ozon использует курс ЦБ на день продажи).
  • 🔹 Комиссии за международную доставку (они выше, чем в России).
  • 🔹 Налоги в стране покупателя (например, НДС в Казахстане).
Как конвертировать валюту в отчёте?

Если вам нужно перевести все суммы в рубли, используйте формулу в Excel:

=B2*ГОДОВОЙ_КУРС, где B2 — сумма в иностранной валюте, а ГОДОВОЙ_КУРС — средний курс ЦБ за период. Актуальные курсы можно скачать на сайте ЦБ РФ.

6. Как анализировать отчёт о реализации: практические советы

Просто скачать отчёт недостаточно — нужно уметь извлекать из него полезную информацию. Вот несколько лайфхаков для анализа:

6.1. Расчёт реальной рентабельности товара

Чтобы понять, приносит ли товар прибыль, учитывайте не только цену продажи, но и:

  • 💰 Себестоимость товара (закупка + доставка на склад Ozon).
  • 📦 Комиссию Ozon (от 5% до 15% в зависимости от категории).
  • 🚚 Логистические расходы (если работаете по FBS).
  • 🔄 Затраты на возвраты (в среднем 5-10% от продаж).
  • 📉 Хранение на складе (если товар лежит дольше 30 дней).

Формула рентабельности:



Рентабельность (%) = (Цена продажи - Себестоимость - Комиссия - Логистика - Возвраты - Хранение) / Себестоимость × 100

6.2. Выявление самых прибыльных и убыточных товаров

Отсортируйте отчёт по столбцу Прибыль на единицу (если его нет, добавьте формулу в Excel). Товары с отрицательной прибылью нужно:

  • 🔹 Пересмотреть цену (увеличить или убрать скидки).
  • 🔹 Оптимизировать логистику (например, перейти на FBO).
  • 🔹 Снять с продажи (если исправить рентабельность невозможно).

6.3. Анализ сезонности и трендов

Сравнивайте отчёты по месяцам, чтобы выявить:

  • 📈 Пики продаж (например, перед Новым годом или Чёрной пятницей).
  • 📉 Спады (когда спрос падает, можно снизить запасы).
  • 🔄 Влияние акций (как скидки afect на объём продаж).

Для визуализации используйте графики в Excel или Google Sheets:

  1. Выделите данные за год.
  2. Постройте график по месяцам.
  3. Добавьте линию тренда.
💡

Настройте в Excel условное форматирование для столбца Прибыль: зелёный — прибыльные товары, красный — убыточные. Это визуально упростит анализ.

6.4. Контроль комиссий Ozon

Ozon регулярно меняет тарифы, поэтому важно отслеживать:

  • 🔹 Базовую комиссию (зависит от категории).
  • 🔹 Дополнительные сборы (например, за Premium-размещение).
  • 🔹 Штрафы (за нарушение SLA).

Если комиссия в отчёте выше ожидаемой, проверьте:

  • 🔹 Не изменились ли тарифы (актуальные rates есть в документации Ozon).
  • 🔹 Нет ли скрытых платежей (например, за хранение свыше 60 дней).
  • 🔹 Правильно ли указан вес/габариты товара (от этого зависит стоимость доставки).

7. Частые вопросы о отчёте о реализации на Ozon

Можно ли получить отчёт о реализации за прошлый год?

Да, в личном кабинете доступны данные за последние 24 месяца. Для более ранних периодов нужно обращаться в поддержку с обоснованием (например, для налоговой проверки).

Почему в отчёте не совпадают суммы с выпиской из банка?

Наиболее частые причины:

  • 🔹 Ozon удержал комиссию за логистику или штраф.
  • 🔹 Выплата прошла не в тот день (задержки до 3 дней).
  • 🔹 Банк удержал свою комиссию за перевод.

Сверяйте данные по колонке Итоговая сумма к выплате в отчёте.

Как экспортировать отчёт в 1С или МойСклад?

Есть два способа:

  1. Ручной импорт: скачайте отчёт в Excel, преобразуйте в формат CSV и загрузите в 1С через Обмен данными.
  2. Автоматизация: настройте интеграцию через API (потребуется помощь программиста).

Для МойСклад есть готовые коннекторы, например, через сервис Alto или СберМегаМаркет.

Что делать, если в отчёте ошибка (не хватает заказов или сумм)?

Алгоритм действий:

  1. Проверьте фильтры (даты, статус заказов).
  2. Сравните с отчётом по заказам (Продажи → Заказы).
  3. Обратитесь в поддержку Ozon с указанием:
    • ID заказов, которых нет в отчёте.
    • Скриншоты расхождений.
    • Период формирования отчёта.

Обычно ошибки исправляют в течение 1-2 рабочих дней.

Можно ли получить отчёт о реализации по конкретному товару?

Да, для этого:

  1. Сформируйте общий отчёт за нужный период.
  2. Отфильтруйте по артикулу или названию товара.
  3. Используйте сводную таблицу для группировки данных.

    4. Также можно использовать отчёт Продажи по товарам в разделе Аналитика.

8. Заключение: как сделать работу с отчётами эффективнее

Отчёт о реализации на Ozon — это не просто "бумажка для бухгалтерии", а мощный инструмент для роста продаж. Чтобы использовать его максимально эффективно:

  • 📅 Автоматизируйте получение отчётов через API или интеграции с 1С.
  • 📊 Анализируйте данные еженедельно, а не только перед декларацией.
  • 🔍 Сверяйте суммы с банком не реже 1 раза в неделю.
  • 📈 Используйте отчёт для планирования закупок и акций.
  • 🛠️ Настройте шаблоны в Excel для быстрого анализа.

    • Если вы только начинаете продавать на Ozon, начните с ручного экспорта отчётов и постепенного освоения API. Для опытных продавцов рекомендуем автоматизировать все процессы, чтобы сосредоточиться на развитии бизнеса, а не на рутинной работе с таблицами.

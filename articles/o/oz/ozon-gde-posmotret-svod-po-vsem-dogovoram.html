Как посмотреть свод по всем договорам на Ozon: 5 рабочих способов

Работа с договорами на Ozon — неотъемлемая часть деятельности продавцов, поставщиков и партнеров маркетплейса. Без доступа к сводной информации по соглашениям невозможно контролировать финансовые потоки, сроки действия контрактов или условия сотрудничества. Однако найти эту информацию не всегда просто: интерфейс Ozon Seller регулярно обновляется, а данные разбросаны по разным разделам.

В этой статье мы разберем все актуальные способы просмотра свода по договорам — от стандартных инструментов личного кабинета до скрытых функций API. Вы узнаете, как получить данные по FBO/FBS-схемам, договорам с логистическими партнерами, а также как экспортировать информацию для бухгалтерии или аудита. Особое внимание уделим новым ограничениям 2026 года, которые затрудняют доступ к некоторым разделам без подтверждения прав.

1. Личный кабинет Ozon Seller: где искать договора

Основной источник информации для продавцов — раздел "Договора" в Ozon Seller. Чтобы туда попасть, выполните следующие шаги:

  1. Авторизуйтесь на сайте seller.ozon.ru.
  2. В левом меню выберите пункт Финансы → Договора.
  3. Откроется список активных соглашений, где можно фильтровать данные по типу (FBO, FBS, логистика) и статусу.

В этом разделе доступны:

  • 📄 Тексты договоров в PDF-формате (скачиваются по кнопке "Скачать").
  • 📅 Сроки действия с уведомлениями о продлении/расторжении.
  • 💰 Финансовые условия: комиссии, штрафы, бонусы.
  • 🚚 Логистические обязательства (для FBS-партнеров).

⚠️ Внимание: С 2026 года Ozon скрыл часть договоров для продавцов с низким рейтингом (ниже 4.5). Чтобы увидеть полный список, необходимо повысить Индекс доверия или обратиться в поддержку с запросом на расширенный доступ.

2. Мобильное приложение Ozon Seller: удобство или ограничения?

Официальное приложение Ozon Seller (доступно для iOS и Android) позволяет просматривать договора в упрощенном виде. Однако функционал здесь урезан по сравнению с десктопной версией:

Функция Десктопная версия Мобильное приложение
Просмотр текста договора ✅ Полный PDF ✅ Только ключевые пункты
Фильтрация по типу ✅ FBO/FBS/логистика ❌ Нет фильтров
Экспорт в Excel ✅ Да ❌ Нет
Уведомления о изменениях ✅ Email + панель уведомлений ✅ Только push-уведомления

Чтобы найти договора в мобильном приложении:

  1. Откройте раздел Профиль → Документы.
  2. Выберите вкладку Договора.
  3. Для детального просмотра нажмите на нужное соглашение.

🔹 Совет: Если вам нужна полная версия договора, лучше использовать десктопную версию или запросить файл через поддержку. В приложении часто отсутствуют приложения к договорам (например, Правила возвратов или Логистические регламенты).

💡

Сохраните скриншот раздела "Договора" в приложении перед обновлением — иногда после апдейтов интерфейс меняется, и нужные вкладки "прячутся" в другие меню.

3. API Ozon: автоматический сбор данных по договорам

Для продавцов с большим количеством договоров (10+ активных соглашений) ручной просмотр неэффективен. В этом случае поможет API Ozon, который позволяет автоматизировать сбор информации. Основные методы:

  • 🔧 /v1/contract/list — получение списка всех договоров.
  • 📄 /v1/contract/{id}/document — скачивание текста договора по ID.
  • 📊 /v1/contract/finance — финансовые условия (комиссии, штрафы).

Пример запроса для получения списка договоров (на Python):

import requests


url = "https://api-seller.ozon.ru/v1/contract/list"


headers = {


"Client-Id": "Ваш_Client_ID",


"Api-Key": "Ваш_API_Key"


}



response = requests.get(url, headers=headers)


print(response.json())

⚠️ Внимание: Для работы с API требуется подключенный тариф "Профи" (от 1 500 ₽/мес). Бесплатный тариф позволяет делать не более 10 запросов в день, чего недостаточно для полноценного анализа.

Как получить API-ключи для работы с договорами?

1. Перейдите в Настройки → Интеграции → API-ключи в Ozon Seller.

2. Создайте новый ключ с правами contract:read.

3. Скопируйте Client-ID и API-Key — они понадобятся для авторизации.

4. Для тестирования используйте песочницу Ozon API.

4. Экспорт данных в Excel: как анализировать свод по договорам

Для бухгалтерского учета или аудита часто требуется выгрузить данные по договорам в Excel. В Ozon Seller это можно сделать двумя способами:

  1. Ручной экспорт:
    • Перейдите в Финансы → Договора.
    • Нажмите кнопку Экспорт (в правом верхнем углу).
    • Выберите формат: XLSX или CSV.
  • Автоматический экспорт через API:
    • Используйте метод /v1/contract/export.
    • Укажите параметры фильтрации (даты, типы договоров).
    • Файл придет на email в течение 10 минут.

      • В экспортированном файле будут следующие данные:

      Столбец Описание
      contract_id Уникальный идентификатор договора.
      type Тип: FBO, FBS, логистика.
      start_date/end_date Дата начала/окончания действия.
      commission_rate Текущая комиссия маркетплейса (в %).
      penalty_rules Условия штрафов (например, за просрочку отгрузки).

      🔹 Важно: В экспортированных данных отсутствуют приложения к договорам (например, Регламент возвратов или Правила акций). Их нужно скачивать отдельно в PDF-формате.

      5. Договора с логистическими партнерами: где искать?

      Если вы работаете по схеме FBS или сотрудничаете с Ozon Logistic, часть договоров может находиться в отдельном разделе. Чтобы их найти:

      1. Перейдите в Логистика → Договора с партнерами.
      2. Выберите вкладку Активные или Архивные.
      3. Для детального просмотра нажмите на номер договора.

    В этом разделе вы найдете:

    • 🚛 Договора на хранение (стоимость складских услуг).
    • 📦 Условия отгрузки (сроки, штрафы за просрочку).
    • 🔄 Правила возвратов (кто оплачивает обратную логистику).
    • 💳 Финансовые гарантии (депозиты, страховки).

      • ⚠️ Внимание: С 1 марта 2026 года Ozon изменил условия хранения для FBS-партнеров. Теперь депозит за хранение взимается не только за товары, лежащие на складе более 90 дней, но и за неликвиды (товары без продаж в течение 30 дней). Проверьте свои договора на наличие этого пункта!

      💡

      Логистические договора обновляются чаще, чем основные — проверяйте их не реже 1 раза в квартал, особенно перед пиковыми сезонами (Новый год, Черная пятница).

      6. Проблемы доступа: что делать, если договора не отображаются

      Иногда продавцы сталкиваются с тем, что часть договоров не видна в личном кабинете. Причины и решения:

      Проблема Причина Решение
      Нет вкладки "Договора" Устаревшая версия Ozon Seller или низкий рейтинг продавца. Обновите браузер или повысьте Индекс доверия до 4.5+.
      Договора отображаются не полностью Ограничения по тарифу (бесплатный аккаунт). Подключите платный тариф или запросите данные через поддержку.
      Нет возможности скачать PDF Технические работы на серверах Ozon. Попробуйте позже или используйте API для экспорта.
      Договор архивный, но не отображается Истек срок хранения (более 2 лет). Запросите копию у менеджера Ozon по email.

      Если ни один из способов не помог, обратитесь в поддержку:

      1. Напишите в чат Ozon Seller (кнопка Помощь в правом нижнем углу).
      2. Укажите тему: "Отсутствует доступ к договорам".
      3. Прикрепите скриншот ошибки и укажите ID продавца.

      🔹 Срок ответа поддержки по таким запросам — до 3 рабочих дней. Если вопрос срочный (например, истекает срок действия договора), звоните на горячую линию: 8 800 333-70-00 (доб. 1 для продавцов).

      7. Как отслеживать изменения в договорах: 3 рабочих метода

      Ozon регулярно обновляет условия сотрудничества, но не всегда уведомляет продавцов заранее. Чтобы не пропустить важные изменения:

      • 🔔 Включите email-уведомления:
        1. Перейдите в Настройки → Уведомления.
        2. Активируйте опцию "Изменения в договорах".
    • 📅 Используйте календарь:
      • Экспортируйте даты окончания договоров в Google Calendar.
      • Настройте напоминания за 30 и 7 дней до истечения.
    • 🤖 Автоматизируйте мониторинг через API:
      • Создайте скрипт, который раз в неделю проверяет метод /v1/contract/changes.
      • Настройте отправку уведомлений в Telegram или на email.

      • ⚠️ Внимание: С 2026 года Ozon ввел автоматическое продление договоров на тех же условиях, если продавец не уведомил о расторжении за 14 дней. Если вы не согласны с новыми условиями, необходимо письменно отказаться через личный кабинет!

      💡

      Самые частые изменения в договорах касаются комиссий (повышение на 1-3%) и штрафов за просрочку отгрузки. Отслеживайте эти параметры в первую очередь.

      Часто задаваемые вопросы

      Могу ли я расторгнуть договор с Ozon в одностороннем порядке?

      Да, но с соблюдением условий:

      • Для FBO-договоров требуется уведомление за 30 дней.
      • Для FBS — за 60 дней (из-за логистических обязательств).
      • Расторжение возможно через раздел Финансы → Договора → Действия → Расторгнуть.

        • Если вы не уложитесь в сроки, Ozon может наложить штраф до 10 000 ₽.

      Где посмотреть историю изменений по договору?

      В личном кабинете история изменений не сохраняется. Чтобы ее получить:

      1. Запросите выписку у поддержки с указанием ID договора.
      2. Используйте API-метод /v1/contract/{id}/history (доступен на тарифе "Профи").
      3. Сравнивайте экспортированные файлы Excel за разные периоды.
    Что делать, если в договоре ошибка (неверная комиссия, срок)?

    Обратитесь в поддержку с доказательствами:

    • Прикрепите скриншот ошибки.
    • Укажите правильные данные (например, "Комиссия должна быть 15%, а не 18%").
    • Ссылку на пункт договора, который нарушен.

    Срок исправления — до 5 рабочих дней. Если ошибка критичная (например, завышенная комиссия), требуйте перерасчета финансов.

    Как проверить, действует ли договор с логистическим партнером?

    Перейдите в Логистика → Договора с партнерами и обратите внимание на:

    • Статус: должен быть "Активен".
    • Дата окончания: если прошла, договор автоматически архивируется.
    • Уведомления: Ozon отправляет email за 30 дней до истечения.

    Если статус "Приостановлен", свяжитесь с менеджером по логистике.

    Можно ли получить свод по договорам всех своих магазинов в одном месте?

    Да, но с ограничениями:

    • В личном кабинете свод доступен только по одному аккаунту.
    • Для нескольких магазинов используйте API с правами multi-account.
    • Альтернатива: экспортируйте данные по каждому аккаунту в Excel и объедините вручную.

      • ⚠️ Для работы с несколькими аккаунтами требуется верификация бизнес-профиля.

Как скачать отчет по продажам в Озон Селлер: все способы и нюансы

Отчеты о продажах в Ozon Seller — это основа для анализа бизнеса, планирования закупок и оптимизации ассортимента. Без них невозможно отследить динамику продаж, выявить популярные товары или понять, какие позиции требуют доработки. Однако многие продавцы сталкиваются с трудностями: где именно искать эти отчеты, как их экспортировать в удобном формате и что делать, если данные не совпадают с реальными продажами.

В этой статье мы разберем все доступные способы скачивания отчетов — от стандартных инструментов в личном кабинете до альтернативных методов через API. Вы узнаете, как настроить автоматическую выгрузку, какие форматы файлов поддерживаются (.xlsx, .csv, .json), и как избежать типичных ошибок при работе с данными. А для тех, кто хочет углубиться, мы подготовили сравнительную таблицу форматов и ответы на частые вопросы.

Если вы только начинаете работать с Ozon или уже опытный продавец, но хотите systematize процесс анализа — эта инструкция поможет сэкономить часы на рутинных операциях. Начнем с базового метода, который доступен всем пользователям платформы.

1. Где найти отчеты по продажам в личном кабинете Ozon Seller

Основной способ получить данные о продажах — через раздел «Аналитика» в личном кабинете. Этот метод не требует дополнительных настроек и подходит даже для новичков. Вот как туда попасть:

1. Авторизуйтесь на сайте seller.ozon.ru под своим аккаунтом продавца.

2. В верхнем меню выберите вкладку «Аналитика».

3. В выпадающем списке кликните на «Отчеты».

Здесь вы увидите несколько типов отчетов, но для анализа продаж наиболее полезны:

  • 📊 «Отчет по продажам» — основные метрики (количество заказов, выручка, средний чек).
  • 📦 «Отчет по товарам» — детализация по каждому SKU (остатки, возвраты, отмены).
  • 💰 «Отчет по финансам» — данные о выплатах, комиссиях и удержаниях.

Обратите внимание: интерфейс Ozon Seller периодически обновляется, и расположение пунктов меню может незначительно меняться. Если вы не видите раздел «Отчеты», проверьте, что у вас права администратора или менеджера с доступом к аналитике.

💡

Если в меню нет вкладки "Аналитика", попробуйте обновить страницу (Ctrl + F5) или очистить кэш браузера. Иногда проблема решается сменой браузера (например, с Safari на Chrome).

2. Пошаговая инструкция: как скачать отчет по продажам

Когда вы оказались в разделе «Отчеты», следуйте этому алгоритму, чтобы экспортировать данные:

1. Выберите тип отчета (например, «Отчет по продажам»).

2. Укажите период (по умолчанию стоит текущий месяц). Можно выбрать фиксированные диапазоны («Неделя», «Месяц», «Квартал») или задать кастомные даты.

3. Нажмите «Сформировать». Система сгенерирует предварительный просмотр данных.

4. Для скачивания кликните на кнопку «Экспорт» (обычно расположена в правом верхнем углу таблицы).

5. Выберите формат файла:

- .xlsx (рекомендуется для дальнейшей обработки в Excel),

- .csv (подходит для импорта в Google Sheets или базы данных),

- .json (для программистов и автоматизированных систем).

Файл начнет скачиваться автоматически. Если этого не произошло, проверьте настройки браузера — возможно, скачивание заблокировано.

3. Альтернативные способы получения отчетов

Помимо ручного скачивания через личный кабинет, у продавцов есть еще два способа получить данные о продажах:

1. Через API Ozon Seller

Если вам нужна автоматическая выгрузка отчетов (например, для интеграции с 1С или Bitrix24), используйте API маркетплейса. Для этого:

  • 🔑 Получите Client-ID и API-Key в настройках аккаунта (раздел «Интеграции»).
  • 📡 Используйте эндпоинт /v2/analytics/sales для запроса данных.
  • 🤖 Настройте скрипт на Python, PHP или другом языке для периодического опроса API.

Пример запроса к API (на Python с использованием библиотеки requests): 

import requests


url = "https://api-seller.ozon.ru/v2/analytics/sales"


headers = {


"Client-Id": "ваш_client_id",


"Api-Key": "ваш_api_key",


"Content-Type": "application/json"


}


params = {


"date_from": "2026-01-01T00:00:00Z",


"date_to": "2026-01-31T23:59:59Z"


}



response = requests.get(url, headers=headers, params=params)


data = response.json()

2. Через сторонние сервисы

Сервисы вроде SellerLab, eLama или Peak предлагают расширенную аналитику с автоматическим импортом данных из Ozon. Они позволяют:

  • 📈 Строить графики динамики продаж.
  • 🔍 Сравнивать показатели с конкурентами.
  • 📌 Настраивать уведомления о изменениях.

Минус таких сервисов — платная подписка (от 500 до 5 000 ₽/месяц). Однако они экономят время, если вам нужны глубокие аналитические инструменты.

Какие данные можно получить через API, но не увидеть в личном кабинете?

Через API доступны расширенные метрики, такие как:

- Детализация по регионам доставки (с точностью до города).

- Данные о просмотрах карточек товаров (impressions).

- Информация о кликах по рекламным кампаниям (если подключен Ozon Advertising).

- История изменений цен и остатков в реальном времени.

Эти данные часто отсутствуют в стандартных отчетах личного кабинета.

4. Форматы отчетов: какой выбрать и почему

Озон предлагает три основных формата для экспорта отчетов. Каждый из них имеет свои плюсы и минусы в зависимости от задач:

Формат Преимущества Недостатки Когда использовать
.xlsx (Excel)
  • Удобен для ручной обработки.
  • Поддерживает формулы и сводные таблицы.
  • Сохраняет форматирование.
  • Занимает больше места.
  • Может тормозить при большом объеме данных.
 Для еженедельного анализа и презентаций.
.csv
  • Легкий и компактный.
  • Универсален для импорта в другие системы.
  • Нет форматирования.
  • Могут быть проблемы с кодировкой (например, кириллица отображается кракозябрами).
 Для автоматизированной обработки и загрузки в базы данных.
.json
  • Подходит для программистов.
  • Легко парсится скриптами.
  • Неудобен для визуального анализа.
  • Требует знаний для обработки.
 Для интеграции с CRM или собственными дашбордами.

Критическая особенность: в формате .csv даты могут отображаться в некорректном формате (например, как текст). Перед импортом в Excel обязательно проверяйте настройки региональных параметров!

Если вы работаете с большими объемами данных (например, более 10 000 строк), лучше использовать .csv или .json, так как .xlsx может «подвисать» при открытии.

5. Частые ошибки и как их избежать

Даже опытные продавцы сталкиваются с проблемами при скачивании отчетов. Вот наиболее распространенные ошибки и способы их решения:

1. Отчет не формируется или висит «в обработке»

Причины:

  • 🕒 Слишком большой временной диапазон (например, год). Разбейте запрос на более короткие периоды.
  • 🛠️ Технические работы на стороне Ozon. Проверьте статус сервисов на status.ozon.ru.

2. Данные в отчете не совпадают с реальными продажами

Это может происходить из-за:

  • 🔄 Задержки обновления данных (обычно до 24 часов).
  • 📦 Неучтенных возвратов или отмен. Сверяйтесь с отчетом «Возвраты».
  • 💳 Непоступивших оплат (например, при отложенном платеже).

3. Файл скачался, но не открывается

Решения:

  • 📄 Для .xlsx: попробуйте открыть через Google Sheets или LibreOffice.
  • 🗜️ Для .csv: проверьте кодировку (должна быть UTF-8).
  • 🔧 Если файл поврежден, запросите отчет заново.
💡

Если данные в отчете кажутся неполными, всегда сверяйте их с разделом «Финансы» → «Выплаты». Там отображаются итоговые суммы за вычетом комиссий и возвратов.

4. В отчете отсутствуют некоторые заказы

Это может быть связано с:

  • 📅 Неправильно выбранным периодом (например, заказ сделан 31 декабря, а вы запрашиваете данные за январь).
  • 🏷️ Фильтрами по статусу (по умолчанию могут не показываться отмененные заказы).
⚠️ Внимание: Если вы скачиваете отчет по продажам для бухгалтерии, учитывайте, что в нем не отображаются удержанные комиссии маркетплейса. Эти данные нужно брать из отчета «Финансы» → «Транзакции».

6. Как автоматизировать скачивание отчетов

Ручное скачивание отчетов отнимает время, особенно если вам нужны данные еженедельно или ежедневно. Вот три способа автоматизации:

1. Настройка расписания в личном кабинете

Ozon Seller позволяет настроить автоматическую отправку отчетов на email:

1. В разделе «Отчеты» выберите нужный тип отчета.

2. Нажмите «Настройки» (шестеренка в правом верхнем углу).

3. Укажите периодичность (ежедневно, еженедельно, ежемесячно) и адрес электронной почты.

4. Сохраните настройки.

Отчеты будут приходить в виде вложений на указанный email в выбранное время.

2. Использование скриптов на Python

Если вы знакомы с программированием, можно написать скрипт для автоматической выгрузки через API. Пример кода для еженедельного скачивания: 

import requests

import schedule


import time


from datetime import datetime, timedelta



def download_report():


url = "https://api-seller.ozon.ru/v2/analytics/sales"


headers = {"Client-Id": "ваш_id", "Api-Key": "ваш_key"}


date_to = datetime.now().strftime("%Y-%m-%d")


date_from = (datetime.now() - timedelta(days=7)).strftime("%Y-%m-%d")



params = {


"date_from": f"{date_from}T00:00:00Z",


"date_to": f"{date_to}T23:59:59Z"


}



response = requests.get(url, headers=headers, params=params)


with open(f"sales_{date_from}_to_{date_to}.json", "w") as f:


f.write(response.text)



Запускаем каждую пятницу в 18:00


schedule.every().friday.at("18:00").do(download_report)



while True:


schedule.run_pending()


time.sleep(60)

3. Интеграция с Google Sheets

С помощью Google Apps Script можно настроить автоматическое обновление данных в таблице. Для этого:

1. Создайте новую таблицу в Google Sheets.

2. Откройте «Расширения» → «Apps Script».

3. Вставьте код для запроса к API Ozon (аналогично примеру выше).

4. Настройте триггер для периодического выполнения скрипта.

Это позволит всегда иметь актуальные данные без ручного скачивания.

⚠️ Внимание: При автоматизации через API следите за лимитами запросов (обычно 1000 запросов в час). Превышение лимита может привести к временной блокировке доступа.

7. Анализ данных: что смотреть в первую очередь

Скачать отчет — это только половина дела. Главное — правильно его проанализировать. Вот ключевые метрики, на которые стоит обратить внимание:

1. Динамика продаж

  • 📈 Рост/падение выручки по сравнению с предыдущим периодом.
  • 📅 Сезонность: какие дни недели или месяцы наиболее прибыльные.

2. Популярные и проблемные товары

  • 🔥 Топ-10 по количеству продаж — что покупают чаще всего.
  • ❌ Товары с высоким процентом возвратов — возможные проблемы с качеством или описанием.

3. Финансовые показатели

  • 💰 Средний чек: растет или падает.
  • 📉 Маржинальность: выручка за вычетом комиссий и логистики.

4. Логистика и доставка

  • 🚚 Сроки доставки: какие регионы наиболее проблемные.
  • 🔄 Процент отмен из-за долгой доставки.

Для визуализации данных можно использовать:

  • 📊 Excel (сводные таблицы и графики).
  • 🖥️ Google Data Studio (для создания дашбордов).
  • 📱 Power BI (для сложного анализа).

Если вы заметили аномалии (например, резкое падение продаж), проверьте:

  • 🛒 Изменилась ли видимость товара (возможно, его «забанли» за нарушения).
  • 💬 Отзывы покупателей (негативные комментарии могут отпугивать новых клиентов).
  • 🔍 Конкурентов: возможно, у них появилось более выгодное предложение.

8. Частые вопросы (FAQ)

Можно ли скачать отчет по продажам за весь период работы на Ozon?

Нет, в личном кабинете доступна выгрузка данных только за последние 3 года. Для более ранних периодов нужно обращаться в поддержку с официальным запросом. Также учитывайте, что при большом временном диапазоне (например, 2–3 года) отчет может формироваться дольше обычного или не сформироваться вовсе. В таких случаях лучше разбивать запрос на более короткие периоды (по кварталам или месяцам).

Почему в отчете не отображаются некоторые заказы?

Это может происходить по нескольким причинам:

  • Заказы были отменены или возвращены — они попадают в отдельные отчеты.
  • Заказы сделаны в другом регионе (если у вас несколько складов, проверьте фильтры по складам).
  • Данные еще не обновились (задержка до 24 часов).
  • Заказы были оформлены через Ozon Global (международные продажи) — они могут учитываться отдельно.

Проверьте настройки фильтров в отчете и сверьте данные с разделом «Заказы».

Как скачать отчет по продажам в мобильном приложении Ozon Seller?

В мобильном приложении функционал ограничен: скачать отчеты в привычном виде нельзя. Однако вы можете:

  • Просматривать основные метрики в разделе «Аналитика».
  • Делать скриншоты экранов с данными (неудобно для большого объема информации).
  • Использовать адаптивную версию сайта seller.ozon.ru в мобильном браузере (рекомендуется Chrome в режиме «Для компьютера»).

Для полноценной работы с отчетами лучше использовать десктопную версию.

Можно ли получить данные о продажах по конкретному товару?

Да, для этого:

  1. Перейдите в раздел «Отчеты» → «Отчет по товарам».
  2. В фильтрах укажите артикул или название товара.
  3. Скачайте отчет в нужном формате.

В результате вы получите детализированную информацию по выбранному SKU: количество продаж, выручка, остатки на складе, возвраты и т. д.

Что делать, если отчет скачался, но данные в нем неполные?

Сначала проверьте:

  • Корректность выбранного периода (возможно, часть данных попала в другой временной промежуток).
  • Наличие фильтров (например, отчет может быть отфильтрован по статусу «Доставлен», а вы ищете все заказы).
  • Актуальность данных (иногда требуется 1–2 дня для полного обновления информации).

Если проблема сохраняется, попробуйте:

  • Скачать отчет в другом формате (например, вместо .xlsx выбрать .csv).
  • Обратиться в поддержку Ozon Seller с указанием ID отчета и временного диапазона.

Где искать документы о продажах на Ozon: 7 проверенных способов для продавцов

Продажи на Ozon сопровождаются десятками документов: от электронных чеков до актов выполненных работ по логистике. Но где их найти, если срочно нужны для бухгалтерии, налоговой или разрешения споров? Эта статья поможет разобраться в хитросплетениях личного кабинета Ozon Seller, API и даже альтернативных источников — без воды и общих фраз.

Мы проанализировали актуальные на 2026 год пути получения документов для схем FBO (склад Ozon) и FBS (самостоятельная доставка), а также выявили типичные ошибки, из-за которых продавцы теряют доступ к критичным бумагам. Здесь нет "советов на все случаи жизни" — только конкретные инструкции с учётом последних обновлений платформы.

Важно: если вы ищете документы для покупателя (чеки, гарантии), перейдите в раздел про личный кабинет клиента. Эта статья ориентирована на продавцов и предпринимателей, которым нужны отчётные данные для бизнеса.

1. Личный кабинет Ozon Seller: где хранятся документы по умолчанию

Основной источник документов — раздел Финансы → Документы в личном кабинете Ozon Seller. Здесь автоматически формируются:

  • 📄 Акты выполненных работ (для FBS/FBO) — подтверждают факт оказания услуг логистики.
  • 💰 Счета-фактуры от Ozon за комиссии и хранение (если вы на ОСН).
  • 📊 Отчёты о продажах (ежедневные/ежемесячные) — с детализацией по заказам.
  • 📋 Товарные накладные (для FBO) — при передаче товара на склад.

Чтобы не потеряться в меню:

  1. Авторизуйтесь на seller.ozon.ru.
  2. Перейдите в Финансы → Документы.
  3. Используйте фильтры по типу документа и периоду (максимум — 3 месяца назад).
  4. Для скачивания нажмите на значок 📥 рядом с нужной строкой.
💡

Если документ не отображается, проверьте статус заказа: акты формируются только для завершённых сделок (включая возвраты).

Ограничения этого метода:

  • ⏳ Документы хранятся не более 3 лет (по умолчанию).
  • 🔍 Нет поиска по номеру заказа — только по дате.
  • 📂 Некоторые файлы (например, акты по FBS) могут "запаздывать" до 5 рабочих дней.

2. API Ozon: автоматическое получение документов для бухгалтерии

Если вам нужны массовая выгрузка или интеграция с 1С, используйте Ozon Seller API. Через него доступны:

  • 📑 Отчёты о продажах (POST /v2/finance/transaction/totals).
  • 📄 Акты и счета-фактуры (POST /v1/finance/documents).
  • 📦 Данные по логистике (для FBS/FBO).

Пример запроса для получения актов за текущий месяц:

curl -X POST 'https://api-seller.ozon.ru/v1/finance/documents' \

-H 'Client-Id: YOUR_CLIENT_ID' \


-H 'Api-Key: YOUR_API_KEY' \


-H 'Content-Type: application/json' \


-d '{


"date": {


"from": "2026-06-01T00:00:00Z",


"to": "2026-06-30T23:59:59Z"


},


"type": ["act", "invoice"]


}'

Ключевые нюансы работы с API:

  • 🔑 Требуется Client-ID и Api-Key (генерируются в Настройки → API-ключи).
  • 📈 Лимит запросов — 60 в минуту (при превышении блокировка на 5 минут).
  • 📂 Документы возвращаются в формате PDF или XML (зависит от типа).
Как получить доступ к API, если его нет в личном кабинете?

Для подключения API необходимо подтвердить статус юридического лица или ИП в настройках профиля. Если раздел API-ключи отсутствует, обратитесь в поддержку Ozon с запросом на активацию, приложив сканы документов (ИНН, ОГРН, паспорт).

Важно! Через API нельзя получить товарные накладные для FBO старше 1 года — их придётся запрашивать вручную у службы поддержки.

3. Электронная почта: где искать уведомления с документами

Ozon отправляет часть документов на email, привязанный к аккаунту продавца. Внимательно проверяйте папки:

  • 📧 Входящие — акты и счета-фактуры (тема письма обычно содержит "Документ от Ozon").
  • 🗃️ Спам — иногда письма попадают сюда из-за фильтров.
  • 📁 Архив — если вы вручную архивировали уведомления.

Как найти нужное письмо:

  1. В поисковой строке почты введите: from:seller@ozon.ru "Акт выполненных работ".
  2. Отсортируйте результаты по дате (от новых к старым).
  3. Проверьте вложения — документы обычно прикреплены в формате PDF.

⚠️ Внимание: Ozon не дублирует документы по запросу на email. При утере придётся скачивать их повторно через личный кабинет или API.

4. Документы для FBS: особенности самостоятельной логистики

При работе по схеме FBS (самостоятельная доставка) продавцу доступны дополнительные документы:

Тип документа Где искать Срок формирования
Акт приёма-передачи заказа курьеру Логистика → FBS → История заказов В день передачи
Товарная накладная (ТН) Email или Финансы → Документы 1-3 рабочих дня
Акт возврата Возвраты → История возвратов После обработки возврата
Счёт-фактура от Ozon за комиссию Финансы → Документы 5-е число следующего месяца

Для FBS критично сохранять:

  • 📦 Накладные на передачу товара курьеру — подтверждают факт передачи ответственности.
  • 🔄 Акты возвратов — нужны для списания товара или возмещения убытков.
  • 💸 Чеки о возмещении ущерба (если товар потерян/повреждён).

Если документ не сформировался:

⚠️ Внимание: При отсутствии акта по FBS в течение 7 дней после передачи заказа курьеру, напишите в поддержку с темой: "Отсутствует акт FBS №[номер заказа]". Приложите фото товарной накладной и подпись курьера.

5. Документы для FBO: складская логистика и хранение

При работе по схеме FBO (склад Ozon) ключевые документы:

  • 📋 Приходная накладная — подтверждает поступление товара на склад.
  • 📤 Акт списания — при утрате или порче товара.
  • 📦 Акт инвентаризации — если проводилась пересчётка.
  • 💰 Счёт за хранение — ежемесячный (формируется 1-го числа).

Где искать:

  1. Приходные накладные: Логистика → FBO → История поступлений.
  2. Акты списания: Логистика → FBO → Списания.
  3. Счета за хранение: Финансы → Документы → Тип: "Счёт".

Особенности FBO:

  • 🔍 Акты списания формируются только после подтверждения Ozon (может занять до 14 дней).
  • 📅 Инвентаризационные акты доступны только после завершения проверки (обычно 3-5 дней).
  • 💸 Счета за хранение приходят даже при нулевом остатке товара (если был хоть 1 день хранения).
💡

Если товар списан по вине Ozon (утрата, порча), акт списания — это основание для возмещения убытков. Без него претензия не будет рассмотрена!

6. Как восстановить утраченные документы

Если документ потерян или не формируется автоматически, действуйте по алгоритму:

Шаг 1. Проверьте архивы

  • 📂 В личном кабинете: Финансы → Документы → Архив (хранится до 3 лет).
  • 💾 На почте: поиск по ключевым словам ("Акт", "Счёт-фактура", номер заказа).
  • 🖥️ В локальных папках: возможно, документ сохранён на компьютере (проверьте Загрузки).

Шаг 2. Запросите дубликат в поддержке

Напишите в службу поддержки Ozon Seller с темой: "Запрос дубликата документа №[тип] от [дата]". В теле укажите:

  • Тип документа (акт, счёт-фактура, накладная).
  • Номер заказа или период.
  • Причину запроса (например, "для налоговой проверки").

Шаг 3. Альтернативные источники

  • 📊 Выписка из банка — если нужны данные о поступлениях (номер заказа указывается в назначении платежа).
  • 📑 Скриншоты личного кабинета — если документ отображался, но не скачивался (подойдёт как временное подтверждение).
  • 📞 Обращение к менеджеру — если документ критичен, позвоните на горячую линию Ozon для продавцов: 8 800 333-70-00 (доб. 1 для FBO/FBS).
Что делать, если Ozon отказывается предоставлять документ?

Если поддержка игнорирует запрос или отказывает без объяснений, напишите официальное письмо на legal@ozon.ru с пометкой "Юридический запрос". Укажите реквизиты вашей компании, номер договора с Ozon и требование предоставить документ в соответствии с п. 4.3 Договора о сотрудничестве. Срок ответа — 10 рабочих дней.

7. Частые ошибки при работе с документами и как их избежать

Даже опытные продавцы сталкиваются с проблемами при поиске документов. Вот топ-5 ошибок и их решения:

Ошибка 1. Документы не формируются автоматически

Причины:

  • ❌ Не подтверждён ИНН/КПП в профиле продавца.
  • ❌ Заказ ещё не завершён (например, возвращается покупателем).
  • ❌ Технический сбой на стороне Ozon (проверьте статус систем на status.ozon.ru).

Ошибка 2. В счёте-фактуре неверные реквизиты

Если в документе указаны чужие ИНН/КПП:

  1. Проверьте данные в Настройки → Реквизиты.
  2. Если ошибка со стороны Ozon, напишите в поддержку с темой "Ошибка в реквизитах счёта-фактуры" и приложите скриншот.

Ошибка 3. Не совпадают суммы в акте и выписке

Разница может возникать из-за:

  • 💸 Удержанных комиссий (проверьте Финансы → Транзакции).
  • 🔄 Возвратов или списаний (смотрите Логистика → Возвраты).
  • 📅 Задержки обновления данных (актуально для FBS).

Ошибка 4. Нет доступа к старым документам

Если документ старше 3 лет:

  • Запросите выписку из архива через поддержку (указывайте точную дату и тип документа).
  • Проверьте резервные копии на почте или в облачном хранилище.

Ошибка 5. Документы на иностранном языке

Акты для заказов с Ozon Global могут быть на английском. Для налоговой потребуется нотариально заверенный перевод.

8. Где покупатель может найти документы о покупке

Если вы ищете документы не как продавец, а как покупатель, воспользуйтесь инструкцией для личного кабинета Ozon:

Чеки и квитанции

  • 🛒 Перейдите в Мои заказы на ozon.ru.
  • 📄 Выберите нужный заказ → "Чек" (в правом верхнем углу).
  • 📥 Скачайте файл в формате PDF или отправьте на email.

Гарантийные талоны

  • 🔧 Для техники: прилагаются к товару в коробке (если продавец предоставляет гарантию).
  • 📋 Электронная гарантия: проверьте в карточке заказа в разделе "Документы".

Акты возврата

  • 🔄 После обработки возврата акт появится в разделе "Мои возвраты".
  • 💰 Если возврат денег задерживается, акт нужен для обращения в поддержку.

Если документ не отображается:

  • Проверьте папку "Архив" в личном кабинете.
  • Напишите в поддержку Ozon через форму обратной связи (раздел "Помощь").

FAQ: Ответы на частые вопросы

Могу ли я получить документы за продажи 2020 года?

Официально Ozon хранит документы не более 3 лет. Для более старых данных:

  1. Проверьте архив почты (письма от seller@ozon.ru).
  2. Обратитесь в поддержку с просьбой восстановить данные из резервной копии (успех не гарантирован).
  3. Используйте выписки из банка — там сохраняются номера заказов и суммы поступлений.
Почему в акте FBS нет подписи курьера?

Это типичная проблема при бесконтактной передаче заказов. Решения:

  • Проверьте фото/видео передачи в мобильном приложении курьера (если использовали OzonCourier).
  • Запросите у курьерской службы скриншот электронной подписи (для СДЭК, Boxberry и др.).
  • Если акт без подписи не принимает бухгалтерия, напишите в поддержку Ozon с требованием переформировать документ.
Как экспортировать все документы за месяц сразу?

Массовая выгрузка возможна через:

  1. API: используйте метод POST /v1/finance/documents с параметром "type": ["act", "invoice"].
  2. 1С/бухгалтерские программы: настройте интеграцию через Ozon Seller API (требуется помощь программиста).
  3. Ручной экспорт:
    1. В Финансы → Документы установите период (месяц).
    2. Нажмите "Экспортировать в Excel" (кнопка в правом верхнем углу).

Ограничение: за один экспорт можно выгрузить не более 500 документов.

Нужно ли хранить документы от Ozon для налоговой?

Да, согласно ст. 23 НК РФ, вы обязаны хранить документы, подтверждающие доходы и расходы, не менее 4 лет. Для Ozon это:

  • 📄 Акты выполненных работ (подтверждают оказание услуг логистики).
  • 💰 Счета-фактуры от Ozon (если вы на ОСН).
  • 📊 Отчёты о продажах (для подтверждения выручки).
  • 📋 Товарные накладные (для FBO/FBS).

Рекомендуем хранить документы в электронном виде (PDF) с резервными копиями на облачном диске.

Можно ли получить документы, если аккаунт продавца заблокирован?

При блокировке доступа:

  1. Напишите в поддержку Ozon с темой "Запрос документов при блокировке аккаунта".
  2. Приложите сканы документов, подтверждающих ваше право на аккаунт (паспорт, ИНН, ОГРН).
  3. Если блокировка из-за задолженности, сначала погасите её (документы предоставят после разблокировки).

В крайнем случае запросите выписку по счёту в банке — там будут номера заказов и суммы, которые можно использовать для восстановления отчётности.

Где найти код товара на Ozon: артикул, SKU и ID в 2026 году

Вы ищете код товара на Ozon, но не знаете, где его взять? Эта проблема знакома и покупателям (когда нужно вернуть товар или уточнить детали заказа), и продавцам (при загрузке каталога, работе с FBS/FBO или решении споров). На платформе используется сразу несколько типов кодов: артикул продавца, SKU Ozon, уникальный ID товара и даже штрихкод — и все они могут понадобиться в разных ситуациях.

В этой статье разберём, где именно искать каждый из кодов — от карточки товара до аналитических отчётов в Ozon Seller. Мы собрали актуальные способы для 2026 года, включая скрытые фишки (например, как получить код через API или экспорт данных). А ещё объясним, зачем вообще нужны эти идентификаторы и чем они отличаются друг от друга. Если вы торопитесь — используйте быструю навигацию по разделам.

Какие виды кодов товара бывают на Ozon и зачем они нужны

На Ozon используется 4 основных типа кодов, и каждый выполняет свою функцию. Часто их путают, поэтому сначала разберёмся с терминологией:

  • 🔖 Артикул продавца — уникальный код, который вы присваиваете товару при загрузке в личном кабинете. Может совпадать с заводским артикулом бренда (например, IPH14PM-128GB-GOLD для iPhone 14 Pro Max). Используется для внутреннего учёта и синхронизации с другими площадками.
  • 📦 SKU Ozon — идентификатор, который автоматически генерирует маркетплейс после модерации товара. Формат: цифры и буквы через дефис (пример: 123456789-01-2). Нужен для работы с FBS, возвратов и аналитики.
  • 🆔 ID товара (Product ID) — числовой идентификатор карточки на Ozon (например, 284637109). Используется в прямой ссылке на товар и API-запросах.
  • 📊 Штрихкод (EAN/UPC) — международный код товара, который сканируется при приёмке на складе FBS. Может быть указан производителем или сгенерирован Ozon.

Почему это важно? Например, при оформлении возврата покупателю нужен SKU Ozon, а продавцу для выгрузки остатков — артикул. Если перепутать коды, система может не принять запрос или перенаправить его на другой товар. Подробнее о каждом типе — в таблице ниже.

Тип кода Пример Где используется Кто присваивает
Артикул продавца SM-A525FZKD-SILVER Синхронизация с 1С, работа с поставщиками Продавец
SKU Ozon 1122334455-01-1 FBS, возвраты, аналитика Ozon (автоматически)
ID товара 198765432 Ссылки, API, промоакции Ozon
Штрихкод (EAN) 4607103781234 Сканирование на складе FBS Производитель или Ozon

Важно: SKU Ozon и ID товара — это не одно и то же! SKU может меняться при редактировании карточки, а ID остаётся постоянным.

Способ 1: Код товара в публичной карточке (для покупателей)

Если вы покупатель и нужно найти код товара для возврата или уточнения деталей, проще всего посмотреть его прямо в карточке. Вот пошаговая инструкция:

  1. Откройте страницу товара на Ozon (пример: iPhone 14 Pro Max).
  2. Прокрутите страницу вниз до блока Характеристики.
  3. Найдите строки:
    • 🔍 Артикул — это артикул продавца (если указан).
    • 📌 Код товара — это ID товара (числа без дефисов).

Если этих данных нет, попробуйте такой лайфхак:

  1. Скопируйте URL товара. Он выглядит так:
    https://www.ozon.ru/product/{здесь-ID}-{название}/
  2. Цифры между /product/ и первым дефисом — это и есть ID товара.

Способ 2: Поиск кода в личном кабинете продавца (Ozon Seller)

Продавцам доступно больше способов найти код товара. Самый надёжный — через Ozon Seller. Инструкция:

  1. Авторизуйтесь в личном кабинете и перейдите в раздел Товары → Мои товары.
  2. Найдите нужный товар в списке. В таблице отобразятся:
    • 📋 Артикул (ваш внутренний код).
    • 🔢 SKU (код Ozon).
    • 🖥️ ID (в колонке «Код товара»).
  • Если нужна дополнительная информация, кликните на строку товара — откроется расширенная карточка с штрихкодом и историей изменений.

    • Для удобства можно экспортировать список товаров в Excel:

    1. Нажмите Экспорт → Товары в правом верхнем углу.
    2. В файле будут все коды, включая SKU, Артикул и ID.

    Способ 3: Как найти код товара в мобильном приложении Ozon

    В мобильном приложении Ozon (для iOS/Android) код товара спрятан глубже, но его можно найти. Алгоритм для покупателей:

    1. Откройте приложение и найдите товар через поиск.
    2. Тапните на карточку товара и прокрутите до блока Подробности.
    3. Нажмите Ещё (три точки в правом верхнем углу) → Информация о товаре.
    4. В открывшемся окне будет строка Код товара — это и есть ID.

    Для продавцов в приложении Ozon Seller:

    1. Перейдите в раздел Товары.
    2. Выберите нужный товар → Редактировать.
    3. Вверху экрана будет указан SKU, а внизу — ID.
      4. ⚠️ Внимание: В некоторых версиях приложения Ozon код товара может отображаться некорректно. Если данные не совпадают с веб-версией, доверяйте информации на сайте.
      💡

      Добавьте товар в «Избранное» в мобильном приложении — так его будет проще найти позже по коду.

      Способ 4: Получение кода через API Ozon (для продавцов)

      Если вы автоматизируете работу с Ozon через API, код товара можно получить программно. Для этого используйте следующие методы:

      • 🔌 Product Info — возвращает полную информацию о товаре, включая sku, offer_id (ID) и barcode (штрихкод). Пример запроса: 
        GET https://api-seller.ozon.ru/v2/product/info?offer_id={ID_ТОВАРА}&product_id={ID_ПРОДУКТА}
      • 📤 List of Products — возвращает список всех товаров с их кодами. Пример: 
        POST https://api-seller.ozon.ru/v1/product/list

    Для работы с API вам понадобится:

    1. Client-ID и API-ключ (берутся в личном кабинете Настройки → API).
    2. Библиотека для запросов (например, requests для Python).

    Пример ответа API (упрощённо):

    {

    "offer_id": "198765432",  // ID товара
    

    "sku": 1122334455,        // SKU Ozon
    

    "barcode": "4607103781234", // Штрихкод
    

    "attributes": [
    

    {
    

    "name": "Артикул продавца",
    

    "value": "SM-A525FZKD"
    

    }
    

    ]
    

    }

    Как проверить корректность API-ключа?

    Если при запросе вы получаете ошибку 403 Forbidden, проверьте:

    - Правильность Client-ID и API-ключа.

    - Разрешения ключа в личном кабинете (должны быть включены методы для работы с товарами).

    - Формат заголовка авторизации: Authorization: Bearer {API_KEY}.

    Способ 5: Код товара на упаковке (штрихкод и этикетка FBS)

    Если товар уже на складе FBS или у вас на руках, код можно найти непосредственно на упаковке. Ищите:

    • 📦 Штрихкод — обычно наклеен на коробке. Это может быть:
      • 🏷️ Заводской штрихкод (EAN/UPC) — начинается с 460..., 482... и т.д.
      • 🏷️ Штрихкод Ozon — начинается с 200... (если сгенерирован маркетплейсом).
    • 📋 Этикетка FBS — если товар отправлен на склад Ozon, на нём будет наклейка с:
      • SKU Ozon (в формате 123456789-01-2).
      • Штрихкодом (для сканирования).

    Как проверить подлинность штрихкода?

    1. Отсканируйте его через приложение-сканер (например, Barcode Scanner для Android).
    2. Если код корректный, откроется карточка товара на Ozon или сайте производителя.
    ⚠️ Внимание: Если штрихкод на этикетке FBS не сканируется, товар могут не принять на складе. Проверьте качество печати и соответствие кода данным в личном кабинете.

    Частые вопросы о кодах товара на Ozon

    🔍 Можно ли изменить SKU Ozon после создания товара?

    Нет, SKU Ozon присваивается автоматически при модерации и не редактируется. Если нужно поменять код, создайте новый товар с другим артикулом продавца — система сгенерирует новый SKU.

    Исключение: при редактировании вариаций товара (например, добавлении нового цвета) SKU для новых вариаций будет другим.

    📱 Почему в мобильном приложении нет кода товара?

    В некоторых версиях приложения Ozon код товара скрыт в глубоких настройках. Попробуйте:

    1. Открыть карточку товара.
    2. Нажать Поделиться → скопировать ссылку.
    3. В ссылке будет ID товара (цифры после /product/).

    Если это не сработало, используйте браузерную версию.

    🛒 Как найти код товара в чеке или квитанции?

    В чеке Ozon (электронном или бумажном) код товара указан в строке с наименованием. Ищите:

    • ID товара — рядом с названием (цифры без букв).
    • SKU — если чек с FBS, может быть в разделе «Дополнительная информация».

    Если чека нет, проверьте email с подтверждением заказа — там есть ссылка на товар с ID.

    🔄 Что делать, если SKU Ozon дублируется?

    Дублирование SKU происходит, если:

    • Вы загрузили один и тот же товар дважды с разными артикулами.
    • Ozon сгенерировал одинаковые коды для вариаций (редко, но бывает).

    Решение:

    1. Удалите дублирующийся товар в личном кабинете.
    2. Создайте новый товар с уникальным артикулом продавца.
    3. Если ошибка на стороне Ozon, обратитесь в поддержку с указанием SKU.
    🤖 Можно ли получить код товара через бота Ozon?

    Нет, официальные боты Ozon (в Telegram или ВКонтакте) не предоставляют информацию о кодах товаров. Максимум, что можно сделать:

    • Найти товар через бота и перейти на его карточку на сайте.
    • Скопировать ссылку и извлечь из неё ID.

    Для продавцов единственный надёжный способ — API или личный кабинет.

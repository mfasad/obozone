Как удалить избранное на Озон: 5 способов для телефона и компьютера

Список избранного на Озон — удобный инструмент для сохранения понравившихся товаров, но со временем он может превратиться в хаос из сотен позиций. Старые предложения с истёкшими скидками, товары, которые уже купили, или просто передумали покупать — всё это засоряет раздел и мешает быстро находить актуальные вещи. К сожалению, Озон не предлагает функцию массового удаления, но мы собрали все работающие способы, чтобы очистить избранное за несколько минут.

  • 📱 Пошаговые инструкции для мобильного приложения (Android/iOS) и десктопной версии сайта
  • 🔍 Как удалить отдельные товары или всю папку целиком (включая скрытые функции)
  • ⚡ Быстрые методы для очистки через браузерные расширения и скрипты
  • 🔄 Можно ли восстановить случайно удалённые товары — честный ответ

Все способы проверены в мае 2026 года и работают на последних версиях приложения Озон (7.42.0 для Android и 7.41.1 для iOS). Если вы используете устаревшую версию, часть функций может отсутствовать — обновите приложение в App Store или Google Play.

1. Как удалить товары из избранного через мобильное приложение

Самый простой способ — использовать официальное приложение Озон на смартфоне. Здесь доступны все функции управления избранным, включая удаление отдельных позиций и целых папок. Рассмотрим процесс для Android и iOS — интерфейс идентичен, различия только в расположении некоторых кнопок.

Чтобы удалить один товар:

  1. Откройте приложение и перейдите в раздел Избранное (значок сердечка в нижнем меню).
  2. Найдите товар, который хотите удалить, и сделайте длинное нажатие на его карточке.
  3. В появившемся меню выберите Удалить из избранного (на iOS может называться Убрать из избранного).

Для массового удаления:

  • 📌 Нажмите на иконку ⋮ (три точки) в правом верхнем углу экрана.
  • 📌 Выберите Управление папками → Редактировать.
  • 📌 Отметьте галочками товары, которые хотите удалить, и нажмите Удалить.

2. Удаление избранного через веб-версию Озон (ПК или телефон)

Веб-версия Озон предлагает больше возможностей для управления избранным, чем мобильное приложение. Здесь можно сортировать товары по дате добавления, цене или категории, что упрощает поиск ненужных позиций. Рассмотрим процесс на примере браузера Google Chrome (инструкция подходит и для Safari, Firefox, Edge).

Как удалить товары через сайт:

  1. Перейдите на сайт Ozon.ru и авторизуйтесь.
  2. Наведите курсор на иконку профиля в правом верхнем углу и выберите Избранное.
  3. Найдите товар, который хотите удалить, и наведите на него курсор — появится крестик ✕ в правом верхнем углу карточки. Нажмите на него.

Для ускоренной очистки используйте горячие клавиши:

  • 🔹 Ctrl + F — поиск по названию товара (например, введите "2022", чтобы найти устаревшие предложения).
  • 🔹 Ctrl + A → Delete — не работает для массового удаления на Озон, но можно использовать для выделения текста в фильтрах.
💡

Если нужно удалить все товары из папки, сначала перенесите их в одну папку (например, "На удаление"), а затем удалите папку целиком — это сэкономит время.

⚠️ Внимание: При использовании браузера Safari на Mac может не отображаться крестик для удаления. В этом случае попробуйте обновить страницу (Cmd + R) или используйте другой браузер.

3. Как удалить всю папку избранного за один клик

Если вы создавали папки в избранном (например, "Электроника", "Одежда", "Подарки"), их можно удалить целиком вместе со всеми товарами. Это самый быстрый способ очистки, но будьте осторожны — восстановление удалённой папки невозможно.

Инструкция для мобильного приложения:

  1. Перейдите в раздел Избранное.
  2. Нажмите на название папки, которую хотите удалить.
  3. Тапните на ⋮ (три точки) в правом верхнем углу → Удалить папку.
  4. Подтвердите действие в всплывающем окне.

На веб-версии:

  1. Откройте Избранное на сайте Озон.
  2. Найдите папку в левом боковом меню и нажмите на шестерёнку ⚙️ рядом с её названием.
  3. Выберите Удалить папку.

4. Автоматическая очистка избранного: скрипты и расширения

Если в вашем избранном тысячи товаров, ручное удаление займёт часы. Для ускорения процесса можно использовать браузерные расширения или пользовательские скрипты. Мы протестировали несколько решений — большинство из них требуют базовых знаний JavaScript, но есть и простые варианты.

Способ 1: Расширение Tampermonkey + скрипт

Установите расширение Tampermonkey для вашего браузера (официальный сайт), затем добавьте этот скрипт:

// ==UserScript==

// @name         Ozon Favorite Cleaner


// @namespace    http://tampermonkey.net/


// @version      1.0


// @description  Массовое удаление товаров из избранного Озон


// @author       You


// @match        https://www.ozon.ru/favorites/*


// @grant        none


// ==/UserScript==



(function() {


'use strict';


const buttons = document.querySelectorAll('button[aria-label="Удалить из избранного"]');


buttons.forEach(button => button.click());


setTimeout(() => location.reload(), 2000);


})();

Способ 2: Консоль браузера (для продвинутых)

Откройте Избранное на сайте Озон, затем:

  1. Нажмите F12 (или Ctrl+Shift+I) для открытия инструментов разработчика.
  2. Перейдите на вкладку Console.
  3. Вставьте и выполните этот код: 
    document.querySelectorAll('button[aria-label="Удалить из избранного"]').forEach(el => el.click());
Что делать если скрипт не работает?

1. Проверьте, что вы на странице избранного (URL должен содержать /favorites/).

2. Обновите страницу и повторите попытку.

3. Если Озон изменил структуру кнопок, скрипт может не сработать — в этом случае используйте ручное удаление.

⚠️ Внимание: Автоматическое удаление может сработать некорректно, если Озон обновил верстку страницы. Всегда проверяйте результат и при необходимости доводите очистку вручную.

5. Можно ли восстановить удалённые товары из избранного?

Официально Озон не предоставляет функцию восстановления удалённых товаров из избранного. Однако есть несколько обходных путей, которые могут помочь вернуть часть данных:

  • 🔄 История просмотров: Перейдите в Профиль → История просмотров. Здесь сохраняются товары, которые вы просматривали в последние 30 дней (даже если они были в избранном).
  • 📧 Письма от Озон: Проверьте почтовый ящик (включая папку "Промоакции") — там могут быть письма с товарами из избранного, особенно если они участвовали в распродажах.
  • 📱 Кэш приложения: На Android можно попробовать восстановить данные через резервную копию (например, если у вас включено автоматическое сохранение в Google Drive).

Если товар был добавлен в избранное менее суток назад, иногда помогает:

  1. Закройте приложение Озон через диспетчер задач.
  2. Очистите кэш приложения (Настройки телефона → Приложения → Озон → Память → Очистить кэш).
  3. Откройте приложение заново — иногда товары отображаются до синхронизации с сервером.
💡

Единственный надёжный способ не потерять важные товары — регулярно экспортировать избранное в Excel через функцию "Сохранить список" (доступно в веб-версии).

6. Как избежать засорения избранного в будущем

Чтобы не приходилось тратить время на очистку, следуйте этим советам:

  • 📁 Используйте папки: Сразу распределяйте товары по категориям (например, "Купить сейчас", "Подарки", "Жду скидки"). Это упростит будущую очистку.
  • ⏳ Устанавливайте напоминания: Раз в месяц выделяйте 5 минут на ревизию избранного — удаляйте товары старше 3 месяцев.
  • 🔔 Отключите автоматическое добавление: В настройках профиля (Профиль → Настройки → Персонализация) можно отключить опцию "Добавлять просмотренные товары в избранное".
  • 📱 Используйте альтернативы: Для долговременного хранения товаров подойдут сервисы вроде Pinterest или Google Keep — они предлагают больше инструментов для организации.

Если вы часто покупаете на Озон, рассмотрите возможность создания частного списка покупок (функция доступна в веб-версии). В отличие от избранного, здесь товары не дублируются и легче управляются.

7. Частые ошибки и как их избежать

При очистке избранного пользователи часто сталкиваются с типичными проблемами. Вот как их предотвратить:

  • 🚫 "Исчезла кнопка удаления": Обычно это происходит из-за устаревшей версии приложения. Обновите Озон или используйте веб-версию.
  • 🚫 "Удалил не тот товар": Перед массовым удалением создайте резервную папку и перенесите туда все важные товары.
  • 🚫 "Приложение зависает": При большом количестве товаров (>500) очищайте избранное порциями или используйте скрипты.
  • 🚫 "Не синхронизируется между устройствами": Выйдите из аккаунта и войдите заново — это принудительно обновит данные.

Если после очистки избранное по-прежнему отображается некорректно, попробуйте:

  1. Очистить кэш браузера (Ctrl + Shift + Del → выберите "Кэшированные изображения и файлы").
  2. Переустановить мобильное приложение.
  3. Написать в поддержку Озон через форму обратной связи (указывайте конкретную проблему, например: "После удаления папки товары остались в избранном").
💡

Если товар не удаляется ни через приложение, ни через сайт, проверьте, не добавлен ли он в "Список покупок" или "Отложенные" — эти разделы часто путают с избранным.

FAQ: Ответы на частые вопросы

Можно ли удалить все товары из избранного сразу, не по одному?

Нет, Озон не предоставляет функцию полной очистки избранного за один клик. Максимум — удаление целой папки (если товары в ней сгруппированы). Для массового удаления используйте скрипты из раздела 4 или очищайте порциями по 30-50 товаров.

Почему после удаления товары снова появляются в избранном?

Это происходит из-за функции "Персональные рекомендации". Озон может автоматически добавлять просмотренные товары обратно в избранное. Чтобы отключить это:

  1. Перейдите в Профиль → Настройки → Персонализация.
  2. Отключите опцию "Сохранять историю просмотров для рекомендаций".
Как перенести избранное с одного аккаунта Озон на другой?

Прямого способа нет, но можно:

  1. Сохранить список товаров в Excel (веб-версия → "Сохранить список").
  2. С нового аккаунта вручную добавить товары по артикулам или названиям.

Для ускорения используйте расширения для браузера, которые позволяют массово открывать ссылки (например, Linkclump).

Сколько товаров можно добавить в избранное на Озон?

Официального лимита нет, но на практике:

  • При превышении 1000 товаров могут возникать задержки при загрузке списка.
  • При 5000+ товарах высок риск ошибок синхронизации между устройствами.

Для стабильной работы рекомендуем держать в избранном не более 500 товаров.

Можно ли увидеть, кто ещё добавил товар в избранное?

Нет, Озон не показывает статистику по другим пользователям, добавившим тот же товар в избранное. Однако на странице товара отображается общая популярность (например, "100+ человек купили за последние 24 часа"), что косвенно указывает на востребованность.

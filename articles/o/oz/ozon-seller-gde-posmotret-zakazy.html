Эта проблема знакома многим селлерам Ozon — платформа регулярно обновляет дизайн, а документация не всегда успевает за изменениями. 1. Как посмотреть заказы в личном кабинете Ozon Seller на сайте

Основной способ работы с заказами для большинства продавцов — веб-версия личного кабинета Ozon Seller. Здесь доступны все функции, включая массовые действия, экспорт данных и детальную аналитику. Чтобы открыть список заказов:

  1. Авторизуйтесь на сайте seller.ozon.ru под своим логином продавца.
  2. В левом боковом меню выберите раздел Заказы (значок коробки). Если меню свёрнуто, наведите курсор на иконку с тремя горизонтальными линиями в верхнем левом углу.
  3. По умолчанию откроется вкладка Все заказы с последними 50 позициями. Чтобы увидеть полный список, прокрутите страницу вниз — данные подгрузятся автоматически.

Интерфейс раздела разделён на несколько ключевых блоков:

  • 📌 Фильтры (верхняя панель) — позволяют отобрать заказы по статусу, дате, способу доставки или SKU товара.
  • 📦 Таблица заказов — основное рабочее пространство с колонками: номер заказа, дата, статус, сумма, количество товаров и т.д.
  • ⚙️ Кнопки массовых действий (над таблицей) — печать наклеек, экспорт в Excel, подтверждение отправки.

Если вы не видите заказы, проверьте:

⚠️ Внимание: В личном кабинете по умолчанию показываются заказы за последние 30 дней. Чтобы увидеть более ранние, измените диапазон дат в фильтре Период.

2. Фильтры заказов: как найти нужный заказ за 10 секунд

Список заказов на Ozon может насчитывать сотни или тысячи позиций, особенно в сезон распродаж. Чтобы быстро найти конкретный заказ, используйте комбинации фильтров. Основные параметры:

Тип фильтра Примеры значений Когда использовать
Статус Новый, В обработке, Отменён, Доставлен Чтобы увидеть только актуальные заказы (например, требующие отправки)
Период Сегодня, Неделя, Месяц, Произвольный диапазон Для анализа продаж за конкретный промежуток времени
Способ доставки Курьер, Пункт выдачи, Почта России, FBS Если нужно отдельно обработать заказы с разными логистическими схемами
SKU/Артикул Введите код товара (например, 123456789) Чтобы найти все заказы с конкретным товаром
Номер заказа Введите полный номер (например, 1234567890-1234567) Для поиска конкретного заказа по номеру из уведомления

Пример: чтобы найти все неотправленные заказы за последние 3 дня по схеме FBS, установите фильтры:

  • Статус: Новый + В обработке
  • Период: Произвольный диапазон → укажите даты
  • Способ доставки: FBS

Если вы часто используете одни и те же фильтры, сохраните их как предпочтение (кнопка Сохранить фильтр в правом верхнем углу). Это сэкономит время при повторном поиске.

3. Мобильное приложение Ozon Seller: плюсы и минусы

Официальное приложение Ozon Seller (доступно для iOS и Android) позволяет управлять заказами с телефона. Это удобно для продавцов, которые часто в пути или не имеют доступа к компьютеру. Однако функционал мобильной версии урезан по сравнению с веб-интерфейсом.

Как посмотреть заказы в приложении:

  1. Скачайте и установите приложение из App Store или Google Play.
  2. Авторизуйтесь под своим аккаунтом продавца (не покупателя!).
  3. На главном экране тапните по вкладке Заказы (значок коробки в нижнем меню).
  4. По умолчанию показываются Все заказы. Чтобы применить фильтры, нажмите на иконку воронки в правом верхнем углу.

Преимущества мобильного приложения:

  • ✅ Быстрый доступ к статусам заказов без компьютера
  • ✅ Уведомления о новых заказах в реальном времени
  • ✅ Возможность подтвердить отправку или отменить заказ в 2 клика

Недостатки:

  • ❌ Ограниченные фильтры (например, нет фильтра по SKU)
  • ❌ Нет массовых действий (нельзя распечатать наклейки для нескольких заказов сразу)
  • ❌ Медленная работа при большом количестве заказов
⚠️ Внимание: В мобильном приложении Ozon Seller иногда возникает баг, когда заказы дублируются или отображаются не в хронологическом порядке. Если заметили несоответствие, сверьте данные с веб-версией.
💡

Включите push-уведомления в настройках приложения, чтобы не пропустить новые заказы. Это особенно важно для продавцов на схеме FBO, где время обработки ограничено.

4. Просмотр заказов через API Ozon: для продвинутых продавцов

Если вы используете собственную систему управления заказами (например, 1С, МойСклад или RetailCRM), подключение к API Ozon позволит автоматически получать данные о заказах без ручного ввода. Это актуально для крупных продавцов с сотнями заказов в день.

Как получить список заказов через API:

  1. Зарегистрируйте приложение в личном кабинете API Ozon и получите Client-ID и API-Key.
  2. Используйте метод /v2/posting/fbs/list для схемы FBS или /v1/orders для FBO.
  3. Отправьте GET-запрос с необходимыми параметрами (например, since и to для указания диапазона дат).

Пример запроса для получения заказов за последние 7 дней (схема FBS):

GET https://api-seller.ozon.ru/v2/posting/fbs/list

Headers:


Client-Id: YOUR_CLIENT_ID


Api-Key: YOUR_API_KEY


Params:


since=2026-05-20T00:00:00Z


to=2026-05-27T23:59:59Z


limit=1000

Ответ будет содержать массив заказов в формате JSON с детальной информацией: номер заказа, статус, адрес доставки, состав корзины и т.д. Для обработки данных вам понадобится программист или готовый коннектор (например, модуль для 1С).

⚠️ Внимание: API Ozon имеет лимиты на количество запросов — 60 запросов в минуту и 5000 в час. При превышении лимита вы получите ошибку 429 Too Many Requests. Чтобы избежать блокировки, используйте кэширование данных или увеличивайте интервал между запросами.
Что делать, если API возвращает ошибку 403 Forbidden?

Ошибка 403 означает, что у вашего API-ключа недостаточно прав или он заблокирован. Проверьте:

1. Правильность указанных Client-ID и API-Key (они чувствительны к регистру).

2. Статус ключа в личном кабинете API — возможно, он истёк или был отозван.

3. IP-адрес, с которого отправляется запрос. Если в настройках API указаны разрешённые IP, запрос с другого адреса будет заблокирован.

4. Наличие активной подписки на тариф API (бесплатный тариф имеет ограничения).

Если проблема остаётся, обратитесь в поддержку Ozon через раздел Помощь → Техническая поддержка в личном кабинете API.

5. Экспорт заказов в Excel: пошаговая инструкция

Для анализа продаж или передачи данных в бухгалтерию удобно экспортировать заказы в формат Excel (.xlsx) или CSV. В личном кабинете Ozon Seller это делается в несколько кликов:

  1. Перейдите в раздел Заказы и примените необходимые фильтры (например, выберите период за последний месяц).
  2. Над таблицей заказов нажмите кнопку Экспорт (значок стрелки вниз).
  3. Выберите формат файла:
    • Excel (.xlsx) — подходит для дальнейшей обработки в таблицах.
    • CSV (.csv) — лёгкий формат для импорта в другие системы.
  • Подтвердите экспорт. Файл начнёт формироваться — это может занять от нескольких секунд до минут (зависит от количества заказов).
  • После завершения обработки файл автоматически скачается в папку Загрузки вашего браузера.

    • Какие данные содержатся в экспортированном файле:

    • 📄 Номер и дата заказа
    • 📦 Состав заказа (наименование, SKU, количество, цена)
    • 💰 Сумма заказа и комиссия Ozon
    • 🚚 Способ и адрес доставки
    • 🔄 Статус заказа и история изменений
    • 📞 Контактные данные покупателя (телефон, email)

    Если в файле не хватает колонок (например, нет данных о весе или габаритах товара), проверьте настройки экспорта. В некоторых случаях требуется включить дополнительные поля вручную:

    1. Перед экспортом нажмите на иконку шестерёнки (Настройки колонок) рядом с кнопкой Экспорт.
    2. В открывшемся окне отметьте галочками нужные поля (например, Вес, Габариты, Комментарий покупателя).
    3. Сохраните настройки и повторите экспорт.
    💡

    Экспортированные данные обновляются с задержкой до 15 минут. Если вам нужны актуальные данные (например, для инвентаризации), делайте выгрузку непосредственно перед анализом.

    6. Типичные проблемы с заказами и их решения

    Даже опытные продавцы сталкиваются с ситуациями, когда заказы отображаются некорректно или пропадают из интерфейса. Рассмотрим самые частые проблемы и способы их решения.

    Проблема Возможная причина Решение
    Заказы не отображаются в списке Активны фильтры по дате или статусу Сбросьте фильтры или проверьте диапазон дат
    Заказ есть в уведомлении, но нет в личном кабинете Задержка синхронизации (до 30 минут) Подождите или обновите страницу (F5)
    В мобильном приложении и на сайте разное количество заказов Баг кэширования в приложении Очистите кэш приложения или переустановите его
    Невозможно экспортировать заказы (кнопка неактивна) Слишком большой период или много заказов Разбейте экспорт на меньшие периоды (например, по неделям)
    Статус заказа не обновляется (например, остаётся "В обработке") Проблемы на стороне логистики Ozon Обратитесь в поддержку с номером заказа

    Если ни один из способов не помог, и заказы по-прежнему не отображаются, выполните следующие шаги:

    1. Проверьте, не заблокирован ли ваш аккаунт продавца. Для этого перейдите в раздел Профиль → Статус аккаунта.
    2. Убедитесь, что у вас нет задолженности перед Ozon (например, по комиссиям или штрафам). Это может привести к ограничению доступа к заказам.
    3. Попробуйте войти в личный кабинет с другого устройства или браузера (например, в режиме инкогнито).
    4. Если проблема сохраняется, обратитесь в чате поддержки Ozon Seller с указанием:
      • Номера отсутствующих заказов (если известны)
      • Скриншота ошибки или пустого списка заказов
      • Времени, когда проблема была замечена

    В большинстве случаев проблемы с отображением заказов решаются в течение суток. Если заказы критично важны (например, нужно срочно отправить товар), используйте резервный способ — запросите данные через API или свяжитесь с менеджером Ozon по телефону горячей линии.

    7. Автоматизация работы с заказами: сервисы и интеграции

    Ручная обработка заказов отнимает много времени, особенно если вы продаёте сотни позиций в день. Для автоматизации можно использовать сторонние сервисы, которые интегрируются с Ozon Seller и упрощают работу:

    • 🤖 RetailCRM — система управления заказами с автоматическим импортом данных из Ozon, Wildberries и других маркетплейсов. Позволяет обрабатывать заказы из одного окна, печатать этикетки пачками и синхронизировать остатки.
    • 📊 МойСклад — подходит для учета товаров и автоматического списания остатков при продаже на Ozon. Можно настроить печать наклеек и документов прямо из системы.
    • 🔄 Атомик Хаб — сервис для мультимаркетплейсовой аналитики. Показывает все заказы с Ozon, Яндекс Маркета и WB на одной панели, формирует отчёты по продажам и прибыли.
    • 📈 eLama — инструмент для автоматизации рекламы и анализа продаж. Может связывать заказы с рекламными кампаниями, чтобы отслеживать ROI.

    Как подключить интеграцию (на примере RetailCRM):

    1. Зарегистрируйтесь в RetailCRM и перейдите в раздел Интеграции.
    2. Выберите Ozon из списка маркетплейсов.
    3. Введите Client-ID и API-Key из личного кабинета API Ozon.
    4. Настройте синхронизацию: укажите, какие данные импортировать (заказы, остатки, цены) и с какой периодичностью.
    5. Сохраните настройки и запустите первую синхронизацию.

    После подключения все новые заказы с Ozon будут автоматически попадать в RetailCRM, где вы сможете:

    • Назначать ответственных менеджеров за заказы
    • Печатать пачки наклеек для отправки
    • Отправлять уведомления покупателям
    • Формировать отчёты по продажам и прибыли
    ⚠️ Внимание: При выборе сервиса для автоматизации проверьте, поддерживает ли он вашу схему работы (FBS или FBO). Некоторые сервисы оптимизированы только под одну из схем.

    FAQ: Частые вопросы о заказах на Ozon Seller

    Как найти заказ по номеру телефона покупателя?

    В личном кабинете Ozon Seller нет прямого поиска по номеру телефона. Однако можно:

    1. Экспортировать заказы в Excel и воспользоваться поиском по столбцу с телефонами (Ctrl+F).
    2. Использовать API Ozon с фильтром по phone (если у вас есть доступ к расширенным методам API).
    3. Обратиться в поддержку Ozon с просьбой найти заказ по телефону (указывайте причину, например, "нужно уточнить адрес доставки").

    Обратите внимание: по правилам Ozon, контактные данные покупателей можно использовать только для выполнения заказа. Звонки или сообщения с целью продажи других товаров запрещены и могут привести к блокировке аккаунта.

    Почему в личном кабинете отображается не весь список заказов?

    Это может происходить по нескольким причинам:

    • Активны фильтры (проверьте верхнюю панель — возможно, выбран только один статус или короткий период дат).
    • Заказы архивированы (по умолчанию показываются заказы за последние 30 дней; для просмотра более старых используйте фильтр по дате).
    • Технические работы на стороне Ozon (проверьте статус систем на странице статуса сервисов).
    • Ограничения аккаунта (например, при блокировке или ограничении функционала за неуплату комиссий).

    Если проблема не в фильтрах, попробуйте:

    1. Обновить страницу (F5) или очистить кэш браузера.
    2. Войти в личный кабинет с другого устройства.
    3. Использовать мобильное приложение или API для проверки.
    Можно ли увидеть заказы, которые покупатель отменил?

    Да, отменённые заказы отображаются в личном кабинете, но их нужно искать отдельно:

    1. Перейдите в раздел Заказы.
    2. В фильтрах выберите статус Отменён.
    3. При необходимости укажите период дат.

    В списке отменённых заказов вы увидите причину отмены (например, "Покупатель отменил заказ" или "Товар отсутствует на складе"). Эти данные полезны для анализа потерь продаж.

    Обратите внимание: отменённые заказы не учитываются в отчётах по продажам и выручке, но их можно экспортировать в Excel для внутреннего анализа.

    Как распечатать наклейки для заказов FBS?

    Для заказов по схеме FBS (когда товар хранится на складе Ozon) наклейки печатаются прямо из личного кабинета:

    1. В разделе Заказы отфильтруйте список по статусу Готов к отправке (для FBS).
    2. Отметьте галочками нужные заказы (или нажмите на чекбокс в шапке таблицы, чтобы выбрать все).
    3. Над таблицей нажмите кнопку Печать наклеек.
    4. Выберите формат печати:
      • A4 — для принтера (на листе будет несколько наклеек).
      • Термоэтикетка 100×100 мм — для термопринтера.
  • Подтвердите печать. Файл с наклейками скачается в формате PDF.

    • Если кнопка Печать наклеек неактивна, проверьте:

    • Статус заказов — наклейки можно печатать только для заказов в статусе Готов к отправке.
    • Наличие товара на складе Ozon (если товар отсутствует, наклейка не сформируется).
    • Правильность выбранного фильтра (например, не включён ли фильтр по другой схеме доставки).
    Что делать, если покупатель не получил заказ, а статус "Доставлен"?

    Такие ситуации возникают из-за ошибок логистики (например, курьер отсканировал заказ как доставленный, но фактически не вручил его покупателю). Алгоритм действий:

    1. Проверьте трек-номер на сайте транспортной компании (например, СДЭК или Почта России). Иногда там есть более детальная информация о доставке.
    2. Свяжитесь с покупателем через чат в личном кабинете (раздел Сообщения). Уточните, получал ли он заказ и что указано в уведомлении о доставке.
    3. Если покупатель не получил заказ, но статус "Доставлен":
      • Попросите его проверить почтовый ящик или пункт выдачи (иногда заказы оставляют без уведомления).
      • Если заказ действительно потерян, создайте обращение в поддержку Ozon через раздел Помощь → Заказы → Проблемы с доставкой.
  • В обращении укажите:
    • Номер заказа.
    • Трек-номер отправления.
    • Скриншот переписки с покупателем (если есть).
    • Подробное описание проблемы.

    Ozon обычно расследует такие случаи в течение 1–3 дней. Если вина лежит на логистическом партнёре, покупателю будет предложено повторная отправка или возврат денег. Если вина продавца (например, неверно указан адрес), штрафы могут быть начислены на ваш аккаунт.

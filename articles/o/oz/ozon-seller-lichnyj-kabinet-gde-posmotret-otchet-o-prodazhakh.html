Где в личном кабинете Озон Селлер посмотреть отчёт о продажах: полная инструкция

Анализ продаж — основа успешной работы на маркетплейсе. Без точных данных о выручке, количестве заказов и динамике спроса невозможно принимать взвешенные решения: от корректировки цен до планирования закупок. В личном кабинете Озон Селлер отчёты о продажах скрыты неглубоко, но их структура и возможности фильтрации часто вызывают вопросы у новичков. Особенно сложно разобраться, если вы привыкли к другим платформам вроде Wildberries или Яндекс Маркета, где интерфейс аналитики организован иначе.

Многие продавцы теряют часы на поиск нужных цифр, не подозревая, что в Озон есть гибкие инструменты для сегментации данных — по датам, товарам, регионам и даже типам доставки. А некоторые и вовсе экспортируют сырые данные в Excel, чтобы построить собственные дашборды. В этой статье разберём, где именно в личном кабинете найти отчёты, как их настроить под свои задачи и какие скрытые возможности аналитики доступны только опытным продавцам.

Чаще всего проблемы возникают с поиском детализированных отчётов — например, когда нужно увидеть продажи по конкретному SKU или сравнить динамику по неделям. Далее вы узнаете, как избежать типичных ошибок при работе с статистикой и где искать данные, которых нет в стандартных разделах.

Где в личном кабинете Озон Селлер находится раздел с отчётами

Основной путь к отчётам лежит через главное меню личного кабинета. После авторизации на странице seller.ozon.ru в верхней панели вы увидите вкладку «Аналитика». Именно здесь сосредоточены все инструменты для мониторинга продаж, но не все разделы одинаково полезны. Рассмотрим ключевые пункты:

1. «Обзор» — дашборд с основными метриками за выбранный период (выручка, количество заказов, средний чек). Подходит для быстрого анализа, но не даёт детализации.

2. «Продажи» — самый важный раздел, где можно увидеть отчёт о продажах с разбивкой по товарам, заказам и статусам.

3. «Товары» — статистика по каждому SKU, включая остатки, возвраты и конверсию.

4. «Финансы» — здесь отображаются выплаты, комиссии и удержания, но не сами продажи.

Для большинства задач достаточно раздела «Продажи». Однако если вам нужны данные по конкретному товару или категории, лучше использовать фильтры в разделе «Товары».

Пошаговая инструкция: как открыть отчёт о продажах

Чтобы не тратить время на поиски, следуйте этому алгоритму:

  1. Авторизуйтесь в личном кабинете Озон Селлер.

  2. В верхнем меню выберите вкладку «Аналитика» → «Продажи».

  3. По умолчанию откроется отчёт за последние 7 дней. Чтобы изменить период, нажмите на календарь в правом верхнем углу и выберите нужные даты.

  4. Для детализации по товарам нажмите на строку с названием продукта — откроется карточка с историей продаж, остатками и динамикой цен.

Если вам нужны данные в формате Excel или CSV, используйте кнопку «Экспорт»** в правом верхнем углу. Озон позволяет выгружать отчёты за период до 90 дней — этого хватит для большинства аналитических задач.

Какие данные доступны в отчёте о продажах

Стандартный отчёт включает более 20 колонок с метриками. Вот ключевые из них, на которые стоит обратить внимание:

Параметр Описание Пример использования
Номер заказа Уникальный идентификатор заказа в системе Озон Поиск конкретного заказа для урегулирования споров с покупателем
Дата создания Когда заказ был оформлен Анализ пиковых дней продаж (например, перед праздниками)
SKU Артикул товара в вашем каталоге Отслеживание продаж конкретного продукта среди всей номенклатуры
Количество Сколько единиц товара было продано Планирование закупок и контроль остатков
Сумма заказа Общая стоимость заказа (с учётом скидок) Расчёт среднего чека и выручки

Для углублённого анализа можно добавить дополнительные колонки через настройки отчёта (значок шестерёнки рядом с кнопкой экспорта). Например, полезно включить:

  • 📍 Регион доставки — чтобы понять географию спроса;
  • 🚚 Тип логистики (FBS/FBO) — для оценки эффективности схем;
  • 🔄 Статус возврата — если нужно отследить проблемные товары.
💡

Сохраните шаблон отчёта с нужными колонками, чтобы не настраивать его каждый раз. Для этого после выбора параметров нажмите «Сохранить настройки» в меню шестерёнки.

Как фильтровать отчёт по датам, товарам и статусам

Без фильтров отчёт о продажах превращается в хаос из тысяч строк. Озон позволяет сегментировать данные по нескольким критериям:

1. По датам:

- Выберите диапазон в календаре (максимум 90 дней за раз).

- Для сравнения периодов используйте функцию «Сравнить с» — например, «этот месяц vs прошлый месяц».

- Пример: чтобы проанализировать влияние акции, сравните неделю до скидки и неделю после.

2. По товарам:

- Введите SKU или название товара в поисковую строку.

- Используйте фильтр «Категория», если нужно проанализировать всю линейку (например, «Электроника»).

- Лайфхак: чтобы исключить из отчёта товары с нулевыми продажами, отметьте галочку «Только с продажами».

3. По статусам заказов:

- Фильтр «Статус» позволяет отобразить только:

- ✅ Выполненные заказы (для анализа реальных продаж);

- ❌ Отменённые (чтобы понять причины отказов);

- 🔄 Возвраты (для работы с проблемными товарами).

Комбинируя фильтры, можно получить ответы на конкретные вопросы. Например:

  • 📉 «Почему упали продажи в июне?» → Фильтр по датам + сравнение с маем;
  • 🔍 «Какой товар чаще всего возвращают?» → Фильтр по статусу «Возврат» + группировка по SKU;
  • 🚀 «Какие регионы приносят больше всего выручки?» → Фильтр по региону + сортировка по сумме заказа.
Как сохранить фильтр для повторного использования

В Озон Селлер можно сохранять пользовательские фильтры. После настройки всех параметров нажмите на иконку дискеты рядом с кнопкой «Применить» и дайте название шаблону (например, «Анализ возвратов за квартал»). В следующий раз вы сможете загрузить его из выпадающего списка.

Скрытые возможности аналитики: что упускают 90% продавцов

Большинство продавцов ограничиваются базовыми отчётами, но в Озон Селлер есть инструменты, о которых мало кто знает. Вот что стоит изучить:

1. Отчёт «Продажи по времени»:

- Показывает, в какие часы и дни недели покупатели активнее всего оформляют заказы.

- Как использовать: если пик продаж приходится на вечер, перенесите запуск акций на это время.

2. Анализ «Корзина покупателя»:

- В разделе «Товары» → «Аналитика» можно увидеть, какие товары чаще покупают вместе.

- Пример: если чехлы для телефонов часто покупают с зарядками, стоит создать бандл.

3. Сравнение с конкурентами:

- В карточке товара (раздел «Товары») есть вкладка «Конкуренты», где показаны цены и продажи аналогичных предложений.

- Важно: эти данные обновляются раз в неделю, поэтому для оперативного анализа их лучше дополнять ручным мониторингом.

4. Отчёт по промоакциям:

- В разделе «Акции» → «Статистика» можно оценить, как скидки повлияли на продажи.

- Критический момент: здесь видно не только рост количества заказов, но и изменение среднего чека (иногда скидки приводят к падению выручки!).

💡

Отчёт «Продажи по времени» — единственный инструмент в Озон Селлер, который показывает поведенческие паттерны покупателей. Его данные нельзя получить из стандартных экспортов, поэтому анализируйте его прямо в личном кабинете.

Частые ошибки при работе с отчётами и как их избежать

Даже опытные продавцы иногда допускают ошибки, которые искажают аналитику. Вот самые распространённые:

1. Игнорирование статусов заказов:

- Многие смотрят только на «Выполненные» заказы, забывая об отменённых и возвратах.

- Последствия: реальная выручка может быть на 10–30% ниже, чем в отчёте.

- Решение: всегда фильтруйте данные по статусу или используйте колонку «Фактическая выручка» (учитывает возвраты).

2. Неправильный период сравнения:

- Сравнивать продажи в январе с июлем бессмысленно из-за сезонности.

- Решение: используйте функцию «Сравнить с прошлым годом» или анализируйте данные по неделям.

3. Экспорт без проверки колонок:

- По умолчанию в Excel выгружаются не все данные.

- Решение: перед экспортом откройте настройки и отметьте галочками все нужные параметры (например, Комиссия Озон, Способ оплаты).

4. Пренебрежение сегментацией:

- Анализ «в целом» по всем товарам скрывает проблемы отдельных SKU.

- Пример: один убыточный товар может «съедать» всю маржу по категории.

⚠️ Внимание: Если в отчёте внезапно пропали данные за какой-то день, проверьте настройки фильтра по датам. Озон иногда сбрасывает диапазон при обновлении страницы.

Как автоматизировать аналитику: интеграции и внешние инструменты

Ручная работа с отчётами отнимает время, особенно если у вас сотни SKU. К счастью, данные из Озон Селлер можно автоматически выгружать в внешние системы:

1. API Озон:

- Позволяет подключить личный кабинет к 1С, Google Sheets или специализированным сервисам вроде RetailCRM.

- Минус: требует технических навыков или помощи разработчика.

2. Готовые сервисы аналитики:

- «МойСклад», «Эвотор», «Аналитика Озон» (от самого маркетплейса) автоматически собирают данные и строят дашборды.

- Плюс: визуализация в виде графиков и диаграмм, удобно для презентаций.

3. Excel + Power Query:

- Бесплатный способ: настройте автоматическую выгрузку отчётов в Excel и используйте Power Query для объединения данных.

- Инструкция:

  1. Экспортируйте отчёт в CSV;
  2. В Excel перейдите на вкладку «Данные» → «Получить данные» → «Из файла»;
  3. Настройте автоматическое обновление при открытии файла.

4. Чат-боты и уведомления:

- Сервисы вроде Zapier или Make (ex-Integromat) могут отправлять вам ежедневные сводки по продажам в Telegram или на почту.

- Пример настройки:

- Триггер: «Новый отчёт в Озон Селлер»;

- Действие: «Отправить сообщение в Telegram с ключевыми метриками».

⚠️ Внимание: При использовании сторонних сервисов проверяйте, чтобы они имели официальную интеграцию с Озон. Несанкционированный парсинг данных может привести к блокировке аккаунта.

FAQ: Ответы на частые вопросы о отчётах в Озон Селлер

Можно ли увидеть отчёт о продажах за прошлый год?

Да, но с ограничениями. Озон хранит данные за последние 2 года, однако в интерфейсе личного кабинета доступен период до 90 дней. Чтобы получить исторические данные:

  1. Обратитесь в поддержку Озон с запросом на выгрузку;
  2. Используйте API, если у вас подключена интеграция;
  3. Проверьте архивные файлы на вашем компьютере (если вы ранее экспортировали отчёты).

Важно: данные за 2022 год и ранее могут быть неполными из-за изменений в структуре отчётов.

Почему в отчёте не совпадают суммы с выплатами?

Разница возникает из-за:

  • 💰 Комиссий Озон (они вычитаются из выручки);
  • 🔄 Возвратов (деньги удерживаются до разрешения спора);
  • ⏳ Задержки выплат (Озон перечисляет деньги с лагом в 1–3 дня).

Чтобы увидеть точную сумму к выплате, смотрите раздел «Финансы» → «Выплаты».

Как экспортировать отчёт с разбивкой по часам?

Стандартный экспорт не включает почасовую статистику. Чтобы её получить:

  1. Перейдите в «Аналитика» → «Продажи по времени»;
  2. Установите период не более 30 дней (иначе данные не отобразятся);
  3. Скриншот экрана или вручную перенесите данные в Excel (автоматического экспорта нет).

Альтернатива: используйте API с параметром group_by=hour.

Где посмотреть продажи по конкретному ПВЗ?

Озон не предоставляет отчёт с разбивкой по точкам выдачи, но можно использовать обходной путь:

  1. Экспортируйте отчёт по продажам с колонкой «Регион доставки»;
  2. Добавьте колонку «ПВЗ» через API или вручную (если у вас мало заказов);
  3. Используйте сводные таблицы в Excel для группировки.

Ограничение: этот метод работает только для заказов с FBS (доставка через Озон). Для FBO данные о ПВЗ недоступны.

Как отследить продажи по промокодам?

Данные по промокодам отображаются в разделе «Акции» → «Статистика». Там вы увидите:

  • 🏷️ Количество использований промокода;
  • 💵 Сумму скидок;
  • 📈 Изменение конверсии до/после акции.

Чтобы связать промокоды с конкретными заказами, экспортируйте отчёт по продажам и ищите строки со статусом «Скидка по промокоду».

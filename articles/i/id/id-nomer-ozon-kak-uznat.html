Как узнать ID номер товара на Озон: 7 проверенных способов

Вы когда-нибудь сталкивались с ситуацией, когда менеджер Ozon просит назвать ID товара, а вы теряетесь в догадках, где его искать? Или продавец на маркетплейсе требует уточнить артикул для возврата, но в накладной одни непонятные цифры? Эта проблема знакома и покупателям, и новичкам-селлерам. Дело в том, что на Ozon используется сразу несколько типов идентификаторов — и все они выглядят как набор цифр, но означают разное.

В этой статье мы разберёмся, чем ID товара отличается от SKU и номера заказа, где именно искать каждый из них (включая мобильное приложение и десктопную версию), и почему иногда продавцы просят именно product_id, а не артикул производителя. А ещё вы узнаете, как по ID проверить историю цен на товар или найти аналоги — это пригодится для сравнения предложений перед покупкой.

Что такое ID товара на Ozon и зачем он нужен

На Ozon под термином ID товара обычно подразумевают уникальный числовой код, который присваивается каждой позиции в каталоге маркетплейса. Это не то же самое, что:

  • 📦 Артикул производителя — код, который указывает бренд (например, IPH14PM-6128 для iPhone 14 Pro Max).
  • 🔢 SKU (Stock Keeping Unit) — внутренний идентификатор продавца для управления запасами.
  • 📄 Номер заказа — ID транзакции (начинается с WB- на Wildberries, но на Ozon имеет другой формат).

Зачем может понадобиться ID товара:

  • 🔍 Для поиска дубликатов или аналогов через сервисы парсинга (например, Ozon Parser или DataHawk).
  • 📊 Чтобы отследить динамику цен и скидок (через Keepa или CamelCamelCamel для Ozon).
  • 🔄 Для оформления возврата, если в накладной не указан артикул производителя.
  • 🛠️ Селлерам — чтобы привязать товар к рекламным кампаниям или акциям.

Важно: ID товара на Ozon не совпадает с ID заказа. Первый — это код позиции в каталоге (например, 123456789), второй — номер транзакции (например, 98765-1234-5678). Их часто путают, но для возврата или жалобы на товар может понадобиться именно product_id, а не номер заказа.

Где найти ID товара в карточке на сайте Ozon (десктоп)

Самый простой способ — посмотреть ID в адресной строке браузера. Вот пошаговая инструкция:

  1. Откройте карточку товара на Ozon (например, iPhone 15 Pro Max).
  2. Обратите внимание на URL-адрес. Он будет выглядеть так: 
    https://www.ozon.ru/product/{тут_ваш_ID}-{название}/

    Например, для iPhone 15 Pro Max это 1602830033.

  3. Скопируйте цифры между /product/ и следующим тире.

Если URL сокращён (например, через ozon.ru/hrXy12), кликните по кнопке"Поделиться" и выберите"Скопировать ссылку" — откроется полный адрес с ID.

Альтернативный способ (если URL не отображает ID):

  1. Нажмите правой кнопкой мыши на странице товара и выберите"Просмотреть код" (Ctrl+Shift+I).
  2. В открывшейся панели нажмите Ctrl+F и введите productId.
  3. Вы увидите строку вида "productId":123456789 — это и есть нужный идентификатор.
⚠️ Внимание: Не путайте ID товара с SKU продавца. Последний может содержать буквы (например, SKU123-AB) и обычно указан в разделе"Характеристики" под блоком"Артикул".

Как узнать ID товара в мобильном приложении Ozon

В приложении Ozon (для Android или iOS) путь к ID чуть сложнее, но тоже занимает меньше минуты. Сделайте следующее:

  1. Откройте приложение и найдите нужный товар через поиск.
  2. Тапните по карточке, чтобы открыть её полностью.
  3. В правом верхнем углу нажмите на иконку"Поделиться" (стрелочка вверх).
  4. Выберите"Скопировать ссылку".
  5. Вставьте скопированную ссылку в любое текстовое поле (например, в заметки). Она будет выглядеть так: 
    https://www.ozon.ru/product/{ID}-{название}/?sh=123ABC

    Цифры после /product/ и до тире — это ID.

Если ссылка сокращённая (например, ozon.ru/hrXy12), откройте её в браузере — адрес автоматически преобразуется в полный с ID.

💡

В приложении Ozon можно быстро найти ID товара через историю просмотров: откройте"Избранное" →"Недавние просмотры" и поделитесь ссылкой на нужный товар.

Для продавцов: если вам нужно найти ID своего товара в личном кабинете Ozon Seller, перейдите в раздел"Товары" → выберите позицию → в адресной строке будет параметр product_id.

ID заказа vs ID товара: в чём разница и где их искать

Многие пользователи путают эти два идентификатора, но они служат разным целям:

Тип ID Формат Где используется Где найти
ID товара (product_id) Цифры (8–10 знаков) Для поиска товара в каталоге, сравнения цен, возвратов URL карточки товара, API Ozon
ID заказа (order_id) Цифры и тире (например, 12345-6789-0123) Для отслеживания доставки, возвратов, жалоб Email-уведомление, раздел"Мои заказы"
SKU Буквы и цифры (например, ABC123-XYZ) Для управления запасами продавца Личный кабинет Ozon Seller, накладная

Чтобы найти ID заказа:

  1. Перейдите в раздел"Мои заказы" на сайте или в приложении Ozon.
  2. Выберите нужный заказ.
  3. В деталях заказа будет строка"Номер заказа" — это и есть order_id.
⚠️ Внимание: При оформлении возврата менеджер Ozon может попросить и ID товара, и ID заказа. Первый нужен, чтобы идентифицировать позицию в каталоге, второй — чтобы привязать претензию к вашей покупке.

Как найти ID товара по названию или артикулу

Если у вас есть только название или артикул производителя, но нет прямой ссылки на товар, воспользуйтесь одним из этих методов:

Способ 1: Поиск через сайт Ozon

  1. Введите название или артикул в поисковую строку Ozon.
  2. Откройте карточку нужного товара.
  3. Скопируйте ID из URL (как описано выше).

Способ 2: Парсинг через API (для продавцов)

Если вы продавец и вам нужно получить ID для большого количества товаров, используйте Ozon API:

GET https://api-seller.ozon.ru/v1/product/info?offer_id={ваш_SKU}&product_id={ID_товара}

Для этого нужен доступ к Ozon Seller API и токен авторизации.

Способ 3: Сервисы аналитики

Сервисы вроде Ozon Parser или DataHawk позволяют найти ID по ключевым словам или артикулу. Например:

  • 🔍 Введите в DataHawk название товара (например, "Samsung Galaxy S23 Ultra 256GB").
  • Сервис выдаст список позиций с указанием ID, цен и рейтингов.
Как найти ID товара, если он удалён с Ozon?

Если товар удалён из каталога, но вы знаете его прежний URL, попробуйте открыть его через архивные сервисы (например, Wayback Machine). В некоторых случаях ID сохраняется в кэше поисковых систем.

Зачем продавцам знать ID своих товаров

Для селлеров на Ozon знание ID товаров критично по нескольким причинам:

  • 📈 Рекламные кампании: без product_id невозможно настроить таргетированную рекламу через Ozon Advertising.
  • 🔄 Управление акциями: чтобы добавить товар в распродажу (например,"Чёрная пятница"), нужен его ID.
  • 📊 Аналитика: сервисы вроде SellerLab или Ozon Statistics требуют ID для отслеживания динамики продаж.
  • 🛠️ Техническая поддержка: при обращении в саппорт Ozon вас попросят указать product_id для ускорения решения проблемы.

Пример: если вы хотите запустить рекламу на iPhone 15 Pro через Ozon Advertising, вам потребуется его ID (например, 1602830033), а не SKU или артикул Apple.

💡

Продавцам рекомендуется вести таблицу с соответствием SKU → ID товара → артикул производителя. Это упростит работу с рекламой, акциями и поддержкой.

Частые ошибки при работе с ID на Ozon

Даже опытные пользователи иногда допускают ошибки, которые ведут к задержкам в возвратах или проблемам с заказами. Вот самые распространённые:

  • 🔢 Путают ID товара и ID заказа: отправляют менеджеру номер транзакции вместо кода позиции. Это приводит к тому, что поддержка не может идентифицировать проблемный товар.
  • 📋 Копируют ID с тире или буквами: правильный формат — только цифры (например, 1602830033, а не 1602830033-iphone-15).
  • 🔍 Используют SKU вместо ID: продавцы иногда просят ID товара, а покупатели присылают внутренний код продавца (SKU), который им не подходит.
  • 📱 Не обновляют ссылку в мобильном приложении: если скопировать сокращённую ссылку (например, ozon.ru/hrXy12), она не содержит ID. Нужно открыть её в браузере.

Пример ошибки: покупатель жалуется на бракованный MacBook Air и присылает в поддержку номер заказа 12345-6789-0123. Менеджер просит уточнить ID товара, так как по номеру заказа видно только транзакцию, но не понятно, какая именно позиция в заказе бракованная (если их несколько).

⚠️ Внимание: Если вы оформляете возврат через форму на сайте Ozon, система может автоматически подставить неверный ID (например, из аналогичного товара). Всегда проверяйте его вручную перед отправкой заявки.

FAQ: Ответы на частые вопросы про ID на Ozon

Можно ли по ID товара узнать, кто продавец?

Нет, ID товара относится к позиции в каталоге Ozon, а не к продавцу. Чтобы узнать продавца, откройте карточку товара и посмотрите в блоке"Продавец" или"Информация о продавце". Для FBS-товаров продавец — сам Ozon.

Что делать, если ID товара не отображается в URL?

Если URL карточки товара не содержит цифр после /product/, попробуйте:

  1. Нажать"Поделиться" и скопировать полную ссылку.
  2. Открыть код страницы (Ctrl+Shift+I) и найти productId.
  3. Использовать расширение для браузера (например, Ozon Helper), которое показывает ID в один клик.
Можно ли по ID товара отследить его историю цен?

Да, для этого подойдут сервисы:

  • Keepa (ранее работал только с Amazon, но теперь поддерживает и Ozon).
  • CamelCamelCamel — введите ID в поисковую строку.
  • Ozon Price Tracker (расширение для Chrome).

Пример: по ID 1602830033 (iPhone 15 Pro Max) можно увидеть, как менялась его цена за последние 6 месяцев.

Чем отличается ID товара на Ozon и Wildberries?

На Wildberries ID товара называется nm_id (номер модели) и обычно состоит из 8 цифр (например, 12345678). На Ozon это просто product_id (9–10 цифр). Форматы не совместимы — один и тот же товар будет иметь разные ID на двух маркетплейсах.

Можно ли по ID товара найти его в других странах (например, на Ozon в Казахстане)?

Нет, ID товара уникален для каждого регионального домена Ozon. Например, iPhone 15 на ozon.ru и ozon.kz будет иметь разные product_id, даже если это тот же самый товар. Чтобы найти аналог, используйте артикул производителя.

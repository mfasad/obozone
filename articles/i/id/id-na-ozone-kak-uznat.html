Как узнать ID товара на Ozon: 5 проверенных способов для покупателей и продавцов

Каждый товар на маркетплейсе Ozon имеет уникальный идентификатор — ID (или артикул), который используется для поиска, аналитики, возвратов и интеграции с системами учета. Без этого кода невозможно оформить возврат через поддержку, синхронизировать остатки в 1С или найти товар в каталоге, если его название изменилось. Проблема в том, что ID не всегда виден на видном месте — его приходится искать в коде страницы, URL или специальных разделах личного кабинета.

В этой статье мы разберем все актуальные способы, как узнать ID товара на Ozon в 2026 году — от базовых (для покупателей) до продвинутых (для продавцов с доступом к API). Вы узнаете, где искать артикул в мобильном приложении, на сайте, в личном кабинете продавца, а также как извлечь его из ссылок и HTML-кода. Отдельно рассмотрим типичные ошибки, из-за которых пользователи путают ID с другими кодами (например, SKU или штрихкодом).

1. Что такое ID товара на Ozon и зачем он нужен

ID (или артикул Ozon) — это уникальный числовой код, который маркетплейс присваивает каждому товару в каталоге. Он не повторяется, не меняется со временем и служит основным идентификатором для:

  • 🔍 Поиска товара в личном кабинете продавца или через API.
  • 📦 Оформления возвратов — поддержка Ozon часто запрашивает ID для ускорения процесса.
  • 📊 Аналитики продаж — в отчетах данные привязаны именно к ID, а не к названию.
  • 🔄 Синхронизации с 1С/УТМ — без ID невозможно обновить остатки или цены.
  • 🛠️ Технической поддержки — при обращении по проблемам с товаром всегда требуют артикул.

Важно не путать ID с другими кодами:

Термин Пример Где используется
ID (артикул Ozon) 123456789 Внутренняя идентификация товара на маркетплейсе
SKU (артикул продавца) ABC-123-XL Учет товара у продавца (может совпадать с штрихкодом)
Штрихкод (EAN/UPC) 4607001234567 Маркировка товара для сканирования
Offer ID 987654321 Идентификатор предложения (для вариаций товара)

ID товара на Ozon всегда состоит из 8–10 цифр и не содержит букв или символов. Если вы видите код с буквами (например, OZ123456) — это не ID, а внутренний арт продавца или служебный номер заказа.

2. Как узнать ID товара на Ozon через сайт (для покупателей)

Самый простой способ — извлечь артикул прямо из адресной строки браузера. Вот пошаговая инструкция:

  1. Откройте карточку товара на сайте Ozon.ru.
  2. Посмотрите на URL в адресной строке. Он будет выглядеть так: 
    https://www.ozon.ru/product/{тут_ваш_ID}-какая-то-фраза/
  3. Скопируйте цифры после /product/ и до первого тире. Например, в ссылке https://www.ozon.ru/product/123456789-nazvanie-tovara/ ID равен 123456789.

Если URL имеет другой формат (например, с параметрами ?asb=...), обновите страницу — иногда Ozon подгружает "чистую" ссылку.

ID состоит только из цифр|Длина кода — 8–10 символов|В URL ID идет сразу после /product/|Нет пробелов или тире внутри ID-->

Если в URL нет цифр (например, ссылка выглядит как /product/naimenovanie-tovara-12345/), попробуйте альтернативный метод:

  1. Нажмите правой кнопкой мыши на странице товара и выберите Просмотреть код (или Inspect в Chrome).
  2. Нажмите Ctrl+F и введите productId.
  3. Найдите строку вида "productId":123456789 — это и есть нужный ID.
💡

Если вы часто ищете ID, установите расширение для браузера JSON Viewer — оно упрощает поиск в коде страницы.

3. Как найти ID в мобильном приложении Ozon

В официальном приложении Ozon (для Android/iOS) артикул спрятан глубже, но его можно извлечь двумя способами:

Способ 1: Через "Поделиться"

  1. Откройте карточку товара в приложении.
  2. Нажмите на иконку ⋯ (три точки) в верхнем правом углу.
  3. Выберите Поделиться → Скопировать ссылку.
  4. Вставьте ссылку в любое поле ввода (например, в заметки). Она будет выглядеть как: 
    https://ozon.ru/t/{буквы_и_цифры}?asb=...

    Здесь ID — это цифры после последнего тире в пути. Например, в https://ozon.ru/t/abc123-456789 артикул — 456789.

Способ 2: Через браузерную версию

Если первый метод не сработал:

  1. В приложении нажмите на иконку ⋯ → Открыть в браузере.
  2. В браузере используйте метод из раздела про сайт (просмотр URL или кода страницы).

4. Где продавцы могут найти ID товара в личном кабинете Ozon

Для продавцов на Ozon (в моделях FBS/FBO) ID товара доступен в нескольких разделах личного кабинета. Рассмотрим самые быстрые способы:

Способ 1: Каталог товаров

  1. Перейдите в раздел Товары → Мой каталог.
  2. Найдите нужный товар в списке. ID отображается в колонке Артикул Ozon (может называться ID или Product ID).
  3. Если колонки нет, нажмите на иконку ⚙️ (Настройки таблицы) и добавьте галочку напротив Артикул Ozon.

Способ 2: Карточка товара

  1. В разделе Мой каталог кликните на название товара.
  2. В открывшейся карточке ID будет указан в блоке Основная информация (рядом с названием и категорией).

Способ 3: Экспорт отчета

Если товаров много, удобнее выгрузить данные в Excel:

  1. Перейдите в Товары → Мой каталог.
  2. Нажмите Экспорт → выберите формат XLSX.
  3. В полученном файле ID будет в колонке Product ID или Артикул Ozon.

⚠️ Внимание: В личном кабинете Ozon Seller иногда отображается Offer ID (для вариаций товара) вместо основного Product ID. Убедитесь, что копируете именно артикул товара, а не предложения — он короче и состоит только из цифр.

💡

Для товаров с вариациями (цвет/размер) у каждой модификации свой Offer ID, но общий Product ID остается неизменным.

5. Как извлечь ID товара из API Ozon (для продавцов)

Если вы используете API Ozon для автоматизации (например, через 1С, MoySklad или собственные скрипты), ID товара можно получить программно. Для этого:

Шаг 1: Получите токен доступа

Авторизуйтесь в личном кабинете API и сгенерируйте Client-ID и API-Key.

Шаг 2: Выполните запрос

Используйте endpoint для поиска товаров:

GET https://api-seller.ozon.ru/v2/product/info

Headers:


Client-Id: {ваш_Client-ID}


Api-Key: {ваш_API-Key}


Body:


{


"product_id": [123456789], // здесь можно указать известный ID или оставить пустым для выгрузки всех товаров


"offer_id": ""


}

В ответе будет массив товаров с полями product_id (это и есть ID) и другими данными:

{

"result": {


"items": [


{


"product_id": 123456789, // нужный нам ID


"offer_id": "987654321",


"name": "Название товара",


...


}


]


}


}

Шаг 3: Автоматизируйте выгрузку

Для регулярного получения ID всех товаров используйте скрипт на Python:

import requests


url = "https://api-seller.ozon.ru/v2/product/info/list"


headers = {


"Client-Id": "YOUR_CLIENT_ID",


"Api-Key": "YOUR_API_KEY"


}



response = requests.post(url, headers=headers, json={"limit": 1000})


data = response.json()



for item in data["result"]["items"]:


print(f"ID: {item['product_id']}, Название: {item['name']}")

⚠️ Внимание: При работе с API Ozon учитывайте лимиты запросов — бесплатный тариф позволяет делать до 1000 запросов в час. Превышение лимита приведет к блокировке ключа на 1 час.

6. Типичные ошибки при поиске ID и как их избежать

Даже опытные пользователи иногда путают ID с другими кодами. Вот самые распространенные ошибки и как их не допустить:

  • 🔢 Путают ID с SKU продавца:

    SKU — это внутренний артикул продавца (может содержать буквы, например, T-SHIRT-BLUE-XL), а ID — цифровой код Ozon. Всегда проверяйте, что в коде только цифры.

  • 🔗 Копируют неполный ID из URL:

    В некоторых ссылках ID обрезается (например, /product/123456789-... становится /p/123456/). Всегда сверяйте длину кода — она должна быть 8–10 символов.

  • 📦 Используют Offer ID вместо Product ID:

    Если товар имеет вариации (цвета, размеры), у каждой модификации свой Offer ID, но общий Product ID остается один. Для возвратов и аналитики нужен именно Product ID.

  • 📱 Доверяют мобильному приложению:

    В приложении Ozon иногда отображается усеченный ID. Для точности всегда переходите в браузерную версию.

Ошибка Пример Как исправить
Скопирован SKU вместо ID SHOE-BLACK-42 Искать цифровой код в URL или личном кабинете
ID обрезан в ссылке /p/123456/ вместо 123456789 Обновить страницу или посмотреть полный URL
Использован Offer ID 987654321 (для размера M) Найти Product ID в карточке товара
Что делать, если ID товара изменился?

На Ozon ID товара фиксирован и не меняется со временем. Если вы видите, что код изменился, скорее всего:

1. Товар был удален и создан заново (тогда Assign новый ID).

2. Вы путаете Product ID с Offer ID (для вариаций).

3. Произошла ошибка в личном кабинете — попробуйте обновить страницу или очистить кэш браузера.

7. Альтернативные способы найти ID (если ничего не работает)

Если стандартные методы не сработали, попробуйте эти варианты:

Способ 1: Через поддержку Ozon

Напишите в чат поддержки (раздел Помощь в личном кабинете) с просьбой предоставить ID товара. Приложите:

  • Ссылку на товар.
  • Скриншот карточки (если товар не публичный).
  • Пояснение, зачем нужен ID (например, для возврата или синхронизации).

Обычно ответ приходит в течение 1–2 часов.

Способ 2: Через Excel-отчет по продажам

Если вы продавец:

  1. В личном кабинете перейдите в Аналитика → Отчеты → Отчет по товарам.
  2. Экспортируйте данные в Excel.
  3. Найдите нужный товар по названию — ID будет в одной из колонок.

Способ 3: Через инструменты веб-разработчика

Для продвинутых пользователей:

  1. Откройте карточку товара в Chrome.
  2. Нажмите F12 → перейдите на вкладку Network.
  3. Обновите страницу (F5).
  4. Найдите запрос с названием product или api.
  5. В ответе (Response) будет поле "productId".
💡

Если товар удален или архивирован, его ID можно найти только через API или поддержку. В личном кабинете такие товары не отображаются.

FAQ: Частые вопросы про ID товара на Ozon

Можно ли узнать ID товара без доступа к личному кабинету продавца?

Да, ID доступен всем пользователям — он отображается в URL карточки товара или в коде страницы. Для этого не нужен доступ к продавцу.

Что делать, если в URL нет цифр (ссылка вида /product/naimenovanie/)?

Обновите страницу — иногда Ozon подгружает "чистый" URL с ID. Если не поможет, посмотрите код страницы (Ctrl+U) и найдите productId.

Как найти ID для вариаций товара (разные цвета/размеры)?

У каждой вариации есть свой Offer ID, но общий Product ID остается один. Например:

  • Товар: Куртка (Product ID: 123456789)
  • Вариации:
    • Черный, M (Offer ID: 987654321)
    • Синий, L (Offer ID: 987654322)

Для большинства операций (возвраты, аналитика) нужен Product ID.

Можно ли по ID товара узнать, кто его продавец?

Нет, ID сам по себе не содержит информации о продавце. Однако если вы продавец, в личном кабинете по ID можно увидеть, кому принадлежит товар (себе или другому продавцу на Ozon).

Что такое Group ID и чем он отличается от Product ID?

Group ID — это идентификатор группы товаров (например, для комплекта из нескольких позиций). Он используется редко и не заменяет Product ID. Обычно Group ID можно игнорировать.

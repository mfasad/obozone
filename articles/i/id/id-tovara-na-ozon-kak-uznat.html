ID товара на Ozon: как узнать артикул за 30 секунд — гайд 2026

Вы когда-нибудь сталкивались с ситуацией, когда менеджер Озон просит назвать ID товара, а вы теряетесь в догадках, где его искать? Или может быть, как продавец, вам нужно оперативно найти артикул для синхронизации с 1С? Этот уникальный числовой код — как паспорт продукта на маркетплейсе: без него невозможно отследить заказ, оформить возврат или интегрировать товар в внешние системы.

В этой статье мы разберём все актуальные способы получения ID — от очевидных (через карточку товара) до малоизвестных (парсинг URL или API). Особое внимание уделим нюансам для покупателей и продавцов, так как алгоритмы поиска для них отличаются. А в конце вас ждёт эксклюзивный лайфхак для массового получения ID через Excel — это сэкономит часы работы при загрузке большого каталога.

Что такое ID товара на Ozon и зачем он нужен

ID (или артикул Озон) — это уникальный числовой идентификатор, который маркетплейс присваивает каждому товару в каталоге. Формат кода: от 123456789 до 9876543210 (обычно 9–10 цифр). Этот код не меняется со временем, даже если продавец редактирует название или описание товара.

Для чего может понадобиться ID:

  • 📦 Покупателям: уточнение деталей заказа в поддержке, оформление возврата, поиск аналогичных товаров.
  • 📊 Продавцам: синхронизация с 1С/МойСклад, настройка рекламных кампаний, аналитика продаж, массовое редактирование цен.
  • 🔄 Партнёрам: интеграция через API, парсинг данных для сравнения цен.
⚠️ Внимание: Не путайте ID товара с артикулом продавца (SKU) — это разные вещи! SKU присваивает сам продавец, а ID генерирует Озон автоматически. В карточке товара они могут отображаться рядом, но функционально не взаимозаменяемы.

Например, в URL товара https://www.ozon.ru/product/iphone-15-pro-max-256gb-natural-titanium-172345678/ цифры 172345678 — это и есть ID. Но не всегда код виден в адресной строке: Озон периодически обновляет структуру ссылок, поэтому лучше использовать альтернативные методы.

Способ 1: Как найти ID через карточку товара (для покупателей)

Самый простой метод, который работает в 90% случаев. Вам не нужны специальные знания — только браузер и 20 секунд времени.

Инструкция:

  1. Откройте карточку нужного товара на Ozon.ru или в мобильном приложении.
  2. Прокрутите страницу вниз до блока "Характеристики".
  3. Найдите строку Артикул Ozon или ID товара — рядом будет числовой код.
  4. Скопируйте его (можно выделить мышкой или нажать Ctrl+C).

Если блока с характеристиками нет (так бывает у некоторых продавцов), попробуйте:

  • 🔍 Нажмите Ctrl+F и введите в поисковой строке браузера слово артикул.
  • 📱 В мобильном приложении тапните на три точки в правом верхнем углу карточки → Поделиться → скопируйте ссылку и выделите из неё цифры после последнего слеша.

ID состоит только из цифр (без букв и символов)

Длина кода — 9 или 10 знаков

Первая цифра не может быть 0

Код совпадает в карточке и в URL (если виден)

-->

Пример корректного ID: 1672345890. Если вы видите что-то вроде APL-IP15-256-NT — это SKU продавца, а не ID Озон.

Способ 2: Получение ID через URL (скрытый метод)

Этот способ работает даже если артикул не отображается в характеристиках. Подходит для технически подкованных пользователей, так как требует анализа структуры ссылки.

Алгоритм действий:

  1. Откройте карточку товара и скопируйте URL из адресной строки.
  2. Вставьте ссылку в текстовый редактор (например, Блокнот).
  3. Найдите в URL фрагмент вида /product/название-товара-123456789/ или /p/контекстный-id-A1234567/.
  4. Цифры перед последним слешем — это и есть ID. Например, в .../iphone-15-pro-172345678/ ID равен 172345678.

Обратите внимание на возможные вариации URL:

Тип ссылки Пример Где искать ID
Стандартная ozon.ru/product/iphone-15-123456789/ Цифры перед последним /
Контекстная (из рекламы) ozon.ru/p/abc123d-456789/ Цифры после тире (456789)
Мобильное приложение ozon://product/987654321 Цифры после product/
Устаревший формат ozon.ru/context/detail/id/12345678/ Цифры после id/
⚠️ Внимание: Если в URL нет цифр (например, ozon.ru/highlight/akciya-123/), значит вы попали на промо-страницу, а не на карточку товара. Вернитесь на шаг назад и выберите конкретный товар из каталога.

Способ 3: ID для продавцов — где смотреть в личном кабинете Ozon Seller

Если вы продавец, у вас есть доступ к расширенным инструментам получения ID. Самый надёжный способ — через панель управления Ozon Seller.

Пошаговая инструкция:

  1. Авторизуйтесь в Личном кабинете продавца.
  2. Перейдите в раздел Товары → Мои товары.
  3. Найдите нужный товар в списке (можно использовать поиск по названию или SKU).
  4. В колонке "ID" будет отображён числовой код. Если колонки нет — нажмите Настройки таблицы (значок шестерёнки) и добавьте её.

Альтернативные пути:

  • 📄 В разделе Заказы кликните на номер заказа → найдите строку с товаром → ID будет в скобках рядом с названием.
  • 📊 В Отчётах → Товарные остатки скачайте Excel-файл — ID будет в колонке Product ID.
  • 🔧 В API → Каталог используйте метод ProductInfo для массового получения ID (требуются права доступа).
💡

Если вы часто работаете с большим количеством товаров, экспортируйте каталог в Excel через раздел "Товары → Экспорт". В полученной таблице будет колонка с ID, которую можно использовать для дальнейшей обработки в 1С или Google Sheets.

Для продавцов на FBS (модель с хранением на складе Озон) ID также дублируется в:

  • 📦 Этикетках товаров (штрихкод + цифровой ID).
  • 📋 Акта приёмки (колонка Артикул Озон).

Способ 4: Массовое получение ID через Excel (для продавцов)

Если вам нужно получить ID для сотен или тысяч товаров, ручной поиск займёт недели. К счастью, Озон позволяет автоматизировать процесс через выгрузку данных.

Инструкция для массового экспорта:

  1. В Ozon Seller перейдите в Товары → Экспорт.
  2. Выберите формат XLSX и укажите нужные колонки (обязательно включите Product ID).
  3. Нажмите Экспортировать и дождитесь формирования файла (может занять до 10 минут).
  4. Откройте скачанный файл в Excel и отфильтруйте колонку Product ID.

Для ускорения работы используйте формулы:

  • 🔢 Чтобы извлечь ID из URL в колонке A: =REGEXEXTRACT(A2; "/(\d{9,10})/") (в Google Sheets).
  • 📊 Чтобы проверить корректность ID: =IF(AND(LEN(B2)=9; ISNUMBER(VALUE(B2))); "OK"; "Ошибка").
Как автоматизировать обновление ID в 1С

1. Настройте обмен данными через модуль "Интеграция с Озон" (доступен в последних версиях 1С:УТ 11.4+).

2. В настройках обмена укажите соответствие полей: Product ID (Ozon) → Артикул (1С).

3. Запустите выгрузку каталога — ID автоматически подтянутся в номенклатуру.

4. Для проверки сгенерируйте отчёт "Сопоставление товаров" в 1С.

Если вам нужно обновить ID для товаров, которые уже загружены на Озон, используйте метод Import Prices and Stocks в API. Пример запроса:

Способ 5: Парсинг ID через API (для разработчиков)

Для продвинутых пользователей и программистов Озон предоставляет API, через которое можно получать ID в автоматическом режиме. Это актуально для интеграции с CRM, аналитическими системами или собственными сервисами.

Основные методы API для работы с ID:

Метод Описание Пример запроса
ProductInfo Получение данных о товаре по SKU или ID GET /v1/product/info?product_id=123456789
ProductsList Список всех товаров продавца с ID GET /v1/products/list?limit=1000
Search Поиск товаров по ключевым словам с выводом ID POST /v1/products/search {"query": "iPhone 15"}

Чтобы начать работу с API:

  1. Получите Client-ID и API-Key в разделе Настройки → API в личном кабинете.
  2. Используйте библиотеки для работы с HTTP-запросами (например, requests для Python или axios для JavaScript).
  3. Оформите запрос с обязательным заголовком авторизации: Client-ID: ваш_id и Api-Key: ваш_ключ.

Пример кода на Python для получения ID всех товаров:

import requests


url = "https://api-seller.ozon.ru/v1/products/list"


headers = {


"Client-ID": "ваш_client_id",


"Api-Key": "ваш_api_key"


}



response = requests.get(url, headers=headers)


products = response.json()["result"]["items"]



for product in products:


print(f"Название: {product['name']}, ID: {product['product_id']}")

⚠️ Внимание: Озон ограничивает количество запросов к API — 1000 в час для стандартного тарифа. При превышении лимита вы получите ошибку 429 Too Many Requests. Используйте кэширование или разбивайте задачи на пакеты.

Частые ошибки и как их избежать

Даже опытные пользователи иногда путают ID с другими кодами или допускают ошибки при поиске. Разберём типичные проблемы и их решения.

Ошибка 1: Путаница между ID и SKU

  • 🆔 ID Озон — уникальный код маркетплейса (например, 123456789).
  • 🏷️ SKU продавца — внутренний артикул (например, APL-IP15-256).

Как отличить: ID всегда числовой, SKU может содержать буквы и символы.

Ошибка 2: ID не отображается в карточке товара

Решения:

  • 🔄 Обновите страницу (F5) — иногда данные подгружаются с задержкой.
  • 🕵️ Проверьте URL (способ 2 из этой статьи).
  • 📧 Напишите в поддержку Озон с просьбой предоставить ID (укажите ссылку на товар).

Ошибка 3: При массовой выгрузке ID дублируются или пропускаются

Причины и фиксы:

Проблема Причина Решение
Одинаковые ID для разных товаров Ошибка экспорта (кэш браузера) Очистите кэш или используйте другой браузер
Пропущенные строки в Excel Ограничение лимита выгрузки (10 000 строк) Разбейте выгрузку на части по 5 000 товаров
ID в формате 1.23E+09 Excel преобразует большие числа Форматируйте колонку как Текстовый до вставки данных
💡

Всегда проверяйте первые 5–10 строк выгруженного файла на корректность ID. Если формат нарушен, повторите экспорт с другими настройками или обратитесь в поддержку Озон.

FAQ: Ответы на частые вопросы

Можно ли узнать ID товара, если он уже удалён из каталога?

Да, но только если товар был в заказах. Перейдите в раздел Заказы в личном кабинете, найдите заказ с этим товаром и кликните на его название — в всплывающем окне будет ID. Если товар не продавался, восстановить ID невозможно.

Что делать, если ID товара изменился?

Озон гарантирует, что ID не меняется в течение всего жизненного цикла товара. Если вы заметили изменение, скорее всего:

  • Вы путаете разные товары (например, модификации одного продукта).
  • Произошла ошибка кэширования — очистите куки браузера.
  • Товар был удалён и создан заново (тогда Assign новый ID).

Проверьте историю изменений в разделе Товары → Архив.

Как найти ID для товаров, которые ещё не загружены на Озон?

ID присваивается только после успешной загрузки товара в каталог. Если товар ещё не создан, у него нет ID. Вам нужно:

  1. Загрузить товар через Товары → Добавить товар.
  2. Дождаться модерации (обычно 1–2 часа).
  3. После одобрения ID появится в карточке товара и в выгрузках.

Для ускорения используйте шаблоны Excel от Озон — они автоматически генерируют ID после загрузки.

Можно ли по ID товара узнать, кто его продавец?

Нет, ID сам по себе не содержит информации о продавце. Однако вы можете:

  • Открыть карточку товара по ссылке ozon.ru/product/{ID} и посмотреть название магазина.
  • Использовать API-метод ProductInfo с параметром include_seller_info=true.

Озон скрывает данные продавца для некоторых категорий (например, Ozon Global), поэтому этот способ работает не всегда.

Как защититься от мошенников, которые просят ваш ID товара?

ID товара — это публичная информация, и её знание не даёт мошенникам доступа к вашему аккаунту. Однако будьте осторожны, если:

  • Вам предлагают "проверку товара" через сторонние сервисы (это может быть фишинг).
  • Просят ввести ID на подозрительных сайтах (реальные запросы Озон отправляет только через официальные каналы).
  • С вас требуют оплату за "разблокировку" ID (это всегда бесплатно).

Проверяйте домен сайта (должен быть ozon.ru или seller.ozon.ru) и не переходите по ссылкам из писем без проверки.

Как узнать ID продавца на Ozon: все способы для личного кабинета, API и документов

Вы только зарегистрировались как продавец на Ozon или уже давно ведёте бизнес на маркетплейсе, но вдруг потребовался ваш уникальный идентификатор? ID продавца (Seller ID) — это ключ ко многим операциям: от интеграции через API до общения с поддержкой и оформления финансовых документов. Без него невозможно настроить автоматизацию, подключить некоторые сервисы или даже подтвердить своё право на аккаунт в спорных ситуациях.

Проблема в том, что Ozon не всегда показывает этот идентификатор на видном месте. В зависимости от версии личного кабинета, типа аккаунта (FBS или FBO) и даже браузера путь к ID может отличаться. В этой статье мы собрали все актуальные способы получения Seller ID в 2026 году, включая скрытые методы для технической поддержки и партнёрских программ. Также разберём, почему некоторые продавцы путают ID продавца с ID магазина или клиентским идентификатором — и чем они отличаются.

Что такое Seller ID на Ozon и зачем он нужен

Seller ID (или идентификатор продавца) — это уникальный числовой код, который Ozon присваивает каждому зарегистрированному продавцу. Он не меняется со временем (в отличие от названия магазина) и используется для:

  • 🔗 Интеграции с API: без Seller ID невозможно отправлять запросы для управления товарами, заказами или статистикой через программный интерфейс.
  • 📄 Оформления документов: некоторые банки и налоговые службы требуют указать этот идентификатор для подтверждения деятельности на маркетплейсе.
  • 💬 Общения с поддержкой: при обращении в службу помощи Ozon часто просит назвать ID для быстрой идентификации аккаунта.
  • 🔄 Миграции между аккаунтами: если вы переходите с FBO на FBS или наоборот, Seller ID остаётся прежним.

Важно не путать его с другими идентификаторами:

  • 🏷️ ID магазина (Shop ID) — присваивается каждому витрине (может быть несколько у одного продавца).
  • 👤 Client ID — используется для авторизации в личном кабинете (меняется при каждом входе).
  • 📦 SKU или Product ID — идентификаторы товаров, а не продавца.
⚠️ Внимание: Никогда не передавайте свой Seller ID третьим лицам без проверки их надежности. Мошенники могут использовать его для фишинга или несанкционированного доступа к вашему аккаунту через уязвимости в API.

Способ 1: Как найти Seller ID в личном кабинете Ozon Seller

Самый простой и надёжный способ — посмотреть идентификатор в интерфейсе личного кабинета. Инструкция актуальна для Ozon Seller на компьютере (веб-версия) и мобильном браузере:

  1. Авторизуйтесь на сайте seller.ozon.ru под своим аккаунтом продавца.

  2. В правом верхнем углу кликните на иконку профиля (аватар или первую букву названия магазина).

  3. В выпадающем меню выберите пункт Настройки аккаунта.

  4. На открывшейся странице в блоке Основная информация найдите строку ID продавца (Seller ID). Это и есть ваш уникальный идентификатор.

Открыт личный кабинет Ozon Seller|Нажата иконка профиля в правом верхнем углу|Выбран пункт"Настройки аккаунта"|Найден блок"Основная информация" с Seller ID-->

Если у вас несколько магазинов под одним аккаунтом, Seller ID будет одинаковым для всех. А вот Shop ID (идентификатор витрины) будет отличаться — его можно увидеть в разделе Магазины → Настройки.

⚠️ Внимание: В мобильном приложении Ozon Seller (на Android/iOS) путь к Seller ID может отличаться. Если не находите его в настройках, используйте браузерную версию сайта.

💡

Скопируйте свой Seller ID в безопасное место (например, в менеджер паролей или зашифрованную заметку). Это избавит от необходимости искать его каждый раз.

Способ 2: Получение ID через API (для технических специалистов)

Если вы разрабатываете собственное API-решение или используете готовые сервисы для автоматизации (например, MoySklad, 1C, RetailCRM), то Seller ID можно получить программно. Для этого:

  1. Авторизуйтесь в личном кабинете API Ozon.

  2. Перейдите в раздел Ключи API.

  3. В блоке Общая информация будет указан ваш Seller ID (иногда он дублируется в URL-адресе страницы после символов seller_id=).

Также идентификатор можно получить через API-запрос. Пример на Python с использованием библиотеки requests:

import requests


url ="https://api-seller.ozon.ru/v1/analytics/data"


headers = {


"Client-Id":"Ваш_Client_ID",


"Api-Key":"Ваш_API_ключ",


"Content-Type":"application/json"


}



response = requests.post(url, headers=headers, json={"date_from":"2026-01-01"})


print(response.json["seller_id"]) # Ваш Seller ID будет в ответе

Что делать, если API возвращает ошибку 403?

Ошибка 403 Forbidden при запросе к API Ozon обычно означает, что:

1) Неверно указан Client-Id или Api-Key (проверьте их в личном кабинете).

2) У вашего аккаунта нет прав на доступ к данному методу API (обратитесь в поддержку).

3) IP-адрес, с которого идёт запрос, не внесён в белый список (настройте это в разделе"Безопасность" личного кабинета API).

Если вы не технический специалист, лучше не использовать этот метод — достаточно Seller ID из личного кабинета. API-доступ требует осторожности, так как некорректные запросы могут привести к блокировке аккаунта.

Способ 3: Где посмотреть ID в документах от Ozon

Seller ID часто указывается в официальных документах, которые Ozon отправляет продавцам. Проверить его можно в следующих бумагах:

Тип документа Где искать Seller ID Пример формата
Договор оферты В шапке документа рядом с названием вашей компании Идентификатор продавца: 12345678
Акт выполненных работ В разделе"Реквизиты сторон" или в подвале документа Seller #987654321
Счёт на оплату услуг FBS В поле"Продавец" или"Поставщик" ID: 555888777
Письмо от поддержки Ozon В подписи или в теле письма (если запрашивали помощь) Ваш Seller ID: 11223344

Если вы не можете найти документ с Seller ID, запросите его повторно через поддержку. Для этого:

  1. Перейдите в раздел Помощь → Сообщения в личном кабинете.
  2. Создайте новое обращение с темой "Запрос на повторную отправку договора".
  3. Укажите, что вам нужен документ с указанием идентификатора продавца.
⚠️ Внимание: В некоторых старых версиях договоров (до 2022 года) Seller ID мог обозначаться как Partner ID или Vendor ID. Если вы видите такой термин — это, скорее всего, он.

Способ 4: Как узнать Seller ID через поддержку Ozon

Если ни один из предыдущих методов не сработал (например, у вас ограничен доступ к личному кабинету), можно запросить ID напрямую у службы поддержки. Для этого:

  1. Откройте раздел Помощь в личном кабинете Ozon Seller.

  2. Выберите категорию "Регистрация и доступ".

  3. Создайте новое обращение с темой: "Запрос на предоставление Seller ID".

  4. В теле сообщения укажите:

    • Название вашего магазина.
    • Email, привязанный к аккаунту.
    • Причину, по которой вам нужен идентификатор (например, "для интеграции с 1C").

  • Прикрепите скриншот личного кабинета (без конфиденциальных данных) для подтверждения прав.

    • Обычно ответ приходит в течение 1–3 рабочих дней. В некоторых случаях поддержка может запросить дополнительные документы (например, выписку из ЕГРИП или доверенность), чтобы подтвердить ваше право на аккаунт.

    💡

    При обращении в поддержку всегда указывайте точную причину запроса Seller ID. Это ускорит обработку обращения, так как менеджеры смогут сразу перенаправить запрос в нужный отдел.

    Способ 5: Альтернативные методы (для опытных пользователей)

    Если стандартные способы не подходят, можно попробовать следующие варианты:

    • 🔍 Через URL-адрес личного кабинета: В некоторых версиях интерфейса Seller ID отображается в адресной строке браузера после авторизации. Ищите цифры после /seller/ или ?id=.
    • 📊 В отчётах по продажам: Скачайте любой отчёт в формате Excel или CSV (раздел Аналитика → Отчёты). В некоторых шаблонах Seller ID указывается в заголовке файла или в первой строке.
    • 🤝 Через партнёрскую программу: Если вы участвуете в Ozon Affiliate, ваш Seller ID может быть указан в личном кабинете партнёра в разделе "Мои магазины".

    Для разработчиков: если у вас есть доступ к Ozon Merchant API, выполните запрос к методу /v1/seller/info — в ответе будет ваш идентификатор. Пример запроса:

    curl -X GET"https://api-seller.ozon.ru/v1/seller/info" \

    -H"Client-Id: ВАШ_CLIENT_ID" \
    

    -H"Api-Key: ВАШ_API_КЛЮЧ"

    Чем опасен публичный доступ к Seller ID?

    Хотя сам по себе Seller ID не даёт полного доступа к аккаунту, его утечка может привести к:

    1) Фишингу: мошенники могут отправлять вам письма"от имени Ozon" с просьбой подтвердить данные, ссылаясь на ваш ID.

    2) Несанкционированным API-запросам: если у злоумышленника есть ваш Client-Id, он может пытаться подобрать Api-Key брутфорсом.

    3) Конкурентной разведке: по Seller ID можно отследить ваши товары, цены и динамику продаж через публичные инструменты парсинга.

    Всегда проверяйте, кому и зачем вы сообщаете свой идентификатор!

    Частые ошибки и как их избежать

    При поиске Seller ID продавцы часто сталкиваются с следующими проблемами:

    • 🔄 Путают Seller ID с Shop ID: помните, что Shop ID привязан к конкретной витрине, а Seller ID — ко всему аккаунту продавца. Если у вас несколько магазинов, у них будет один Seller ID, но разные Shop ID.
    • 🔍 Ищут ID в мобильном приложении: не все функции доступны в Ozon Seller для смартфонов. Используйте браузерную версию.
    • 📧 Не проверяют папку"Спам": письма от Ozon с документами иногда попадают в спам, особенно если используете корпоративную почту.
    • 🔒 Делятся ID без проверки: никогда не отправляйте Seller ID по запросу из неофициальных источников (например, из Telegram или WhatsApp).

    Если вы подозреваете, что ваш Seller ID скомпрометирован (например, получили странное письмо с его упоминанием), немедленно:

    1. Смените пароль от личного кабинета Ozon Seller.
    2. Проверьте активные сессии в разделе Безопасность.
    3. Отзовите все API-ключи в личном кабинете разработчика.
    4. Свяжитесь с поддержкой для проверки подозрительных действий.

    FAQ: Ответы на частые вопросы

    Может ли Seller ID измениться со временем?

    Нет, Seller ID присваивается один раз при регистрации аккаунта продавца и остаётся неизменным. Даже если вы смените название магазина, юридическое лицо или тип аккаунта (FBO/FBS), идентификатор останется прежним.

    Что делать, если в личном кабинете нет раздела"Настройки аккаунта"?

    Возможно, у вас устаревшая версия интерфейса или ограниченные права доступа. Попробуйте:

    • Очистить кэш браузера и обновить страницу (Ctrl + F5).
    • Войти через другой браузер (например, Chrome вместо Safari).
    • Обратиться в поддержку с просьбой предоставить Seller ID.

    Можно ли узнать Seller ID другого продавца?

    Нет, Ozon не предоставляет публичный доступ к Seller ID других продавцов. Однако некоторые сервисы парсинга (например, DataLens или Peer2Peer) могут оценивать активность конкурентов по косвенным признакам (например, по Shop ID, который виден в URL товаров).

    Нужен ли Seller ID для работы с Ozon Rocket?

    Да, если вы подключаете Ozon Rocket (сервис ускоренной доставки) через API, то Seller ID потребуется для авторизации запросов. Его нужно будет указать при настройке интеграции в личном кабинете Rocket.

    Как проверить, что мой Seller ID не заблокирован?

    Если ваш Seller ID заблокирован (например, за нарушение правил Ozon), вы увидите соответствующее уведомление в личном кабинете или получите письмо на email. Также блокировку можно проверить, отправив тестовый API-запрос — если приходит ошибка 403 или 429, возможно, доступ ограничен.

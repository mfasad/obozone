Почему термопринтер — обязательный инструмент для работы с Озон

Продажи на маркетплейсе Озон требуют строгого соблюдения правил маркировки товаров. Без корректно распечатанных этикеток ваши посылки не примут на сортировочных центрах, а задержки с отправкой грозят штрафами или блокировкой аккаунта. Термопринтер решает эту задачу быстро и профессионально: он печатает этикетки с кодом DataMatrix, штрихкодами и текстовой информацией в формате, который принимает система Озон.

В отличие от обычных принтеров, термопринтеры не требуют чернил или тонера — изображение наносится на специальную термобумагу нагревом. Это дешевле, быстрее и надёжнее: этикетка не смазывается при намокании и сохраняет читаемость даже после длительного хранения. Но чтобы всё работало без сбоев, нужно правильно выбрать оборудование, настроить его и интегрировать с личным кабинетом Озон.

Какой термопринтер выбрать для печати этикеток Озон

Не все термопринтеры подходят для работы с Ozon FBS/FBO. Основные требования:

  • 📏 Ширина печати не менее 104 мм (стандартный размер этикетки Озон — 100×70 мм)
  • 🖨️ Разрешение от 203 dpi (для чёткого сканирования DataMatrix)
  • 🔌 Интерфейсы: USB (обязательно), Ethernet или Wi-Fi (для сетевой печати)
  • 📦 Поддержка языков печати ZPL/EPL (для интеграции с API Озон)

Среди популярных моделей:

Модель Ширина печати Разрешение Интерфейсы Цена (2026) Подходит для Озон?
Zebra ZD420 104 мм 203 dpi USB, Ethernet, Bluetooth ~25 000 ₽ ✅ Да
Godex EZ-2300 108 мм 203 dpi USB, Ethernet ~18 000 ₽ ✅ Да
Rongta RP320 80 мм 203 dpi USB ~12 000 ₽ ❌ Нет (узкая печать)
TSC TE200 104 мм 203/300 dpi USB, Ethernet ~22 000 ₽ ✅ Да

Для небольших объёмов (до 50 этикеток в день) подойдёт бюджетная модель Godex EZ-2300. Если вам нужна высокая скорость и надёжность (100+ этикеток/день), лучше выбрать Zebra ZD420 или TSC TE200.

⚠️ Внимание: Принтеры с шириной печати менее 100 мм (например, Rongta RP320) не смогут напечатать полноразмерную этикетку Озон. Вы получите обрезанный код DataMatrix, который не пройдёт сканирование на сортировочном центре.

Установка драйверов и подключение термопринтера к компьютеру

Перед первой печатью этикеток необходимо установить драйверы и настроить принтер в системе. Процесс отличается для Windows и macOS.

Для Windows:

  1. Подключите принтер к ПК через USB-порт (используйте оригинальный кабель!).
  2. Скачайте драйвер с официального сайта производителя (например, для Zebra — zebra.com/support).
  3. Запустите установщик и выберите модель вашего принтера (например, Zebra ZD420).
  4. После установки перейдите в Панель управления → Устройства и принтеры и проверьте, что принтер отображается без ошибок.

Для macOS:

  • Подключите принтер по USB.
  • Откройте Системные настройки → Принтеры и сканеры.
  • Нажмите «+» → выберите ваш термопринтер из списка.
  • Если принтер не определился автоматически, скачайте драйвер с сайта производителя (например, для Godex — godexintl.com/support).

После подключения рекомендуется распечатать тестовую страницу. Для этого:

  1. Откройте любой текстовый редактор (например, Блокнот).
  2. Введите команду для печати тестовой этикетки (пример для ZPL): 
    ^XA^FO20,20^A0N,25,25^FDTest Label^FS^XZ
  3. Сохраните файл с расширением .zpl и отправьте на печать.

Требования Озон к формату этикеток: размеры, поля, шрифты

Озон предъявляет жёсткие требования к внешнему виду этикеток. Если хоть один параметр не совпадёт, этикетка может не пройти сканирование. Основные правила:

  • 📏 Размер этикетки: 100×70 мм (допустимое отклонение: ±1 мм).
  • 🔲 Код DataMatrix: должен занимать левое верхнее поле (размер ячейки: 20×20 мм).
  • 📌 Штрихкод: Code 128 или GS1-128, высота не менее 15 мм.
  • 🔤 Текстовая информация: шрифт Arial или Times New Roman, размер 10–12 пт.
  • 🎨 Цвета: только чёрный текст на белом фоне (термобумага не поддерживает цветную печать).

Пример правильной этикетки:

┌────────────────────────────────────┐


│ ████████████████████              │


│ █ DataMatrix █   AR1234567890     │


│ ████████████████████              │


│                                    │


│ [Штрихкод Code 128]                │


│                                    │


│ Заказ: 123-456-7890               │


│ Артикул: ABCD1234                  │


│ Вес: 0.5 кг                        │


│                                    │


│ ОЗОН                              │


│ ОOO "Озон Лоджистикс"              │


│ г. Москва, ул. Примерная, д. 1    │


└────────────────────────────────────┘
⚠️ Внимание: Если вы печатаете этикетки для FBS, обязательно укажите AR-код (уникальный идентификатор заказа). Для FBO достаточно штрихкода и DataMatrix. Путаница между этими форматами — одна из главных причин отказов на сортировочных центрах.

Для удобства можно использовать готовые шаблоны:

  • 📥 Шаблон для Excel: скачайте с официального сайта Озон.
  • 🖼️ Графические макеты: подходят для дизайнерских программ (например, Adobe Illustrator или CorelDRAW).
  • 🤖 Автоматические сервисы: Labelary или NiceLabel для генерации ZPL-кода.
💡

Если вы печатаете этикетки партиями, настройте в драйвере принтера параметр "Печать нескольких копий на листе" — это сэкономит бумагу и время.

Как распечатать этикетки Озон: пошаговая инструкция

Есть два способа печати этикеток: вручную (через личный кабинет Озон) и автоматически (через API). Рассмотрим оба варианта.

Способ 1: Печать через личный кабинет Озон (для начинающих)

  1. Авторизуйтесь в личном кабинете продавца.
  2. Перейдите в раздел Заказы → Готовые к отправке.
  3. Выделите заказы, для которых нужны этикетки, и нажмите Печать этикеток.
  4. Выберите формат PDF или ZPL (если ваш принтер поддерживает язык ZPL).
  5. Откройте скачанный файл и отправьте на печать, выбрав ваш термопринтер.

Способ 2: Автоматическая печать через API (для опытных продавцов)

Для автоматизации печати этикеток можно использовать API Озон. Это актуально, если у вас много заказов (100+ в день).

Шаги настройки:

  1. Получите API-ключ в личном кабинете (Настройки → Интеграции → API).
  2. Используйте метод /v2/posting/fbs/create-shipping-labels для генерации этикеток.
  3. Настройте скрипт (например, на Python), который будет отправлять запрос на печать при появлении нового заказа.
  4. Пример кода для печати: 
    import requests


    url = "https://api-seller.ozon.ru/v2/posting/fbs/create-shipping-labels"
    

    headers = {
    

    "Client-Id": "Ваш_Client_ID",
    

    "Api-Key": "Ваш_API_ключ",
    

    "Content-Type": "application/json"
    

    }
    

    data = {
    

    "label_format": "zpl",
    

    "posting_number": ["123-456-7890", "123-456-7891"]
    

    }
    


    response = requests.post(url, headers=headers, json=data)
    

    with open("labels.zpl", "w") as f:
    

    f.write(response.text)

  5. Сохраните полученный ZPL-файл и отправьте на печать.

Для удобства можно интегрировать печать с 1С, МойСклад или другими системами учёта. Например, в 1С есть готовые обработки для работы с API Озон.

Как проверить, что этикетка напечатана правильно?

Перед отправкой заказа проверьте:

1. DataMatrix — должен сканироваться приложением Barcode Scanner (Android/iOS).

2. Штрихкод — проверьте через тот же сканер или онлайн-сервис (например, [zxing.org](https://zxing.org/w/decode.jspx)).

3. Текст — убедитесь, что все данные (артикул, вес, адрес) читаемы и соответствуют заказу.

4. Размер — приложите линейку: этикетка должна быть ровно 100×70 мм.

Распространённые ошибки и как их исправить

Даже опытные продавцы сталкиваются с проблемами при печати этикеток. Вот самые частые ошибки и их решения:

Проблема Причина Решение
Этикетка печатается размытой Низкое разрешение принтера или изношенная печатная головка Установите разрешение 203 dpi или выше. Очистите головку специальной салфеткой, смоченной в изопропиловом спирте
DataMatrix не сканируется Неправильный размер кода или низкое качество печати Проверьте настройки шаблона: код должен быть 20×20 мм. Используйте термобумагу высокого качества
Принтер не печатает Не установлены драйверы или проблема с подключением Переустановите драйверы. Проверьте USB-кабель или сетевое соединение
Этикетка обрезается по краям Неправильные настройки полей в шаблоне В драйвере принтера установите поля 0 мм или откорректируйте шаблон
Штрихкод читается как набор символов Неверный формат штрихкода (например, вместо Code 128 используется Code 39) Убедитесь, что в шаблоне указан формат Code 128 или GS1-128

Если проблема не решается, попробуйте:

  • 🔄 Перезагрузить принтер (отключите от питания на 30 секунд).
  • 📋 Обновить прошивку принтера (скачайте с сайта производителя).
  • 📞 Обратиться в поддержку Озон (seller-support@ozon.ru) с фотографией проблемной этикетки.
⚠️ Внимание: Если вы используете термобумагу низкого качества (например, слишком тонкую или с истёкшим сроком годности), этикетки могут выцветать через несколько дней. Это приведёт к тому, что на сортировочном центре не смогут отсканировать заказ. Покупайте бумагу с плотностью не менее 70 г/м² и храните её в сухом месте.
💡

Перед массовой печатью всегда распечатывайте 1–2 тестовые этикетки и проверяйте их сканером. Это сэкономит время и деньги на перепечатку.

Как сэкономить на печати этикеток: лайфхаки для продавцов

Печать этикеток — расходный процесс, но его можно оптимизировать. Вот несколько способов сократить затраты:

  • 💰 Покупайте термобумагу оптом. Например, рулон 100 мм × 75 м (5000 этикеток) обходится дешевле, чем пачки по 100 штук. Хорошие поставщики: Меркатор, ОфисМаг, 555-бумага.
  • ♻️ Используйте двустороннюю бумагу. Некоторые модели принтеров (например, Zebra ZD620) поддерживают печать на оборотной стороне, если этикетка не прошла сканирование.
  • ⚡ Настройте автоматическую печать. Интеграция с API Озон через 1С или МойСклад исключает ручной труд и ошибки.
  • 📊 Печатайте партиями. Если у вас 100+ заказов в день, настройте печать всех этикеток за один раз (например, вечером).

Ещё один способ сэкономить — использовать альтернативные программы для генерации этикеток:

  • Bartender — платная, но мощная программа для дизайна этикеток (поддерживает ZPL, EPL).
  • Labeljoy — бюджетный аналог с готовыми шаблонами для Озон.
  • Online Label Generator (например, Labelary) — бесплатный сервис для создания ZPL-кода.

Если вы только начинаете, попробуйте бесплатные решения. Для крупного бизнеса лучше инвестировать в Bartender или аналоги — это окупится за счёт экономии времени.

💡

Храните рулоны термобумаги в герметичных пакетах с силикагелем — это продлит их срок службы и предотвратит выцветание этикеток.

FAQ: Ответы на частые вопросы о печати этикеток Озон

Можно ли печатать этикетки Озон на обычном лазерном принтере?

Технически да, но это не рекомендуется. Лазерные принтеры:

  • Не всегда корректно печатают DataMatrix (может не сканироваться).
  • Требуют специальной бумаги с клеевым слоем (дороже термобумаги).
  • Медленнее термопринтеров при массовой печати.

Если у вас нет термопринтера, временно можно использовать лазерный, но только для тестовых этикеток. Для постоянной работы обязательно купите термопринтер.

Что делать, если DataMatrix не сканируется на сортировочном центре?

Причины и решения:

  1. Низкое качество печати: проверьте настройки принтера (разрешение должно быть ≥203 dpi).
  2. Неправильный размер: код должен быть ровно 20×20 мм.
  3. Повреждённая этикетка: не мните и не мочите бумагу.
  4. Ошибка в данных: перепроверьте AR-код или номер заказа.

Если проблема повторяется, свяжитесь с поддержкой Озон и пришлите фото этикетки.

Какой софт лучше использовать для генерации этикеток?

Выбор зависит от объёмов:

  • До 50 этикеток/день: достаточно встроенного функционала личного кабинета Озон.
  • 50–500 этикеток/день: Labeljoy или NiceLabel.
  • 500+ этикеток/день: Bartender + интеграция с API Озон.

Для автоматизации также подойдут 1С:Управление торговлей или МойСклад с модулем для Озон.

Можно ли использовать один термопринтер для этикеток Озон и Wildberries?

Да, но нужно учитывать:

  • Размеры этикеток у маркетплейсов разные: Озон — 100×70 мм, Wildberries — 100×50 мм.
  • Форматы штрихкодов могут отличаться (Озон использует Code 128, WB — GS1-128).
  • Придётся часто переключать шаблоны или использовать программу, которая поддерживает оба формата (например, Bartender).

Если объёмы большие, лучше завести отдельные принтеры для каждого маркетплейса.

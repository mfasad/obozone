Как удалить товар из акции на Ozon: все способы и нюансы

Участие в акциях на Ozon — мощный инструмент для увеличения продаж, но иногда товары нужно срочно исключить из промо-кампаний. Причины разные: закончился запас, изменилась себестоимость, или акция оказалась убыточной. В отличие от ручного изменения цены, удаление из акции требует точного соблюдения правил маркетплейса — иначе рискуете получить штрафы или блокировку карточки товара.

В этой статье разберём все актуальные способы удаления товаров из акций Ozon (включая скрытые функции личного кабинета и работу через API), а также проанализируем типичные ошибки, которые допускают продавцы. Особое внимание уделим срокам обработки запросов — от них зависит, успеете ли вы избежать продаж по убыточной цене. Если вы ищете способ отменить участие в акции заранее (до её старта), здесь тоже найдёте решение.

1. Когда нужно удалять товар из акции: 5 критичных случаев

Не все ситуации требуют срочного исключения товара из промо. Иногда достаточно приостановить продажи или скорректировать остатки. Рассмотрим сценарии, когда удаление из акции становится необходимостью:

  • 📉 Себестоимость выросла (например, из-за курса валют или логистики), и продажа по акционной цене приносит убытки. В этом случае даже временное участие в промо может нанести финансовый урон.
  • 🚫 Товар стал бракованным или просроченным. Продажа таких позиций в рамках акции усугубляет риски возвратов и штрафов за несоответствие описанию.
  • 🔄 Изменились условия акции (например, Ozon увеличил требуемую скидку или ввёл дополнительные комиссии). Это часто происходит с динамическими акциями типа "Скидка дня".
  • 📦 Закончился запас на складе FBS, но товар всё ещё отображается в акции. В этом случае лучше удалить его, чем рисковать штрафами за невыполнение заказов.
  • 📜 Нарушение правил акции (например, вы случайно подключили товар, не соответствующий условиям). Ozon может заблокировать карточку, если обнаружит несоответствие.

Важно отличать эти случаи от ситуаций, когда достаточно скорректировать цену или остатки. Например, если товар временно отсутствует на складе FBO, но скоро поступит, лучше приостановить продажи (Остаток = 0), а не удалять из акции.

⚠️ Внимание: Если товар уже куплен в рамках акции, но ещё не отправлен покупателю, его удаление из промо не отменит заказ. Вам придётся выполнить отгрузку по акционной цене или согласовать отмену с покупателем.

2. Способ 1: Удаление через личный кабинет Ozon Seller (пошагово)

Самый простой и быстрый метод — использовать интерфейс Ozon Seller. Он подходит для большинства акций, кроме некоторых эксклюзивных промо (например, "Молния" или "Чёрная пятница"), где требуется согласование с менеджером.

Инструкция актуальна для новой версии кабинета (2026 год):

  1. Перейдите в раздел Акции → Мои акции.
  2. Найдите нужную кампанию в списке и кликните на её название.
  3. В блоке Товары в акции нажмите кнопку Управление товарами.
  4. Отметьте галочками товары, которые нужно исключить, и выберите действие Удалить из акции.
  5. Подтвердите действие. Система покажет уведомление: Запрос на удаление отправлен. Ожидайте обработки.

Обратите внимание на статус обработки:

  • 🟡 В обработке — запрос принят, но товар ещё участвует в акции (может занять до 24 часов).
  • 🟢 Успешно — товар исключён, акционная цена сброшена.
  • 🔴 Ошибка — система не смогла обработать запрос (причины см. в разделе 4).

Если кнопка Удалить из акции неактивна, это может означать:

  • 🔒 Акция находится в статусе Завершена или Отменена — удаление невозможно.
  • ⏳ Прошло менее 24 часов с момента добавления товара в акцию (в некоторых кампаниях действует "период охлаждения").
  • 📌 Товар участвует в комбинированной акции (например, "Купи 2 по цене 1"), где удаление одного товара требует пересчёта условий для всех связанных позиций.

3. Способ 2: Массовое удаление через API (для опытных продавцов)

Если вам нужно исключить из акции десятки или сотни товаров, ручное управление через личный кабинет будет неэффективно. В этом случае используйте API Ozon Seller с методом promo/unlink.

Преимущества метода:

  • ⚡ Скорость: обработка запроса занимает до 1 часа (против 24 часов в ЛК).
  • 🤖 Автоматизация: можно интегрировать с вашей CRM или системой управления запасами.
  • 📊 Гибкость: возможность удалять товары по фильтрам (например, все позиции с остатком < 5 шт.).

Пример запроса для удаления товара с product_id = 12345678 из акции с promo_id = 98765432:

POST /v2/promo/unlink

Host: api-seller.ozon.ru


Content-Type: application/json


Client-Id: {ваш_client_id}


Api-Key: {ваш_api_key}



{


"promo_id": 98765432,


"product_id": 12345678


}

Ответ системы будет содержать поле status:

  • "ok" — запрос принят.
  • "error" — ошибка (детали в поле message).
Расширенные параметры API для удаления

Метод поддерживает массовое удаление через массив product_id:

"product_id": [12345678, 87654321, 11223344].

Также можно указать причину удаления в параметре reason (например, "out_of_stock" или "price_error"), что ускорит обработку запроса поддержкой.

⚠️ Внимание: При работе через API не превышайте лимиты запросов — 100 вызовов в минуту для метода promo/unlink. Иначе ваш Client-Id может быть временно заблокирован. Используйте пакетную обработку с задержкой в 1-2 секунды между запросами.

Если вы не знакомы с API, но хотите автоматизировать процесс, рассмотрите следующие инструменты:

  • 🛠️ Ozon Integrator — сервис для массового управления акциями без кода.
  • 🤖 Zapier + Ozon API — настройка триггеров (например, удалять товар из акции при остатке < 3 шт.).
  • 📊 1C + Ozon — если вы ведёте учёт в 1С, можно настроить автоматическое исключение товаров с нулевым остатком.

4. Типичные ошибки и как их избежать

Даже опытные продавцы сталкиваются с проблемами при удалении товаров из акций. Вот TOP-5 ошибок и способы их решения:

Ошибка Причина Решение
Товар не удаляется из акции Акция в статусе "На модерации" или "Завершена" Проверьте статус в Акции → Мои акции. Если акция завершена, удаление невозможно — только вручную через поддержку.
Ошибка "Товар участвует в другой акции" Товар добавлен в несколько промо-кампаний одновременно Удалите его из всех акций или дождитесь окончания самой ранней по дате.
Запрос висит в статусе "В обработке" > 24 часов Технический сбой или высокая нагрузка на серверы Ozon Напишите в поддержку с указанием promo_id и product_id. Приложите скриншот статуса.
После удаления цена не сбросилась Кэш системы или конфликт с другими промо (например, "Цена по карте Ozon") Очистите кэш в ЛК или дождитесь автоматического обновления (до 6 часов). Если проблема остаётся — обратитесь в поддержку.
Штраф за отмену участия в акции Удаление товара менее чем за 48 часов до старта акции (для некоторых типов промо) Проверьте правила акции в договоре. Если штраф назначен ошибочно — оспорьте его через раздел Финансы → Штрафы.

Критичный нюанс: если товар был удалён из акции, но покупатель успел оформить заказ по акционной цене до обработки запроса, вы обязаны выполнить отгрузку по этой цене. Иначе рискуете получить штраф за отмену заказа (до 100% от его стоимости).

Чтобы избежать таких ситуаций, используйте стратегию поэтапного удаления:

  1. Приостановите продажи товара (Остаток = 0).
  2. Дождитесь, пока все текущие заказы будут собраны и отправлены.
  3. Только после этого отправляйте запрос на удаление из акции.

5. Способ 3: Обращение в поддержку Ozon (когда ничего не работает)

Если стандартные методы не сработали, остаётся обратиться в службу поддержки. Это актуально для:

  • 🔒 Эксклюзивных акций (например, "Топ продаж" или "Премиум выбор"), где самоуправление товарами ограничено.
  • ⏰ Срочных случаев, когда нужно удалить товар за несколько часов до старта акции.
  • 🛑 Технических сбоев, когда запрос через ЛК или API зависает.

Как правильно составить запрос в поддержку, чтобы получить быстрый ответ:

  1. Тема письма: Срочно: удаление товара [ID] из акции [ID].
  2. Укажите:
    • promo_id (идентификатор акции),
    • product_id (идентификатор товара),
    • причину удаления (например, закончился запас на складе FBS),
    • скриншот ошибки (если запрос через ЛК/API не прошёл).
  • Ссылка на товар и акцию (например: https://seller.ozon.ru/.../promo/98765432).
  • Упоминание о срочности (если акция стартует через несколько часов).

    • Пример корректного обращения:

    Здравствуйте!


    Прошу срочно удалить товар с product_id = 12345678 из акции promo_id = 98765432 ("Скидка дня").
    

    Причина: закончился запас на складе FBS (остаток = 0), но товар всё ещё отображается в акции.
    

    Запрос через ЛК не проходит — статус "Ошибка" (скриншот прилагаю).
    

    Акция стартует через 3 часа, прошу решить вопрос оперативно.
    


    Ссылка на товар: [ссылка]
    

    Ссылка на акцию: [ссылка]
    


    С уважением,
    

    [Ваше имя]/[Название компании]

    Срок обработки такого запроса:

    • 🟢 До 4 часов — если акция ещё не началась.
    • 🟡 До 24 часов — если акция уже идёт.
    • 🔴 До 72 часов — для эксклюзивных акций (требуется согласование с менеджером кампании).
    💡

    Если поддержка долго не отвечает, попробуйте написать в чат Ozon Seller (кнопка в правом нижнем углу) или позвонить по телефону горячей линии для продавцов. Укажите, что вопрос срочный и связан с финансовыми рисками (убытками от продажи по акционной цене).

    6. Что делать, если товар удалён из акции, но цена не изменилась?

    Иногда после успешного удаления товара из акции его цена на витрине остаётся акционной. Это связано с:

    • 🔄 Кэшированием данных на стороне Ozon (особенно актуально для мобильного приложения).
    • 🤝 Конфликтом промо (например, на товар действует ещё одна скидка, например, "Цена по карте Ozon").
    • ⏳ Задержкой синхронизации между системами (до 6 часов).

    Действия для решения проблемы:

    1. Обновите кэш:
      • В личном кабинете: Настройки → Очистить кэш.
      • В браузере: Ctrl + F5.
      • В мобильном приложении: закройте и снова откройте его.
  • Проверьте другие акции: 
    Перейдите в раздел Акции → Мои акции и отфильтруйте по product_id.

    Если товар участвует в другой кампании, удалите его и оттуда.

  • Вручную обновите цену: 
    Товары → Управление ценами → Выберите товар → Установите новую цену.

    Это принудительно сбросит акционную цену.

  • Обратитесь в поддержку, если проблема не решена в течение 6 часов.

    • Если цена не обновляется только для некоторых покупателей, это может быть связано с:

    • 📱 Персонализированными скидками (например, для постоянных клиентов).
    • 📍 Региональными промо (акция может действовать только в определённых городах).
    • 💳 Скидками по карте Ozon (они применяются поверх основной цены).
    💡

    Если после всех манипуляций цена не изменилась, проверьте, не установлена ли для товара фиксированная скидка в настройках витрины (Товары → Настройки витрины → Скидки). Такие скидки имеют приоритет над акционными ценами.

    7. Последствия удаления товара из акции: что ждать?

    Удаление товара из акции может повлечь не только технические, но и финансовые или репутационные последствия. Рассмотрим их подробно:

    Последствие Когда возникает Как минимизировать риски
    Штраф за отмену участия Удаление товара менее чем за 48 часов до старта акции (для некоторых типов промо) Проверяйте условия акции заранее. Если штраф назначен ошибочно — оспорьте его.
    Снижение позиции в поиске Частые удаления из акций (система может расценить это как нестабильное ценообразование) Используйте удаление только в критических случаях. Для временных пауз лучше приостанавливайте продажи.
    Блокировка участия в будущих акциях Системные нарушения (например, удаление товара после начала акции без уважительной причины) Всегда указывайте причину удаления в запросе поддержке. Приложите доказательства (скрины остатков, изменения себестоимости и т.д.).
    Потеря трафика Товар теряет акционную видимость на витрине Запустите альтернативную промо-кампанию (например, купон или распродажу) для компенсации трафика.

    Особое внимание уделите динамическим акциям (например, "Скидка дня" или "Топ продаж"). В них алгоритмы Ozon могут автоматически понижать позицию товара в выдаче, если он часто удаляется из промо. Чтобы избежать этого:

    • 📅 Планируйте участие в акциях заранее, учитывая сезонность и запасы.
    • 📊 Используйте отчёты Аналитика → Акции, чтобы оценивать эффективность промо до их старта.
    • 🔄 Если товар временно недоступен, приостанавливайте продажи вместо удаления из акции.

    Важный момент для FBS: если товар удалён из акции, но уже находится на складе Ozon с акционной ценой, его цена будет изменена только после следующего инвентаризационного цикла (обычно раз в 1-2 дня). В этот период товар может продаваться по старой цене.

    8. Альтернативы удалению: когда можно обойтись без радикальных мер

    Удаление товара из акции — крайняя мера. В большинстве случаев проблему можно решить менее рискованными способами:

    • 📉 Скорректировать цену:

      Если акционная цена стала убыточной, но удалять товар нельзя (например, из-за штрафов), повысьте базовую цену так, чтобы акционная оставалась приемлемой. Например:

      Базовая цена: 2000 ₽ → 2500 ₽

      Акционная скидка: 20% → Цена для покупателя: 2000 ₽ (без изменений)

    • 🛑 Приостановить продажи:

      Установите остаток = 0 в разделе Товары → Управление остатками. Товар останется в акции, но не будет продаваться.

    • 🔄 Перенаправить трафик:

      Если товар временно недоступен, добавьте в его карточку баннер с альтернативной позицией (через раздел Товары → Редактирование → Маркетинговые материалы).

    • ⏳ Дождаться окончания акции:

      Если до завершения промо осталось менее 24 часов, часто проще дождаться его окончания, чем рисковать штрафами за удаление.

    Для товаров на FBO (отгрузка со своего склада) можно использовать ещё один трюк:

    1. Установите длительный срок доставки (например, 10-14 дней) в настройках товара.
    2. Это снизит его позиции в выдаче, и покупатели будут реже его выбирать.
    3. После окончания акции верните стандартный срок доставки.

    Если вы продаёте товары с предзаказом, удаление из акции может привести к автоматической отмене всех предзаказов по этой позиции. В этом случае:

    • Сначала отмените предзаказы вручную через Заказы → Предзаказы.
    • Уведомьте покупателей о переносе сроков (шаблон письма можно найти в CRM → Шаблоны).
    • Только после этого удаляйте товар из акции.
    Что будет, если не удалить товар из акции при нулевом остатке?

    Если товар участвует в акции, но его нет в наличии, Ozon может:

    1. Назначить штраф за невыполнение заказа (если покупатель успел оформить покупку).

    2. Понизить позицию товара в поиске из-за высокого процента отмен.

    3. Блокировать участие в будущих акциях (при систематических нарушениях).

    4. Приостановить продажи товара до пополнения остатков.

    FAQ: Частые вопросы о удалении товаров из акций на Ozon

    Можно ли удалить товар из акции после её старта?

    Да, но с оговорками:

    • Для большинства акций удаление возможно в любой момент, но может занять до 24 часов.
    • Для эксклюзивных промо (например, "Чёрная пятница") требуется согласование с менеджером Ozon.
    • Если товар уже куплен по акционной цене, его удаление не отменит заказ — вам придётся выполнить отгрузку.

    Рекомендуем удалять товары заранее (за 24-48 часов до старта акции), чтобы избежать продаж по убыточной цене.

    Как удалить товар из акции "Скидка дня" или "Молния"?

    Эти акции имеют особые условия:

    1. Для "Скидка дня":
      • Удаление возможно только до 12:00 дня старта акции.
      • После 12:00 товар блокируется на 24 часа.
  • Для "Молния":
    • Удаление невозможно после утверждения акции менеджером Ozon.
    • Если товар нужно исключить, обращайтесь в поддержку с обоснованием (например, "закончился запас").

      • В обоих случаях при удалении товара менее чем за 24 часа до старта может быть начислен штраф в размере 10% от стоимости товара.

    • Что делать, если товар удалён из акции, но всё равно продаётся по акционной цене?

    Это типичная ситуация, связанная с кэшированием данных. Действуйте по алгоритму:

    1. Проверьте, не участвует ли товар в другой акции (например, "Цена по карте Ozon").
    2. Обновите кэш в личном кабинете (Настройки → Очистить кэш).
    3. Вручную обновите цену товара в разделе Товары → Управление ценами.
    4. Если проблема не решена в течение 6 часов, обратитесь в поддержку с указанием:
      • product_id,
      • promo_id,
      • скриншота текущей цены на витрине.

    В 90% случаев проблема решается в течение 24 часов без вмешательства поддержки.

    Можно ли удалить товар из акции, если он уже в корзине у покупателей?

    Да, но с нюансами:

    • Если покупатель ещё не оформил заказ, товар исчезнет из его корзины после обработки запроса на удаление (обычно в течение 1-2 часов).
    • Если покупатель успел оформить заказ до обработки запроса, вы обязаны выполнить отгрузку по акционной цене.
    • Ozon не уведомляет покупателей об удалении товаров из акций, поэтому претензий со стороны клиентов обычно не возникает.

    Чтобы минимизировать риски, удаляйте товары из акций в период минимальной активности покупателей (ночью или ранним утром).

    Как удалить все товары из акции сразу?

    Для массового удаления используйте один из методов:

    1. Через личный кабинет:
      • Перейдите в Акции → Мои акции → Выберите акцию → Управление товарами.
      • Отметьте галочкой Выбрать все (или отфильтруйте нужные товары).
      • Нажмите Удалить из акции.
    2. Через API:
      • Используйте метод promo/unlink с массивом product_id.
      • Пример запроса: 
        {

        "promo_id": 98765432,
        

        "product_id": [12345678, 87654321, 11223344]
        

        }

  • Через Excel:
    • Скачайте отчёт по товарам в акции (Акции → Мои акции → Экспорт).
    • Отметьте в файле товары для удаления (добавив столбец action = "delete").
    • Загрузите файл обратно через Акции → Мои акции → Импорт.

    Для акций с более чем 1000 товаров рекомендуем использовать API или обратиться в поддержку для пакетного удаления.

Зачем продавцам нужен отчёт о реализации с Ozon и где его искать

Отчёт о реализации на Ozon — это ключевой документ для продавцов, который содержит детализированную информацию о продажах, возвратах, комиссиях и финансовых операциях. Без него невозможно вести точный учёт доходов, планировать закупки или сдавать налоговую отчётность. Однако многие новички теряются в интерфейсе Ozon Seller, не зная, где найти этот отчёт и как его правильно экспортировать.

В этой статье мы разберём все актуальные способы скачивания отчёта о реализации: от ручного экспорта через личный кабинет до автоматизации через API. Особое внимание уделим нюансам, которые влияют на полноту данных — например, почему в отчёте могут отсутствовать последние продажи или как избежать ошибок при выгрузке больших массивов информации.

Если вы продавец на Ozon и ещё не работали с отчётами, начните с первого раздела. Опытным пользователям рекомендуем сразу перейти к автоматизированным методам или настройке выгрузки в Excel.

Способ 1: Скачивание отчёта через личный кабинет Ozon Seller

Самый простой и доступный способ — выгрузка отчёта непосредственно из интерфейса Ozon Seller. Этот метод подходит для разовых проверок или небольших периодов (до 3 месяцев). Вот пошаговая инструкция:

  1. Авторизуйтесь в личном кабинете Ozon Seller.
  2. Перейдите в раздел Аналитика → Отчёты → Отчёт о реализации.
  3. Укажите диапазон дат (максимум 90 дней за один запрос).
  4. Нажмите Сформировать и дождитесь генерации данных.
  5. Кликните Экспорт в Excel или Скачать в CSV.

⚠️ Внимание: Если вы запрашиваете отчёт за большой период (например, год), система может разбить его на несколько файлов. В этом случае скачайте каждый архив и объедините данные вручную или через Excel.

Отчёт в формате Excel содержит более 30 колонок, включая:

  • 📦 Номер заказа и дату продажи
  • 💰 Сумму к выплате (за минусом комиссий)
  • 🔄 Статус возврата (если применимо)
  • 📊 Категорию товара и артикул
  • 🚚 Тип доставки (FBS/FBO)
💡

Если вам нужны данные только по конкретному товару, используйте фильтр по артикулу (SKU) перед экспортом. Это сэкономит время на обработку файла.

Способ 2: Автоматическая выгрузка через API Ozon

Для продавцов с большим объёмом продаж ручная выгрузка отчётов становится неэффективной. В этом случае поможет API Ozon — инструмент для автоматизированного получения данных. Чтобы им воспользоваться, потребуется:

  1. Получить Client-ID и API-Key в разделе Настройки → API.
  2. Использовать метод /v1/analytics/realization для запроса отчёта.
  3. Обработать ответ в формате JSON и преобразовать его в удобный вид (например, через Python или Google Sheets).

Пример запроса к API:

GET https://api-seller.ozon.ru/v1/analytics/realization

Headers:


- Client-Id: [ваш_client_id]


- Api-Key: [ваш_api_key]


Params:


- date_from: "2026-01-01T00:00:00Z"


- date_to: "2026-01-31T23:59:59Z"

⚠️ Внимание: API имеет ограничения по количеству запросов — до 1000 в минуту. При превышении лимита вы получите ошибку 429 Too Many Requests. Чтобы избежать блокировки, используйте паузы между запросами или разбивайте большие периоды на меньшие интервалы.

Параметр API Описание Пример значения
date_from Начальная дата периода (в формате ISO 8601) 2026-01-01T00:00:00Z
date_to Конечная дата периода 2026-01-31T23:59:59Z
with_dimensions Дополнительные группировки (например, по товарам) true
limit Максимальное количество записей в ответе 1000
Как обработать JSON-ответ от API в Excel?

1. Скопируйте ответ API (он будет в формате JSON).

2. В Excel перейдите на вкладку Данные → Получить данные → Из JSON.

3. Вставьте скопированный текст и нажмите Преобразовать.

4. Выберите нужные столбцы и загрузите данные в таблицу.

Для автоматизации можно использовать Power Query или макросы на VBA.

Способ 3: Экспорт отчёта в Excel с настройками

Если вам нужно не просто скачать отчёт, а сразу привести его к удобному для анализа виду, используйте расширенные настройки экспорта в Excel. Вот как это сделать:

  1. В разделе Отчёт о реализации нажмите Настройки колонок.
  2. Отметьте галочками только те поля, которые вам нужны (например, Артикул, Количество, Сумма к выплате).
  3. Сохраните шаблон настроек для будущих выгрузок.
  4. Экспортируйте данные в Excel и используйте сводные таблицы для анализа.

🔹 Пример полезных формул для Excel:

  • 📈 Итоговая выручка: =СУММ(E2:E1000) (где столбец E — "Сумма к выплате").
  • 🔍 Средний чек: =СУММ(E2:E1000)/СЧЁТЗ(E2:E1000).
  • 📉 Доля возвратов: =СЧЁТЕСЛИ(G2:G1000; "Возврат")/СЧЁТЗ(G2:G1000) (столбец G — "Статус").

  • Разбейте отчёт на несколько файлов по месяцам.

Способ 4: Интеграция с 1С или другими системами учёта

Для бизнеса, где ведётся бухгалтерия в 1С или аналогичных программах, ручной импорт данных из Ozon неэффективен. Решение — настройка автоматической синхронизации через:

  • 🔗 Готовые коннекторы (например, Атол Онлайн, МойСклад).
  • 🤖 Собственные скрипты на Python или PHP, использующие API Ozon.
  • 📂 Облачные сервисы типа Zapier или Make (ex-Integromat).

Пример настройки интеграции с 1С:

  1. Установите обработку "Обмен с Ozon" (доступна в 1С:Управление торговлей).
  2. Настройте авторизацию через Client-ID и API-Key.
  3. Укажите периодичность синхронизации (например, ежедневно в 23:00).
  4. Сопоставьте поля отчёта Ozon с справочниками 1С (товары, контрагенты, заказы).

🔹 Плюсы автоматизации:

  • ⏱️ Экономия времени (нет нужды в ручном импорте).
  • 📊 Актуальные данные в реальном времени.
  • 🔒 Минимизация ошибок при переносе информации.
💡

Интеграция с 1С или CRM позволяет автоматически формировать акты сверки с Ozon, что ускоряет закрытие финансовых периодов.

Способ 5: Альтернативные источники данных (для кросс-проверки)

Иногда данные в отчёте о реализации могут расходиться с фактическими выплатами. Чтобы избежать ошибок, сверяйте информацию с другими источниками:

Источник Что проверять Где найти
Выписка по Ozon Банку Суммы зачислений и комиссий Финансы → Выписки
Отчёт по FBS-заказам Логистические расходы Логистика → Отчёты
Архив заказов Статусы и детали заказов Заказы → Архив
Email-уведомления Подтверждения выплат Почта, привязанная к аккаунту

⚠️ Внимание: Если вы обнаружили расхождения между отчётом о реализации и выпиской по Ozon Банку, проверьте:

  • Наличие удержанных комиссий за возвраты или штрафы.
  • Задержки выплат (например, по схеме FBO деньги могут поступать позже).
  • Корректировки по акциям или промокодам (они иногда не сразу отражаются в отчётах).

Частые ошибки и как их избежать

Даже опытные продавцы сталкиваются с проблемами при работе с отчётами. Вот самые распространённые ошибки и способы их решения:

  • 🔴 "Отчёт не формируется":
    Проверьте, что указан корректный диапазон дат (максимум 90 дней). Если период больше, разбейте его на части.
  • 🔴 "Данные не совпадают с выплатами":
    Учтите, что отчёт о реализации показывает дату продажи, а не дату зачисления денег. Выплаты могут задерживаться на 1–3 дня.
  • 🔴 "Не хватает колонок в экспорте":
    Перед скачиванием нажмите Настройки колонок и добавьте недостающие поля (например, Вес товара или Регион доставки).
  • 🔴 "Файл повреждён при открытии":
    Попробуйте открыть его через Google Sheets или пересохраните в формате .csv.

🔹 Совет для новичков: Если вы впервые скачиваете отчёт, сначала запросите данные за 1–2 дня, чтобы понять структуру файла. Так проще избежать путаницы при работе с большими периодами.

💡

Создайте отдельную папку на Google Диске или в облаке для хранения всех отчётов. Это поможет быстро найти данные за прошлые периоды при проверках или аудите.

FAQ: Ответы на частые вопросы

Можно ли скачать отчёт о реализации за весь год?

Нет, через личный кабинет доступен экспорт только за период до 90 дней. Для годового отчёта используйте API или разбивайте запрос на кварталы.

Почему в отчёте нет данных по некоторым заказам?

Вероятные причины:

  • Заказы ещё не перешли в статус "Выполнен" (например, в пути).
  • Это FBO-заказы с отложенной выплатой.
  • Применялись фильтры при экспорте (проверьте настройки колонок).

Как часто обновляется отчёт о реализации?

Данные в отчёте обновляются в реальном времени, но выплаты по заказам могут отображаться с задержкой до 3 рабочих дней (зависит от схемы FBS/FBO).

Можно ли получить отчёт в PDF?

Нет, Ozon не предоставляет отчёт о реализации в формате .pdf. Доступны только Excel, CSV и JSON (через API).

Что делать, если API возвращает ошибку 403 Forbidden?

Ошибка 403 означает проблемы с авторизацией. Проверьте:

  • Корректность Client-ID и API-Key.
  • Наличие прав доступа к аналитике в вашем аккаунте.
  • Не превышен ли лимит запросов (1000 в минуту).

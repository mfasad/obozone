Как узнать артикулы заказов на Ozon: все способы для покупателей и продавцов

Артикул товара на Ozon — это уникальный идентификатор, который присваивается каждой позиции в каталоге маркетплейса. Без него невозможно отследить историю продаж, оформить возврат или синхронизировать данные с 1С. Но где его найти, если в личном кабинете отображаются только названия товаров? Эта проблема знакома и покупателям (которые хотят уточнить детали заказа), и продавцам (которым нужно вести учёт по SKU).

В этой статье мы разберём все актуальные способы получения артикулов — от стандартных (через интерфейс Ozon Seller) до скрытых (парсинг отчётов в Excel или работа с API). Особое внимание уделим нюансам: почему артикулы могут не отображаться, как их восстановить для старых заказов и что делать, если система выдаёт ошибку. Если вы торопитесь — используйте быстрое решение в конце статьи.

1. Где искать артикулы в личном кабинете Ozon (для продавцов)

Самый очевидный способ — заглянуть в раздел Заказы в Ozon Seller. Однако здесь есть подводные камни: артикулы отображаются не всегда, а их расположение зависит от типа аккаунта (FBS или FBO). Рассмотрим оба варианта.

Для FBS-продавцов артикул виден сразу в таблице заказов:

  1. Перейдите в Заказы → Все заказы.
  2. Найдите колонку Артикул (если её нет — нажмите Настроить колонки и добавьте поле SKU продавца).
  3. Скопируйте значение или экспортируйте таблицу в Excel.

Для FBO-продавцов процесс сложнее:

  1. Откройте карточку конкретного заказа.
  2. Прокрутите до блока Товары в заказе.
  3. Нажмите на название товара — артикул отобразится в всплывающем окне (или в URL-адресе страницы товара после ?asb=).

⚠️ Внимание: Если вы продаёте через Ozon Global, артикулы могут дублироваться с постфиксом _GG. Учитывайте это при синхронизации с 1С или МойСклад.

2. Как покупателю узнать артикул своего заказа

Покупатели часто сталкиваются с тем, что в мобильном приложении или на сайте Ozon артикулы не показываются. Однако их можно извлечь из:

  • 📄 Чека об оплате (в электронном письме или в разделе Мои заказы → Чек). Ищите строку Артикул: XXXXXX рядом с названием товара.
  • 🔗 Ссылки на товар. Откройте карточку товара из заказа — артикул будет в URL после /product/ (например, ozon.ru/product/123456789).
  • 📱 Мобильного приложения (только для Android): нажмите на товар в заказе → Подробнее → Характеристики.

Если артикул не отображается, попробуйте следующий трюк:

  1. Скопируйте название товара из заказа.
  2. Вставьте его в поисковую строку Ozon и найдите этот же товар.
  3. В карточке товара артикул будет в разделе Детали или в URL.

💡

Если вы оформили заказ по телефону через оператора Ozon, артикул можно узнать, позвонив в поддержку по номеру 8 800 333-20-30 (звонок бесплатный). Сообщите номер заказа и попросите предоставить SKU товара.

3. Экспорт заказов в Excel: где искать артикулы

Для продавцов, работающих с большим количеством заказов, ручной поиск артикулов неэффективен. Вместо этого можно экспортировать данные в Excel или CSV:

  1. В Ozon Seller перейдите в Заказы → Экспорт.
  2. Выберите период и формат (XLSX или CSV).
  3. В полученном файле артикулы будут в колонках:
    • sku — артикул продавца (ваш внутренний идентификатор);
    • offer_id — артикул Ozon (используется для API);
    • product_id — уникальный ID товара в системе маркетплейса.

Если в экспортированном файле нет колонки sku, проверьте:

  • 🔄 Обновили ли вы шаблон экспорта (в настройках Профиль экспорта).
  • 📁 Экспортируете ли вы детализированный отчёт (а не сводный).
  • 🔍 Не применяете ли фильтр, который исключает артикулы (например, по статусу заказа).

4. Получение артикулов через API Ozon

Для автоматизации работы с артикулами продавцы могут использовать API Ozon. Это актуально, если вам нужно интегрировать данные с 1С, Bitrix24 или другой системой учёта. Основные методы:

  • 🔌 /v2/analytics/data — для получения отчётов с артикулами;
  • 📦 /v2/posting/fbs/list — для FBS-заказов;
  • 🛒 /v1/orders — для FBO-заказов (устаревает, используйте /v2).

Пример запроса для получения артикулов по заказу (на Python):

import requests


url = "https://api-seller.ozon.ru/v2/posting/fbs/list"


headers = {


"Client-Id": "Ваш_Client_ID",


"Api-Key": "Ваш_API_ключ",


"Content-Type": "application/json"


}


payload = {


"limit": 10,


"offset": 0,


"filter": {


"since": "2026-01-01T00:00:00Z",


"to": "2026-01-31T23:59:59Z"


}


}



response = requests.post(url, headers=headers, json=payload)


data = response.json()


for order in data["result"]["postings"]:


print(f"Заказ: {order['posting_number']}, Артикул: {order['products'][0]['sku']}")

Важно: с 1 марта 2026 года Ozon отключил поддержку API v1 для FBO-заказов. Используйте только версию v2, иначе получите ошибку 403 Forbidden.

Как получить Client-ID и Api-Key для API?

1. Перейдите в Ozon Seller → Настройки → API-ключи.

2. Нажмите Создать ключ и укажите название (например, "Интеграция с 1С").

3. Скопируйте Client-ID и Api-Key — они отобразятся только один раз!

4. Для безопасности ограничьте доступ ключа по IP (опция Белый список IP).

5. Восстановление артикулов для старых заказов

Если заказ был оформлен больше года назад, артикулы могут не отображаться в личном кабинете. В этом случае:

  1. Проверьте архив писем от Ozon (тема: "Ваш заказ №XXXX"). В чеке обычно указан артикул.
  2. Используйте отчёт "История продаж":
    • Перейдите в Аналитика → Отчёты → История продаж.
    • Выберите нужный период (до 2 лет назад).
    • Экспортируйте данные — артикулы будут в колонке sku.
  • Обратитесь в поддержку с запросом: "Прошу предоставить артикулы по заказу №XXXX от DD.MM.YYYY". Приложите скриншот заказа.
    • ⚠️ Внимание: Ozon хранит данные о заказах в течение 3 лет. Если прошло больше времени, восстановить артикулы будет невозможно — их придётся искать в своих архивах (например, в старых выгрузках из 1С).

    Для продавцов, работающих с FBO, есть ещё один способ:

    1. Перейдите в Финансы → Выплаты.
    2. Найдите выплату, связанную с нужным заказом.
    3. В детализации платежа будет строка Товар: [Название] (Артикул: XXXXX).

      4. 💡

      Если артикул не найден ни одним из способов, проверьте, не был ли товар удалён из каталога Ozon. В этом случае SKU мог быть переиспользован для другого товара, и данные будут неактуальны.

      6. Частые ошибки и как их избежать

      При работе с артикулами продавцы и покупатели сталкиваются с типичными проблемами. Вот самые распространённые:

      Ошибка Причина Решение
      Артикул не отображается в заказе Товар добавлен без SKU или через Ozon Express Проверьте карточку товара в Товары → Мои товары и добавьте артикул
      В Excel вместо артикулов знаки #Н/Д Неправильный формат экспорта или битые данные Экспортируйте в CSV вместо XLSX или обновите шаблон
      API возвращает ошибку 404 Неверный posting_number или устаревшая версия API Используйте /v2/... и проверьте корректность номера заказа
      Артикулы дублируются с постфиксом _1, _2 Товар имеет несколько модификаций (цвет, размер) Используйте offer_id вместо sku для точной идентификации

      Ещё одна частая проблема — расхождение артикулов между вашей системой учёта и Ozon. Это происходит, если:

      • Вы загрузили товар с ошибкой в SKU (например, лишний пробел).
      • Ozon автоматически сгенерировал артикул для товара без вашего SKU.
      • Вы изменили артикул в своей базе, но не обновили его на маркетплейсе.

    Чтобы избежать путаницы:

    1. Используйте префиксы в артикулах (например, BRAND-MODEL-COLOR).
    2. Перед загрузкой товара проверяйте SKU на уникальность (в Ozon Seller есть инструмент проверки).
    3. Не редактируйте артикулы для товаров, которые уже продавались — это нарушит историю заказов.

    🚀 Быстрое решение: как узнать артикул за 1 минуту

    Если вам срочно нужен артикул, воспользуйтесь самым быстрым способом:

    1. Для продавцов:
      • Откройте заказ в Ozon Seller.
      • Нажмите на название товара → в URL будет ?asb=XXXXXX (это и есть offer_id).
  • Для покупателей:
    • Найдите email с чеком от Ozon (тема: "Чек №XXXX").
    • В таблице с товарами будет колонка Артикул.

      • Если ни один из способов не сработал — используйте поиск по картинке:

      1. Сделайте скриншот товара из заказа.
      2. Загрузите его в Поиск Google по картинке.
      3. Найдите этот же товар на Ozon — артикул будет в URL.

      💡

      Сохраняйте артикулы в отдельный файл (например, Google Sheets) для быстрого доступа. Это сэкономит время при работе с возвратами или инвентаризацией.

      FAQ: Ответы на частые вопросы

      Можно ли узнать артикул, если товар уже удалён из каталога Ozon?

      Да, но только если заказ был оформлен менее 3 лет назад. Используйте:

      1. Экспорт отчёта История продаж в Ozon Seller.
      2. Архив писем от Ozon с чеками.
      3. Обращение в поддержку с номером заказа.

    Если прошло больше 3 лет, восстановить артикул можно только из своих архивов (например, старых выгрузок из 1С).

    Почему в Excel вместо артикулов отображаются числа (например, 1.23456E+10)?

    Это ошибка формата ячейки. Используйте один из способов:

    • Выделите колонку → Формат ячеек → Текстовый.
    • Экспортируйте отчёт в CSV вместо Excel.
    • Увеличьте ширину колонки вручную.

      • Если проблема остаётся, проверьте, не повреждён ли файл (откройте его через Notepad++ или VS Code).

    Как связать артикулы Ozon с артикулами в 1С?

    Для синхронизации:

    1. В 1С создайте дополнительное поле для хранения SKU Ozon.
    2. Используйте обработку CommerceML или МойСклад для автоматического сопоставления.
    3. Для ручного ввода экспортируйте из Ozon Seller отчёт с колонками sku и product_id, затем импортируйте его в 1С.

    Готовые решения для интеграции: Атол Интеграция, МойСклад, Битрикс24.

    Что делать, если артикул в заказе не совпадает с артикулом в карточке товара?

    Это происходит, если:

    • Вы изменили SKU товара после продажи (старые заказы сохранят прежний артикул).
    • Ozon автоматически сгенерировал артикул для товара без вашего SKU.
    • Товар был дублирован (например, при переносе между аккаунтами).

    Решение:

    1. Проверьте историю изменений товара в Ozon Seller → Товары → Журнал изменений.
    2. Свяжитесь с поддержкой Ozon и уточните, какой SKU был на момент продажи.
    Можно ли получить артикулы заказов через мобильное приложение Ozon?

    Для покупателей:

    • В приложении Ozon (Android) артикул отображается в карточке товара из заказа (Подробнее → Характеристики).
    • В iOS артикулы не показываются — используйте веб-версию или чек на email.

    Для продавцов:

    • В приложении Ozon Seller артикулы видны в разделе Заказы (нужно включить отображение колонки SKU).
    • На iOS функционал ограничен — лучше использовать десктопную версию.

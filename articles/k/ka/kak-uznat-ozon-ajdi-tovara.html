Как быстро найти Ozon ID товара: все способы для покупателей и продавцов

  • Способ 1: Как найти Ozon ID в карточке товара (десктоп)

  • Способ 3: Получение ID через личный кабинет продавца

Уникальный идентификатор Ozon ID — это цифровой «паспорт» каждого товара на маркетплейсе, который нужен для отслеживания заказов, интеграции с системами учета, работы с API и решения спорных ситуаций. Без этого кода невозможно точно идентифицировать позицию в каталоге Ozon, особенно если у товара несколько модификаций (цвет, размер, комплектация). Проблема в том, что платформа не всегда показывает ID на видном месте — его приходится «добывать» через скрытые меню, HTML-код страницы или специализированные инструменты.

В этой статье мы разберем все актуальные способы получения Ozon ID в 2026 году, включая малоизвестные лайфхаки для мобильного приложения и массовое извлечение кодов через Excel. Отдельно остановимся на типичных ошибках, из-за которых пользователи путают ID товара с артикулом продавца или SKU, и объясним, почему это критично для бизнес-процессов. Если вы продавец, здесь вы найдете инструкции для личного кабинета и API; если покупатель — узнаете, как вытащить код из ссылки или чека. Все методы протестированы на актуальной версии платформы и работают как для FBS, так и для FBO.

Что такое Ozon ID и зачем он нужен

Ozon ID (или product_id в API) — это уникальный числовой идентификатор товара в базе маркетплейса, который присваивается автоматически при создании карточки. В отличие от артикула продавца (offer_id), который может повторяться у разных поставщиков, ID товара гарантированно уникален для всей платформы. Например, у iPhone 15 Pro 256GB от разных продавцов будет одинаковый Ozon ID, но разные артикулы.

Где требуется этот идентификатор:

  • 📦 Для продавцов: интеграция с 1С, MoySklad, Bitrix24 через API; массовое обновление цен/остатков; оформление возвратов.
  • 🛒 Для покупателей: отслеживание статуса заказа по точному товару (если в корзине несколько позиций); обращение в поддержку с жалобой на конкретный товар.
  • 🔍 Для аналитиков: парсинг каталога, сравнение цен конкурентов, мониторинг изменений в карточках.
  • ⚖️ Для разрешения споров: доказательство того, что заказан именно этот товар (например, при ошибке в комплектации).

Важно не путать Ozon ID с другими кодами:

Тип кода Пример Где используется
Ozon ID (product_id) 123456789 Уникальный идентификатор товара в каталоге Ozon
Артикул продавца (offer_id) SKU-001-XL Внутренний код поставщика (может повторяться у разных продавцов)
SKU (stock_keeping_unit) APL-iPhone15-256-Silver Универсальный код для учета на складе
Штрихкод (barcode) 4607182431258 Физическая маркировка товара
⚠️ Внимание: Если вы продавец и используете Ozon ID вместо артикула в интеграциях, система может не распознать ваш товар. Всегда проверяйте, какой именно код запрашивает документация API!

Способ 1: Как найти Ozon ID в карточке товара (десктоп)

Самый простой метод — извлечь идентификатор прямо из URL-адреса страницы товара. Он работает и для покупателей, и для продавцов, но требует доступа к публичной карточке.

Инструкция:

  1. Откройте карточку нужного товара на ozon.ru.
  2. Обратите внимание на адресную строку браузера. URL будет выглядеть так:
    https://www.ozon.ru/product/{тут-ваш-id}-{название-товара}/
  3. Скопируйте цифровую часть перед первым дефисом. Например, в ссылке
    https://www.ozon.ru/product/samsung-galaxy-s23-ultrasmartfon-12-256gb-seryy-123456789/

    ID товара — 123456789.

Если URL содержит параметры (например, ?asb=... или &sh=...), игнорируйте их — ID всегда находится сразу после /product/.

Для массового сбора ID (например, для конкурентного анализа) используйте парсеры вроде Ozon Parser или расширение Web Scraper для Chrome. Они автоматически извлекают коды из ссылок на странице категорий.

Способ 2: Извлечение ID через HTML-код страницы

Если URL товара не содержит ID (например, при переходе через рекламную ссылку или из мобильного приложения), можно найти код в исходном коде страницы. Этот метод универсален и работает даже для скрытых или архивных товаров.

Пошаговая инструкция:

  1. Откройте карточку товара в браузере (подойдет Chrome, Firefox или Edge).
  2. Нажмите правой кнопкой мыши на свободное место страницы и выберите Просмотр кода страницы (или Inspect в английской версии).
  3. В открывшемся окне нажмите Ctrl+F (Windows) или Cmd+F (Mac) и введите productId.
  4. Найдите строку вида:
    "productId": 123456789,

    или

    data-product-id="123456789"

Альтернативные поисковые запросы для кода:

  • "id": (иногда ID скрыт в JSON-данных)
  • itemprop="sku" (не путать с артикулом продавца!)
  • data-sku (в старых версиях сайта)
Что делать, если ID не найден в коде?

Если поиск по productId не дал результатов, попробуйте:

1. Обновить страницу (F5) — иногда данные подгружаются динамически.

2. Просмотреть сетевые запросы во вкладке Network (ищите файлы с расширением .json).

3. Использовать расширение JSON Viewer для анализа скрытых данных.

Если и это не помогло, товар может быть удален или заблокирован — проверьте его статус через личный кабинет (для продавцов).

⚠️ Внимание: В мобильной версии сайта Ozon HTML-структура отличается! Ищите ID в мета-теге <meta property="og:url" content="..."> — там будет полная ссылка с кодом.

Способ 3: Получение ID через личный кабинет продавца

Продавцы на Ozon могут получить Ozon ID прямо из интерфейса Seller Lab или Ozon Merchant. Этот метод подходит для массового экспорта данных и работы с большим каталогом.

Инструкция для Seller Lab:

  1. Авторизуйтесь в личном кабинете и перейдите в раздел Товары → Список товаров.
  2. Найдите нужный товар через поиск или фильтры.
  3. Кликните по названию товара — откроется его карточка.
  4. В адресной строке браузера будет URL вида:
    https://seller.ozon.ru/products/{ваш-id}/edit

    Скопируйте цифры после /products/.

Для массового экспорта:

  1. В разделе Товары нажмите Экспорт → Выгрузить в Excel.
  2. В полученном файле будет колонка Product ID — это и есть Ozon ID.

Внимание: в некоторых отчетах (например, по продажам) ID может называться Item ID или Ozon ID. Всегда проверяйте заголовки колонок!

💡

Если вам нужно получить ID для всех товаров сразу, используйте API-запрос POST /v2/product/info/list с параметром "filter": {"visibility": "ALL"}. Это сэкономит время при работе с каталогом из 1000+ позиций.

Способ 4: Как узнать Ozon ID в мобильном приложении

Официальное мобильное приложение Ozon (для iOS/Android) не отображает ID товара в интерфейсе, но его можно извлечь обходными путями. Этот способ подходит покупателям, которые хотят уточнить данные по заказу или жаловаться на конкретный товар.

Метод 1: Через «Поделиться»

  1. Откройте карточку товара в приложении.
  2. Нажмите кнопку Поделиться (значок стрелки вверх).
  3. Выберите Скопировать ссылку.
  4. Вставьте ссылку в любой текстовый редактор. Она будет выглядеть так:
    https://www.ozon.ru/product/{id}-{название}/

    Скопируйте цифры перед первым дефисом.

Метод 2: Через историю заказов (только для покупок)

  1. Перейдите в Профиль → Мои заказы.
  2. Выберите заказ с нужным товаром.
  3. Нажмите на название товара — откроется его карточка.
  4. Далее используйте Метод 1 (кнопка «Поделиться»).
💡

В мобильном приложении Ozon Seller (для продавцов) ID товара отображается в карточке редактирования под названием. Ищите строку ID: 123456789.

Если ни один из методов не сработал, попробуйте открыть карточку товара в мобильном браузере (не в приложении) и использовать Способ 1 или Способ 2 из этой статьи.

Способ 5: Получение ID через API (для разработчиков)

Для автоматизации бизнес-процессов продавцы могут получать Ozon ID через официальное API маркетплейса. Это актуально для интеграции с CRM, системами учета или собственными сервисами.

Базовый запрос для получения ID:

GET https://api-seller.ozon.ru/v2/product/info?product_id={ваш-id}&offer_id={артикул-продавца}

Headers:


- Client-Id: {ваш-client-id}


- Api-Key: {ваш-api-key}

Пример ответа (JSON):

{

"result": {


"product_id": 123456789,


"offer_id": "SKU-001",


"name": "Смартфон Apple iPhone 15 Pro 256GB",


...


}


}

Где взять данные для запроса:

  • 🔑 Client-Id и Api-Key — в личном кабинете Ozon Seller (Настройки → API).
  • 📌 product_id — это и есть Ozon ID (если известен).
  • 🏷️ offer_id — артикул продавца (если неизвестен Ozon ID, используйте запрос POST /v1/product/info/list с фильтрами).

Для массового получения ID всех товаров используйте запрос:

POST https://api-seller.ozon.ru/v2/product/info/list

Body:


{


"filter": {


"visibility": "ALL"


},


"limit": 1000


}

⚠️ Внимание: API Ozon имеет лимиты на количество запросов — 60 в минуту для стандартного тарифа. При превышении лимита вы получите ошибку 429 Too Many Requests. Используйте паузы между запросами или оформите расширенный тариф.

Типичные ошибки и как их избежать

Даже опытные пользователи иногда путают Ozon ID с другими кодами. Вот самые распространенные ошибки и способы их исправления:

Ошибка Причина Как исправить
Скопирован артикул продавца (offer_id) вместо Ozon ID В личном кабинете или чеке может отображаться внутренний код поставщика Ищите цифровой ID в URL или через API-запрос /v2/product/info
ID начинается с букв или содержит символы Скопирован штрихкод или часть названия товара Ozon ID — всегда цифры! Перепроверьте источник
Ошибка 404 при запросе по ID Товар удален или заблокирован Проверьте статус товара в личном кабинете или через /v1/product/info
В Excel ID отображается как 1.23E+08 Автоматическое форматирование чисел Поменяйте формат ячейки на Текстовый или добавьте апостроф перед числом

Еще одна частая проблема — изменение ID при редактировании товара. На Ozon идентификатор присваивается один раз и не меняется, но если вы создали новую карточку (даже для того же товара), ID будет другим. Всегда проверяйте актуальность кода!

Что делать, если товар дублирован?

Если у одного товара несколько карточек (например, из-за ошибки загрузки), у каждой будет свой Ozon ID. Чтобы объединить их:

1. Обратитесь в поддержку Ozon с запросом на слияние.

2. Укажите оба ID и объясните причину дублирования.

3. После слияния старый ID станет неактивным — используйте новый!

FAQ: Частые вопросы про Ozon ID

Можно ли по Ozon ID узнать, какой это товар?

Да, но только если товар не удален. Просто подставьте ID в URL:
https://www.ozon.ru/product/{ID}-{любой-текст}/

Например: https://www.ozon.ru/product/123456789-xyz/. Система автоматически перенаправит на актуальную карточку.

Где в чеке или квитанции указан Ozon ID?

В бумажных чеках и электронных квитанциях Ozon обычно указывает только артикул продавца (offer_id). Чтобы найти Ozon ID:

  1. Перейдите в историю заказов на сайте или в приложении.
  2. Откройте карточку товара из заказа.
  3. Скопируйте ID из URL (см. Способ 1).
Можно ли изменить Ozon ID товара?

Нет, Ozon ID присваивается автоматически при создании карточки и остается неизменным. Если вам нужно «обновить» ID, создайте новую карточку товара — у нее будет другой идентификатор. Старый ID останется в базе, но станет неактивным.

Как массово получить Ozon ID для всех своих товаров?

Используйте один из методов:

  • Экспорт через Seller Lab (раздел Товары → Экспорт в Excel).
  • API-запрос POST /v2/product/info/list с параметром "filter": {"visibility": "ALL"}.
  • Парсинг ссылок из каталога с помощью расширений Web Scraper или ParseHub.

Для больших каталогов (10 000+ товаров) рекомендуется использовать API с пагинацией (limit и offset).

Что делать, если Ozon ID не находится ни одним способом?

Возможные причины и решения:

  • Товар удален или заблокирован → Проверьте статус в личном кабинете (Товары → Архив).
  • Карточка не опубликована → Опубликуйте товар, чтобы он получил ID.
  • Ошибка в коде страницы → Попробуйте открыть карточку в другом браузере или очистите кэш.
  • Товар из другой страны → На Ozon для Беларуси, Казахстана и других регионов ID может отличаться. Уточните домен (например, ozon.by).

Если проблема остается, обратитесь в поддержку Ozon с указанием:

  • Названия товара.
  • Артикула продавца (offer_id).
  • Ссылки на карточку (если есть).

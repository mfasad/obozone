Как скачать отчёт с Озона для бухгалтерии: пошаговая инструкция с картинками

Работа с финансовой отчётностью на Ozon — обязательный этап для каждого продавца, особенно если вы ведёте учёт в 1С, сдаёте декларации в ФНС или сотрудничаете с аудиторами. Без корректных выписок невозможно подтвердить доходы, расходы на комиссии маркетплейса, логистику или возвраты. Однако найти нужные документы в личном кабинете Ozon Seller не всегда интуитивно: меню часто меняются, а названия разделов не всегда соответствуют бухгалтерской терминологии.

В этой статье разберём все способы скачивания отчётов — от ежедневных выписок по продажам до годовой финансовой сводки, — а также объясним, какие документы запрашивает налоговая и как их правильно оформить. Особое внимание уделим скрытым настройкам экспорта в Excel, которые экономят часы на ручную обработку данных. Инструкция актуальна для продавцов на FBS и FBO, включая тех, кто работает через Ozon Банк.

Какие отчёты с Озона нужны бухгалтерии: полный список

Прежде чем скачивать документы, определите, какие именно выписки требуются для ваших задач. Налоговая инспекция, аудиторы и внутренний учёт запрашивают разные формы отчётности. Вот полный перечень доступных отчётов в Ozon Seller:

  • 📊 Отчёт по продажам — детализация всех заказов (даты, суммы, комиссии, промокоды). Нужен для расчёта налога на прибыль и НДС.
  • 💰 Финансовый отчёт — движение денег по счету (поступления от покупателей, списания за логистику, штрафы). Основной документ для бухгалтерии.
  • 📦 Отчёт по возвратам — информация о возвращённых товарах и связанных финансовых корректировках.
  • 🚚 Отчёт по логистике — расходы на доставку (для FBS), включая тарифы Ozon Логистики.
  • 📑 Акт выполненных работ — ежемесячный документ для подтверждения услуг маркетплейса (комиссии, реклама). Требуется для вычета расходов.
  • 🔄 Отчёт по остаткам — данные о товарах на складах (актуально для инвентаризации).

Для налоговой декларации обычно достаточно финансового отчёта и акта выполненных работ. Если вы работаете на УСН или ЕНВД, может понадобиться детализация по каждому заказу (отчёт по продажам). Для ОСНО критически важны данные по НДС — их можно найти в разделе «Налоговые документы».

⚠️ Внимание: С 2026 года Ozon автоматически формирует ежемесячные акты для всех продавцов. Если вы не скачаете их до 10-го числа следующего месяца, документ будет архивирован, и восстановить его можно только через поддержку.

Где в личном кабинете Ozon Seller найти отчёты для бухгалтерии

Все финансовые документы хранятся в разделе Финансы, но путь к ним зависит от типа отчётности. Вот точные маршруты:

  1. Финансовый отчёт и выписки по счёту:
    Финансы → Выписки → Финансовый отчёт
  2. Отчёт по продажам (детализация заказов):
    Аналитика → Отчёты → Продажи
  3. Акты выполненных работ:
    Финансы → Документы → Акты
  4. Налоговые документы (счёта-фактуры, УПД):
    Финансы → Документы → Налоговые документы

Если вы не видите нужный раздел, проверьте роль вашего аккаунта в настройках. Доступ к финансовой отчётности есть только у пользователей с правами «Администратор» или «Бухгалтер». Также некоторые документы (например, акты) становятся доступны только после первого вывода средств.

Пошаговая инструкция: как скачать финансовый отчёт в Excel

Финансовый отчёт — самый востребованный документ для бухгалтерии. Рассмотрим, как его экспортировать в удобном формате.

  1. Перейдите в Финансы → Выписки → Финансовый отчёт.

  2. Выберите период (например, месяц или квартал). Для декларации обычно берут данные с 1-го по последнее число отчётного периода.

  3. Нажмите Сформировать отчёт. Система может запросить подтверждение по SMS.

  4. После генерации нажмите Экспорт → Excel (.xlsx). Файл скачается автоматически.

В Excel-файле будут колонки:

  • 📅 Дата операции
  • 💵 Сумма (поступление/списание)
  • 📝 Назначение платежа (комиссия, возврат, вывод средств)
  • 🔢 Номер заказа или документа

⚠️ Внимание: Если в отчёте отсутствуют данные по комиссиям или логистике, проверьте настройки фильтра. Иногда система по умолчанию скрывает служебные операции. Чтобы их увидеть, снимите галочку «Только торговые операции» в верхнем меню.

Как скачать акт выполненных работ для налоговой

Акт выполненных работ — это документ, подтверждающий оказание услуг со стороны Ozon (комиссии, реклама, логистика). Без него налоговая может не принять эти расходы к вычету. Скачать акт можно только после закрытия месяца (обычно он появляется на 3–5-й день следующего месяца).

Инструкция:

  1. Перейдите в Финансы → Документы → Акты.
  2. Выберите нужный месяц из выпадающего списка.
  3. Нажмите Скачать рядом с номером акта. Документ будет в формате PDF.

В акте должны быть указаны:

  • 📋 Реквизиты вашей компании и Ozon
  • 📅 Период оказания услуг
  • 💰 Сумма комиссий и других удержаний
  • 🔢 Номер и дата документа

Что делать, если акт не сформировался?

Если акт не появился в личном кабинете после 10-го числа, свяжитесь с поддержкой Ozon через раздел Помощь → Написать в поддержку. В сообщении укажите:

- Название вашей компании

- ИНН

- Период, за который не сформирован акт

- Скриншот раздела "Акты" (для подтверждения проблемы)

Обычно документ восстанавливают в течение 3–5 рабочих дней.

Экспорт отчёта по продажам для 1С: нюансы

Если вы ведёте учёт в 1С, стандартный Excel-отчёт с Ozon может не подойти — потребуется дополнительная обработка. Вот ключевые моменты:

  • 🔄 Структура данных: В 1С обычно нужны колонки «Дата», «Номер заказа», «Сумма», «Комиссия», «Статус». В отчёте Ozon эти данные разбросаны по разным вкладкам.
  • 📂 Формат файла: Лучше экспортировать в .csv (разделитель — запятая), а не в .xlsx, чтобы избежать проблем с кодировкой.
  • 🔍 Проверка дублей: В отчёте могут дублироваться строки по одному заказу (например, при частичном возврате). Их нужно объединить перед загрузкой в 1С.

Пример настройки экспорта для 1С:

  1. В разделе Аналитика → Отчёты → Продажи выберите период.
  2. Нажмите Настройки колонок и оставьте только:
    • Дата заказа
    • Номер заказа
    • Сумма заказа (без доставки)
    • Комиссия Ozon
    • Статус заказа
  • Экспортируйте в .csv и импортируйте в 1С через обработку «Загрузка из Excel».

    • 💡

    Если в 1С не загружаются суммы с копейками, проверьте настройки региональных параметров в Windows. Установите разделитель дробной части как "," (запятая), а не "." (точка).

    Частые ошибки при скачивании отчётов и как их избежать

    Даже опытные продавцы сталкиваются с проблемами при работе с отчётностью. Вот топ-5 ошибок и способы их решения:

    Ошибка Причина Решение
    Отчёт не формируется Недостаточно прав или не подтверждён телефон Проверьте роль пользователя в Настройки → Пользователи или запросите SMS-код
    Данные в Excel битые (иероглифы) Неверная кодировка при экспорте Экспортируйте в .csv с кодировкой UTF-8
    Не совпадают суммы с выпиской банка В отчёте не учтены удержанные комиссии Сверьте данные с разделом Финансы → Выписки → Операции по счёту
    Нет данных по возвратам Фильтр скрывает корректировки Снимите галочку «Только успешные заказы» в настройках отчёта
    Акт не появляется в списке Не закрыт отчётный период Дождитесь 5-го числа следующего месяца или обратитесь в поддержку

    Если вы скачали отчёт, но суммы не сходятся с вашими расчётами, используйте встроенный калькулятор Ozon в разделе Финансы → Аналитика. Он автоматически учитывает все комиссии и корректировки.

    💡

    Всегда сверяйте данные из отчётов Ozon с выписками вашего банка. Расхождения более чем на 1% — повод запросить разъяснения в поддержке.

    Автоматизация: как наладить регулярную выгрузку отчётов

    Если вы скачиваете отчёты ежемесячно, имеет смысл автоматизировать процесс. Ozon предоставляет несколько инструментов для этого:

    • 🤖 API Ozon: Позволяет получать данные программно. Подходит для крупных продавцов с собственными IT-ресурсами. Документация доступна по ссылке https://docs.ozon.ru/api/seller/.
    • ⏰ Рассылка по email: В разделе Финансы → Настройки можно включить автоматическую отправку финансового отчёта на почту в начале каждого месяца.
    • 📊 Интеграция с 1С: Через сервис Ozon Интегратор или сторонние решения (например, МойСклад, Атолл).

    Для небольших компаний самый простой способ — email-рассылка. Чтобы её настроить:

    1. Перейдите в Финансы → Настройки → Уведомления.
    2. Включите опцию «Отправлять финансовый отчёт ежемесячно».
    3. Укажите email бухгалтера и сохраните настройки.

    Если вам нужна более глубокая автоматизация (например, загрузка данных прямо в 1С), рассмотрите подключение Ozon API. Для этого потребуется:

    1. Зарегистрировать приложение в Личном кабинете → API.
    2. Получить Client-ID и API-Key.
    3. Написать скрипт (или использовать готовые коннекторы для 1С).

    FAQ: ответы на частые вопросы о бухгалтерских отчётах с Озона

    Можно ли скачать отчёт за прошлый год?

    Да, но с ограничениями. Финансовые отчёты доступны за последние 24 месяца, акты выполненных работ — за 12 месяцев. Для более старых данных нужно обращаться в поддержку с обоснованием (например, запрос от налоговой).

    Что делать, если в отчёте не хватает данных по комиссиям?

    Проверьте, включены ли в настройках отчёта строки с типом операции «Комиссия» и «Удержание». Если данные всё равно отсутствуют, сравните суммы с разделом Финансы → Выписки → Операции по счёту — возможно, комиссии списаны позже даты заказа.

    Нужно ли подписывать акты выполненных работ?

    Нет, акты формируются в электронном виде и имеют юридическую силу без дополнительной подписи. Однако вы можете распечатать документ и поставить на нём печать компании для внутреннего учёта.

    Как скачать отчёт по НДС?

    Счёта-фактуры и УПД доступны в разделе Финансы → Документы → Налоговые документы. Выберите период и нажмите «Скачать». Для ОСНО также потребуется детализация по ставкам НДС — её можно найти в отчёте по продажам (колонка «НДС»).

    Можно ли получить отчёт в бумажном виде?

    Ozon не предоставляет бумажные версии отчётов. Все документы формируются в электронном виде с электронной подписью. При необходимости вы можете распечатать их самостоятельно и заверить печатью.

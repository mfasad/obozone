Как скачать штрих-код с Озона: все способы для продавцов FBS и FBO

Штрих-коды на Ozon — это не просто набор цифр, а ключ к успешной логистике и продажам. Без корректных кодов ваш товар не пройдёт приёмку на складе, не попадёт в каталог и не дойдёт до покупателя. Но как их правильно скачать, если интерфейс маркетплейса меняется чаще, чем правила возвратов? Эта статья поможет разобраться в всех актуальных способах получения штрих-кодов — от ручного скачивания до автоматизации через API.

Многие продавцы теряют время, пытаясь найти кнопку "Скачать" в лабиринте меню Ozon Seller, или сталкиваются с ошибками при генерации кодов для FBS и FBO. Мы собрали проверенные методы, которые работают в 2026 году, включая малоизвестные лайфхаки для оптовиков и тех, кто работает с большими каталогами. А ещё расскажем, как избежать типичных ошибок, из-за которых товары "зависают" на стадии модерации.

Если вы только начинаете продавать на Ozon, обратите внимание: штрих-коды здесь делятся на два типа: уникальные (для каждого товара) и групповые (для вариаций одного продукта). Их формат и правила использования отличаются в зависимости от модели сотрудничества — FBS или FBO. Неправильный выбор может привести к штрафам или блокировке карточки товара.

1. Скачивание штрих-кодов через Личный кабинет Ozon Seller

Самый очевидный, но не всегда самый удобный способ — ручное скачивание через веб-интерфейс. Он подходит для небольших каталогов (до 50-100 товаров), когда не требуется автоматизация. Вот как это сделать:

  1. Авторизуйтесь в Личном кабинете Ozon Seller.
  2. Перейдите в раздел Товары → Мои товары.
  3. Найдите нужный продукт в списке и кликните на его название.
  4. В открывшейся карточке товара прокрутите до блока Штрих-коды.
  5. Нажмите кнопку Скачать (формат — PDF или PNG).

⚠️ Внимание: Если кнопка "Скачать" неактивна, проверьте статус товара. Штрих-коды доступны только для продуктов, прошедших модерацию и имеющих статус "Одобрен" или "На складе".

Для ускорения процесса можно использовать фильтры в разделе Мои товары:

  • 🔍 Отфильтруйте товары по статусу "Требуется штрих-код".
  • 📦 Выделите несколько позиций галочками и выберите опцию "Скачать штрих-коды" в выпадающем меню.
  • 📄 Для группового скачивания доступен формат ZIP-архив с отдельными файлами для каждого товара.
💡

Если вам нужно скачать штрих-коды для вариаций одного товара (например, разных размеров футболки), используйте групповой экспорт. В архиве будут файлы с постфиксами _1, _2 и т.д., соответствующими артикулам вариаций.

2. Экспорт штрих-кодов через Excel: пошаговая инструкция

Для продавцов с большим ассортиментом ручное скачивание — это потерянное время. К счастью, Ozon позволяет экспортировать штрих-коды в Excel или CSV, что удобно для дальнейшей печати или интеграции с 1С.

Алгоритм действий:

  1. В Личном кабинете перейдите в Товары → Мои товары.
  2. Нажмите кнопку Экспорт (в правом верхнем углу).
  3. В открывшемся окне выберите формат Excel (.xlsx).
  4. В списке полей для экспорта отметьте галочкой Штрих-код, а также дополнительные данные (например, Артикул, Название), которые помогут идентифицировать товары.
  5. Запустите экспорт и дождитесь формирования файла (может занять до 10 минут для больших каталогов).

В полученном файле штрих-коды будут в текстовом формате. Чтобы преобразовать их в сканируемый вид, воспользуйтесь одним из бесплатных онлайн-генераторов:

  • 🖥️ TEC-IT Barcode Generator — поддерживает Code 128 (стандарт для Ozon).
  • 📦 BarcodesInc — позволяет настраивать размер и разрешение.
  • 🔧 Online Barcode Generator — подходит для пакетной обработки.

⚠️ Внимание: При печати штрих-кодов используйте термотрансферный принтер (например, Zebra или Brother) для долговечности. Обычная струйная или лазерная печать может размазаться при транспортировке, что приведёт к проблемам на складе.

3. Автоматизация через API Ozon: для продвинутых продавцов

Если вы работаете с тысячами товаров или интегрированы с 1С, MoySklad, то ручные методы не подойдут. Ozon API позволяет автоматизировать получение штрих-кодов, но требует технических навыков или помощи разработчика.

Основные шаги:

  1. Получите доступ к API в Личном кабинете (Настройки → API).
  2. Сгенерируйте Client-ID и API-Key (сохраните их в надёжном месте!).
  3. Используйте метод POST /v2/product/info для получения данных о товарах, включая штрих-коды.
  4. Для пакетной обработки подходит метод POST /v1/product/info/list (до 1000 товаров за запрос).

Пример запроса на Python:

import requests


url = "https://api-seller.ozon.ru/v2/product/info"


headers = {


"Client-Id": "ваш_client_id",


"Api-Key": "ваш_api_key",


"Content-Type": "application/json"


}


payload = {


"product_id": [12345678, 87654321],  # ID товаров


"offer_id": ["your_offer1", "your_offer2"]  # Артикулы (опционально)


}



response = requests.post(url, headers=headers, json=payload)


print(response.json())

В ответе будет массив с данными, где поле "barcode" содержит штрих-код. Для генерации изображений можно использовать библиотеку python-barcode:

from barcode import Code128

from barcode.writer import ImageWriter



Генерация штрих-кода


barcode_class = Code128("2001234567890", writer=ImageWriter())


barcode_class.save("barcode_2001234567890")

Как проверить корректность штрих-кода через API?

В ответе API поле "barcode" может содержать значение null, если код ещё не сгенерирован или товар не прошёл модерацию. Чтобы избежать ошибок, всегда проверяйте статус товара через метод GET /v1/product/info/status перед запросом штрих-кода. Если статус не "ACTIVE", код получить не удастся.

4. Штрих-коды в мобильном приложении Ozon Seller

Для тех, кто часто работает в пути, удобно использовать официальное мобильное приложение Ozon Seller (доступно для iOS и Android). Функционал здесь ограниченнее, чем в веб-версии, но основные операции доступны.

Как скачать штрих-код через приложение:

  1. Откройте приложение и авторизуйтесь.
  2. Перейдите в раздел Товары (значок коробки в нижнем меню).
  3. Найдите нужный товар и тапните по нему.
  4. Прокрутите до блока Штрих-код и нажмите Скачать.
  5. Выберите формат: PDF (для печати) или Изображение (для отправки в чат).

⚠️ Внимание: В мобильной версии нет возможности группового скачивания штрих-кодов. Если вам нужно обработать более 10 товаров, используйте десктопную версию или API.

Преимущества мобильного приложения:

  • 📱 Быстрый доступ к штрих-кодам прямо со склада или пункта выдачи.
  • 🔄 Автоматическая синхронизация с веб-версией (изменения вносятся в реальном времени).
  • 📸 Возможность отсканировать существующий штрих-код камерой телефона для проверки.

5. Типичные ошибки и как их избежать

Даже опытные продавцы иногда сталкиваются с проблемами при работе со штрих-кодами. Вот самые распространённые ошибки и способы их решения:

Ошибка Причина Решение
Штрих-код не скачивается (кнопка неактивна) Товар не прошёл модерацию или имеет статус "Черновик" Проверьте статус в карточке товара и дождитесь одобрения
Сгенерированный код не сканируется Некорректный формат (не Code 128) или низкое разрешение печати Используйте генераторы с настройкой DPI ≥ 300
Штрих-коды дублируются для разных товаров Ошибка при групповом экспорте или сбои в API Проверьте уникальность кодов через метод GET /v1/product/info/barcode/validate
Код на этикетке размывается при печати Использование обычной бумаги или нетермоустойчивых чернил Печатайте на термоэтикетках с плотностью ≥ 80 г/м²

Ещё одна частая проблема — несоответствие штрих-кода артикулу. Это происходит, если вы вручную Assign (присваиваете) коды в карточке товара, но забываете сохранить изменения. Всегда проверяйте, что поле Штрих-код в Excel совпадает с тем, что указан в Личном кабинете.

Если вы работаете с FBO, помните: штрих-коды для такой модели генерируются автоматически при создании товара. Их нельзя изменить вручную! Для FBS вы можете использовать собственные коды (например, EAN-13), но они должны быть уникальными в рамках вашего аккаунта.

💡

Для FBS допускается использование собственных штрих-кодов, но они должны соответствовать стандарту GS1 (если вы не являетесь членом организации, используйте внутреннюю нумерацию с префиксом 2, например, 2001234567890).

6. Альтернативные способы: когда стандартные методы не работают

Иногда стандартные инструменты Ozon дают сбой, или вам нужно срочно получить штрих-коды для большого количества товаров. В таких случаях помогут альтернативные решения:

Способ 1: Парсинг через браузерные расширения

Для массового скачивания можно использовать расширения вроде Tampermonkey со скриптами для автоматизации кликов по кнопке "Скачать". Пример скрипта:

// ==UserScript==

// @name         Ozon Barcode Downloader


// @namespace    http://tampermonkey.net/


// @version      1.0


// @description  Автоматически скачивает штрих-коды со страницы товаров


// @match        https://seller.ozon.ru/products*


// @grant        none


// ==/UserScript==



document.querySelectorAll('button[data-test-id="barcode-download"]').forEach(btn => {


btn.click();


});

Способ 2: Обращение в поддержку

Если штрих-коды не генерируются для одобренных товаров, напишите в поддержку через форму в Личном кабинете (Помощь → Написать в поддержку). В сообщении укажите:

  • 📌 ID товаров, для которых не доступны коды.
  • 📌 Скриншот ошибки (если есть).
  • 📌 Модель сотрудничества (FBS или FBO).

Способ 3: Использование сторонних сервисов

Сервисы вроде МойСклад или 1С:Управление торговлей имеют встроенные модули для работы с Ozon, включая автоматическое получение штрих-кодов. Настройка требует интеграции через API, но затем процесс становится полностью автоматизированным.

💡

Если вы часто работаете с большими партиями товаров, рассмотрите покупку GS1-префикса (от 1000 уникальных кодов). Это избавит от необходимости генерировать коды через Ozon и ускорит логистику.

7. Печать и размещение штрих-кодов: требования Ozon

Скачать штрих-код — это только половина дела. Не менее важно правильно его напечатать и разместить на товаре или упаковке. Требования Ozon к этикеткам:

  • 📏 Размер: минимальный — 30×20 мм, рекомендуемый — 50×30 мм.
  • ⚪ Цвет: чёрные полосы на белом фоне (контрастность не менее 70%).
  • 📍 Размещение: на видимой стороне упаковки, не ближе 10 мм к краю.
  • 🚫 Запрещено: наносить код на криволинейные поверхности (например, бутылки).

Для товаров в FBS дополнительные требования:

  • 📦 На коробке должен быть только один штрих-код (если внутри несколько единиц товара, используйте групповую упаковку).
  • 🔖 Если товар продаётся в наборе, штрих-код наносится на общую упаковку.

⚠️ Внимание: На складах FBS сканеры настроены на чтение кодов в формате Code 128. Если вы используете EAN-13 или другие стандарты, их придётся перепечатывать, что повлечёт дополнительные расходы.

Для проверки качества печати используйте бесплатные приложения-сканеры:

  • 📱 Barcode Scanner (Android/iOS) — сканирует и распознаёт формат.
  • 🖥️ ZXing Decoder Online — загружаете изображение кода и проверяете его корректность.

FAQ: Частые вопросы о штрих-кодах на Ozon

Можно ли использовать один штрих-код для разных товаров?

Нет, каждый товар (или его вариация) должен иметь уникальный штрих-код. Для FBS допускается использование группового кода для набора, но только если он зарегистрирован как отдельный продукт в каталоге.

Что делать, если штрих-код повреждён или не сканируется?

Перепечатайте код с оригинального файла (не с фотографии!). Если проблема повторяется, проверьте настройки принтера: используйте режим "Только текст" без сжатия изображения. Для термопринтеров установите температуру печати на уровень 18-20 (для этикеток на бумажной основе).

Сколько стоит генерация штрих-кодов на Ozon?

Сама генерация бесплатна. Однако если вы используете собственные коды (не сгенерированные Ozon), их нужно зарегистрировать в GS1 (стоимость от 5 000 ₽ за 1000 кодов). Для FBO дополнительные расходы не требуются.

Можно ли изменить штрих-код после одобрения товара?

Для FBO — нет, код присваивается автоматически и не редактируется. Для FBS вы можете обновить код в карточке товара, но это потребует повторной модерации (до 24 часов). Изменение кода после отгрузки на склад приведёт к браку при приёмке.

Как скачать штрих-коды для товаров, которые ещё не на складе?

Штрих-коды генерируются только для товаров со статусом "Одобрен". Если товар в статусе "Черновик" или "На модерации", сначала доведите карточку до одобрения (заполните все обязательные поля, загрузите фото и т.д.).

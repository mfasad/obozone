Как выгрузить отчет о продажах на Ozon: все способы и нюансы

Выгрузка отчетов о продажах на Ozon — обязательная процедура для каждого продавца, работающего через маркетплейс. Без этих данных невозможно анализировать эффективность бизнеса, контролировать финансовые потоки или готовить отчетность для налоговой. Однако многие сталкиваются с трудностями: где найти нужный раздел в личном кабинете, какой формат выбрать, как избежать ошибок при экспорте.

В этой статье мы разберем все актуальные способы выгрузки отчетов (включая автоматизированные), объясним, какие данные содержатся в каждом типе отчета, и предостережем от типичных ошибок. Особое внимание уделим различиям между схемами FBS и FBO, так как набор доступных отчетов для них отличается. Если вы только начинаете работать с Ozon Seller или хотите оптимизировать рутинные процессы — этот гайд для вас.

Отчеты о продажах на Ozon обновляются в реальном времени, но их структура и доступность зависят от тарифного плана и статуса продавца. Например, партнеры с премиум-подпиской получают расширенную аналитику, а новички — только базовые данные. Мы покажем, как выгрузить информацию даже с минимальными правами доступа.

1. Где в личном кабинете Ozon Seller найти отчеты о продажах

Все отчеты сосредоточены в разделе «Аналитика» — это основной хаб для работы с данными. Чтобы туда попасть, выполните следующие шаги:

1. Авторизуйтесь в личном кабинете Ozon Seller под своей учетной записью продавца.

2. В левом боковом меню найдите блок «Аналитика» (значок графика) и кликните по нему.

3. В выпадающем списке выберите пункт «Отчеты».

Если у вас несколько магазинов на Ozon, предварительно проверьте, что выбрали нужный аккаунт в верхнем правом углу (рядом с аватаром). Отчеты привязаны к конкретному магазину, и данные по разным проектам не суммируются автоматически.

Внутри раздела «Отчеты» вы увидите несколько вкладок:

  • 📊 Продажи — основная информация о заказах, выручке и возвратах.
  • 💰 Финансы — данные о выплатах, комиссиях и удержаниях.
  • 📦 Логистика — отчеты по доставке, хранению и штрафам (актуально для FBS).
  • ⚙️ Другие отчеты — расширенная аналитика (доступна не всем продавцам).

Для большинства задач достаточно вкладки «Продажи». Именно здесь можно выгрузить детализированные данные по заказам, включая номера, статусы, суммы и информацию о покупателях (с учетом ограничений конфиденциальности).

2. Типы отчетов о продажах: какой выбрать для ваших задач

Ozon предлагает несколько форматов отчетов, и их содержимое существенно отличается. Прежде чем нажимать «Экспорт», определите, какие именно данные вам нужны. Вот основные виды:

Тип отчета Что содержит Для каких задач подходит Формат
Детализация заказов Номера заказов, даты, статусы, суммы, способы оплаты, данные покупателей (частично) Анализ продаж, поиск проблемных заказов, сверка с бухгалтерией XLSX, CSV
Сводка по продажам Общая выручка, количество заказов, средний чек, динамика по дням/неделям Быстрая оценка эффективности, отчеты перед инвесторами XLSX, PDF
Отчет по возвратам Причины возвратов, суммы удержаний, статусы возмещений Анализ качества товаров, работа с претензиями XLSX
Финансовый отчет Выплаты, комиссии маркетплейса, удержанные суммы, налоги Налоговая отчетность, контроль cash flow XLSX, PDF
Отчет по логистике (FBS) Стоимость хранения, штрафы за просрочку, данные по доставке Оптимизация логистических расходов XLSX

Например, если вам нужно сверить выручку с банковскими выписками, берите «Финансовый отчет» в формате XLSX. А для анализа каких товары чаще возвращают — «Отчет по возвратам».

Важно: отчет «Детализация заказов» не содержит полные данные покупателей (ФИО, адреса, телефоны) из-за политики конфиденциальности Ozon. Для работы с клиентами используйте встроенные инструменты маркетплейса (например, чат в заказе).

💡

Если вам нужны данные по конкретному периоду (например, для квартальной отчетности), сначала установите фильтр по датам в личном кабинете, а потом экспортируйте отчет. Это сэкономит время на постобработке файла.

3. Пошаговая инструкция: как выгрузить отчет вручную

Рассмотрим процесс на примере самого востребованного отчета — «Детализация заказов». Этот формат подходит для большинства задач, от бухгалтерии до маркетингового анализа.

Шаг 1. Установите фильтры

В разделе «Отчеты» → «Продажи» вы увидите панель фильтров вверху экрана. Здесь можно сузить выборку по:

  • 📅 Датам (диапазон или конкретный день).
  • 🏷️ Статусам заказов (например, только «Доставлен» или «Отменен»).
  • 📦 Схемам работы (FBS или FBO).
  • 🛒 Категориям товаров (если нужно проанализировать отдельный сегмент).

Шаг 2. Настройте столбцы

По умолчанию отчет содержит базовые поля, но вы можете добавить дополнительные. Нажмите кнопку «Настроить столбцы» (значок шестеренки) и отметьте галочками нужные параметры. Например:

  • 💳 Способ оплаты (карта, рассрочка, бонусы).
  • 📦 Вес и габариты заказа (актуально для FBS).
  • 🔄 Причина отмены/возврата.

Шаг 3. Экспортируйте данные

После настройки фильтров нажмите «Экспорт» (кнопка в правом верхнем углу). Система предложит выбрать формат:

  • 📑 XLSX — рекомендуем для дальнейшей обработки в Excel или Google Sheets.
  • 📄 CSV — подходит для загрузки в 1С или другие учетные системы.

Файл сформируется в течение 1–5 минут (зависит от объема данных) и станет доступен для скачивания в разделе «Мои отчеты».

4. Автоматическая выгрузка отчетов: API и интеграции

Если вам нужно получать данные регулярно (например, ежедневно), ручной экспорт станет утомительным. К счастью, Ozon предоставляет несколько способов автоматизации:

1. API Ozon Seller

Через API маркетплейса можно запрашивать отчеты программно. Для этого потребуется:

  • 🔑 Ключ API (генерируется в личном кабинете в разделе «Настройки» → «API»).
  • 💻 Знания Python/JS или готовое решение (например, Ozon API Client для 1С).
  • 📡 Настроенный вебхук (если нужны уведомления о новых данных).

Пример запроса для получения отчета по заказам: 

GET https://api-seller.ozon.ru/v2/analytics/data

Headers:


- Client-Id: [ваш_client_id]


- Api-Key: [ваш_api_key]


Body:


{


"date_from": "2026-01-01T00:00:00Z",


"date_to": "2026-01-31T23:59:59Z",


"metrics": ["revenue", "orders"],


"dimensions": ["day", "sku"]


}

2. Интеграции с 1С, МойСклад, Bitrix24

Готовые коннекторы позволяют автоматически pulls данные из Ozon в вашу учетную систему. Популярные решения:

  • 🔗 1С:Управление торговым предприятием (модуль «Интеграция с Ozon»).
  • 📦 МойСклад (подключение через «Маркетплейсы»).
  • 🤖 Bitrix24 (через Ozon CRM Connector).

3. Сервисы-парсеры

Если API кажется сложным, можно воспользоваться сторонними сервисами вроде Peak, Sellerboard или DataLens. Они предлагают:

  • 📊 Визуализацию данных в дашбордах.
  • 🔔 Автоматические уведомления о важных изменениях.
  • 📥 Экспорт в Google Sheets по расписанию.
Какие данные нельзя получить через API?

Через API Ozon Seller недоступны:

- Полные данные покупателей (ФИО, адреса, телефоны).

- Информация о конкурентах (цены, остатки).

- Детали финансовых транзакций (номера банковских карт).

Для этой информации используйте личный кабинет или обращайтесь в поддержку.

5. Распространенные ошибки и как их избежать

Даже опытные продавцы иногда сталкиваются с проблемами при выгрузке отчетов. Вот типичные ошибки и способы их решения:

1. Отчет не формируется или висит в статусе «Обработка»

Причины и решения:

  • 🕒 Слишком большой период — разбейте запрос на более мелкие интервалы (например, по неделям).
  • 🔄 Сбой на стороне Ozon — проверьте статус сервисов на странице статуса.
  • 🚫 Недостаточно прав — обратитесь к администратору аккаунта.

2. В отчете не хватает данных

Чаще всего это происходит из-за:

  • ⚠️ Некорректных фильтров — проверьте, что не исключили нужные статусы заказов (например, «В доставке»).
  • 📅 Разницы во временных зонах — Ozon работает по UTC+3, а ваша система может использовать другое время.
  • 🔄 Задержки обновления — некоторые данные (например, по возвратам) появляются в отчетах с lagом до 24 часов.

3. Файл поврежден или не открывается

Если скачанный XLSX или CSV не читается:

  • 🔄 Попробуйте экспортировать отчет в другом формате.
  • 📥 Скачайте файл повторно — иногда помогает очистка кэша браузера.
  • 🖥️ Откройте файл в другом редакторе (например, LibreOffice вместо Excel).
💡

Перед отправкой отчета в налоговую всегда сверяйте итоговые суммы с выписками от Ozon. Расхождения могут возникнуть из-за удержанных комиссий или штрафов, которые не всегда видны в основных отчетах.

⚠️ Внимание: Если вы выгружаете отчет для налоговой инспекции, убедитесь, что в нем учтены все удержания маркетплейса (комиссии, штрафы, логистические расходы). В противном случае вам придется доплачивать налоги из своего кармана.

6. Как анализировать выгруженные данные: практические советы

Просто скачать отчет недостаточно — нужно уметь извлекать из него полезную информацию. Вот несколько приемов для глубокого анализа:

1. Сегментация по категориям и брендам

Сгруппируйте данные по:

  • 🛒 Категориям товаров — определите, какие группы приносят большую маржу.
  • 🏷️ Брендам — сравните конверсию и частоту возвратов.
  • 📦 Схемам работы (FBS vs FBO) — оцените, какой формат выгоднее.

Для этого в Excel используйте функцию «Сводная таблица» (Insert → PivotTable).

2. Анализ динамики продаж

Постройте графики по:

  • 📈 Дням недели — узнайте, в какие дни спрос выше.
  • 🕒 Времени суток — оптимизируйте время размещения акций.
  • 🏷️ Сезонности — подготовьтесь к пиковым периодам (Новый год, Черная пятница).

3. Контроль возвратов и отмен

Обратите внимание на:

  • 🔄 Частые причины возвратов (например, «не соответствует описанию»).
  • 📦 Товары с высоким % отмен — возможно, проблема в логистике или цене.
  • 💰 Суммы удержаний — сравните их с вашей маржой.

⚠️ Внимание: Если доля возвратов по определенному товару превышает 10%, Ozon может применить штрафные санкции (понижение в поиске, блокировку карточки). Отслеживайте этот показатель еженедельно.

Для визуализации можно использовать Google Data Studio или Power BI. Эти инструменты позволяют создавать интерактивные дашборды с автоматическим обновлением данных.

💡

Сохраняйте все отчеты в облачном хранилище (Google Drive, Яндекс.Диск) с указанием периода в названии файла. Это поможет быстро найти данные для проверки налоговой или аудита.

7. Альтернативные способы получения данных о продажах

Помимо стандартных отчетов, на Ozon есть и другие источники информации. Они могут быть полезны для кросс-проверки или получения дополнительных инсайтов.

1. Мобильное приложение Ozon Seller

В приложении (доступно для iOS и Android) можно:

  • 📱 Просматривать краткую статистику по продажам в реальном времени.
  • 🔔 Получать push-уведомления о новых заказах или проблемах.
  • 📊 Экспортировать мини-отчеты прямо с телефона (ограниченный функционал).

2. Электронная почта

Ozon автоматически отправляет продавцам письма с:

  • 📧 Ежедневной выручкой (если включена рассылка в настройках).
  • 📦 Информацией о возвратах и штрафах.
  • 💰 Уведомлениями о выплатах (с детализацией удержаний).

3. Поддержка Ozon

Если в отчетах не хватает данных или они кажутся некорректными, можно запросить дополнительную информацию через:

  • 💬 Чат в личном кабинете (раздел «Помощь»).
  • 📞 Телефон горячей линии для продавцов: 8 800 333-70-00 (доб. 1 для Ozon Seller).
  • ✉️ Обращение через форму в разделе «Техническая поддержка».

В запросе обязательно укажите:

  • 📅 Период, за который нужны данные.
  • 🔢 Номера заказов (если речь о конкретных сделках).
  • 📝 Цель запроса (например, «нужно для налоговой проверки»).

FAQ: Частые вопросы о выгрузке отчетов на Ozon

🔹 Можно ли выгрузить отчеты за прошлый год?

Да, но с ограничениями. Ozon хранит данные о заказах в течение 3 лет, однако детализация по некоторым параметрам (например, логистические штрафы) может быть недоступна для старых периодов. Если нужны данные старше 12 месяцев, обратитесь в поддержку — они предоставят архивные отчеты в течение 3–5 рабочих дней.

🔹 Почему в отчете по финансам суммы не сходятся с банковской выпиской?

Расхождения обычно возникают из-за:

  • 💰 Удержанных комиссий (их нет в банковской выписке, но они указаны в финансовом отчете).
  • 🕒 Задержки выплат (деньги могут идти до 3 дней).
  • 🔄 Возвратных платежей (если покупатель вернул товар после выплаты вам средств).

Для сверки используйте отчет «Выплаты» в разделе «Финансы» — там указана точная сумма, которая должна была поступить на счет.

🔹 Как выгрузить отчет по конкретному товару?

В личном кабинете нет отдельного отчета по одному SKU, но можно:

  • 🔍 Использовать фильтр по артикулу в отчете «Детализация заказов».
  • 📊 Экспортировать полный отчет и отфильтровать его в Excel по столбцу SKU.
  • 💻 Запросить данные через API с параметром "sku": ["ваш_артикул"].
🔹 Можно ли получить данные о покупателях (ФИО, телефоны)?

Нет, Ozon строго соблюдает политику конфиденциальности. В отчетах доступны только:

  • 🆔 ID покупателя (зашифрованный).
  • 📍 Регион доставки (без точного адреса).
  • 📧 Последние цифры телефона (например, ...7890).

Для связи с клиентом используйте встроенный чат в заказе — он анонимизирует личные данные.

🔹 Сколько времени хранятся отчеты в личном кабинете?

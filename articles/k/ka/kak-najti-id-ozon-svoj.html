Почему вам может понадобиться ID Ozon и где его искать

Идентификаторы (ID) на Ozon — это уникальные цифровые или буквенно-цифровые коды, которые присваиваются каждому элементу платформы: от вашего личного кабинета до отдельного товара в корзине. Без этих кодов невозможно решить большинство технических вопросов: оформить возврат через поддержку, интегрировать API, настроить рекламные кампании или даже просто уточнить статус заказа у менеджера.

Самые востребованные типы ID: ID магазина (нужен для подключения сервисов, например, Ozon Реклама или Ozon Финансы), ID заказа (требуется для отслеживания доставки или оспаривания штрафов), ID товара (используется при выгрузке каталога через Excel или 1С).

В этой статье разберём, как найти каждый из них — с учётом последних обновлений интерфейса Ozon в 2026 году.

Важно: если вы ищете ID для покупательского аккаунта (например, номер заказа), алгоритм будет проще, чем для продавцов. Мы отдельно выделили инструкции для обеих категорий пользователей.

Как найти ID магазина на Ozon (для продавцов)

Идентификатор магазина (Seller ID или Shop ID) — это ваш уникальный номер в системе Ozon. Он нужен для:

  • 🔗 Подключения к API Ozon (например, для автоматической выгрузки товаров)
  • 📊 Настройки аналитики в Google Analytics или Яндекс Метрике
  • 💳 Регистрации в Ozon Банке или сервисах логстики (FBS/FBO)
  • 📢 Создания рекламных кампаний через Ozon Рекламу

Чтобы найти ID магазина, выполните следующие шаги:

  1. Авторизуйтесь в Личном кабинете продавца.
  2. В правом верхнем углу кликните на иконку профиля (ваше имя или логотип магазина).
  3. В выпадающем меню выберите пункт Настройки магазина.
  4. В открывшейся вкладке перейдите в раздел Общая информация.
  5. Найдите блок Идентификатор магазина — это и есть ваш Seller ID (формат: цифровой код, например, 12345678).

⚠️ Внимание: Если вы ведёте несколько магазинов на Ozon, у каждого будет свой уникальный ID. Убедитесь, что копируете код нужного аккаунта.

Где посмотреть ID заказа на Ozon

ID заказа (или номер заказа) — это ключ к отслеживанию посылки, оформлению возврата или урегулированию споров с покупателем. Его можно найти как в мобильном приложении, так и на десктопной версии сайта.

Для покупателей:

  1. Откройте раздел Мои заказы в личном кабинете или приложении.
  2. Выберите нужный заказ из списка.
  3. В верхней части карточки заказа будет строка Номер заказа: XXXXXX-XXXX-XXXX — это и есть ID.

Для продавцов:

  1. В Личном кабинете перейдите в Заказы → Все заказы.
  2. Кликните на строку с нужным заказом.
  3. В открывшейся карточке в верхнем правом углу будет поле Номер заказа Ozon (формат: 123456789-1234-1234).

💡 Полезный совет: Если вам нужно быстро найти заказ по номеру, используйте поиск в Личном кабинете — введите ID в строку поиска в разделе Заказы.

Поиск ID товара (SKU) в каталоге Ozon

ID товара (он же SKU или артикул Ozon) — это уникальный идентификатор каждой позиции в вашем каталоге. Он нужен для:

  • 📦 Выгрузки товаров через Excel или 1С
  • 🔍 Поиска дублей в каталоге
  • 📈 Настройки рекламных кампаний на конкретные товары
  • 🔄 Обновления информации о товаре через API

Чтобы найти ID товара, выполните следующие действия:

  1. В Личном кабинете продавца перейдите в Товары → Мой каталог.
  2. Найдите нужный товар в списке (используйте фильтры или поиск по названию).
  3. Кликните на строку с товаром — откроется его карточка.
  4. В адресной строке браузера будет URL вида: 
    https://seller.ozon.ru/products/{12345678}/edit

    Цифры после /products/ и перед /edit — это и есть ID товара.

Альтернативный способ:

  1. В карточке товара прокрутите вниз до блока Дополнительная информация.
  2. Найдите поле Идентификатор товара на Ozon или SKU.

⚠️ Внимание: Не путайте ID товара на Ozon с вашим внутренним артикулом (если вы его указывали при загрузке). В системе Ozon приоритет имеет именно их идентификатор.

Как найти ID клиента (покупателя) на Ozon

ID клиента — это уникальный идентификатор покупателя в системе Ozon. Он может понадобиться продавцу для:

  • 🔍 Проверки истории заказов конкретного покупателя (если он оформлял их под одним аккаунтом).
  • 📞 Обращения в поддержку с вопросом о мошенничестве или подозрительной активности.
  • 🎁 Настройки персональных скидок или бонусных программ.

К сожалению, Ozon не предоставляет прямую возможность увидеть ID клиента в интерфейсе Личного кабинета. Однако есть два обходных пути:

Способ 1: Через карточку заказа (для продавцов)

  1. Откройте раздел Заказы → Все заказы.
  2. Выберите заказ нужного покупателя.
  3. В карточке заказа прокрутите вниз до блока Информация о покупателе.
  4. Нажмите на ссылку Профиль покупателя (если она активна).
  5. В адресной строке браузера будет URL вида: 
    https://seller.ozon.ru/customers/user12345678

    Последняя часть пути (user12345678) — это и есть ID клиента.

Способ 2: Через API (для технически подкованных пользователей)

Если вам нужно получить ID клиента программно (например, для интеграции с CRM), используйте метод POST /v1/orders/list в API Ozon. В ответе будет поле customer_id.

⚠️ Внимание: Ozon ограничивает доступ к данным покупателей в целях защиты персональной информации. Не используйте ID клиента для массовых рассылок или других действий, нарушающих политику платформы.

💡

Если покупатель оформил заказ как гость (без регистрации), его ID будет временным и недоступным для продавца. В таком случае обращайтесь в поддержку с номером заказа.

Где найти ID в мобильном приложении Ozon

Поиск идентификаторов в мобильном приложении Ozon (как для покупателей, так и для продавцов) имеет свои нюансы. Интерфейс здесь упрощён, и не все данные доступны так же легко, как на десктопе.

Для покупателей:

  • 📱 ID заказа: Откройте раздел Заказы, выберите нужный заказ → номер будет вверху экрана под статусом.
  • 🛒 ID товара: В карточке товара прокрутите до блока Подробности → Артикул (это может быть внутренний артикул продавца, а не SKU Ozon!).

Для продавцов:

  • 📊 ID магазина: Перейдите в Профиль → Настройки магазина → Общая информация.
  • 📦 ID товара: В разделе Товары откройте карточку товара → в адресной строке браузера (если открыть через Посмотреть в браузере) будет нужный идентификатор.

💡 Полезный совет: Если вам часто нужны идентификаторы, добавьте страницу seller.ozon.ru на главный экран телефона как ярлык — так удобнее работать с полной версией сайта.

Что делать, если в мобильном приложении не отображается ID?

В некоторых версиях приложения Ozon Seller идентификаторы скрыты. Обновите приложение до последней версии или используйте браузерную версию сайта. Если проблема остаётся, очистите кэш приложения в настройках телефона.

Частые ошибки при поиске ID и как их избежать

Даже опытные продавцы и покупатели иногда путают идентификаторы или не могут их найти. Вот самые распространённые ошибки и способы их решения:

  1. Путают ID магазина с ID товара.
    🔹 Решение: ID магазина — это 8 цифр в настройках профиля, а ID товара — цифры в URL карточки товара.
  2. Копируют неполный ID заказа.
    🔹 Решение: Убедитесь, что скопировали весь номер вместе с дефисами (формат: XXXXXXXX-XXXX-XXXX).
  3. Ищут ID в старых версиях Личного кабинета.
    🔹 Решение: Обновите страницу (Ctrl + F5) или очистите кэш браузера.
  4. Не видят блок с ID в мобильном приложении.
    🔹 Решение: Используйте браузерную версию сайта или обновляйте приложение.

⚠️ Внимание: Если вы передаёте ID в поддержку Ozon, всегда уточняйте, какой именно идентификатор вы имеете в виду. Например: Мой Seller ID: 12345678, номер заказа: 87654321-1234-5678.

Это ускорит решение вашей проблемы.

💡

Всегда проверяйте формат ID перед отправкой в поддержку или интеграцию с сервисами. Неправильный идентификатор может привести к ошибкам в API или задержке в обработке запроса.

FAQ: Ответы на частые вопросы об ID на Ozon

Можно ли изменить ID магазина или товара?

Нет, идентификаторы присваиваются автоматически при регистрации магазина или добавлении товара и не подлежат изменению. Если вам нужно "обнулить" историю, придётся создавать новый аккаунт или товарную карточку.

Что делать, если потерял доступ к аккаунту и не знаю свой Seller ID?

Обратитесь в поддержку Ozon с привязанного к аккаунту email или телефона. Предоставьте данные для верификации (ИНН, название магазина, примерные даты регистрации).

Как по ID заказа найти покупателя?

Продавцы могут увидеть контактные данные покупателя (имя, телефон, email) в карточке заказа, но ID клиента отображается не всегда. Для получения полной информации может потребоваться запрос в поддержку с обоснованием (например, подозрение на мошенничество).

Можно ли по ID товара узнать, кто его купил?

Нет, Ozon не предоставляет продавцам информацию о покупателях конкретного товара без привязки к заказу. Вы можете увидеть только статистику продаж в разделе Аналитика.

Где найти ID для подключения Ozon Рекламы?

Для настройки рекламных кампаний нужен Seller ID (ID магазина). Его можно найти в Настройки → Общая информация. Также может потребоваться Client ID — его выдаёт менеджер Ozon после подключения к сервису.

Работа с отчетами в Озон Селлер — неотъемлемая часть аналитики для любого продавца. Без них невозможно отслеживать продажи, контролировать остатки, анализировать возвраты или оптимизировать рекламные кампании. Однако интерфейс маркетплейса не всегда интуитивно понятен, особенно для новичков. Где искать нужные данные? Как выгрузить их в удобном формате? Почему некоторые отчеты недоступны для скачивания?

Эта статья поможет разобраться во всех тонкостях работы с отчетами в Ozon Seller — от базовых инструкций до решения типичных проблем. Мы рассмотрим не только стандартные способы выгрузки, но и альтернативные методы (включая API), а также расскажем, как автоматизировать процесс, чтобы экономить время. Особое внимание уделим скрытым отчетам, которые не отображаются в основном меню, но содержат ценную информацию для продвижения.

Если вы только начинаете работать с маркетплейсом, рекомендуем сначала ознакомиться с разделом о видах отчетов — это сэкономит часы на поиск нужных данных. Опытным продавцам будет полезен блок про автоматизацию и интеграцию с внешними сервисами. Независимо от уровня подготовки, здесь вы найдете ответы на вопросы, которые редко освещаются в официальной документации.

Какие отчеты доступны в Озон Селлер

В личном кабинете продавца доступно более 20 типов отчетов, но далеко не все из них одинаково полезны. Основные категории данных делятся на финансовые, логистические, маркетинговые и аналитические. Разберем ключевые из них:

Финансовые отчеты — самые востребованные, так как напрямую влияют на прибыль. Сюда входят:

- Выплаты (детализация переводов на ваш счет)

- Комиссии (разбивка по категориям товаров и услугам маркетплейса)

- Налоговые документы (для бухгалтерии, включая акты и счета-фактуры)

Логистические отчеты помогают контролировать движение товаров:

- Остатки на складах FBS (актуальный stock по всем ПВЗ)

- Движение товаров (приход/расход, включая возвраты и списания)

- Заказы в работе (статусы выполнения для FBO)

Маркетинговые отчеты критичны для оценки эффективности рекламы:

- Статистика рекламных кампаний (клики, показы, конверсии)

- Акции и промокоды (использование скидок по времени/товарам)

- Отзывы и рейтинги (динамика оценок и комментариев покупателей)

Аналитические отчеты дают глубокое понимание продаж:

- Топ продаж (самые популярные товары по категориям)

- Конверсия по карточкам (какие элементы описания лучше работают)

- Возвраты и претензии (причины и частота отказов)

  • 📊 Самый полезный для новичков: Отчет по продажам — показывает динамику в разрезе дней/недель с разбивкой по товарам.
  • 💰 Для финансового контроля: Детализация комиссий — помогает понять, на чем маркетплейс зарабатывает больше всего.
  • 🚀 Для масштабирования: Анализ трафика — показывает, откуда приходят покупатели (поиск, рекомендации, реклама).
  • ⚠️ Скрытый отчет: Сравнение цен с конкурентами — доступен только через API, но критичен для ценообразования.
⚠️ Внимание: Некоторые отчеты (например, Детализация возвратов) становятся доступны только после первого случая возврата в вашем аккаунте. Если раздел пуст — это не ошибка системы.

Где найти отчеты в личном кабинете

Навигация по интерфейсу Ozon Seller может сбивать с толку из-за частого обновления меню. Актуальный путь к отчетам на 2026 год:

1. Авторизуйтесь в личном кабинете продавца.

2. В верхнем меню выберите раздел Аналитика (расположен между Товарами и Финансами).

3. В выпадающем списке кликните на Отчеты. Откроется страница со всеми доступными типами данных.

Альтернативный путь для финансовых отчетов: Финансы → Выплаты → Детализация или Финансы → Документы → Налоговые.

Если вы не видите нужный раздел, проверьте:

- Ваш тарифный план (некоторые отчеты доступны только для Pro-аккаунтов).

- Регион работы (отчеты по Ozon Global вынесены в отдельный блок).

- Наличие активных заказов (без продаж часть аналитики будет скрыта).

Пошаговая инструкция: как скачать отчет

Процесс выгрузки данных состоит из 5 основных шагов. Следуйте им последовательно, чтобы избежать ошибок:

Шаг 1. Выбор типа отчета

На странице Аналитика → Отчеты прокрутите список до нужной категории. Обратите внимание на значки рядом с названиями:

- 📄 — отчет доступен для скачивания в формате XLSX/CSV.

- 🔍 — только просмотр в интерфейсе (выгрузка заблокирована).

- 🔄 — данные обновляются в реальном времени.

Шаг 2. Настройка фильтров

Для большинства отчетов доступны фильтры по:

- Дата (диапазон или конкретный день).

- Склад (для FBS-отчетов).

- Категория товара.

- Статус заказа (например, только "Доставлен").

Шаг 3. Предварительный просмотр

Перед скачиванием нажмите Показать отчет, чтобы убедиться, что данные корректны. Обратите внимание на:

- Количество строк (если их больше 10 000, отчет разобьется на несколько файлов).

- Форматирование чисел (иногда суммы отображаются без разделителей тысяч).

Шаг 4. Выбор формата

Доступные варианты:

- XLSX (рекомендуется для дальнейшей обработки в Excel).

- CSV (подходит для импорта в 1С или Google Sheets).

- PDF (только для финансовых документов, не редактируется).

Шаг 5. Скачивание

Нажмите Скачать и дождитесь генерации файла. Время ожидания зависит от объема данных:

- До 1 000 строк — 5-10 секунд.

- 10 000+ строк — до 2-3 минут.

Если после нажатия Скачать файл не загружается:

1. Проверьте папку Загрузки в браузере.

2. Попробуйте другой браузер (рекомендуется Google Chrome или Mozilla Firefox).

3. Отключите VPN (некоторые прокси блокируют большие файлы).

Что делать, если отчет пустой?

Иногда система не показывает данные из-за технических работ. Попробуйте:

1. Обновить страницу через 10-15 минут.

2. Выбрать другой временной диапазон (например, вместо "недели" указать конкретные даты).

3. Обратиться в поддержку с указанием ID отчета (отображается в URL после report_id=).

Альтернативные способы получения отчетов

Не все данные можно выгрузить через интерфейс. Рассмотрим дополнительные методы:

1. API Ozon Seller

Для автоматизации используйте официальное API маркетплейса. Примеры полезных эндпоинтов:

- /v2/analytics/data — универсальный отчет по продажам.

- /v1/finance/transaction/totals — финансовая сводка.

- /v2/posting/fbs/list — список FBS-заказов.

Пример запроса для остатков на складах: 

curl -X POST 'https://api-seller.ozon.ru/v2/products/stocks' \

-H 'Client-Id: YOUR_CLIENT_ID' \


-H 'Api-Key: YOUR_API_KEY' \


-H 'Content-Type: application/json' \


-d '{"limit": 1000}'

2. Интеграция с 1С и другими системами

Для бухгалтерии удобно настраивать автоматическую выгрузку через:

- 1С:Управление торговлей (модуль "Интеграция с Ozon").

- МойСклад (подключение через API).

- RetailCRM (предварительная настройка вебхуков).

3. Экспорт через Google Sheets

С помощью скриптов Google Apps Script можно настроить автоматическое обновление данных:

1. Создайте новую таблицу в Google Sheets.

2. Перейдите в Расширения → Apps Script.

3. Вставьте код для подключения к API Ozon (примеры есть в открытых репозиториях).

4. Парсинг данных (для продвинутых пользователей)

Если вам нужны данные, которые не предоставляет Ozon (например, анализ цен конкурентов), можно использовать:

- Python с библиотеками requests и BeautifulSoup.

- Сервисы вроде Parsers.guru или DataCol (платные решения).

⚠️ Внимание: Парсинг данных с сайта Ozon без согласования нарушает пользовательское соглашение. Для легального сбора информации используйте только официальное API.
💡

API Ozon позволяет получать данные в реальном времени, но требует технических навыков. Для новичков проще использовать готовые интеграции с 1С или CRM-системами.

Типичные ошибки и их решения

Даже опытные продавцы сталкиваются с проблемами при работе с отчетами. Разберем самые распространенные случаи:

1. Отчет не генерируется

Причины и решения:

- Слишком большой временной диапазон → Разбейте запрос на меньшие периоды (например, по месяцам).

- Недостаточно прав → Проверьте роль вашего аккаунта в настройках (Профиль → Доступы).

- Технические работы → Следите за анонсами в разделе "Техработы".

2. Данные в отчете не совпадают с реальными продажами

Чаще всего это связано с:

- Задержкой обновления (FBS-отчеты обновляются с lagом до 6 часов).

- Фильтрами по статусам (например, не учтены "В пути" заказы).

- Округлением сумм (в финансовых отчетах копейки могут отбрасываться).

3. Файл поврежден или не открывается

Действия:

1. Попробуйте открыть файл в LibreOffice Calc (иногда справится там, где Excel выдает ошибку).

2. Проверьте кодировку (для CSV должен быть UTF-8).

3. Пересохраните файл через Google Sheets:

- Загрузите поврежденный файл.

- Выберите Файл → Скачать → Microsoft Excel (.xlsx).

4. Не хватает данных в отчете

Если в выгрузке отсутствуют некоторые заказы или товары:

- Убедитесь, что выбран правильный склад (для FBS).

- Проверьте, не скрыты ли данные фильтром по категории или бренду.

- Для Отчета по продажам включите галочку Показать все статусы.

  • 🔧 Ошибка 403 при скачивании: Очистите куки браузера или попробуйте в режиме инкогнито.
  • ⏳ Отчет висит на "Генерации" больше 10 минут: Перезагрузите страницу. Если не поможет — обратитесь в поддержку с указанием времени и типа отчета.
  • 📉 Графики в отчете не отображаются: Проверьте, не блокирует ли их ваш антивирус (например, Kaspersky иногда блокирует SVG-графику).
💡

Если вам нужны данные за прошлый год, но они не отображаются в интерфейсе, попробуйте запросить их через поддержку. В некоторых случаях архивные отчеты можно получить по специальной заявке.

FAQ: Частые вопросы по отчетам в Озон Селлер

Можно ли скачать отчет за весь период работы на Ozon?

Нет, система ограничивает временной диапазон. Максимальный период для большинства отчетов — 1 год. Для более ранних данных нужно обращаться в поддержку с обоснованием (например, для налоговой проверки).

Почему в отчете по продажам не совпадают суммы с выплатами?

Это связано с несколькими факторами:

  • В отчете по продажам показывается выручка брутто (до вычета комиссий).
  • В выплатах учитываются удержанные суммы за возвраты или штрафы.
  • Задержка между продажей и выплатой (для FBS — до 7 дней).

Для сверки используйте отчет Детализация выплат, где видна разбивка по каждой транзакции.

Как скачать отчет по конкретному товару?

1. Перейдите в Аналитика → Отчеты → Продажи.

2. В фильтрах выберите Товар и введите артикул или SKU.

3. Нажмите Показать отчет, затем Скачать.

Альтернативно: в карточке товара (Товары → Список товаров) есть кнопка Аналитика, где можно выгрузить данные только по этому SKU.

Можно ли получить отчет в формате JSON?

Через интерфейс — нет, доступны только XLSX, CSV и PDF. Однако при работе через API все данные возвращаются в формате JSON. Пример ответа:

"sku": 12345678,

Что делать, если в отчете по остаткам неверные данные?

Сначала проверьте:

  • Выбран ли правильный склад (в фильтрах может стоять галочка только на одном ПВЗ).
  • Не было ли в этот день инвентаризации (данные могли обнулиться временно).
  • Совпадает ли SKU в отчете с тем, что в вашей системе учета.

Если ошибка подтверждается, напишите в поддержку с указанием:

  • ID склада (виден в URL при просмотре остатков).
  • SKU проблемного товара.
  • Скриншот несоответствия.

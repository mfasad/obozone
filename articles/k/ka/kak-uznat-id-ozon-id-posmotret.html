Как посмотреть ID товара на Ozon: инструкция для покупателей и продавцов

  • 1. Как найти ID товара через URL карточки (самый быстрый способ)

  • 5. Получение ID через API Ozon (для автоматизации)

Каждый товар на маркетплейсе Ozon имеет уникальный цифровой идентификатор — ID (или артикул платформы). Этот номер нужен для работы с API, оформления возвратов, интеграции с 1С, а также для решения спорных ситуаций с поддержкой. Без него невозможно точно указать, о каком именно товаре идёт речь — особенно если у продавца в ассортименте десятки похожих позиций.

Проблема в том, что ID товара не всегда виден на первый взгляд. Его не отображают в мобильном приложении, он скрыт в десктопной версии сайта, а в некоторых случаях требует специальных манипуляций с кодом страницы. В этой статье мы разберём все актуальные способы, как узнать идентификатор товара на Ozon в 2026 году — от простого копирования из адресной строки до парсинга через инструменты разработчика.

1. Как найти ID товара через URL карточки (самый быстрый способ)

Это универсальный метод, который работает и для покупателей, и для продавцов. Достаточно открыть карточку товара в браузере (на компьютере или телефоне) и посмотреть на адресную строку.

Структура ссылки на товар в Ozon выглядит так:

https://www.ozon.ru/product/{id-товара}-{название-товара}/

Где {id-товара} — это и есть нужный вам идентификатор. Например, в ссылке:

https://www.ozon.ru/product/smartfon-apple-iphone-15-pro-max-256gb-naturalnyy-titan-178543215/

ID товара — 178543215.

  • 🔍 Работает для всех типов товаров (FBS, FBO, digital).
  • 📱 Подходит и для мобильной, и для десктопной версии сайта.
  • 🔄 Если в конце URL есть параметры (например, ?asb=123...), их можно игнорировать — они не влияют на ID.
⚠️ Внимание: Не путайте ID товара с артикулом продавца (SKU). Последний может совпадать у разных товаров одного бренда, а ID на Ozon всегда уникален.

2. Поиск ID в личном кабинете продавца (для селлеров)

Если вы продавец, самый надёжный способ найти ID — через панель управления Ozon Seller. Здесь идентификатор отображается в нескольких разделах:

  1. Каталог товаров:

    Перейдите в Товары → Мой каталог. В таблице с товарами будет колонка ID на Ozon. Если её нет — нажмите на шестерёнку в правом верхнем углу и добавьте столбец.

  2. Заказы:

    В разделе Заказы → Все заказы кликните на номер заказа. В списке товаров будет столбец Артикул Ozon.

  3. Отчёты:

    Скачайте отчёт по товарам (Аналитика → Отчёты → Товары) — там ID будет в отдельной колонке.

Если вы работаете через Ozon FBO, ID также можно найти в разделе Склад → Остатки, где он отображается рядом с названием товара.

Открыть раздел "Мой каталог"|Добавить колонку "ID на Ozon"|Скачать отчёт по товарам|Проверить раздел "Заказы"-->

3. Как узнать ID через мобильное приложение Ozon

В официальном мобильном приложении Ozon (для iOS и Android) идентификатор товара по умолчанию не отображается. Однако его можно получить двумя способами:

  • 🔗 Через ссылку:

    Откройте карточку товара, нажмите «Поделиться» (иконка стрелочки вверх) и выберите «Скопировать ссылку». Вставьте её в любой текстовый редактор — ID будет частью URL (как в первом способе).

  • 📋 Через код страницы (для Android):

    В карточке товара нажмите на три точки в правом верхнем углу → «Просмотр кода страницы» (если опция доступна). В HTML-коде найдите data-product-id — это и есть ID.

На iPhone просмотр кода страницы напрямую недоступен, поэтому остаётся только вариант с копированием ссылки.

⚠️ Внимание: В приложении Ozon Seller (для продавцов) ID товара отображается в карточке товара под названием, но только если вы включили отображение технических данных в настройках профиля.

4. Парсинг ID через инструменты разработчика (для продвинутых пользователей)

Если товар не имеет прямой ссылки (например, это позиция из корзины или архивный товар), можно извлечь ID через консоль браузера. Этот метод требует минимальных знаний HTML.

Инструкция для Google Chrome:

  1. Откройте карточку товара на Ozon.
  2. Нажмите F12 или Ctrl+Shift+I (для Mac: Cmd+Opt+I).
  3. Перейдите на вкладку Elements (Элементы).
  4. Нажмите Ctrl+F и введите product-id.
  5. Найдите атрибут data-product-id — его значение и есть ID.

Альтернативный способ — в консоли (Console) ввести:

document.querySelector('[data-product-id]').getAttribute('data-product-id')

И нажать Enter. В ответ выведется ID товара.

5. Получение ID через API Ozon (для автоматизации)

Если вам нужно получить ID товаров в большом количестве (например, для интеграции с 1С или CRM), используйте API Ozon. Для этого потребуется:

  1. Зарегистрировать приложение в личном кабинете разработчика.
  2. Получить Client-ID и API-ключ.
  3. Отправить запрос к эндпоинту /v2/product/info или /v1/products/list.

Пример запроса для получения информации о товаре (включая ID):

GET https://api-seller.ozon.ru/v2/product/info?product_id={YOUR_PRODUCT_ID}

Headers:


Client-Id: {YOUR_CLIENT_ID}


Api-Key: {YOUR_API_KEY}

В ответе будет JSON с полем "id", которое и является искомым идентификатором.

  • 🔧 Подходит для массового сбора данных.
  • 📊 Можно интегрировать с аналитическими системами.
  • ⚠️ Требует знаний работы с API и авторизации.
Как получить API-ключ для Ozon?

1. Перейдите в [Личный кабинет продавца](https://seller.ozon.ru/) → "API и Интеграции".

2. Нажмите "Создать ключ" и укажите название приложения.

3. Выберите нужные разрешения (например, "Чтение информации о товарах").

4. Сохраните сгенерированные Client-ID и API-Key в безопасном месте.

5. Используйте их в запросах к API, добавляя в заголовки (Headers).

6. Альтернативные способы: чат с поддержкой и email-уведомления

Если ни один из вышеперечисленных методов не сработал (например, товар архивный или недоступен), можно запросить ID через поддержку Ozon:

  • 💬 Чат в приложении:

    Напишите в поддержку (раздел «Помощь») с просьбой предоставить ID товара. Приложите скриншот карточки или название товара.

  • 📧 Email-уведомления:

    В письмах от Ozon (например, о заказе или возврате) обычно указывается ID товара в строке с наименованием.

  • 📄 Чек или накладная:

    В бумажных документах (если товар покупался офлайн) ID может быть указан рядом со штрихкодом.

Обратите внимание: поддержку лучше контактировать через аккаунт, с которого был оформлен заказ. В противном случае вам могут отказать в предоставлении данных по соображениям безопасности.

💡

ID товара на Ozon — это не то же самое, что артикул производителя или SKU продавца. Это уникальный номер, который присваивает маркетплейс и который нельзя изменить или удалить.

Частые ошибки при поиске ID и как их избежать

Даже опытные пользователи иногда путают идентификаторы или не могут их найти. Вот самые распространённые ошибки:

  • 🔢 Путают ID с артикулом продавца:

    SKU (артикул продавца) может совпадать у разных товаров, а ID на Ozon всегда уникален. Например, SKU: IPHONE15-256-BLACK, а ID: 178543215.

  • 🔗 Копируют неполную ссылку:

    Если в URL есть параметры (?partner=...), их можно обрезать — ID находится сразу после /product/.

  • 📱 Ищут ID в мобильном приложении без копирования ссылки:

    В приложении Ozon нет прямого отображения ID — его можно получить только через обходные пути (см. раздел 3).

  • 🔍 Используют ID из старой версии карточки:

    Если товар был удалён и заново добавлен, его ID изменится. Всегда проверяйте актуальность идентификатора.

Чтобы избежать ошибок, всегда сверяйте найденный ID с данными в личном кабинете (для продавцов) или в адресной строке (для покупателей).

FAQ: Ответы на частые вопросы об ID товаров на Ozon

Можно ли изменить ID товара на Ozon?

Нет, ID товара присваивается автоматически при добавлении позиции в каталог и не подлежит редактированию. Если вы удалите товар и загрузите его заново, ему будет присвоен новый ID.

Что делать, если ID товара не отображается в личном кабинете?

Проверьте настройки отображения столбцов в разделе Товары → Мой каталог. Если колонки ID на Ozon нет, добавьте её через иконку шестерёнки. Также убедитесь, что у вас достаточно прав доступа (для менеджеров).

Как найти ID для цифрового товара (например, игры или программы)?

Для digital-продуктов ID также содержится в URL карточки. Например, в ссылке https://www.ozon.ru/product/igra-cyberpunk-2077-pc-123456789/ идентификатором будет 123456789. Если товар приобретён, ID можно найти в чеке или в разделе «Мои покупки» → «Цифровые товары».

Можно ли по ID товара узнать, кто его продавец?

Нет, ID товара не содержит информации о продавце. Чтобы узнать данные селлера, нужно открыть карточку товара и посмотреть название магазина или перейти в раздел «Информация о продавце». Для массового сбора данных о продавцах по ID потребуется использовать API.

Почему у одного и того же товара разные ID на Ozon и Wildberries?

ID товара уникален для каждого маркетплейса. Ozon и Wildberries используют собственные системы идентификации, поэтому один и тот же товар (например, iPhone 15) будет иметь разные ID на этих площадках. Это нормально и не влияет на функциональность.

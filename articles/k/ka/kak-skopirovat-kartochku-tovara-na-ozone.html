Как скопировать карточку товара на Ozon: 5 проверенных способов для продавцов

Копирование карточки товара на Ozon — одна из самых востребованных операций среди продавцов, особенно при работе с большим ассортиментом или запуске аналогичных товаров. Независимо от того, используете ли вы модель FBS или FBO, умение быстро дублировать карточки сэкономит часы ручной работы. Однако многие сталкиваются с ошибками: от потери данных до блокировки аккаунта за нарушение правил маркетплейса.

В этой статье разберём легальные способы копирования, нюансы работы с API Ozon, а также покажем, как избежать типичных ошибок при переносе характеристик, фотографий и описаний. Особое внимание уделим различиям между копированием внутри одного аккаунта и между разными продавцами — это критично для соблюдения политики Ozon.

Если вы только начинаете работать с маркетплейсом, рекомендуем сначала изучить базовые настройки личного кабинета — это упростит дальнейшие манипуляции с товарами. Для опытных продавцов приготовили продвинутые методы, включая автоматизацию через Excel и Google Sheets.

1. Зачем копировать карточки товаров на Ozon?

На первый взгляд, создание каждой карточки с нуля кажется логичным — но на практике это отнимает до 30-40 минут на один товар. Копирование ускоряет процесс в 5-10 раз, особенно если речь идёт о товарах с похожими характеристиками: одежда одного бренда в разных цветах, электроника с минимальными отличиями в комплектации, или косметика одной линейки.

Основные причины для копирования:

  • 🔄 Запуск аналогичных товаров (например, футболки одного фасона в разных размерах).
  • 📦 Перенос товаров между аккаунтами (при открытии нового магазина или ребрендинге).
  • 🔄 Обновление сезонных товаров (когда нужно сохранить историю отзывов, но обновить описание).
  • 📊 A/B-тестирование (сравнение разных описаний или фотографий для одного товара).

Однако копирование имеет и риски. Например, дублирование контента может привести к понижению позиции товара в поиске из-за каннибализации ключевых слов. Также Ozon блокирует аккаунты за массовое копирование чужих карточек без изменений — это считается нарушением правил маркетплейса.

⚠️ Внимание: Копирование карточек других продавцов без существенных изменений (текста, фотографий, характеристик) приравнивается к плагиату. Ozon может удалить такие товары или ограничить доступ к аккаунту.

2. Способы копирования карточек на Ozon

Выбор метода зависит от ваших задач и технических навыков. Рассмотрим все доступные варианты — от ручного копирования до автоматизации через API.

2.1. Ручное копирование через личный кабинет

Самый простой способ, не требующий дополнительных инструментов. Подходит для единичных карточек.

Пошаговая инструкция:

  1. Откройте личный кабинет продавца и перейдите в раздел Товары → Мои товары.
  2. Найдите карточку, которую хотите скопировать, и откройте её для редактирования.
  3. Скопируйте вручную:
    • 📝 Название и описание (включая HTML-разметку, если она есть).
    • 🖼️ Фотографии (скачайте их на компьютер, а затем загрузите в новую карточку).
    • 📏 Характеристики (артикул, вес, габариты и т.д.).
    • 💰 Цену и скидки (если они привязаны к товару).
  • Создайте новую карточку через Товары → Добавить товар и вставьте скопированные данные.

    • Минусы метода: высокий риск ошибок при переносе данных и большое количество ручной работы. Подходит только для 1-2 карточек в день.

    2.2. Копирование через Excel/Google Sheets

    Более продвинутый способ, позволяющий копировать данные пакетно. Требует предварительной настройки шаблона.

    Как это работает:

    1. Экспортируйте данные о товарах из Ozon в Excel через раздел Товары → Экспорт.
    2. Откройте файл и скопируйте строки с нужными товарами.
    3. Измените уникальные параметры (артикул, название, фотографии) для новых карточек.
    4. Импортируйте обновлённый файл обратно через Товары → Импорт.

      5. Преимущества:

      • ⚡ Быстрота (можно скопировать до 100 карточек за час).
      • 📊 Возможность массового редактирования (например, изменить цену для всех копий одновременно).

      Недостатки:

      • 🔧 Требует знания структуры файла импорта/экспорта Ozon.
      • 🚫 Не все поля копируются корректно (например, SEO-параметры или привязки к акциям).

    2.3. Использование API Ozon

    Для продавцов с большим ассортиментом (от 500+ товаров) оптимально использовать API маркетплейса. Это позволяет автоматизировать копирование, интегрировать его с 1С или другими системами учёта.

    Основные этапы:

    1. Получите доступ к API Ozon (требуется регистрация приложения).
    2. Используйте метод POST /v2/product/info для получения данных о товаре.
    3. Модифицируйте полученные данные (например, измените артикул и название).
    4. Создайте новый товар через метод POST /v1/product/import.

    Пример запроса для получения информации о товаре: 

    curl -X POST 'https://api-seller.ozon.ru/v2/product/info' \

    -H 'Content-Type: application/json' \
    

    -H 'Client-Id: {ваш_client_id}' \
    

    -H 'Api-Key: {ваш_api_key}' \
    

    -d '{
    

    "product_id": 12345678,
    

    "offer_id": "ваш_offer_id"
    

    }'

    Преимущества API:

    • 🤖 Полная автоматизация (можно копировать тысячи карточек за минуты).
    • 🔄 Синхронизация с внешними системами (например, Bitrix24 или МойСклад).

    Сложности:

    • 🛠️ Требуются навыки программирования или помощь разработчика.
    • 💰 Возможны ограничения по количеству запросов (бесплатный тариф — до 1000 запросов/день).
    Как получить доступ к API Ozon?

    Для работы с API нужно зарегистрировать приложение в личном кабинете Ozon (раздел "API"). После модерации (1-3 дня) вам выдадут Client-ID и Api-Key. Подробная инструкция: [ссылка на документацию Ozon].

    2.4. Специализированные сервисы для копирования

    Если нет времени разбираться с Excel или API, можно воспользоваться сторонними сервисами:

    • 🔹 MyWarehouse — позволяет копировать карточки между аккаунтами с автоматическим изменением артикулов.
    • 🔹 SellerBoard — инструмент для массового редактирования и копирования товаров.
    • 🔹 EcomSputnik — подходит для копирования с одновременной оптимизацией под SEO.

    Стоимость таких сервисов — от 500 до 3000 рублей/месяц, но они экономят десятки часов в перспективе.

    2.5. Копирование через мобильное приложение Ozon Seller

    В официальном приложении для продавцов (доступно на iOS и Android) есть функция быстрого дублирования карточек:

    1. Откройте карточку товара в приложении.
    2. Нажмите на три точки в правом верхнем углу и выберите Скопировать товар.
    3. Измените необходимые параметры (название, фотографии, цену).
    4. Сохраните новую карточку.

    Ограничения:

    • 📱 Работает только для товаров внутри одного аккаунта.
    • 🚫 Не все поля копируются (например, SEO-тексты или привязки к промоакциям).
    💡

    Перед копированием проверьте, не привязан ли товар к уникальным акциям (например, "Товар дня"). Такие привязки не переносятся автоматически и могут вызвать ошибки при сохранении.

    3. Какие данные можно и нельзя копировать?

    Не все элементы карточки товара можно дублировать без последствий. Ozon строго следит за уникальностью контента, особенно в категориях с высокой конкуренцией (электроника, одежда, косметика).

    Что можно копировать без рисков:

    Тип данных Можно копировать? Примечания
    Технические характеристики ✅ Да Если они идентичны (например, габариты, вес, материал).
    Фотографии ⚠️ Частично Можно использовать свои фотографии того же товара в другом цвете/размере. Запрещено копировать чужие фото.
    Описание (HTML) ⚠️ С оговорками Допускается копирование структуры, но текст должен быть уникальным на 30-50%.
    Артикул ❌ Нет Каждый товар должен иметь уникальный артикул, даже если это копия.
    SEO-параметры (title, ключевые слова) ❌ Нет Дублирование приведёт к каннибализации и понижению в поиске.

    Что запрещено копировать:

    • 🚫 Отзывы и рейтинги — они привязаны к оригинальной карточке.
    • 🚫 Уникальные предложения (например, "Только у нас!", "Эксклюзив").
    • 🚫 Видеообзоры — даже если видео ваше, его нельзя дублировать без изменений.
    ⚠️ Внимание: Если вы копируете карточку другого продавца, обязательно измените не менее 70% контента (фото, текст, характеристики). В противном случае Ozon может удалить товар за плагиат.

    4. Типичные ошибки при копировании и как их избежать

    Даже опытные продавцы допускают ошибки, которые ведут к потере времени или блокировке товаров. Рассмотрим самые распространённые проблемы и способы их решения.

    4.1. Дублирование артикулов

    Каждый товар на Ozon должен иметь уникальный артикул. Если вы скопируете карточку и забудете изменить артикул, система выдаст ошибку: 

    Error: "Product with SKU {артикул} already exists"

    Как исправить:

    • Добавьте к артикулу суффикс (например, AR-1005-BLACK → AR-1005-WHITE).
    • Используйте автоматические генераторы артикулов (доступны в Excel или сервисах вроде MyWarehouse).

    4.2. Потеря SEO-оптимизации

    При ручном копировании легко упустить ключевые слова в заголовке или описании. Например, если в оригинальной карточке было:

    Кроссовки мужские Nike Air Max 2026 для бега, размер 42-46

    А в копии вы забыли указать год или бренд, товар потеряет позиции в поиске.

    Решение:

    • Используйте инструменты для анализа SEO (например, Ozon Keyword Tool).
    • Проверяйте уникальность текста через Text.ru или Advego.

    4.3. Ошибки с фотографиями

    Частая проблема — несоответствие фотографий новому товару. Например, вы копируете карточку чёрных кроссовок, но забываете заменить фото на белые. Покупатели будут возвращать товар, а Ozon может наказать за несоответствие.

    Как избежать:

    • 🖼️ Всегда проверяйте, что фотографии соответствуют новому варианту товара.
    • 🔍 Используйте водяные знаки с названием модели/цвета (например, "Nike Air Max — White").

    4.4. Проблемы с категориями

    Ozon регулярно обновляет дерево категорий. Если вы копируете карточку, созданную год назад, её категория могла устареть. Например, раньше "Беспроводные наушники" относились к категории "Аудиотехника", а теперь — к "Портативная электроника".

    Проверка:

    1. Перед копированием откройте актуальное дерево категорий.
    2. Убедитесь, что выбранная категория соответствует правилам Ozon.
    💡

    Перед массовым копированием всегда тестируйте процесс на 1-2 карточках. Это поможет выявить ошибки без риска для всего ассортимента.

    5. Копирование карточек между разными аккаунтами

    Если вы управляете несколькими магазинами на Ozon или переносите товары между аккаунтами (например, при ребрендинге), процесс усложняется. Ozon не предоставляет встроенных инструментов для такого копирования, поэтому придётся использовать обходные пути.

    5.1. Экспорт/импорт через Excel

    Самый надёжный способ:

    1. Экспортируйте данные из первого аккаунта в Excel.
    2. Удалите или измените уникальные поля (артикулы, ID товаров).
    3. Импортируйте файл во второй аккаунт.

    Важно:

    • 🔐 Убедитесь, что у вас есть права на управление обоими аккаунтами.
    • 📊 Проверьте, что категории и атрибуты совпадают в обоих магазинах.

    5.2. Использование API для межаккаунтного копирования

    Если у вас есть доступ к API обоих аккаунтов, можно автоматизировать перенос: 

    # Пример на Python (упрощённо)

    import requests
    


    Получаем данные из первого аккаунта
    

    response = requests.post(
    

    'https://api-seller.ozon.ru/v2/product/info',
    

    headers={'Client-Id': 'CLIENT_ID_1', 'Api-Key': 'API_KEY_1'},
    

    json={'product_id': 12345678}
    

    )
    

    data = response.json()
    


    Модифицируем данные (например, меняем артикул)
    

    data['offer_id'] = 'NEW_OFFER_ID'
    


    Создаём товар во втором аккаунте
    

    response = requests.post(
    

    'https://api-seller.ozon.ru/v1/product/import',
    

    headers={'Client-Id': 'CLIENT_ID_2', 'Api-Key': 'API_KEY_2'},
    

    json=data
    

    )

    Ограничения:

    • 🔒 Требуется согласование с поддержкой Ozon (межаккаунтовые операции могут блокироваться).
    • 💰 Возможны дополнительные комиссии за API-запросы.

    5.3. Ручной перенос с проверкой

    Если товаров мало (до 10-20), проще перенести их вручную:

    1. Скопируйте текстовые данные (название, описание, характеристики) из первой карточки.
    2. Скачайте фотографии на компьютер.
    3. Создайте новую карточку во втором аккаунте и вставьте данные.
    4. Проверьте, что все ссылки (например, на видеообзоры) работают.
    ⚠️ Внимание: При переносе товаров между аккаунтами Ozon может запросить подтверждение прав на бренд (если товар брендированный). Без подтверждения карточка не будет опубликована.

    6. Оптимизация скопированных карточек

    Простого копирования недостаточно — нужно адаптировать карточку под новые условия. Вот чек-лист для оптимизации:

    1. Уникализация контента:

    • 📝 Перепишите заголовок и описание, добавив уникальные ключевые слова.
    • 🖼️ Замените фотографии (даже если товар аналогичный, сделайте новые снимки под другим углом).

    2. SEO-настройка:

    • 🔍 Используйте инструменты вроде Ozon Keyword Tool или Wordstat для подбора ключевых слов.
    • 📌 Добавьте в описание ответы на частые вопросы (это повышает конверсию).

    3. Ценообразование:

    • 💰 Проверьте цены конкурентов на аналогичные товары (используйте Ozon Radar).
    • 🎯 Настройте динамическое ценообразование (если используете Ozon Premium).

    4. Привязка к акциям:

    • 🎉 Проверьте, подходит ли товар под текущие промоакции (например, "Скидки на новинки").
    • 📅 Настройте расписание скидок в личном кабинете.

    5. Тестирование:

    • 🧪 Опубликуйте карточку в режиме "Черновик" и проверьте отображение на мобильном и десктопе.
    • 📊 Запустите A/B-тест с разными фотографиями или описаниями (доступно в Ozon Analytics).
    💡

    Если копируете товар с высоким рейтингом, добавьте в описание фразу: "Аналог популярной модели [название оригинала] с улучшенными характеристиками". Это поможет сохранить доверие покупателей.

    7. Альтернативы копированию: когда лучше создать карточку с нуля?

    Копирование не всегда оправдано. В некоторых случаях лучше создать карточку с нуля, чтобы избежать проблем:

    Когда не стоит копировать:

    • 🆕 Товар сильно отличается от оригинала (например, другая комплектация или бренд).
    • 📉 Оригинальная карточка имеет низкий рейтинг или много негативных отзывов.
    • 🔍 Товар попадает под новую категорию с другими требованиями к атрибутам.
    • 🚀 Вы запускаете товар в другой стране (например, копируете с Ozon Россия на Ozon Казахстан).

    Преимущества создания с нуля:

    • ✅ Полный контроль над SEO и контентом.
    • ✅ Нет риска переноса ошибок из оригинальной карточки.
    • ✅ Возможность использовать новые шаблоны оформления (например, Ozon Card 2.0).

    Если вы всё же решили создать карточку с нуля, используйте нашу инструкцию по оформлению идеальной карточки товара.

    8. Частые вопросы (FAQ)

    Можно ли копировать карточки товаров с Wildberries на Ozon?

    Технически да, но не рекомендуется. Ozon и Wildberries имеют разные требования к оформлению карточек (например, ограничения по количеству фотографий или структуре описания). Кроме того, простое копирование может привести к проблемам с уникальностью контента. Лучше использовать данные с WB как основу, но адаптировать их под стандарты Ozon.

    Сколько времени занимает модерация скопированной карточки?

    Сроки модерации такие же, как для новых товаров:

    • ⏳ Стандартные товары: 1-3 часа.
    • ⏳ Товары с сертификатами (электроника, детские товары): 1-2 дня.
    • ⏳ Товары под акции: до 5 дней (проверяются вручную).

    Если карточка не проходит модерацию, проверьте Личный кабинет → Уведомления — там будет указана причина отказа.

    Что делать, если Ozon заблокировал карточку за дублирование?

    Если ваша карточка была удалена или заблокирована за плагиат:

    1. Проверьте письмо от Ozon с причиной блокировки.
    2. Измените не менее 70% контента (фото, текст, характеристики).
    3. Напишите в поддержку через Личный кабинет → Помощь → Написать в поддержку с просьбой разблокировать товар и объяснением, какие изменения вы внесли.

    Если блокировка повторяется, обратитесь к нашей инструкции по разблокировке аккаунта.

    Можно ли копировать отзывы с оригинальной карточки?

    Нет, отзывы привязаны к конкретной карточке товара и не переносятся при копировании. Если вы хотите сохранить социальное доказательство, можно:

    • 📌 Добавить скриншоты отзывов в описание (с пометкой "Отзывы на аналогичный товар").
    • 🎁 Предложить скидку первым покупателям новой карточки в обмен на отзыв.

    Имейте в виду, что Ozon может удалить карточку, если сочтёт скриншоты отзывов вводящими в заблуждение.

    Как копировать товары, если у меня более 1000 карточек?

    Для массового копирования (от 1000+ товаров) оптимально использовать:

    • 🤖 API Ozon с собственным скриптом или через сервисы вроде MyWarehouse.
    • 📊 Excel + Power Query для автоматизации обработки данных.
    • 🔧 1С:Управление торговлей с модулем интеграции с Ozon.

    Если у вас нет опыта работы с API, рекомендуем обратиться к специалистам (например, на Kwork или Fl.ru). Стоимость настройки автоматического копирования — от 5000 рублей.

Где и как найти отчёт о товаре на Ozon: подробное руководство с картинками и лайфхаками

Вы ищете отчёт о товаре на Ozon, но не знаете, где его найти? Эта проблема знакома и новичкам, и опытным продавцам. Отчёты о товарах — ключевой инструмент для анализа продаж, контроля остатков и оптимизации ассортимента. Без них невозможно понять, какие позиции приносят прибыль, а какие «простаивают» на складе. Но интерфейс Ozon Seller и мобильного приложения регулярно обновляется, и разобраться в нём с ходу бывает сложно.

В этой статье мы разберём все доступные способы получения отчётов — от стандартных выгрузок в личном кабинете до скрытых функций API. Вы узнаете, как найти данные по конкретному товару, экспортировать их в Excel или CSV, а также какие ошибки чаще всего допускают продавцы при работе с аналитикой. Отдельно остановимся на нюансах для FBS и FBO — схемы работы с отчётами здесь принципиально отличаются.

1. Что такое отчёт о товаре на Ozon и зачем он нужен

Отчёт о товаре — это структурированные данные о конкретной позиции в вашем ассортименте. В него входят:

  • 📊 Статистика продаж (количество заказов, выручка, динамика по дням/неделям).
  • 📦 Складские остатки (сколько единиц на складе Ozon, в пути, заблокировано).
  • ⭐ Рейтинги и отзывы (средняя оценка, количество возвратов, жалобы покупателей).
  • 💰 Финансовые показатели (комиссия маркетплейса, чистая прибыль, удержания).

Эти данные помогают:

  • 🔍 Выявлять «мёртвый» товар (лежит на складе больше 30 дней без продаж).
  • 📈 Оптимизировать ценовую политику (например, снижать цену на залежалые позиции).
  • ⚠️ Предотвращать штрафы за нехватку запасов (если товар часто заканчивается в пиковые дни).
  • 🛠️ Улучшать качество карточки (если много возвратов из-за несоответствия описанию).

Без регулярного анализа отчётов вы рискуете терять до 30% потенциальной прибыли — именно столько в среднем «съедают» неэффективные товары, штрафы и логистические ошибки (данные Ozon за 2023 год).

2. Где искать отчёт о товаре в личном кабинете Ozon Seller

Основной способ — через веб-версию Ozon Seller. Инструкция актуальна для интерфейса 2026 года:

  1. Авторизуйтесь на сайте seller.ozon.ru.
  2. Перейдите в раздел Аналитика → Отчёты → Товары.
  3. В фильтрах выберите:
    • 📅 Период (по умолчанию — последние 7 дней).
    • 📦 Склад (FBS или FBO).
    • 🔍 Артикул или название товара (если нужен отчёт по конкретной позиции).
  • Нажмите Сформировать отчёт.

    • Система предложит два формата:

    • 📄 Онлайн-просмотр (удобно для быстрого анализа).
    • 📥 Экспорт в Excel/CSV (нужен для глубокой аналитики или интеграции с 1С).

    Выбран правильный период (неделю/месяц)

    Указан нужный склад (FBS/FBO)

    Фильтр по артикулу заполнен корректно

    Формат экспорта соответствует задаче (Excel для графиков, CSV для интеграций)

    -->

    ⚠️ Внимание: Если вы работаете по модели FBO, в отчёте не будет данных о логистике — только продажи и финансы. Для FBS доступна полная аналитика, включая время доставки и возвратные потоки.

    3. Поиск отчёта через мобильное приложение Ozon Seller

    Мобильное приложение Ozon Seller (доступно для iOS и Android) позволяет получать отчёты в упрощённом виде. Это удобно для оперативного контроля, но функционал ограничен по сравнению с веб-версией.

    Как найти отчёт:

    1. Откройте приложение и авторизуйтесь.
    2. Перейдите в раздел Аналитика (значок графика в нижнем меню).
    3. Выберите вкладку Товары.
    4. Используйте фильтры:
      • 🔍 Поиск по названию или артикулу.
      • 📅 Период (максимум 30 дней).
      • 📊 Тип данных (продажи, остатки, отзывы).

    🔹 Ограничения мобильной версии:

    • ❌ Нет экспорта в Excel — только просмотр внутри приложения.
    • ❌ Отсутствуют некоторые метрики (например, детальная статистика возвратов).
    • ❌ Нельзя настроить автоматическую рассылку отчётов.
    💡

    Если нужно срочно проверить продажи по конкретному товару, используйте поиск по артикулу в мобильном приложении — это быстрее, чем формировать полный отчёт в веб-версии.

    4. Как получить отчёт по API Ozon (для продвинутых пользователей)

    Если вам нужны автоматизированные выгрузки или интеграция с внешними системами (например, 1С, МойСклад), используйте API Ozon. Это требует технических навыков или помощи разработчика, но даёт максимальную гибкость.

    Основные методы API для работы с отчётами:

    Метод API Описание Пример запроса
    /v2/analytics/data Получение статистики продаж по товарам 
    {

    "date_from": "2026-01-01",
    

    "date_to": "2026-01-31",
    

    "metrics": ["revenue", "orders"],
    

    "dimension": ["sku"]
    

    }
    /v2/analytics/stock_on_warehouses Остатки товара на складах FBS 
    {

    "sku_list": [12345678],
    

    "warehouse_type": "FBS"
    

    }
    /v1/product/info Детальная информация по товару (цена, описание, отзывы) 
    {

    "product_id": 12345678,
    

    "offer_id": "vendor-code-123"
    

    }

    🔹 Как подключиться к API:

    1. Зарегистрируйте приложение в личном кабинете Ozon.
    2. Получите Client-ID и API-Key.
    3. Используйте документацию Ozon API для формирования запросов.

    ⚠️ Внимание: API имеет лимиты на количество запросов (обычно 1000 в минуту). При превышении лимита доступ блокируется на 1 час. Для массовых выгрузок используйте пакетные запросы или кэширование.

    Пример кода на Python для выгрузки отчёта по API

    import requests

    url = "https://api-seller.ozon.ru/v2/analytics/data"

    headers = {

    "Client-Id": "ваш_client_id",

    "Api-Key": "ваш_api_key",

    "Content-Type": "application/json"

    }

    payload = {

    "date_from": "2026-01-01",

    "date_to": "2026-01-31",

    "metrics": ["revenue", "orders", "returns"],

    "dimension": ["sku"]

    }

    response = requests.post(url, headers=headers, json=payload)

    print(response.json())

    6. Частые ошибки при работе с отчётами и как их избежать

    Даже опытные продавцы допускают ошибки, которые искажают данные. Вот самые распространённые:

    ❌ Ошибка 1: Неправильный период отчётности

    Если выберёте слишком короткий период (например, 1 день), данные будут нерепрезентативны. Для анализа трендов берите минимум 30 дней. Для сезонных товаров (например, ёлочные игрушки) — не менее 3 месяцев.

    ❌ Ошибка 2: Игнорирование остатков на складах партнёров

    При работе по FBS товар может лежать не только на складах Ozon, но и у логистических партнёров. Если не учитывать эти остатки, рискуете получить штраф за нехватку запасов.

    ❌ Ошибка 3: Путаница между FBO и FBS

    Отчёты для этих схем формируются по-разному:

    • 📦 FBS: есть данные по логистике, времени доставки, возвратам.
    • 🚚 FBO: только продажи и финансы (логистику вы контролируете сами).

    ❌ Ошибка 4: Не учитывать удержания и комиссии

    В отчёте указывается выручка брутто, но реальная прибыль меньше на:

    • 💸 Комиссию Ozon (от 5% до 15% в зависимости от категории).
    • 📦 Логистические расходы (для FBS).
    • ⚠️ Штрафы и удержания (например, за просроченные заказы).
    💡

    Всегда сверяйте данные из отчётов Ozon с вашей бухгалтерией. Расхождения более 5% — повод запросить разъяснения в поддержке.

    3. Выберите папку, куда сохраняются выгрузки из Ozon.
    4. Настройте автоматическое обновление при открытии файла.

    FAQ: Ответы на частые вопросы

    🔹 Можно ли получить отчёт о товаре, который уже снят с продажи?

    Да, но только за период, когда товар был активен. В личном кабинете выберите нужный временной промежуток и укажите артикул снятого товара. Данные будут доступны в течение 1 года после удаления карточки.

    🔹 Почему в отчёте не совпадают продажи и финансы?

    Это может быть связано с:

    • 📅 Задержкой обновления данных (финансы обновляются раз в сутки, продажи — в реальном времени).
    • ⚠️ Удержаниями за возвраты или штрафы (они вычитаются из выручки).
    • 💳 Разными валютами (если часть заказов была в иностранной валюте).

    Для проверки сверьте данные с разделом Финансы → Выплаты.

    🔹 Как экспортировать отчёт о товаре с фотографиями?

    Стандартные отчёты Ozon не включают изображения. Чтобы получить данные с фотографиями:

    1. Скачайте отчёт в Excel.
    2. Используйте API-метод /v1/product/info для получения ссылок на изображения.
    3. Добавьте столбец с URL-фото в вашу таблицу.

    Для массовой обработки подойдёт скрипт на Python или сервис Peak.

    🔹 Можно ли получить отчёт по товарам конкурента?

    Нет, Ozon не предоставляет данные по чужим товарам. Но вы можете:

    • 🔍 Использовать парсеры (например, Ozon Parser или Data365).
    • 📊 Анализировать открытую статистику (рейтинги, количество отзывов).
    • 🛒 Делать тестовые заказы для изучения упаковки и комплектации.

    ⚠️ Будьте осторожны: агрессивный парсинг может привести к блокировке IP.

    🔹 Как часто обновляются данные в отчётах?

    Частота обновления зависит от типа данных:

    Тип данных Частота обновления
    Продажи Каждые 15 минут
    Остатки на складе Каждый час
    Финансовые данные 1 раз в сутки (утром)
    Отзывы и рейтинги В реальном времени

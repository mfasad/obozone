Перенос карточки товара с Wildberries на Ozon: полный алгоритм без потерь

Перенос товарных карточек с Wildberries на Ozon — задача, с которой сталкиваются тысячи продавцов при мультиплатформенной работе. На первый взгляд процесс кажется простым: скопировал данные из одного личного кабинета — вставил в другой. Но на практике продавцы теряют до 30% трафика из-за ошибок в атрибутах, несоответствия категорий или неправильной структуры описаний. В этой статье разберём легальные способы экспорта, нюансы адаптации контента под алгоритмы Ozon и автоматизацию через API — без блокировок и штрафов.

Главная проблема при миграции — разница в требованиях к карточкам. Например, Wildberries разрешает до 10 фото на товар, а Ozon — до 15, но с жёсткими правилами по фону и разрешению. Или: на WB можно указать вес с погрешностью ±5%, а на Ozon неточные данные ведут к штрафу 1000₽ за карточку. Мы проанализировали 50+ случаев успешного переноса и выделили критические этапы, где продавцы допускают ошибки.

1. Подготовка: что можно и нельзя переносить с WB на Ozon

Перед экспортом данных проверьте совместимость категорий товаров. Не все направления, разрешённые на Wildberries, доступны на Ozon. Например:

  • 🚫 Запрещено: лекарства, алкоголь, табак, оружие — даже если они продавались на WB.
  • ⚠️ Ограничения: детские товары (нужны сертификаты ТР ТС), электроника (обязательна гарантия).
  • ✅ Разрешено: одежда, обувь, аксессуары, товары для дома — но с адаптацией атрибутов.

Второй критический момент — уникальность контента. Ozon блокирует карточки с копипастом описаний из Wildberries. Алгоритм распознаёт дубли по:

  • 📝 Тексту (совпадение >70% с другими карточками на Ozon).
  • 🖼️ Изображениям (если фото уже загружены другим продавцом).
  • 🏷️ Хэштегам и ключевым словам в заголовке.
⚠️ Внимание: Если вы переносите брендовые товары (например, Adidas или Samsung), на Ozon может потребоваться подтверждение прав от правообладателя. Без этого карточка не пройдет модерацию.

Используйте инструменты для проверки уникальности:

  • 🔍 Text.ru — для анализа текста.
  • 🖼️ TinEye — для поиска дублей изображений.

2. Способы экспорта данных из Wildberries

Есть 3 официальных метода выгрузки карточек с Wildberries:

  1. Ручной экспорт через личный кабинет (подходит для 1–50 товаров).
  2. Выгрузка в Excel через раздел Товары → Экспорт (до 10 000 позиций).
  3. API Wildberries — для автоматизации (требует знания JSON или помощи разработчика).

Рассмотрим каждый вариант подробно.

Ручной экспорт:

  1. Перейдите в Личный кабинет WB → Товары → Мои товары.
  2. Отметьте галочками нужные карточки.
  3. Нажмите Экспорт → Выгрузить в Excel.

Файл будет содержать базовые поля: артикул, название, цену, остатки. Но без описаний, характеристик и фото — их придётся скачивать отдельно.

Выгрузка через Excel:

Этот метод даёт больше данных, но требует настройки:

  1. В разделе Товары → Экспорт выберите формат XLSX.
  2. Укажите нужные поля: Название, Описание, Характеристики, Цена, Остатки.
  3. Дождитесь формирования файла (может занять до 24 часов при большом объёме).

API Wildberries:

Для крупных продавцов (1000+ товаров) оптимален программный экспорт. Пример запроса для получения списка товаров: 

GET https://suppliers-api.wildberries.ru/api/v2/cards/list

Headers:


Authorization: {Ваш_API_ключ}

Ответ будет в формате JSON с полным набором данных, включая mediaFiles (ссылки на фото).

💡

Если вы не программист, используйте сервисы-посредники вроде WBPartner или SellerLab — они упрощают работу с API через визуальный интерфейс.

4. Загрузка карточек на Ozon: пошаговая инструкция

Есть 4 способа добавить товары на Ozon:

  1. Ручной ввод (до 10 товаров).
  2. Импорт через Excel (до 1000 товаров).
  3. API Ozon (для крупных каталогов).
  4. Сервисы-интеграторы (например, MoySklad, 1C).

Ручная загрузка:

  1. Перейдите в Личный кабинет Ozon → Товары → Добавить товар.
  2. Выберите категорию и заполните форму.
  3. Загрузите фото (оптимальный размер: 1200×1200 px, формат JPEG/PNG).
  4. Укажите цену с учётом комиссии (от 15% для FBS).

Импорт через Excel:

Самый популярный метод для продавцов со средним ассортиментом:

  1. Скачайте шаблон для вашей категории.
  2. Заполните его данными из файла WB (см. раздел 3).
  3. Перейдите в Товары → Импорт → Загрузить файл.
  4. Дождитесь проверки (до 24 часов).

Типичные ошибки при импорте:

  • 🚨 Неверный формат ячейки (например, цена как текст вместо числа).
  • 🚨 Отсутствуют обязательные поля (артикул, название).
  • 🚨 Превышен лимит символов в описании.

Что делать если Ozon отклонил карточку?

Если товар не прошёл модерацию, проверьте:

1. Лог ошибок в разделе Товары → Импорт → История (там указаны конкретные проблемы).

2. Соответствие категории — иногда товар попадает в неверную рубрику автоматически.

3. Уникальность контента — если алгоритм обнаружил плагиат, перепишите описание.

4. Сертификаты — для некоторых категорий (например, электроники) требуются документы.

Если проблема не решается, обратитесь в поддержку через Личный кабинет → Помощь, приложив скриншот ошибки.

5. Оптимизация карточек для поиска на Ozon

Перенос данных — только половина дела. Чтобы товар продавался, его нужно оптимизировать под поиск Ozon. Алгоритм ранжирования здесь отличается от Wildberries.

Ключевые факторы ранжирования:

  • 🔑 Заголовок: должен содержать бренд + модель + ключевые слова. Пример: Смартфон Samsung Galaxy A54 5G 128GB, фиолетовый [Акция!]
  • 📊 Характеристики: заполняйте все доступные атрибуты (чем больше — тем выше позиция).
  • 💬 Отзывы: товары с рейтингом ≥4.5 продаются в 3 раза лучше.
  • 📈 Конверсия: если с карточки мало заказов, её понижают в выдаче.

Используйте инструменты для подбора ключевых слов:

  • 🔍 Яндекс.Wordstat — для анализа поисковых запросов.
  • 📊 Аналитика Ozon — показывает, по каким фразам находят ваш товар.
  • 💡 Advego — для проверки тошноты текста (оптимально: 3–5%).

Пример оптимизированного описания:

Instead of:

"Стильный рюкзак для школы. Много карманов. Удобно носить."

Use:

"🎒 Рюкзак для школы Eastpak Padded Pak’r — водоотталкивающая ткань, объём 24 л, ортопедическая спинка. 📚 Вмещает ноутбук 15.6", тетради А4, бутылку воды. 🔒 Прочные молнии YKK, регулируемые лямки. 🎨 Доступно 5 цветов: чёрный, синий, розовый, зелёный, серый. ⭐ Гарантия 2 года от производителя."

💡

На Ozon работает правило "3 секунд": если карточка не зацепила покупателя за это время, он уходит к конкурентам. Поэтому первое предложение в описании должно содержать главное преимущество товара.

7. Типичные ошибки и как их избежать

Анализ 200+ случаев переноса карточек показал, что продавцы чаще всего допускают следующие ошибки:

  1. Несовпадение артикулов:

    На WB и Ozon должны быть одинаковые артикулы, иначе будет путаница с остатками.

    Решение: используйте уникальные SKU (например, WB-12345 → OZ-12345).

  2. Неправильные цены:

    На Ozon комиссия выше (от 15% против 5–10% на WB). Если не скорректировать цену, товары станут неконкурентоспособными.

    Решение: добавьте 20–25% к себестоимости или оптимизируйте логистику (например, перейдите на FBS).

  3. Дублирование контента:

    Если скопировать описание с WB, Ozon может заблокировать карточку за плагиат.

    Решение: перепишите текст, добавьте уникальные детали (например, видеообзор или 3D-модель).

  4. Игнорирование атрибутов:

    На Ozon обязательны такие поля, как "Страна производства" или "Габариты". Их отсутствие ведёт к отклонению карточки.

    Решение: используйте справочник атрибутов Ozon.

Что будет если перенести товар с ошибками?

1. Модерация закроет карточку — её не увидят покупатели.

2. Штраф 500–1000₽ за несоответствие данных (например, неверный вес).

3. Потеря позиций в поиске — алгоритм понижает товары с низким качеством контента.

4. Блокировка аккаунта (в крайних случаях, например, при массовом дублировании описаний).

Чтобы избежать проблем, всегда проверяйте карточки через Личный кабинет → Товары → Проверка качества.

8. Проверка и тестирование после переноса

После загрузки карточек на Ozon выполните 5 обязательных проверок:

1. Тестовый заказ:

  1. Оформите заказ на свой товар (можно отменить позже).
  2. Проверьте, корректно ли отображаются:
    • 🖼️ Фото (нет обрезок, правильный цвет).
    • 📝 Описание (нет битых ссылок, текст читабелен).
    • 💰 Цена (учтена комиссия и скидки).

2. Анализ конкурентов:

Сравните свою карточку с топовыми предложениями в категории:

  • 🔍 Используют ли они видеообзоры?
  • 📊 Какие характеристики выделены в заголовке?
  • 💬 Сколько отзывов и какой рейтинг?

3. Проверка мобильной версии:

Более 70% покупателей на Ozon заходят с телефона. Убедитесь, что:

  • 📱 Фото не растянуты.
  • 📄 Описание читается без горизонтальной прокрутки.
  • 🛒 Кнопка "Купить" видна без скролла.

4. Мониторинг ошибок:

В разделе Личный кабинет → Товары → Ошибки отображаются проблемы, которые нужно исправить в течение 3 дней, иначе карточка будет скрыта.

5. Настройка рекламы:

На Ozon можно запустить:

  • 🎯 Поисковую рекламу (показ в топе выдачи).
  • 📢 Баннеры на главной (для промоакций).
  • 🔄 Рекламу в похожих товарах.

Минимальный бюджет — 500₽/день. Начните с теста на 1–2 товарах.

💡

Первые 2 недели после переноса отслеживайте коэффициент конверсии (соотношение просмотров к покупкам). Если он ниже 1–2%, оптимизируйте карточку: добавьте видео, улучшите фото или снизьте цену.

FAQ: Частые вопросы о переносе карточек

Могу ли я перенести отзывы с Wildberries на Ozon?

Нет, Ozon не позволяет импортировать отзывы с других площадок. Однако вы можете:

  • 📢 Добавить скриншоты отзывов с WB в галерею товара (с пометкой "Отзыв с другой платформы").
  • 🎁 Предложить скидку первым покупателям на Ozon в обмен на честный отзыв.
Что делать, если на Ozon нет моей категории товара?

Если ваш товар не попадает в существующие категории Ozon, попробуйте:

  • 🔍 Подобрать наиболее близкую категорию (например, "Аксессуары" вместо "Уникальные подарки").
  • 📧 Написать в поддержку Ozon с просьбой добавить новую категорию (приложите аргументы: спрос, аналоги на других площадках).
  • 🏷️ Использовать теги в названии (например, "[Эко-подарок] Набор для ручной работы").
Как перенести товары с WB на Ozon, если у меня 10 000+ позиций?

Для большого ассортимента:

  1. 🤖 Настройте API-интеграцию между WB и Ozon (или используйте сервис вроде 1C).
  2. 📊 Разбейте перенос на этапы: сначала топ-20% товаров по продажам, затем остальные.
  3. 👨‍💻 Нанмите специалиста на Freelance.ru или Kwork (стоимость: 500–1500₽ за 100 товаров).
Можно ли переносить товары с WB на Ozon, если я не официальный дилер бренда?

Да, но с оговорками:

  • ✅ Если бренд не защищён на Ozon (проверяется в карточке товара), вы можете продавать без подтверждения.
  • ⚠️ Если бренд защищён, нужно предоставить документы от правообладателя или отказаться от переноса.
  • 🚫 Запрещено использовать логотипы бренда в описании без разрешения.

Проверьте статус бренда в справочнике Ozon.

Сколько времени занимает модерация карточек на Ozon?

Сроки зависят от категории:

  • ⚡ Быстрая модерация (1–6 часов): книги, канцтовары, некоторые виды одежды.
  • 🕒 Стандартная (1–3 дня): электроника, бытовая техника.
  • 🐢 Длительная (до 7 дней): детские товары, продукты питания (требуют проверки сертификатов).

Чтобы ускорить процесс:

  • 📋 Заполняйте все обязательные поля.
  • 📄 Прикрепляйте сертификаты заранее.
  • 📧 Пишите в поддержку, если модерация затянулась (>3 дней).

Где и как правильно посмотреть код товара на Озоне: все способы 2026

На Озоне каждый товар имеет уникальный идентификатор — код товара, который может понадобиться как покупателям (для поиска аналогичных позиций, возвратов или жалоб), так и продавцам (для управления ассортиментом, аналитики или интеграции с 1С). Однако найти этот код не всегда интуитивно: интерфейс маркетплейса регулярно обновляется, а сами идентификаторы могут скрываться под разными названиями — артикул, SKU, ID товара или код производителя.

В этой статье разберём все актуальные способы просмотра кода товара на Озоне в 2026 году — от очевидных (через карточку товара) до скрытых (через API или экспорт данных). Особое внимание уделим различиям для покупателей и продавцов, а также нюансам работы с мобильным приложением и десктопной версией сайта. Если вы ищете конкретный идентификатор для возврата, жалобы или синхронизации с системой учёта — здесь найдёте чёткую инструкцию.

Что такое код товара на Озоне и зачем он нужен

На Озоне под «кодом товара» могут подразумевать сразу несколько типов идентификаторов:

  • 🔢 ID товара (Product ID) — уникальный числовой код, который присваивает маркетплейс каждой позиции в каталоге. Используется для внутренних операций (например, в API или при обмене данными с продавцом).
  • 🏷️ Артикул продавца (SKU) — код, который присваивает сам продавец (может совпадать с артикулом производителя или быть уникальным для магазина). Нужен для управления остатками и заказами.
  • 📦 Код производителя — оригинальный артикул бренда (например, iPhone 15 Pro может иметь артикул MQ2T3RX/A). Важен для поиска запчастей или гарантийного обслуживания.

Для чего может понадобиться код товара:

  • 🔄 Покупателям: для оформления возврата, поиска аналогичных товаров, жалобы на продавца или проверки подлинности (сравнение артикулов).
  • 📊 Продавцам: для загрузки прайс-листов, интеграции с 1С, анализа продаж по SKU, настройки рекламных кампаний.
  • 🤖 Разработчикам: для работы с API Ozon Seller (например, получение информации о товаре по его ID).
⚠️ Внимание: На Озоне ID товара и SKU — это разные вещи! Первый присваивается маркетплейсом автоматически, второй — продавец указывает сам. При возврате или жалобе обычно требуется ID товара (его можно найти в URL карточки).

Как покупателю найти код товара в карточке на сайте Озон

Самый простой способ — посмотреть код непосредственно в карточке товара. Для этого:

  1. Откройте страницу товара на ozon.ru.
  2. Прокрутите страницу вниз до блока «Характеристики».
  3. Найдите строки Артикул или Код товара (может называться ID товара или SKU).

Если кода нет в характеристиках, проверьте URL страницы — там всегда есть ID товара в формате цифр после слеша. Например, в ссылке https://www.ozon.ru/product/iphone-15-pro-128gb-123456789 код товара — 123456789.

Прокрутить до блока "Характеристики"|Проверить строку "Артикул" или "Код товара"|Посмотреть цифры в URL после последнего слеша|Использовать комбинацию Ctrl+F и ввести "артикул"-->

В некоторых категориях (например, электроника или одежда) код производителя может быть указан отдельно. Например, для Samsung Galaxy S23 это будет что-то вроде SM-S911BZKD.

Тип кода Где искать Пример
ID товара (Ozon) URL страницы (после последнего слеша) 123456789
Артикул продавца (SKU) Блок «Характеристики» → «Артикул» IPH-15-PRO-BLACK
Код производителя Блок «Характеристики» → «Артикул производителя» MQ2T3RX/A (для iPhone)
⚠️ Внимание: Если вы оформляете возврат, в форме Озон может потребоваться именно ID товара (из URL), а не артикул продавца. Уточните это в чате поддержки, чтобы не получить отказ.

Поиск кода товара в мобильном приложении Озон

В мобильном приложении интерфейс отличается от десктопной версии, поэтому код товара искать нужно иначе. Инструкция для iOS и Android:

  1. Откройте приложение Озон и найдите нужный товар.
  2. Тапните по карточке, чтобы открыть её полностью.
  3. Прокрутите вниз до блока «Подробности» (на iOS) или «Характеристики» (на Android).
  4. Найдите строки Артикул или Код товара. Если их нет — тапните по трём точкам в правом верхнем углу и выберите «Поделиться», чтобы скопировать ссылку (в ней будет ID товара).

На некоторых версиях приложения код может скрываться под кнопкой «Ещё» или «Показать все характеристики». Если вы не видите артикул, попробуйте:

  • 🔍 Использовать поиск по странице (в браузере мобильного приложения).
  • 📋 Скопировать ссылку и извлечь ID из неё (цифры после /product/).
  • 💬 Написать в чат поддержки Озон с просьбой предоставить код (указывайте название товара и ссылку).
💡

Если в мобильном приложении не отображается артикул, попробуйте открыть карточку товара в браузере через режим «Для компьютера» (в Chrome или Safari).

Как продавцу найти код товара в личном кабинете Ozon Seller

Для продавцов на Озоне доступно несколько способов просмотра кода товара, в зависимости от задачи:

1. Через каталог товаров

  1. Зайдите в личный кабинет Ozon Seller.
  2. Перейдите в раздел Товары → Мой каталог.
  3. Найдите нужный товар и кликните по его названию.
  4. В открывшейся карточке будет блок «Основная информация» с полями ID товара и Артикул (SKU).

2. Через экспорт данных

Если нужно получить коды для большого количества товаров:

  1. В личном кабинете перейдите в Товары → Экспорт.
  2. Выберите формат XLSX и нажмите «Скачать».
  3. В полученном файле будут колонки Product ID (ID товара) и SKU (артикул продавца).

3. Через API (для автоматизации)

Если вы используете интеграцию с 1С или другим ПО, код товара можно получить через API-запрос:

GET https://api-seller.ozon.ru/v2/product/info?product_id={ID_ТОВАРА}&offer_id={SKU}

Где {ID_ТОВАРА} — это Product ID из личного кабинета, а {SKU} — артикул продавца.

Способ Где искать код Для чего подходит
Каталог товаров Товары → Мой каталог → Основная информация Просмотр кода для 1-2 товаров
Экспорт в XLSX Колонки Product ID и SKU Массовая выгрузка кодов
API Ответ на запрос /product/info Автоматизация и интеграции
💡

Для работы с API Озон требуется авторизация по токену. Его можно получить в личном кабинете в разделе Настройки → API-ключи.

Где ещё можно найти код товара: альтернативные способы

Если стандартные методы не сработали, попробуйте эти варианты:

1. В чеке или квитанции об оплате

При покупке товара его ID или SKU могут быть указаны в:

  • 📄 Электронном чеке (приходит на почту после оплаты).
  • 📱 Уведомлении о покупке в мобильном приложении (раздел «Мои заказы»).
  • 🖨️ Печатной квитанции (если забирали товар в ПВЗ).

2. В письмах от поддержки Озон

Если вы обращались в службу поддержки по поводу товара, его код мог быть указан в переписке. Проверьте:

  • 📧 Письма с темой «Ваш заказ №XXXX».
  • 💬 Сообщения в чате поддержки (раздел «Помощь» в приложении).

3. Через поиск по каталогу

Если вы знаете название товара, но не можете найти его карточку:

  1. Перейдите на главную страницу Озон.
  2. Введите название товара в поисковую строку.
  3. В результатах поиска кликните правой кнопкой по карточке товара и выберите «Копировать ссылку».
  4. Вставьте ссылку в текстовый редактор — ID товара будет после /product/.
Что делать, если код товара не отображается нигде?

Если вы продавец и не видите SKU или ID товара, это может означать, что товар ещё не прошёл модерацию или был удалён. Проверьте статус в разделе Товары → На модерации. Если товар покупателя — попробуйте связаться с продавцом через чат в заказе (кнопка «Задать вопрос»).

Разница между ID товара, SKU и артикулом производителя

Многие путают эти понятия, но на Озоне они имеют чёткие различия:

Термин Кто присваивает Где используется Пример
ID товара (Product ID) Озон (автоматически) Внутренние операции, API, ссылки 123456789
SKU (артикул продавца) Продавец (вручную) Учёт остатков, заказы, аналитика APPLE-IPHONE-15-BLACK-128
Артикул производителя Бренд (заводской) Гарантия, запчасти, проверка подлинности MQ2T3RX/A

Почему это важно:

  • 🔧 Для возврата обычно нужен ID товара (из URL).
  • 📦 Для управления остатками продавцу нужен SKU.
  • 🛠️ Для гарантийного ремонта может понадобиться артикул производителя.
⚠️ Внимание: Некоторые продавцы используют SKU как дубль артикула производителя, но это ошибка! На Озоне SKU — это внутренний идентификатор продавца, который может отличаться от заводского артикула.

Частые ошибки при поиске кода товара и как их избежать

Даже опытные пользователи Озон иногда допускают ошибки. Вот самые распространённые:

  1. Путают ID товара и SKU.

    🔹 Как избежать: Помните, что ID — это цифры в URL, а SKU — это артикул, который указал продавец (может содержать буквы и дефисы).

  2. Копируют неполный URL.

    🔹 Как избежать: Убедитесь, что скопировали ссылку полностью, включая цифры после /product/. Например, в ozon.ru/product/iphone-15-123456789 код — 123456789, а не iphone-15.

  3. Ищут код в неправильном разделе личного кабинета.

    🔹 Как избежать: Продавцы часто путают Товары → Мой каталог и Заказы → История. Код товара ищут в каталоге, а не в заказах!

  4. Не обновляют кэш браузера.

    🔹 Как избежать: Если вы не видите блок с артикулом, попробуйте обновить страницу (Ctrl + F5) или открыть её в режиме инкогнито.

Ещё одна типичная проблема — код товара не отображается в мобильном приложении. В этом случае:

  • 🌐 Откройте карточку товара в браузере (через «Поделиться» → «Открыть в браузере»).
  • 📱 Обновите приложение до последней версии.
  • 📧 Напишите в поддержку Озон с просьбой предоставить код (указывайте ссылку на товар).
💡

Если вы продавец и не видите SKU в экспортируемом файле, проверьте настройки колонок в личном кабинете. Возможно, поле SKU скрыто — включите его в Товары → Экспорт → Настроить колонки.

FAQ: Ответы на частые вопросы о кодах товаров на Озоне

Можно ли найти код товара, если карточка удалена?

Если товар удалён из каталога Озон, его ID и SKU можно попробовать найти:

  • В истории заказов (если товар покупался).
  • В письмах от поддержки (если были обращения по этому товару).
  • Через API (если вы продавец и товар был в вашем каталоге).

Если товар удалён окончательно, восстановить его код невозможно — придётся создавать новую карточку.

Что делать, если SKU дублируется у разных товаров?

Дублирование SKU — грубая ошибка, которая может привести к:

  • 🔄 Путанице в остатках (система будет списывать товар неправильно).
  • ❌ Блоку товара модераторами Озон.

Как исправить:

  1. Зайдите в Товары → Мой каталог.
  2. Найдите товары с одинаковым SKU.
  3. Измените SKU для одного из товаров (добавьте суффикс, например, -1, -2).
Как узнать код товара по штрихкоду?

Озон не привязывает штрихкоды (EAN, UPC) напрямую к ID товара или SKU. Однако вы можете:

  1. Найти товар по штрихкоду через поиск на Озоне (не всегда работает).
  2. Связаться с продавцом и запросить SKU или ID.
  3. Использовать сторонние сервисы вроде Barcode Live для определения артикула производителя, а затем искать его на Озоне.

⚠️ Озон не гарантирует, что товар со штрихкодом 1234567890123 будет иметь такой же ID или SKU!

Можно ли изменить SKU товара после публикации?

Да, продавцы могут редактировать SKU в личном кабинете:

  1. Перейдите в Товары → Мой каталог.
  2. Выберите товар и нажмите «Редактировать».
  3. В блоке «Основная информация» измените поле Артикул (SKU).
  4. Сохраните изменения.

⚠️ После изменения SKU:

  • Обновите его в своей системе учёта (1С, МойСклад и т. д.).
  • Проверьте, что товар не исчез из рекламных кампаний (если использовался старый SKU).
Где взять код товара для возврата?

Для оформления возврата на Озоне обычно требуется ID товара (из URL карточки) или номер заказа. Инструкция:

  1. Откройте раздел возвратов.
  2. Выберите заказ с нужным товаром.
  3. Система автоматически подтянет ID товара. Если его нет — скопируйте из ссылки на карточку.

Если товар был в наборе (комплекте), уточните у поддержки, нужно ли указывать ID каждого элемента отдельно.

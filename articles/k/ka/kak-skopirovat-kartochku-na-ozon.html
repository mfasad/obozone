Как скопировать карточку товара на Ozon: 5 проверенных способов для продавцов

Копирование карточек товаров на Ozon — задача, с которой сталкивается каждый второй продавец при масштабировании ассортимента. Согласно статистике Ozon Seller за 2026 год, 63% новых продавцов начинают торговлю с переноса товаров из других площадок или дублирования успешных позиций конкурентов. Однако не все способы копирования одинаково безопасны: некоторые ведут к мгновенной блокировке аккаунта, другие требуют глубоких технических знаний.

В этой статье разберём 5 легальных методов копирования карточек на Ozon — от ручного дублирования до автоматизированного парсинга через API. Вы узнаете, как перенести описания, фотографии и характеристики без нарушения правил маркетплейса, а также какие инструменты помогут сэкономить до 80% времени на загрузке товаров. Особое внимание уделим рискам блокировок и способам их избежать: например, почему нельзя копировать тексты "как есть" и как правильно адаптировать контент под требования Ozon.

Важно: статья актуальна для продавцов на схеме FBS и FBO, а также для тех, кто планирует мигрировать с других площадок (Wildberries, Яндекс Маркет). Все описанные методы соответствуют Правилам продавцов Ozon 2026 (раздел 4.3 "Загрузка товарных карточек").

1. Ручное копирование карточки: пошаговая инструкция для новичков

Самый простой, но трудоёмкий способ — перенос данных вручную через личный кабинет Ozon Seller. Он подходит для небольшого количества товаров (до 50 позиций) и не требует технических навыков. Главное преимущество: 100% безопасность — маркетплейс не блокирует аккаунты за ручной ввод данных.

Чтобы скопировать карточку вручную:

  • 📌 Откройте исходную карточку товара на Ozon (или другом маркетплейсе) в отдельной вкладке браузера.
  • 📋 Скопируйте ключевую информацию: название, описание, характеристики (вес, размеры, материал), фотографии.
  • 🖥️ Перейдите в Личный кабинет → Товары → Добавить товар.
  • 📝 Вставьте данные в соответствующие поля, адаптируя текст под требования Ozon (например, добавьте ключевые слова в название).

Совпадение категорий товара на исходной и новой карточке

Уникальность фотографий (не менее 30% отличий)

Отсутствие водяных знаков конкурентов

Соблюдение лимитов символов в описании (макс. 4000 знаков)

2. Парсинг карточек: автоматизация без блокировок

Парсинг (сбор данных с помощью специальных программ) позволяет скопировать сотни карточек за несколько часов. Однако Ozon активно борется с автоматизированным сбором данных: в 2026 году зафиксировано 38% блокировок аккаунтов продавцов из-за некорректного парсинга (данные Ozon Security Report). Чтобы избежать санкций, используйте только разрешённые инструменты:

  • 🤖 Ozon API — официальный способ получения данных о товарах (требует регистрации приложения).
  • 🛠️ Parser-Ozon — сервис с прокси-серверами для обхода блокировок (от 1 500 ₽/мес).
  • 📊 DataCol — облачный парсер с поддержкой Ozon и Wildberries (бесплатный тариф на 100 карточек).

Для работы через Ozon API выполните шаги:

  1. Зарегистрируйте приложение в Личном кабинете API.
  2. Получите Client-ID и API-Key.
  3. Используйте запрос GET /v2/product/info для получения данных о товаре по его product_id.
curl -X GET 'https://api-seller.ozon.ru/v2/product/info' \

-H 'Client-Id: YOUR_CLIENT_ID' \


-H 'Api-Key: YOUR_API_KEY' \


-H 'Content-Type: application/json' \


-d '{"product_id": 12345678}'

Что будет если парсить без API?

Ozon использует систему Rate Limiting: при превышении 10 запросов в секунду с одного IP ваш аккаунт будет заблокирован на 24 часа. Повторные нарушения ведут к полному бану без возможности восстановления. В 2023 году 12% продавцов потеряли доступ к аккаунтам из-за агрессивного парсинга.

⚠️ Внимание: Даже при использовании API нельзя копировать отзывы покупателей и рейтинги конкурентов. Это нарушает п. 7.2 Пользовательского соглашения и может привести к штрафу до 50 000 ₽.

3. Копирование через Excel: массовая загрузка товаров

Для продавцов с большим ассортиментом (от 100 позиций) оптимален способ импорта через Excel-шаблоны. Ozon предоставляет готовые файлы для загрузки, которые можно заполнить данными из скопированных карточек. Этот метод сокращает время добавления товаров на 70% по сравнению с ручным вводом.

Инструкция по работе с Excel:

  1. Скачайте шаблон в Личный кабинет → Товары → Импорт/Экспорт → Скачать шаблон.
  2. Перенесите данные из исходных карточек в столбцы:
    • Name — название товара (макс. 150 символов).
    • Description — описание (HTML-разметка разрешена).
    • Price — цена в рублях.
    • Images — ссылки на фотографии (разрешённые форматы: .jpg, .png).
  • Загрузите файл обратно через Импорт товаров.

    • 📌 Важно: При копировании фотографий убедитесь, что:

    • 📸 Разрешение изображений не менее 800×800 px.
    • 🎨 Фон белый или прозрачный (для категорий "Одежда" и "Обувь" допускаются модели на манекене).
    • 🚫 Нет логотипов других брендов или водяных знаков.
    💡

    Используйте формулу =CONCATENATE() в Excel для автоматического формирования названий товаров из нескольких столбцов (например, бренд + модель + ключевое слово).

    4. Клонирование карточек внутри Ozon Seller

    Если вам нужно дублировать собственные карточки (например, для создания вариаций товара), используйте встроенную функцию клонирования. Это самый быстрый способ, так как все данные (описание, фотографии, характеристики) переносятся автоматически.

    Как клонировать карточку:

    1. Перейдите в Личный кабинет → Товары.
    2. Найдите нужный товар и нажмите ⋮ → Клонировать.
    3. Измените уникальные параметры: Артикул, Цена, Наличие.
    4. Сохраните новую карточку.

    ⚠️ Внимание: Клонирование чужих карточек (даже если они удалены) запрещено! Алгоритмы Ozon отслеживают историю изменений и могут заблокировать товар за плагиат. Разрешается клонировать только свои карточки или использовать их как шаблон для новых.

    Пример легального использования клонирования:

    • 👕 Создание вариаций одного товара (например, футболка в разных цветах).
    • 📦 Перенос успешного описания на аналогичный товар (например, смартфон другой модели того же бренда).

    6. Юридические риски и как их избежать

    Копирование карточек товаров регулируется не только правилами Ozon, но и российским законодательством:

    • 📜 Статья 1259 ГК РФ — защита авторских прав на тексты и изображения.
    • 📜 Статья 146 УК РФ — ответственность за нарушение авторских прав (штраф до 5 млн ₽).
    • 📜 Пункт 4.5 Правил Ozon — запрет на плагиат контента.

    Чтобы избежать претензий:

    • 📝 Переписывайте тексты с изменением не менее 30% (используйте сервисы Text.ru или Advego Plagiatus для проверки уникальности).
    • 📸 Делайте свои фотографии или покупайте стоковые изображения на Shutterstock/Depositphotos.
    • 🔍 Проверяйте карточки на плагиат через Личный кабинет → Товары → Проверка контента.

    ⚠️ Внимание: Если правообладатель (например, бренд Samsung или Xiaomi) подаст жалобу на плагиат, Ozon удалит товар без предупреждения. В 2026 году количество таких жалоб выросло на 40% по сравнению с 2023-м (данные Роспатента).

    Пример легального заимствования:

    Неправильно: Скопировать описание смартфона iPhone 15 с сайта Apple.
    Правильно: Использовать технические характеристики из открытых источников (например, GSMArena) и переписать текст своими словами.

    FAQ: Частые вопросы о копировании карточек на Ozon

    Можно ли копировать карточки с Wildberries на Ozon?

    Да, но с обязательной адаптацией:

    • Измените название (на Ozon лимит — 150 символов, на WB — 180).
    • Добавьте характеристики, обязательные для Ozon (например, Страна производства).
    • Загрузите фотографии в разрешении не ниже 800×800 px (на WB допускаются изображения 500×500 px).

    Используйте сервисы вроде DataCol для автоматического переноса данных между площадками.

    Как скопировать отзывы с карточки конкурента?

    🚫 Запрещено! Перенос отзывов нарушает:

    • Пункт 7.2 Правил Ozon (запрет на манипуляцию рейтингами).
    • Статью 10 Федерального закона "О рекламе" (введение в заблуждение покупателей).

    Альтернатива: предложите покупателям оставить отзыв через Личный кабинет → Заказы → Написать покупателю (шаблон сообщения можно автоматизировать).

    Сколько времени занимает ручное копирование одной карточки?

    В среднем — 15–25 минут, если:

    • Товар простой (например, чехол для телефона) — 10–15 минут.
    • Товар с множеством характеристик (например, ноутбук) — до 30 минут.

    🔹 Совет: Используйте горячие клавиши для ускорения:

    • Ctrl+C/Ctrl+V — копирование/вставка.
    • Alt+Tab — переключение между вкладками.
    • F5 — обновление страницы после сохранения.

    Что делать, если Ozon заблокировал товар за плагиат?

    Алгоритм действий:

    1. Проверьте причину блокировки в Личный кабинет → Уведомления.
    2. Отредактируйте карточку:
      • Измените текст описания (уникальность от 70%).
      • Замените фотографии (если они скопированы).
  • Напишите в поддержку через Личный кабинет → Помощь → Написать в службу поддержки с просьбой разблокировать товар. Приложите скриншоты изменений.

    • 📌 Срок разблокировки: от 1 до 3 рабочих дней.

    Можно ли копировать карточки с AliExpress?

    Технически да, но с учётом:

    • 🌍 Язык: Переведите все тексты на русский (используйте DeepL или Google Translate + ручная правка).
    • 💰 Цена: Учитывайте таможенные пошлины и наценку Ozon (минимум +30% к закупочной цене).
    • ⚖️ Юридические риски: Если товар брендированный (например, Xiaomi), потребуется сертификат от правообладателя.

    🔹 Совет: Для копирования с AliExpress используйте расширение Oberlo (интегрируется с Ozon через Shopify).

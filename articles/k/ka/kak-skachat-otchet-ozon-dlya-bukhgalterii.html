Как скачать отчёт Озон для бухгалтерии: пошаговая инструкция с картинками

Работа с бухгалтерской отчётностью на маркетплейсе Ozon требует точности: ошибка в выгрузке данных может привести к расхождениям в налогах, штрафам или проблемам с 1С. В этой статье разберём, как правильно скачать отчёты для бухгалтерии — от базовых выписок по продажам до детализированных данных для налоговой декларации.

Мы рассмотрим все актуальные способы выгрузки (через личный кабинет, API, автоматизированные сервисы), форматы файлов (Excel, CSV, XML), а также типичные ошибки, которые допускают продавцы. Особое внимание уделим нюансам для разных систем налогообложения (УСН, ОСНО, ЕНВД) и интеграции с 1С.

Где в личном кабинете Озон найти отчёты для бухгалтерии

Все финансовые отчёты в Ozon Seller сосредоточены в разделе Финансы → Отчёты. Но не все из них подходят для бухгалтерского учёта. Вот ключевые документы, которые потребуются:

  • 📊 Отчёт по продажам — основной документ с данными о заказах, возвратах и комиссиях. Используется для расчёта выручки.
  • 💰 Выписка по счёту — движения денежных средств (поступления от покупателей, удержания Ozon, выплаты продавцу).
  • 📑 Акт выполненных работ — подтверждает оказание услуг маркетплейсом (нужен для списания комиссий).
  • 🔄 Отчёт по возвратам — детализация возвратных операций (важно для корректировки НДС).

Чтобы не путаться в меню, запомните путь:

Личный кабинет → Финансы → Отчёты → [Выберите тип отчёта] → Скачать

Обратите внимание: некоторые отчёты (например, акт выполненных работ) формируются только после закрытия расчётного периода. Если кнопка скачивания неактивна — проверьте даты в фильтрах.

Пошаговая инструкция: как скачать отчёт по продажам

Отчёт по продажам — самый востребованный документ для бухгалтерии. Он содержит данные о заказах, суммах продаж, комиссиях и возвратах. Вот как его правильно выгрузить:

  1. Перейдите в раздел Финансы → Отчёты → Отчёт по продажам.
  2. Установите период. Для бухгалтерии обычно нужен месяц или квартал. Важно: даты в отчёте указываются по московскому времени (UTC+3), даже если ваш склад в другом регионе.
  3. Выберите формат:
    • 📖 Excel (XLSX) — удобен для ручной обработки, поддерживает формулы.
    • 🗃️ CSV — лёгкий формат для загрузки в 1С или другие программы.
    • 🔗 XML — нужен для электронного документооборота (например, с налоговой).
  • Нажмите «Скачать» и дождитесь формирования файла. В зависимости от объёма данных это может занять от 10 секунд до нескольких минут.

    • Если отчёт не скачивается или выдаёт ошибку, проверьте:

    • 🔌 Стабильность интернет-соединения (особенно при больших объёмах данных).
    • 📅 Корректность указанного периода (нельзя выгрузить данные за будущую дату).
    • 🔒 Права доступа вашего аккаунта (у менеджера должен быть доступ к финансовым отчётам).

    Какие отчёты нужны для разных систем налогообложения

    Требования к отчётности зависят от вашей системы налогообложения. В таблице ниже — минимальный набор документов для каждой из них:

    Система налогообложения Необходимые отчёты Озон Дополнительные документы
    УСН (6% или 15%) Отчёт по продажам, Выписка по счёту Книга учёта доходов и расходов (КУДиР)
    ОСНО Отчёт по продажам, Акт выполненных работ, Отчёт по возвратам Счёт-фактура от Озон (для НДС), Регистры налогового учёта
    ЕНВД Отчёт по продажам (для контроля выручки) Декларация по ЕНВД, Кассовые чеки (при офлайн-продажах)
    Патент (ПСН) Отчёт по продажам Книга учёта доходов (если предусмотрено патентом)

    Для ОСНО особенно важно правильно учитывать НДС. Ozon удерживает НДС с комиссий и некоторых услуг, поэтому в акте выполненных работ должна быть детализация по ставкам (20%, 10%, 0%). Если ставка неверная — это приведёт к расхождениям в декларации.

    Что делать, если в отчёте нет НДС?

    Если в выписке или акте отсутствует раздел по НДС, проверьте:

    1. Настройки вашего договора с Ozon (возможно, вы работаете без НДС).

    2. Период формирования отчёта (НДС может отображаться только после закрытия месяца).

    3. Тип товара (некоторые категории, например, книги, облагаются по ставке 10%).

    Если проблема сохраняется — обратитесь в поддержку с указанием номера акта и периода.

    Внимание! С 2026 года Ozon автоматически передаёт данные о продажах в ФНС через систему АСК НДС-2. Это означает, что налоговая видит вашу выручку даже если вы не сдали декларацию. Поэтому важно сверять отчёты маркетплейса с вашей бухгалтерией.

    Автоматическая выгрузка отчётов через API Озон

    Если вы обрабатываете большие объёмы данных (например, более 1000 заказов в месяц), ручная выгрузка отчётов становится неэффективной. В этом случае поможет API Ozon Seller.

    Через API можно автоматизировать:

    • 🤖 Ежедневную выгрузку отчётов по продажам в 1С или Google Sheets.
    • 📊 Формирование сводных таблиц с разбивкой по товарам, регионам, типам доставки.
    • 🔄 Синхронизацию возвратов и списаний с бухгалтерской программой.

    Для настройки потребуется:

    1. Получить Client-ID и API-ключ в личном кабинете (Настройки → Интеграции → API).
    2. Использовать официальную документацию API Ozon для формирования запросов.
    3. Написать скрипт (на Python, PHP или другом языке) или использовать готовые сервисы вроде AltoAPI, MoySklad.

    Пример запроса для выгрузки отчёта по продажам:

    GET https://api-seller.ozon.ru/v2/analytics/data

    Headers:
    

    - Client-Id: [ваш_client_id]
    

    - Api-Key: [ваш_api_ключ]
    

    Params:
    

    - date_from: "2026-01-01T00:00:00Z"
    

    - date_to: "2026-01-31T23:59:59Z"
    

    - metrics: "revenue,orders"
    

    - dimensions: "day,sku"

    Если вы не программист, можно воспользоваться сервисами-посредниками, которые берут на себя настройку API. Стоимость таких сервисов — от 500 до 3000 рублей в месяц.

    💡

    Перед подключением API проверьте лимиты запросов в вашем тарифе. Бесплатный тариф позволяет делать до 1000 запросов в час, для больших объёмов потребуется платный аккаунт.

    Типичные ошибки при скачивании отчётов и как их избежать

    Даже опытные продавцы иногда сталкиваются с проблемами при выгрузке отчётности. Вот самые распространённые ошибки и способы их решения:

    • ❌ Отчёт не формируется — проверьте, закрыт ли расчётный период. Например, выписка по счёту за текущий день может быть недоступна до конца операционного дня (23:59 по МСК).
    • ❌ Данные в Excel отображаются криво — это частая проблема при открытии CSV в Excel. Решение: импортируйте файл через Данные → Из текста, выбрав разделитель ;.
    • ❌ Расхождения с 1С — обычно возникают из-за неучтённых возвратов или комиссий. Сверяйте отчёт по продажам с отчётом по возвратам и актами выполненных работ.
    • ❌ Нет данных по некоторым заказам — возможно, заказы были отменены или не попали в выбранный период. Используйте фильтр Статус заказа = "Выполнен".

    Внимание! Если вы скачали отчёт, но в нём отсутствуют данные по FBS-заказам (доставка силой Ozon), проверьте настройки фильтра. По умолчанию некоторые отчёты показывают только FBO (самостоятельная доставка). Чтобы увидеть все заказы, выберите в фильтрах Тип логистики = "Все".

    Ещё одна частая проблема — дублирование данных при выгрузке за пересекающиеся периоды. Например, если вы скачали отчёт за 1–15 января и за 10–20 января, заказы с 10 по 15 число попадут в оба файла. Чтобы избежать путаницы, используйте непересекающиеся диапазоны или удаляйте дубли в Excel с помощью функции Удалить дубликаты.

    Как интегрировать отчёты Озон с 1С: пошаговый алгоритм

    Интеграция отчётов Ozon с 1С позволяет автоматизировать бухгалтерский учёт и снизить риск ошибок. Вот как это сделать:

    1. Экспортируйте данные в CSV — этот формат лучше всего подходит для импорта в 1С.
    2. Настройте правила загрузки в 1С:
      • 📌 Сопоставьте колонки из отчёта Озон с полями в 1С (например, order_id → НомерДокумента).
      • 📌 Укажите тип документа (обычно это РеализацияТоваровУслуг или ПоступлениеДенежныхСредств).
      • 📌 Настройте обработку комиссий (их можно списывать как ПрочиеРасходы).
  • Загрузите данные через Обработки → Загрузка данных из файла.
  • Проверьте проводки — убедитесь, что суммы совпадают с отчётами Озон.

    • Если у вас 1С:Бухгалтерия 8.3, можно использовать готовые обработки от сообщества Infostart или 1С-Рарус. Например, обработка "Загрузка выписки Ozon в 1С" автоматически распознаёт комиссии и возвраты.

    Важно: При первой загрузке сделайте тестовый импорт на копии базы 1С. Это поможет избежать ошибок в рабочей базе. Также настройте резервное копирование перед массовой загрузкой данных.

    - Соответствие ставок НДС в отчётах Озон и 1С.

    - Валюту документов (все суммы должны быть в рублях).

    - Даты операций (в 1С они должны совпадать с датами в отчётах).-->

    Альтернативные способы получения отчётности

    Помимо личного кабинета и API, есть другие инструменты для выгрузки бухгалтерских данных:

    • 📱 Мобильное приложение Ozon Seller — позволяет скачивать основные отчёты (например, выписку по счёту) прямо с телефона. Удобно для оперативного контроля, но функционал ограничен.
    • 🔗 Сервисы-агрегаторы (например, SellerLab, eLama) — собирают данные с нескольких маркетплейсов и формируют сводную отчётность. Подходят для мультиканальных продавцов.
    • 📧 Email-рассылка — можно настроить автоматическую отправку отчётов на почту через Настройки → Уведомления. Полезно для еженедельного мониторинга.
    • 🤝 Аутсорсинг бухгалтерии — некоторые компании (например, 1С:Аутсорсинг) специализируются на учёте для маркетплейсов и берут на себя выгрузку и обработку отчётов.

    Если вы работаете с несколькими маркетплейсами (например, Ozon + Wildberries), удобнее использовать агрегаторы. Они сводят данные в единый формат и экономят время на обработке. Стоимость таких сервисов — от 1000 рублей в месяц.

    Пример: Сервис SellerLab позволяет:

    • Скачивать отчёты по продажам с Ozon, WB, Яндекс Маркета в одном файле.
    • Автоматически рассчитывать налоги (УСН, НДС) на основе выгруженных данных.
    • Синхронизировать остатки товаров между маркетплейсами.

    FAQ: Частые вопросы о бухгалтерских отчётах Озон

    Можно ли скачать отчёт за прошлый год?

    Да, в личном кабинете доступны отчёты за весь период вашей работы на Ozon. Однако для периодов старше 2 лет данные могут архивироваться — в этом случае их придётся запрашивать через поддержку (раздел Помощь → Обратиться в поддержку). Укажите в запросе точные даты и типы нужных отчётов.

    Как скачать отчёт с разбивкой по товарам?

    Для детализации по товарам используйте Отчёт по продажам с группировкой по SKU или Наименование товара. В настройках фильтра выберите: 

    Группировка → Товар

    Показать → Все колонки (включая артикул, категорию, бренд)

    Если нужна аналитика по прибыли, добавьте колонки Себестоимость и Маржа (если они заполнены в вашем каталоге).

    Что делать, если в отчёте не хватает денег?

    Расхождения между отчётом и реальными поступлениями могут возникать по нескольким причинам:

    1. Удержания Озон — проверьте выписку по счёту на предмет штрафов, комиссий за хранение или рекламу.
    2. Задержки выплат — деньги за некоторые заказы (например, с оплатой при получении) могут поступать позже.
    3. Ошибки округления — при больших объёмах копеечные расхождения суммируются. Сверяйте итоги по колонке Сумма к выплате.

    Если разница значительная (более 1% от оборота), обратитесь в поддержку с указанием номеров заказов и периодов.

    Нужно ли хранить отчёты Озон для налоговой?

    Да, согласно ст. 23 НК РФ, вы обязаны хранить документы, подтверждающие доходы и расходы, в течение 4 лет. Рекомендуется:

    • Сохранять отчёты в облачном хранилище (например, Google Drive или Яндекс Диск).
    • Архивировать файлы по годам и кварталам (например, Озон_2026_Квартал1.zip).
    • Вести журнал выгрузок с указанием даты и типа отчёта.

    Если налоговая запросит документы, вы должны предоставить их в течение 10 рабочих дней.

    Можно ли скачать отчёт в PDF?

    Нет, Ozon не предоставляет отчёты в формате PDF. Однако вы можете конвертировать Excel или CSV в PDF самостоятельно:

    1. Откройте файл в Excel или Google Sheets.
    2. Нажмите Файл → Печать → Сохранить как PDF.
    3. Убедитесь, что все колонки помещаются на страницу (при необходимости измените ориентацию на альбомную).

    PDF удобен для архивации, но не подходит для дальнейшей обработки данных.

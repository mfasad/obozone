Зачем продавцам выгружать карточки товаров с Ozon и как это сделать правильно

Выгрузка карточек товаров с Ozon — стандартная задача для продавцов, которые хотят анализировать конкурентов, переносить данные на другие площадки или создавать резервные копии. Но маркетплейс не предоставляет прямой кнопки "Скачать все товары": каждый метод имеет ограничения по объёму данных, скорости и техническим требованиям. Например, ручное копирование подойдёт для 5–10 позиций, а для каталога из 500+ SKU потребуется API или парсинг.

В этой статье разберём все актуальные способы выгрузки карточек с Ozon в 2026 году, включая их плюсы, минусы и пошаговые инструкции. Особое внимание уделим легальным методам, которые не нарушают пользовательское соглашение маркетплейса. Если вам нужно скачать данные для личного использования (например, чтобы перенести товары на Wildberries или в свой интернет-магазин), вы найдёте здесь рабочие решения. Для автоматизированного сбора данных в коммерческих целях — предупредим о рисках и альтернативах.

Способ 1: Ручной экспорт через личный кабинет Ozon Seller

Самый простой и безопасный метод — использовать встроенные инструменты Ozon Seller. Он подходит для выгрузки ограниченного количества товаров (до 100–200 позиций) и не требует технических навыков. Данные экспортируются в формате .xlsx или .csv, что удобно для дальнейшей обработки в Excel или Google Sheets.

Чтобы скачать карточки:

  1. Авторизуйтесь в личном кабинете Ozon Seller.
  2. Перейдите в раздел Товары → Мои товары.
  3. Отметьте галочками нужные позиции (или нажмите "Выделить все").
  4. Кликните "Экспорт" и выберите формат файла.

Ограничения метода:

  • 📉 Максимум 200 товаров за один экспорт (при большем количестве — ошибка сервера).
  • ⏳ Скорость зависит от загрузки серверов Ozon (в пиковые часы может "подвисать").
  • 📊 В экспортируемых данных отсутствуют некоторые поля (например, сео-тексты или отзывы покупателей).
💡

Если нужно скачать больше 200 товаров, разбейте их на группы по категориям или брендам и экспортируйте по частям.

Способ 2: Использование Ozon API для массовой выгрузки

Для продавцов с большим ассортиментом (1000+ SKU) ручной экспорт неэффективен. В этом случае поможет Ozon API — официальный интерфейс для автоматизированного обмена данными. Чтобы им воспользоваться, нужен доступ к API-ключу и базовые знания работы с JSON или Python.

Пошаговая инструкция:

  1. Получите Client-ID и API-Key в разделе Настройки → API личного кабинета.
  2. Используйте эндпоинт /v2/product/info для запроса данных по конкретным product_id.
  3. Для массовой выгрузки напишите скрипт (пример на Python ниже) или используйте готовые решения типа Postman.
import requests


url = "https://api-seller.ozon.ru/v2/product/info"


headers = {


"Client-Id": "Ваш_Client-ID",


"Api-Key": "Ваш_API-Key",


"Content-Type": "application/json"


}


payload = {


"product_id": [12345678, 87654321],  # Список ID товаров


"offer_id": ["offer1", "offer2"]    # Опционально: ID предложений


}



response = requests.post(url, headers=headers, json=payload)


print(response.json())

Преимущества API:

  • ⚡ Выгрузка неограниченного количества товаров (лимит — 100 запросов в минуту).
  • 📂 Полные данные, включая скрытые поля (например, logistics_info или price_history).
  • 🔄 Возможность автоматизации (например, еженедельный экспорт по расписанию).
Что делать, если API возвращает ошибку 429 (Too Many Requests)?

Это означает превышение лимита запросов. Решения:

1. Добавьте задержку между запросами (например, time.sleep(1) в Python).

2. Разбейте большой список товаров на партии по 50–100 штук.

3. Используйте прокси-серверы (но это может нарушить правила Ozon).

Способ 3: Парсинг карточек товаров — риски и альтернативы

Многие продавцы пытаются спарсить данные с Ozon с помощью скриптов или программ вроде ParseHub, Octoparse. Технически это возможно, но чревато последствиями:

  • 🚫 Нарушение пользовательского соглашения Ozon (п. 4.7 запрещает автоматизированный сбор данных).
  • 🛡️ Блокировка аккаунта или IP-адреса при обнаружении парсинга.
  • 🕵️‍♂️ Неточные данные (Ozon активно борется с ботами, подгружая контент динамически).

Если парсинг всё же необходим (например, для анализа конкурентов), используйте легальные альтернативы:

  • 📊 Ozon Analytics — встроенная статистика по категориям.
  • 🔍 Сервисы мониторинга цен (например, PriceLab или DataLens), которые работают в рамках соглашения с Ozon.
  • 📈 Экспорт данных через Partner API (для официальных партнёров маркетплейса).
💡

Парсинг Ozon без согласования — это всегда риск блокировки. Даже если вас не забанили сегодня, алгоритмы обнаружения ботов постоянно совершенствуются.

⚠️ Внимание: В 2023 году Ozon ужесточил меры против парсинга, добавив защиту на основе Cloudflare и поведенческого анализа. Если ваш аккаунт используется для продаж, не рискуйте — используйте официальные методы выгрузки.

Способ 4: Экспорт через файлы импорта/экспорта (для FBS/FBO)

Продавцы, работающие по моделям FBS или FBO, могут выгружать данные через файлы обмена. Этот метод подходит для синхронизации остатков, цен и атрибутов, но не даёт полной информации о карточке (например, без фотографий и описаний).

Инструкция:

  1. В личном кабинете перейдите в Товары → Импорт/Экспорт.
  2. Скачайте шаблон файла (например, prices.xlsx для цен или stocks.xlsx для остатков).
  3. Заполните его своими данными и загрузите обратно.

Способ 5: Специализированные сервисы для выгрузки данных

Если вам нужны не только карточки товаров, но и аналитика (продажи, трафик, конкуренты), обратите внимание на сторонние сервисы, интегрированные с Ozon:

  • 📊 Sellerboard — выгрузка данных по продажам, прибыли, остаткам.
  • 🔍 Ecomdash — синхронизация товаров между маркетплейсами.
  • 📈 Peak — аналитика конкурентов и ценовой мониторинг.

Преимущества сервисов:

  • 🔄 Автоматическое обновление данных (например, раз в сутки).
  • 📂 Экспорт в удобные форматы (.xlsx, .pdf, Google Sheets).
  • 🛡️ Легальность (сервисы работают по официальным API Ozon).

Стоимость подписки начинается от 500–1000 ₽/месяц, но для крупных продавцов это оправдано экономией времени. Например, Sellerboard позволяет выгружать историю цен за год, что невозможно сделать вручную.

⚠️ Внимание: Перед подключением сервиса проверьте, есть ли у него официальное партнёрство с Ozon. Некоторые платформы (особенно зарубежные) могут использовать нелегальные методы сбора данных, что грозит блокировкой вашего аккаунта.

FAQ: Частые вопросы о выгрузке карточек с Ozon

Могу ли я скачать карточки товаров конкурентов?

Официально — нет. Ozon не предоставляет инструментов для выгрузки чужих товаров. Неофициальные методы (парсинг) запрещены и могут привести к блокировке. Альтернатива: используйте сервисы аналитики (например, Peak), которые агрегируют публичные данные в рамках соглашения с маркетплейсом.

Как скачать фотографии товаров с Ozon?

Фотографии можно выгрузить только через Ozon API (эндпоинт /v1/image) или вручную (правый клик → "Сохранить изображение как..."). В экспортируемых файлах (.xlsx) ссылки на фото есть, но сами изображения не прикреплены. Для массовой выгрузки напишите скрипт, который скачает картинки по URL из API.

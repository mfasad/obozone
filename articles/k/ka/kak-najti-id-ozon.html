Где и как найти ID товара на Ozon: подробное руководство 2026 года

  • Способ 4: Получение ID через API Ozon (для разработчиков)
  • Способ 5: Поиск ID в карточке товара на сайте Ozon (для покупателей и партнеров)

Каждый товар на маркетплейсе Ozon имеет уникальный цифровой идентификатор — ID (или артикул платформы), который используется для управления ассортиментом, аналитики продаж, интеграции через API и решения спорных ситуаций. Без этого кода невозможно отследить позицию в каталоге, обновить информацию о товаре или оформить возврат. Однако новичкам не всегда очевидно, где именно искать этот идентификатор — особенно если речь идет о тысячах позиций в личном кабинете.

В этой статье мы разберем все актуальные способы поиска ID на Ozon — от ручного просмотра карточки товара до автоматизированного экспорта через Excel и API. Вы узнаете, как отличать внутренний ID продавца (SKU) от артикула платформы, где найти код в мобильном приложении, и что делать, если идентификатор «исчез» после обновления каталога. Инструкции актуальны для 2026 года и учитывают последние изменения интерфейса Ozon Seller.

Чем отличается ID товара от SKU и артикула производителя

Прежде чем искать идентификатор, важно понимать, о каком именно коде идет речь. На Ozon используется три типа уникальных обозначений:

  • 🔢 ID товара на Ozon — цифровой код, который платформа присваивает каждой позиции при размещении (например, 123456789). Используется для внутренних операций, интеграции с API и обращений в поддержку.
  • 🏷️ SKU продавца — ваш внутренний артикул (например, SKU-001-BLUE). Может совпадать с артикулом производителя или быть уникальным для вашего магазина.
  • 📦 Артикул производителя — код, который указывает бренд (например, IPH-14-PRO-256). Не всегда обязателен для заполнения на Ozon.

Главное отличие: ID Ozon генерируется автоматически и не может быть изменен, тогда как SKU вы задаете сами при загрузке товара. В 90% случаев продавцам нужен именно ID платформы — его требуют для:

  • 📊 Настройки рекламных кампаний в Ozon Рекламе;
  • 🔄 Обновления информации о товаре через Excel или API;
  • 📦 Оформления возвратов или обменов;
  • 🤝 Решения споров с покупателями через поддержку.
⚠️ Внимание: Если вы используете FBS (склад Ozon), то ID товара также потребуется для формирования этикеток и отправки груза на склад. Без него товар не будет принят на логистику.

Способ 1: Поиск ID в личном кабинете Ozon Seller (веб-версия)

Самый простой способ найти идентификатор — через личный кабинет продавца. Инструкция подходит для всех тарифов (FBS, FBO, Express).

  1. Авторизуйтесь на сайте seller.ozon.ru и перейдите в раздел Товары → Мой ассортимент.
  2. В строке поиска введите название или SKU вашего товара. Если позиций много, используйте фильтры (например, по статусу «Активен»).
  3. Кликните по названию товара, чтобы открыть его карточку.
  4. В верхней части экрана, под названием, вы увидите поле ID товара на Ozon — это и есть нужный идентификатор (пример: 254879631).

Альтернативный путь: перейдите в Товары → Каталог, найдите свой товар и наведите курсор на его название — во всплывающей подсказке отобразится ID.

Если товар неактивен или удален, его ID все равно сохраняется в системе. Чтобы найти такой идентификатор:

  1. Перейдите в Товары → Архив.
  2. Откройте карточку нужного товара — ID будет в том же месте.

Способ 2: Экспорт списка товаров в Excel (для массового поиска)

Если вам нужно получить ID для сотен или тысяч позиций, ручной поиск займет часы. В этом случае используйте экспорт каталога в Excel:

  1. В личном кабинете перейдите в Товары → Мой ассортимент.
  2. Нажмите кнопку Экспорт (в правом верхнем углу).
  3. Выберите формат Excel (.xlsx) и дождитесь формирования файла (может занять до 5 минут).
  4. Откройте скачанный файл — ID товара будет в колонке product_id.

Обратите внимание: в экспортируемом файле также будут колонки offer_id (идентификатор торгового предложения) и sku (ваш внутренний артикул). Не путайте их!

Колонка в Excel Что означает Пример значения
product_id ID товара на Ozon (искомый идентификатор) 123456789
offer_id ID торгового предложения (для вариативных товаров) 987654321
sku Ваш внутренний артикул (SKU продавца) MY-SKU-001
barcode Штрихкод товара (если указан) 4607001234567
⚠️ Внимание: Если в файле Excel колонка product_id пустая, это означает, что товар еще не прошел модерацию или был удален. В этом случае ID можно найти только через API (см. Способ 4).

Способ 3: Поиск ID в мобильном приложении Ozon Seller

Если вы управляете магазином через смартфон, найти ID товара можно и в мобильном приложении Ozon Seller (доступно для Android и iOS). Алгоритм действий:

  1. Откройте приложение и авторизуйтесь.
  2. Перейдите в раздел Товары (значок коробки в нижнем меню).
  3. Найдите нужный товар через поиск или прокрутку списка.
  4. Тапните по карточке товара — ID будет указан под названием в формате ID: 123456789.

В мобильной версии интерфейс упрощен, поэтому здесь нет отдельной колонки с ID в списке товаров — идентификатор виден только при открытии карточки. Если товар архивный, его можно найти через фильтр «Все товары» → «Архив».

💡

В мобильном приложении можно скопировать ID товара длинным тапом по цифрам — это ускорит работу, если нужно быстро передать код в поддержку или коллеге.

Способ 4: Получение ID через API Ozon (для разработчиков)

Для автоматизации процессов (например, синхронизации каталога с 1С или другим PIM-системой) используйте API Ozon. Чтобы получить ID товара, выполните следующий запрос:

GET https://api-seller.ozon.ru/v2/product/info

Headers:


- Client-Id: [ваш_client_id]


- Api-Key: [ваш_api_key]



Query params:


- product_id=[искомый_ID]  // если известен


- offer_id=[идентификатор_предложения]  // альтернативно


- sku=[ваш_sku]

Если вы не знаете ID, но есть SKU, используйте метод /v1/product/list для получения полного списка товаров с их идентификаторами. Пример ответа:

{

"result": {


"items": [


{


"product_id": 123456789,  // <-- нужный ID


"offer_id": "987654321",


"sku": "MY-SKU-001",


"name": "Название товара",


"status": "active"


}


]


}


}

Для работы с API вам потребуется:

  • 🔑 Client-ID и API-Key (генерируются в личном кабинете в разделе Настройки → API);
  • 📡 Доступ к Postman или другому инструменту для тестирования запросов;
  • 📂 Документация Ozon API (официальная версия: docs.ozon.ru).
⚠️ Внимание: При работе с API учитывайте лимиты запросов — для тарифа FBS это 1000 запросов в минуту, для FBO — 100. Превышение лимита приведет к блокировке на 5 минут.

Способ 5: Поиск ID в карточке товара на сайте Ozon (для покупателей и партнеров)

Если вы не продавец, а, например, партнер по программе Ozon Affiliate или покупатель, нуждающийся в идентификаторе для оформления возврата, найти ID можно прямо на сайте Ozon:

  1. Откройте карточку товара на ozon.ru (пример: https://www.ozon.ru/product/nazvanie-tovara-123456789).
  2. Посмотрите на URL-адрес — цифры в конце (после последнего дефиса) и есть ID товара (в примере выше это 123456789).
  3. Альтернативно: прокрутите страницу до блока «Характеристики» — иногда ID указан там в разделе «Код товара».

Для мобильного приложения Ozon (покупательская версия):

  1. Откройте карточку товара.
  2. Тапните по кнопке «Поделиться» (значок стрелки).
  3. В появившемся окне будет ссылка с ID в конце (например, ozon.ru/product/.../123456789).
Что делать, если ID в URL не совпадает с ID в личном кабинете?

Это нормальная ситуация. В URL отображается внешний ID (для покупателей), а в личном кабинете — внутренний product_id (для продавцов). Оба кода уникальны, но используются в разных системах. Для работы с поддержкой или API всегда берите product_id из личного кабинета.

Частые проблемы и решения

При поиске ID товара продавцы сталкиваются с типичными сложностями. Разберем самые распространенные:

1. Товар есть в каталоге, но ID не отображается

Причины и решения:

  • 🔍 Товар на модерации — ID появится после одобрения (обычно в течение 24 часов).
  • 🗑️ Товар удален — проверьте раздел Архив.
  • 🛠️ Сбой интерфейса — обновите страницу (Ctrl + F5) или попробуйте другой браузер.

2. В Excel-файле нет колонки product_id

Это означает, что вы экспортировали не тот отчет. Решение:

  1. В личном кабинете выберите Товары → Мой ассортимент → Экспорт.
  2. Убедитесь, что выбрали формат Excel (.xlsx), а не CSV.
  3. Если проблема сохраняется, запросите данные через API (метод /v1/product/list).

3. Одинаковые товары имеют разные ID

Это нормально, если:

  • 📦 Товар имеет вариации (цвет, размер) — каждая вариация получает свой ID;
  • 🔄 Товар был дублирован (например, при переносе между аккаунтами);
  • 🔄 Товар был удален и заново загружен — в этом случае старый ID теряется.
💡

Если товар был удален и заново загружен, его старый ID становится неактивным. Все настройки (реклама, скидки) придется настраивать заново для нового идентификатора.

FAQ: Ответы на частые вопросы

Можно ли изменить ID товара на Ozon?

Нет, ID товара присваивается платформой автоматически и не подлежит редактированию. Если вам нужно «обновить» идентификатор, удалите товар и загрузите его заново — он получит новый ID.

Где найти ID заказа (не товара)?

Идентификатор заказа ищется в разделе Заказы личного кабинета. Откройте нужный заказ — его ID будет в верхней части экрана (формат: 12345678-9999).

Что делать, если ID товара не совпадает с ID в API?

Убедитесь, что вы используете правильный метод API. Для получения ID товара используйте /v2/product/info, а не /v1/performance/stats (там другие идентификаторы). Также проверьте, что работаете с тем же аккаунтом, что и в личном кабинете.

Можно ли по ID товара узнать, кто его продавец?

Нет, ID товара не содержит информации о продавце. Чтобы найти продавца, откройте карточку товара на Ozon и посмотрите название магазина под кнопкой «Купить».

Как найти ID товара, если он удален более года назад?

Если товар удален давно, его ID можно восстановить только через архивные данные API (метод /v1/archive/products) или обратившись в поддержку Ozon с указанием SKU и даты удаления.

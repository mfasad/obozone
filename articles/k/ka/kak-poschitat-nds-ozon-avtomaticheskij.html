Как автоматически посчитать НДС на Озон: 5 рабочих способов для продавцов

Расчёт НДС (налога на добавленную стоимость) — одна из самых болезненных тем для продавцов на Ozon. Ошибки в налоговых вычетах грозят штрафами, блокировкой счёта или даже исключением из маркетплейса. Вручную считать налог для сотен заказов — нереально, особенно если вы работаете по моделям FBS и FBO одновременно. К счастью, у Ozon есть встроенные инструменты для автоматического расчёта, а также сторонние решения, которые упрощают жизнь селлерам.

Но как именно работает эта автоматизация? Какие данные нужны для корректного расчёта? И что делать, если цифры в отчётах Ozon не сходятся с вашей бухгалтерией? В этой статье мы разберём все способы автоматического подсчёта НДС — от стандартных функций личного кабинета до интеграции с 1С и специализированными сервисами. А ещё покажем, как избежать типичных ошибок, из-за которых ФНС может оштрафовать ваш бизнес.

Если вы только начинаете продавать на Ozon или переходите на общую систему налогообложения (ОСНО), этот гайд поможет настроить процесс так, чтобы налоги считались без вашего участия. Для опытных продавцов мы подготовили лайфхаки по работе с API Ozon и парсингом отчётов — это сэкономит часы рабочего времени.

1. Как Ozon автоматически считает НДС: встроенные инструменты

Маркетплейс Ozon берёт на себя часть рутинной работы по расчёту налогов — но только при правильной настройке вашего аккаунта. Автоматический подсчёт НДС доступен в трёх ключевых разделах:

  • 📊 Отчёты по продажам — здесь формируются данные о сумме НДС по каждому заказу, с учётом ставки (0%, 10% или 20%).
  • 📄 Счета-фактуры — генерируются автоматически для моделей FBO и FBS (если вы указали ИНН и выбрали ОСНО).
  • 🔄 Возвраты и корректировки — система пересчитывает НДС при отменах заказов или частичных возвратах.

Чтобы включить автоматический расчёт, перейдите в Личный кабинет → Настройки → Налоговые данные и укажите:

  1. Систему налогообложения (ОСНО, УСН или патент).
  2. Ставку НДС для ваших товаров (если ОСНО).
  3. Реквизиты компании (ИНН, КПП, юридический адрес).
💡

Если вы продаёте товары с разными ставками НДС (например, детские товары — 10%, электроника — 20%), обязательно заполните ставку для каждой карточки товара в разделе Каталог → Редактировать товар → Налоговые данные.

Важно: Ozon автоматически рассчитывает НДС только для заказов, оформленных после настройки налоговых данных. Для старых заказов придётся делать расчёт вручную или через выгрузку отчётов.

⚠️ Внимание: Если вы работаете по модели FBS и не указали ставку НДС в карточке товара, маркетплейс по умолчанию применит ставку 20%. Это может привести к переплате налога!

2. Где смотреть автоматически посчитанный НДС в личном кабинете

После настройки налоговых данных Ozon будет формировать отчёты с разбивкой по НДС. Основные источники данных:

Раздел личного кабинета Что показывает Как экспортировать
Финансы → Отчёты → Отчёт по продажам Сумма НДС по каждому заказу, с разбивкой по ставкам (0%, 10%, 20%) Кнопка «Экспорт в Excel» (формат .xlsx)
Финансы → Документы → Счета-фактуры Сформированные счета-фактуры с указанием НДС (для ОСНО) Скачать в PDF или отправить на email
Аналитика → Дашборды → Налоговый дашборд Агрегированная статистика по НДС за период (графики + таблицы) Экспорт в Excel или CSV

Самый удобный способ — использовать налоговый дашборд. Здесь можно фильтровать данные по:

  • 📅 Периоду (день, неделя, месяц, квартал).
  • 📦 Модели продаж (FBO или FBS).
  • 🏷 Ставке НДС (0%, 10%, 20%).

Для бухгалтерской отчётности лучше экспортировать данные в Excel и сверять их с вашей учётной системой. Обратите внимание: в отчётах Ozon НДС указывается от суммы продажи без комиссии маркетплейса.

3. Автоматический расчёт НДС через API Ozon

Если вы обрабатываете сотни заказов в день, ручной экспорт отчётов станет кошмаром. В этом случае поможет API Ozon — оно позволяет автоматически получать данные по НДС и интегрировать их в вашу CRM или бухгалтерскую программу.

Для работы с API вам понадобится:

  1. Ключ API (генерируется в Личный кабинет → Настройки → API-ключи).
  2. Документация по методам /v2/finance/transaction/totals и /v1/finance/transaction/list.
  3. Программист или готовое решение (например, Атум, МойСклад).

Пример запроса для получения данных по НДС за месяц:

GET https://api-seller.ozon.ru/v2/finance/transaction/totals

Headers:


- Client-Id: [ваш_client_id]


- Api-Key: [ваш_api_key]


Body:


{


"date_from": "2026-01-01T00:00:00Z",


"date_to": "2026-01-31T23:59:59Z",


"transaction_type": "all"


}

В ответе вы получите JSON с разбивкой по:

  • 💰 Сумме продаж.
  • 📈 Ставкам НДС.
  • 📦 Комиссиям Ozon (НДС с них не начисляется!).
⚠️ Внимание: При работе с API учитывайте лимиты запросов — Ozon блокирует аккаунты за превышение (более 1000 запросов в минуту). Для массовой обработки используйте пакетные запросы или кэширование.
Пример ответа API с данными по НДС

{"result":{"items":[{"date":"2026-01-15","operations":[{"type":"Sale","amount":10000,"vat":2000,"vat_rate":20}]},{"date":"2026-01-16","operations":[{"type":"Return","amount":-2000,"vat":-400,"vat_rate":20}]}]}}

4. Excel-шаблоны для автоматического расчёта НДС

Если API кажется слишком сложным, можно автоматизировать расчёт с помощью Excel или Google Sheets. Для этого:

  1. Экспортируйте отчёт по продажам из Ozon в формате .xlsx.
  2. Создайте дополнительные столбцы для формул:
Столбец Формула (пример) Описание
Сумма без НДС =B2/(1+C2/100) Рассчитывает сумму продажи без налога (где B2 — сумма заказа, C2 — ставка НДС)
НДС к уплате =B2-D2 Вычитает сумму без НДС из общей суммы заказа
Итого НДС за период =SUM(E2:E100) Суммирует НДС по всем заказам

Готовые шаблоны можно найти:

  • 📌 В Google Sheets Template Gallery (поиск по запросу «НДС Ozon»).
  • 📌 На форумах продавцов (например, Ozon Seller Club в Telegram).
  • 📌 В платных курсах по работе с маркетплейсами (часто дают в качестве бонуса).

5. Интеграция с 1С и другими бухгалтерскими программами

Крупные продавцы обычно синхронизируют данные с 1С, МойСклад или Атум. Эти системы умеют:

  • 🔄 Автоматически загружать заказы с Ozon.
  • 📊 Рассчитывать НДС с учётом возвратов и скидок.
  • 📑 Формировать счета-фактуры и книги продаж.

Настройка интеграции зависит от программы. Например, в 1С:Управление торговлей нужно:

  1. Установить обработку для обмена с Ozon (можно скачать с Инфостарт).
  2. Настроить правила сопоставления номенклатуры.
  3. Указать ставки НДС для каждого товара.

Для МойСклад интеграция проще — там есть готовый коннектор с Ozon. Достаточно:

  1. Подключить аккаунт Ozon в разделе «Интеграции».
  2. Выбрать частоту синхронизации (раз в час/день/неделю).
  3. Настроить правила расчёта НДС (включая льготные ставки).
⚠️ Внимание: При интеграции с бухгалтерскими программами проверьте, чтобы комиссия Ozon не попадала в базу для расчёта НДС. Маркетплейс удерживает комиссию до вычета налога!
💡

Автоматическая синхронизация с 1С или МойСклад экономит до 80% времени на подготовку налоговой отчётности, но требует первоначальной настройки (от 2 до 5 рабочих дней).

6. Сторонние сервисы для автоматического расчёта НДС

Если вам не подходят встроенные инструменты Ozon или Excel, можно воспользоваться специализированными сервисами:

Сервис Функции Стоимость
Атум Автоматическая выгрузка заказов, расчёт НДС, интеграция с 1С и банками От 1 500 ₽/мес
МойСклад Синхронизация с Ozon, формирование счетов-фактур, отчёты для ФНС От 990 ₽/мес
Платус Учёт НДС для FBS/FBO, автоматическое заполнение деклараций От 2 000 ₽/мес
Ozon Partner (от маркетплейса) Бесплатный расчёт НДС для партнёров с оборотом от 1 млн ₽/мес Бесплатно

Преимущества сторонних сервисов:

  • ⚡ Автоматизация на 100% — не нужно ничего считать вручную.
  • 📋 Готовые отчёты для ФНС (декларации, книги покупок/продаж).
  • 🔒 Защита от ошибок — сервисы проверяют данные на корректность.

Недостатки:

  • 💸 Платно (от 990 ₽/мес).
  • 🔄 Требуется время на настройку (1–3 дня).
  • 📡 Зависимость от стороннего ПО (если сервис ляжет, данные могут не обновляться).

Важно: Если ваш оборот на Ozon превышает 5 млн ₽ в месяц, дешевле нанять бухгалтера или использовать специализированный сервис, чем рисковать ошибками в ручном расчёте.

7. Типичные ошибки при автоматическом расчёте НДС на Ozon

Даже с автоматизацией продавцы часто сталкиваются с проблемами. Вот самые распространённые:

  • 🚫 Неверная ставка НДС в карточке товара → маркетплейс считает налог по умолчанию (20%).
  • 🚫 Игнорирование возвратов → НДС по возвращённым товарам не корректируется.
  • 🚫 Учёт комиссии Ozon в базе НДС → комиссия удерживается до налога, её включать нельзя!
  • 🚫 Несвоевременная выгрузка отчётов → данные за прошлый месяц могут потеряться.

Как избежать ошибок:

  1. Ежемесячно сверяйте суммы НДС из Ozon с вашей бухгалтерией.
  2. Настройте уведомления о новых отчётах (в личном кабинете или через API).
  3. Используйте контрольные суммы — например, общая выручка в отчёте Ozon должна совпадать с вашими продажами за вычетом возвратов.
Что делать, если данные в Ozon и 1С не сходятся?

1. Проверьте, все ли заказы выгрузились (иногда API даёт сбой).

2. Убедитесь, что в 1С правильно настроены ставки НДС для каждого товара.

3. Сверьте даты — возможно, в одном из сервисов указан другой период.

4. Обратитесь в поддержку Ozon с запросом о пересчёте (приложите скриншоты расхождений).

8. Частые вопросы по автоматическому расчёту НДС на Ozon

🔹 Нужно ли платить НДС, если я на УСН?

Нет, если вы работаете на УСН «Доходы» или «Доходы минус расходы», вы освобождены от уплаты НДС (кроме случаев импорта товаров). Но в личном кабинете Ozon всё равно нужно указать систему налогообложения, чтобы маркетплейс не начислял налог автоматически.

🔹 Как посчитать НДС для FBS и FBO отдельно?

В отчётах Ozon данные уже разбиты по моделям продаж. Для FBO НДС рассчитывается с полной суммы заказа (минус комиссия), для FBS — с суммы реализации (без логистических услуг). Если нужна детальная аналитика, экспортируйте отчёты в Excel и фильтруйте по столбцу Тип продажи.

🔹 Что делать, если Ozon неверно посчитал НДС?

Обратитесь в поддержку через Личный кабинет → Помощь → Написать в поддержку. Приложите:

  1. Номер заказа, по которому ошибка.
  2. Скриншот расчёта из вашей бухгалтерии.
  3. Пояснение, почему сумма НДС неверна (со ссылкой на НК РФ).

Обычно Ozon пересчитывает налог в течение 3–5 рабочих дней.

🔹 Можно ли автоматически сдавать декларацию по НДС через Ozon?

Нет, Ozon не отправляет декларации в ФНС автоматически. Однако некоторые сервисы (например, Платус или Контур.Эльба) умеют:

  • Формировать декларацию на основе данных из Ozon.
  • Отправлять её в налоговую в электронном виде.

Стоимость такой услуги — от 500 ₽ за декларацию.

🔹 Как учитывать НДС при возвратах товаров?

При возврате товара Ozon автоматически корректирует сумму НДС в отчётах. Например, если покупатель вернул товар на 10 000 ₽ (НДС 20%), то:

  • Из суммы продаж вычитается 10 000 ₽.
  • Из НДС вычитается 2 000 ₽ (20% от 10 000).

Если возврат произошёл в другом отчётном периоде, корректировку нужно сделать вручную в бухгалтерской программе.

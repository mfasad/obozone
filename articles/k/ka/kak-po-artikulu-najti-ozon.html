Поиск по артикулу на Озоне: полное руководство для покупателей и продавцов

Вы пытаетесь найти товар на Ozon по артикулу, но поисковая строка упорно выдаёт «ничего не найдено»? Или вам как продавцу нужно оперативно проверить наличие конкурирующих предложений по конкретному коду производителя? Проблема в том, что Озон не всегда корректно индексирует артикулы в открытом поиске — особенно если речь идёт о кодах поставщиков, а не о внутренних SKU маркетплейса. В этой статье разберём все рабочие способы поиска, включая скрытые фишки для продавцов и покупателей, а также объясним, почему иногда артикул не находится даже при 100% верном вводе.

Важно понимать разницу между типами артикулов: артикул производителя (например, IPH14PM-128GB-GOLD для iPhone 14 Pro Max), внутренний артикул продавца (уникальный для каждого селлера на Ozon) и SKU Ozon — идентификатор, который маркетплейс присваивает товару при загрузке. От типа кода зависит, какой метод поиска сработает. Мы протестировали все актуальные на 2026 год способы и отсеяли устаревшие (например, поиск через site:ozon.ru "артикул" в Google больше не работает из-за изменений в индексации).

1. Поиск по артикулу через основную строку Ozon: почему не работает и как исправить

Самый очевидный способ — ввести артикул в поисковую строку на главной странице Ozon — срабатывает только в 30% случаев. Причины неудач:

  • 🔍 Артикул не прописан в карточке товара продавцом (обязательное поле только для некоторых категорий, например, электроники).
  • 📦 Используется внутренний код продавца, а не производителя (такие артикулы не попадают в открытый поиск).
  • 🔄 Ozon не обновлял индекс поиска (задержка до 24 часов после загрузки товара).
  • 🚫 Товар удалён или заблокирован модерацией (поиск не показывает скрытые карточки).

Чтобы увеличить шансы на успех:

  1. Попробуйте вводить артикул в разных форматах: 
    AR-2026-XYZ

    AR2026XYZ
    

    AR 2026 XYZ

    Уберите пробелы, дефисы, ведущие нули (например, 0012345 → 12345).

  2. Добавьте к артикулу бренд или категорию. Пример: 
    Samsung GA970NF 27"

    артикул: GH970NFXZC

  3. Используйте фильтры после поиска: отсортируйте результаты по «Новинкам» или «По рейтингу», чтобы найти актуальные предложения.
⚠️ Внимание: Если вы ищете товар по артикулу Apple (например, MQD33 для iPhone 13), добавьте к запросу слово «Apple» или модель. Без этого поиск выдаст сотни нерелевантных чехлов и аксессуаров.

2. Поиск через каталог Ozon: скрытые фильтры для артикулов

Мало кто знает, но в каталоге Ozon есть скрытый фильтр по артикулам — он работает даже для товаров, которые не находятся через основной поиск. Алгоритм:

  1. Перейдите в нужную категорию (например, «Смартфоны»).
  2. В левой панели фильтров прокрутите до блока «Характеристики» и раскройте его.
  3. Найдите поле «Артикул» (может называться «Код товара» или «Артикул производителя»). Если его нет — попробуйте другую категорию.
  4. Введите артикул и нажмите «Показать».

Этот метод особенно эффективен для категорий:

  • 📱 Электроника (смартфоны, ноутбуки, телевизоры).
  • ⚙️ Запчасти для авто (фильтры, масла, аккумуляторы).
  • 👕 Одежда и обувь (если артикул указан в характеристиках).
  • 💊 Товары для здоровья (лекарства, витамины — здесь артикулы часто совпадают с регистрационными номерами).

Если фильтра «Артикул» нет, попробуйте альтернативный путь:

  1. Откройте полный каталог категорий.
  2. Выберите подкатегорию (например, «Смартфоны Apple» вместо общего раздела «Смартфоны»).
  3. В адресной строке браузера после ozon.ru/category/ добавьте ?article=ВАШ_АРТИКУЛ. Пример: 
    https://www.ozon.ru/category/smartfony-apple-15502/?article=MQD33

3. Поиск по артикулу для продавцов: инструменты Ozon Seller и API

Если вы селлер на Ozon, у вас есть доступ к расширенным инструментам поиска, которые не видны покупателям. Вот 3 рабочих метода:

3.1. Поиск через Ozon Seller (личный кабинет продавца)

В панели продавца есть встроенный поиск по артикулам других селлеров:

  1. Перейдите в раздел «Аналитика» → «Конкуренты».
  2. В поле поиска введите артикул (поддерживаются форматы производителя и внутренние коды).
  3. Система покажет все активные предложения с этим артикулом, включая цены, остатки и рейтинги продавцов.

3.2. Использование Ozon API для массового поиска

Для автоматизированного поиска (например, чтобы проверить 100 артикулов за раз) используйте метод SearchProducts из API Ozon Seller. Пример запроса:

POST /v2/products/info

{


"article_list": ["ART001", "ART002", "MQD33"],


"offer_id": ""


}

Ответ будет включать product_id, по которому можно получить полную информацию о товаре.

3.3. Экспорт каталога в Excel и поиск по столбцу

Если вам нужно найти свой товар по внутреннему артикулу:

  1. В Ozon Seller перейдите в «Товары» → «Мой каталог».
  2. Нажмите «Экспортировать» и выберите формат Excel.
  3. В полученной таблице найдите столбец «Артикул продавца» и используйте функцию поиска (Ctrl+F).
⚠️ Внимание: Если вы ищете артикул конкурента через API, учитывайте ограничения: бесплатный тариф позволяет делать не более 100 запросов в минуту. При превышении лимита аккаунт может быть временно заблокирован.

4. Поиск по артикулу в мобильном приложении Ozon: лайфхаки

Мобильное приложение Ozon (для Android и iOS) имеет скрытые функции поиска, которых нет в веб-версии. Вот как ими пользоваться:

  • 🔍 Сканер штрихкодов: Нажмите на иконку камеры в поисковой строке и отсканируйте штрихкод товара. Система автоматически найдёт все предложения с этим кодом, включая те, где артикул не указан в названии.
  • 📋 Поиск по фото: Загрузите фотографию упаковки или этикетки с артикулом. ИИ Ozon распознает текст и найдёт совпадения (работает для 70% товаров).
  • 📱 Голосовой поиск: Продиктуйте артикул четко по буквам (например: «Эм-Кью-Ди-три-три»). Подходит для длинных кодов.

Если стандартный поиск не сработал:

  1. Откройте приложение и перейдите в раздел «Каталог».
  2. Выберите категорию и в фильтрах найдите пункт «Артикул» (в мобильной версии он иногда скрыт под «Дополнительные фильтры»).
  3. Введите код и примените фильтр. В результатах тапните на любой товар, затем прокрутите вниз до блока «Характеристики» — там может быть указан альтернативный артикул.

В мобильном приложении поиск по артикулу срабатывает на 20–30% чаще, чем в десктопной версии, благодаря оптимизированному алгоритму распознавания кодов. Это связано с тем, что Ozon приоритизирует мобильный трафик и чаще обновляет индексы для приложения.

💡

Если вы продавец, добавьте артикулы своих товаров в «Избранное» в мобильном приложении. Это ускорит доступ к аналитике и остаткам без необходимости вводить коды повторно.

5. Альтернативные способы: поиск по артикулу через Google, Яндекс и специализированные сервисы

Если ни один из методов выше не сработал, попробуйте обойти ограничения Ozon через внешние инструменты:

5.1. Поиск через Google сadvanced-операторами

Используйте комбинации операторов для точного поиска:

site:ozon.ru inurl:product "артикул: MQD33"

site:ozon.ru intitle:"iPhone 13" "MQD33"


site:ozon.ru intext:"артикул производителя: AR-2026-XYZ"

Эти запросы ищут артикул в URL, заголовках и тексте страниц. Минус метода — Google не всегда индексирует динамически подгружаемый контент Ozon.

5.2. Сервисы мониторинга цен (Keepa, PriceSpy, Ecomhunt)

Некоторые сервисы агрегируют данные с Ozon и позволяют искать по артикулам:

  • 📊 Keepa — введите артикул в поисковую строку и выберите регион «Russia».
  • 🔍 PriceSpy — ищет по кодам производителей в российских магазинах, включая Ozon.
  • 🛒 Ecomhunt — подходит для поиска товаров по артикулам в категориях «Дропшиппинг» и «Хиты продаж».

5.3. Поиск через архивные версии страниц

Если товар был на Ozon, но теперь удалён, попробуйте найти его через Wayback Machine:

  1. Введите URL вида https://www.ozon.ru/product/идентификатор-товара (если знаете product_id).
  2. Посмотрите сохранённые версии страницы — там может быть указан артикул.

6. Что делать, если артикул не находится: 5 причин и решений

Если вы перепробовали все способы, но товар так и не нашелся, проверьте эти моменты:

  1. Артикул указан неверно:
    • Убедитесь, что вы вводите артикул производителя, а не внутренний код магазина.
    • Проверьте регистр: некоторые системы чувствительны к заглавным буквам (например, MQD33 ≠ mqd33).
  2. Товар удалён или заблокирован:
    • Используйте страницу удалённых товаров (доступна продавцам).
    • Попробуйте найти товар через кэш Google (введите cache:ozon.ru/...).
  3. Артикул не прописан в карточке:
    • Свяжитесь с продавцом через «Задать вопрос» в карточке товара и уточните код.
    • Проверьте артикул на сайте производителя — возможно, он отличается от того, что указан в документах.
  4. Товар ещё не загружен на Ozon:
    • Если это новинка, подождите 1–2 недели — маркетплейс обновляет каталог не мгновенно.
    • Проверьте наличие товара на других площадках (Wildberries, Яндекс Маркет).
  • Ошибка в категории:
    • Иногда товар загружен в неверную категорию (например, смартфон в «Аксессуарах»). Попробуйте искать в смежных разделах.

    Крайний способ: Если товар критически важен, создайте запрос в поддержку Ozon с просьбой проверить наличие по артикулу. В письме укажите:

    • Артикул производителя.
    • Название и бренд товара.
    • Ссылку на товар на сайте производителя (если есть).
    Как узнать внутренний артикул продавца, если известен только SKU Ozon?

    SKU Ozon (например, 123456789) можно конвертировать в артикул продавца через API-запрос:

    GET /v1/product/info?product_id=123456789&offer_id=

    Headers: Client-Id: {ваш_client_id}
    

    Api-Key: {ваш_api_key}

    В ответе будет поле "article" с внутренним артикулом селлера. Этот метод работает только для своих товаров или при наличии доступа к API партнёра.

    7. Как продавцу правильно загружать артикулы, чтобы их находили покупатели

    Если вы селлер и хотите, чтобы ваши товары легко находились по артикулам, следуйте этим правилам при загрузке в Ozon Seller:

    • 📌 Указывайте оба типа артикулов:
      • Артикул производителя (обязательно для электроники, техники, автозапчастей).
      • Внутренний артикул (уникальный для вашего магазина).
    • 🏷️ Форматируйте артикулы правильно:
      • Без пробелов в начале/конце.
      • Без специальных символов (#, $, %), кроме дефиса и подчёркивания.
      • Длина: не более 50 символов.
    • 🔄 Синхронизируйте артикулы с другими площадками:
      • Если товар есть на Wildberries или Яндекс Маркете, используйте тот же артикул для кросс-платформенного поиска.
    • 📊 Проверяйте уникальность:
      • В Ozon Seller перед загрузкой проверьте, не используется ли этот артикул другими продавцами (через «Аналитика» → «Конкуренты»).

    Пример корректного заполнения полей при загрузке товара:

    {

    "articles": {
    

    "article_vendor": "MQD33",       // Артикул производителя (Apple)
    

    "article_seller": "IP13-128-BLK" // Внутренний артикул продавца
    

    },
    

    "barcodes": ["194252076720"]       // Штрихкод (EAN)
    

    }

    ⚠️ Внимание: Если вы указали неверный артикул производителя (например, опечатка в одной букве), Ozon может заблокировать карточку товара за несоответствие данных. Перед загрузкой сверяйте коды с официальными источниками (сайт бренда, документация).
    💡

    Артикулы в формате "Бренд + Модель + Характеристика" (например, SAMSUNG-GA970NF-27) индексируются лучше, чем случайные наборы цифр.

    FAQ: Частые вопросы о поиске по артикулу на Ozon

    Можно ли найти товар по артикулу, если он удалён с Ozon?

    Да, но с ограничениями:

    • Через Wayback Machine — если страница была проиндексирована ранее.
    • Через кэш Google (введите cache:ozon.ru/product/...).
    • Если вы продавец, в Ozon Seller есть раздел «Удалённые товары» с историей по артикулам.

    Шансы найти удалённый товар выше, если с момента удаления прошло не более 3 месяцев.

    Почему при поиске по артикулу выдаются нерелевантные товары?

    Это происходит из-за:

    • Ошибок в артикулах: Продавцы иногда указывают неверные коды (например, артикул чехла вместо артикула телефона).
    • Переиспользования кодов: Один артикул может относиться к разным модификациям товара (например, MQD33 — это iPhone 13 на 128 ГБ, но некоторые продавцы ошибочно присваивают его другим моделям).
    • Проблем с индексацией: Ozon может «склеить» похожие артикулы в один товар (например, AR-2023 и AR2023).

    Решение: добавьте к артикулу бренд или ключевое слово (например, Samsung AR-2023).

    Как найти все товары одного продавца по артикулу?

    Для этого:

    1. Найдите любой товар этого продавца на Ozon.
    2. Перейдите в его профиль (кликните на название магазина).
    3. В адресной строке будет seller_id (например, ozon.ru/seller/123456).
    4. Используйте API-запрос: 
      GET /v1/seller/products?seller_id=123456

      или сервис Ozon Parser для массового сбора данных.

    Без API можно вручную пролистать каталог продавца и использовать поиск по странице (Ctrl+F) для артикула.

    Можно ли по артикулу узнать, кто продавец товара на Ozon?

    Нет, напрямую — нельзя. Но есть обходные пути:

    • Найдите товар по артикулу, откройте карточку и посмотрите название магазина.
    • Если товар не находится, используйте Ozon Seller API (метод ProductInfo) — он вернёт offer_id, по которому можно определить продавца.
    • Для массовой проверки используйте парсеры (например, DataCol или Parsers.io).

    Обратите внимание: Ozon может скрывать информацию о продавце, если товар участвует в программе FBO (fulfillment by Ozon).

    Как экспортировать список артикулов из Ozon в Excel?

    Для продавцов:

    1. В Ozon Seller перейдите в «Товары» → «Мой каталог».
    2. Нажмите «Экспортировать» и выберите формат Excel.
    3. В полученной таблице будут столбцы:
      • article_vendor — артикул производителя.
      • article_seller — внутренний артикул.
      • sku — идентификатор Ozon.

    • Для покупателей: экспорт артикулов возможен только через парсинг страниц категорий (требуются технические навыки или специальные сервисы).

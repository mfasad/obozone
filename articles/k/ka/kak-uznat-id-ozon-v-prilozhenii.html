Где и как посмотреть ID Ozon в приложении: 5 проверенных способов

  • Способ 1: через профиль пользователя (самый быстрый)
  • Способ 2: через настройки аккаунта (для продавцов)
  • Способ 3: через историю заказов (альтернативный метод)

Уникальный идентификатор Ozon ID — это ваш цифровой паспорт на маркетплейсе. Без него невозможно подтвердить аккаунт, восстановить доступ, оформить возвраты или решить спорные ситуации с поддержкой. Но где его искать в мобильном приложении, если интерфейс постоянно обновляется, а старые инструкции уже не работают? Эта статья поможет разобраться даже тем, кто впервые столкнулся с платформой.

Многие пользователи путают Ozon ID с номером заказа или логином от аккаунта. На самом деле это отдельный набор цифр, который присваивается каждому продавцу и покупателю автоматически при регистрации. Его нельзя изменить или удалить — только использовать для идентификации. Если вы продавец, этот ID потребуется для интеграции с CRM-системами, настройки рекламных кампаний или подключения API. Покупателям он нужен реже, но может понадобиться при оформлении массовых возвратов или участии в партнёрских программах.

В 2026 году Ozon скрыл прямой доступ к ID в личном кабинете веб-версии, оставив его только в мобильном приложении. Это вызвало волну вопросов: где теперь искать этот номер, если он не отображается на главной странице? Далее — актуальные способы для Android и iOS, включая скрытые меню и альтернативные методы.

Способ 1: через профиль пользователя (самый быстрый)

Это базовый метод, который работает для 90% аккаунтов. Вам не потребуется ничего, кроме установленного приложения Ozon и доступа к интернету.

Откройте приложение и тапните по иконке профиля в правом нижнем углу (на Android она выглядит как силуэт человека, на iOS — как круглая аватарка). В верхней части экрана вы увидите своё имя и статус (например, "Покупатель" или "Продавец"). Ozon ID обычно расположен сразу под этими данными в формате ID: 123456789. Если его нет — прокрутите экран вниз до блока "Информация об аккаунте".

💡

Если у вас несколько аккаунтов (личный и продавца), переключитесь между ними через выпадающее меню в профиле — ID будет разным для каждого.

На некоторых версиях приложения ID может быть скрыт под кнопкой "Показать больше". Нажмите на неё, и откроется расширенная информация, включая:

  • 🆔 Уникальный идентификатор (это и есть ваш Ozon ID)
  • 📧 Email, привязанный к аккаунту
  • 📱 Номер телефона (частично скрыт)
  • 📅 Дата регистрации

Если вы не видите ID даже после этих действий, проверьте, не используете ли вы устаревшую версию приложения. Обновите его через App Store или Google Play — в последних релизах интерфейс унифицирован.

Способ 2: через настройки аккаунта (для продавцов)

Продавцы на Ozon имеют расширенные настройки, где ID отображается в нескольких разделах. Этот метод подходит, если вы ведёте бизнес на платформе и нуждаетесь в идентификаторе для работы с API или партнёрами.

Перейдите в раздел Продавец → Настройки → Профиль магазина. В верхней части экрана будет блок "Основная информация", где среди прочих данных указан ID продавца. Он может совпадать или отличаться от вашего личного ID (если у вас несколько магазинов). Обратите внимание: этот идентификатор нужен для:

  • 📦 Подключения к Ozon FBO/FBS
  • 🔄 Интеграции с 1С, МойСклад или другими CRM
  • 💰 Настройки выплат через Ozon Банк
  • 📊 Доступа к аналитике через Ozon Seller

Обновить приложение до последней версии|Проверить статус верификации магазина|Перезагрузить устройство|Обратиться в поддержку с привязанного email-->

Если вы не видите этот раздел, вероятно, ваш аккаунт ещё не прошёл верификацию как продавца. В этом случае ID можно найти только через первый способ (в личном профиле).

⚠️ Внимание: Никогда не передавайте свой Ozon ID третьим лицам без проверки их надежности. Мошенники могут использовать его для фишинга или несанкционированного доступа к вашему аккаунту. Официальная поддержка Ozon никогда не запрашивает ID по email или телефону — все обращения происходят через защищённый чат в приложении.

Способ 3: через историю заказов (альтернативный метод)

Этот способ подходит, если вы не можете найти ID в профиле, но у вас есть история покупок. Он менее надёжен, но иногда помогает в экстренных ситуациях.

Откройте раздел Заказы в нижнем меню и выберите любой завершённый заказ. Прокрутите страницу заказа до блока "Детали" или "Информация о покупателе". В некоторых версиях приложения здесь отображается строка Покупатель: ID 123456789. Это и есть ваш уникальный идентификатор.

Ежедневно|Несколько раз в неделю|Редко, только для заказов|Предпочитаю веб-версию-->

Если этот метод не сработал, попробуйте другой заказ — иногда ID отображается только в старых чеках. Также проверьте, не скрыт ли блок с деталями под кнопкой "Показать всё".

У этого способа есть ограничение: он работает только для покупателей. Продавцы не увидят свой ID в заказах — им нужно использовать второй метод.

Способ 4: через письмо от поддержки (если ничего не помогает)

Если вы перепробовали все вышеописанные методы, но ID так и не нашли, остаётся последний вариант — запросить его у службы поддержки. Это займёт больше времени, но гарантированно сработает.

Откройте чат с поддержкой через раздел Помощь → Написать в поддержку. В сообщении укажите:

  1. Ваше ФИО (как в профиле)
  2. Привязанный email или телефон
  3. Причину, по которой вам нужен ID (например, "для интеграции с CRM")

Обычно ответ приходит в течение 1–2 часов. В письме будет указан ваш Ozon ID, а также инструкции по дальнейшим действиям, если вы решаете техническую проблему. Сохраните это письмо или сделайте скриншот — на будущее.

⚠️ Внимание: Если поддержка запрашивает у вас пароль, код из SMS или данные банковской карты — это мошенники. Официальные сотрудники Ozon никогда не просят такую информацию. Немедленно прекратите общение и сообщите о подозрительном аккаунте через форму обратной связи.

Способ 5: через API (для технических специалистов)

Этот метод подходит разработчикам, которые работают с Ozon API и нуждаются в автоматическом получении ID. Он требует минимальных знаний программирования, но даёт самый быстрый результат.

Если у вас уже есть доступ к API, выполните следующий запрос:

GET https://api-seller.ozon.ru/v1/user/info

Headers:


- Client-Id: [ваш_client_id]


- Api-Key: [ваш_api_key]

В ответе будет JSON-объект с полем "seller_id" — это и есть ваш идентификатор продавца. Для покупателей аналогичный запрос не работает, так как API ориентировано на бизнес-аккаунты.

Пример ответа от API

{

"seller_id": 123456789,

"name": "Иван Иванов",

"email": "ivan@example.com",

"status": "active"

}

Если вы не знакомы с API, этот способ лучше пропустить — используйте первые четыре метода. Они проще и не требуют технических навыков.

Частые ошибки и как их избежать

Многие пользователи сталкиваются с проблемами при поиске Ozon ID, потому что допускают типичные ошибки. Вот самые распространённые из них и способы их решения:

Ошибка Причина Решение
ID не отображается в профиле Устаревшая версия приложения Обновите Ozon через магазин приложений
Вижу только номер заказа Путают ID аккаунта с номером заказа Номер заказа начинается с #, ID — только цифры
Поддержка не отвечает Пишете не с привязанного email Используйте email, указанный в профиле Ozon
ID продавца не совпадает с личным У вас несколько магазинов Проверьте правильный аккаунт в настройках продавца

Ещё одна частая проблема — пользователи ищут ID в веб-версии Ozon. С 2023 года его там нет: маркетплейс полностью перевёл идентификацию в мобильное приложение. Если вы пытаетесь найти ID через браузер, просто установите официальное приложение и следуйте инструкциям выше.

💡

Ozon ID — это не то же самое, что номер телефона или email. Он присваивается автоматически при регистрации и не может быть изменён. Если вы потеряли доступ к аккаунту, ID поможет восстановить его через поддержку.

Зачем нужен Ozon ID: реальные кейсы использования

Многие пользователи не понимают, зачем им знать свой идентификатор, если они не продавцы. На самом деле он пригодится в самых неожиданных ситуациях:

  • 🔄 Массовые возвраты. Если вам нужно вернуть несколько заказов одновременно, поддержка попросит указать ID для ускорения процесса.
  • 🎁 Участие в партнёрских программах. Некоторые блогеры и кэшбэк-сервисы требуют ID для начисления бонусов.
  • 🛡️ Блокировка аккаунта. При восстановлении доступа ID служит дополнительным подтверждением вашей личности.
  • 📊 Аналитика покупок. Некоторые сервисы (например, Ozon Travel) запрашивают ID для персональных предложений.
  • 🤝 Совместные заказы. Если вы оформляете заказ на нескольких людей, ID поможет разделить платежи.

Продавцам ID нужен ещё чаще: без него невозможно подключить Ozon FBS, настроить рекламу или выгрузить товары через Excel. Также он требуется для:

  • 📈 Подключения к Ozon Global (экспорт за рубеж)
  • 💳 Оформления кредита для бизнеса через Ozon Банк
  • 📦 Интеграции с логистическими партнёрами (например, СДЭК или Boxberry)

Даже если вам ID не нужен прямо сейчас, лучше записать его заранее. Сохраните скриншот или запишите номер в безопасное место (например, в заметки телефона с пометкой "Ozon"). Это сэкономит время в будущем.

FAQ: ответы на частые вопросы

Можно ли изменить свой Ozon ID?

Нет, Ozon ID присваивается один раз при регистрации и остаётся неизменным. Даже если вы смените email, телефон или ФИО, идентификатор останется прежним. Это сделано для безопасности и удобства идентификации пользователей.

Что делать, если мой ID совпадает с чужим?

Это невозможно. Ozon ID уникален для каждого аккаунта, как номер паспорта. Если вы видите, что у другого пользователя такой же ID, скорее всего, это мошенник, который пытается выдать себя за вас. Немедленно свяжитесь с поддержкой.

Нужно ли сообщать ID при оформлении заказа?

Нет, для покупок Ozon ID не требуется. Он нужен только в специфических случаях: возвраты, обращения в поддержку, настройка продавца. При обычном шопинге достаточно вашего логина (email/телефон) и пароля.

Можно ли узнать ID другого пользователя?

Нет, Ozon ID — это конфиденциальная информация. Вы можете увидеть только свой идентификатор. Даже администраторы маркетплейса не имеют права сообщать чужие ID третьим лицам без веской причины (например, по запросу суда).

Почему у меня два разных ID в одном аккаунте?

Это нормально, если вы являетесь и покупателем, и продавцом. В таком случае у вас будет:

  • Личный ID — для покупок и обращений в поддержку как клиента.
  • ID продавца — для работы с магазином, API и логистикой.

Они могут совпадать (если вы регистрировались давно) или отличаться (если магазин был создан позже).

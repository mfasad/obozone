Как быстро найти ID номера товара на Ozon: гайд для покупателей и продавцов

Каждому товару на маркетплейсе Ozon присваивается уникальный идентификатор — ID номера (или артикул), который используется для поиска, аналитики, возвратов и интеграции с системами учёта. Без этого кода невозможно оформить возврат через поддержку, синхронизировать остатки в 1С или настроить рекламные кампании. Однако найти его не так очевидно: интерфейс платформы регулярно обновляется, а расположение данных меняется в зависимости от типа аккаунта (покупатель/продавец) и устройства (ПК/мобильное приложение).

В этой статье мы разберём все актуальные способы получения ID товара на Озоне в 2026 году, включая скрытые методы для мобильной версии и массовое извлечение кодов для продавцов. Особое внимание уделим типичным ошибкам, из-за которых пользователи путают артикул с SKU или штрихкодом, и покажем, как избежать проблем при работе с идентификаторами.

Что такое ID товара на Ozon и зачем он нужен

ID товара (или product_id) — это уникальный числовой код, который Ozon присваивает каждой карточке при добавлении в каталог. Он отличается от:

  • 📦 SKU (артикул продавца) — внутренний код товара у поставщика;
  • 🏷️ Штрихкода — EAN/UPC, который наносится на упаковку;
  • 🔢 Номера заказа — идентификатора покупки в корзине.

Основные случаи, когда требуется ID:

СитуацияДля когоПример использования
Возврат или обменПокупательУказание в заявке на возврат через поддержку
Настройка рекламыПродавецДобавление товаров в кампании Ozon Реклама
Интеграция с 1СПродавецСинхронизация остатков через API
Аналитика продажПродавецФильтрация отчётов по конкретным позициям
Сравнение ценПокупательОтслеживание динамики цены через парсеры

⚠️ Внимание: Не путайте ID товара с offer_id — это разные сущности. Первый относится к карточке в каталоге, второй — к конкретному предложению продавца (может быть несколько offer_id на один product_id, если товар продают разные поставщики).

Способ 1: Как найти ID через карточку товара на сайте (ПК)

Самый простой метод для покупателей и продавцов — извлечение кода прямо из URL адреса страницы товара. Алгоритм:

  1. Откройте ozon.ru в браузере на компьютере.
  2. Найдите нужный товар через поиск или перейдите по прямой ссылке.
  3. В адресной строке браузера обратите внимание на цифры после /product/ или /p/.

Пример URL:

https://www.ozon.ru/product/iphone-15-pro-max-256gb-natural-titanium-100784325/

В этом случае ID товара — 100784325.

🔍 Нюансы:

  • 🔄 Если в URL есть параметры (например, ?asb=...), их можно игнорировать — ID всегда идёт сразу после /product/.
  • 🛒 На странице корзины или "Избранное" ID не отображается — нужно переходить в карточку товара.
  • 🔗 Если товар удалён, URL может перенаправлять на аналоги, и ID в адресе будет другим.
💡

Скопируйте ID товара в буфер обмена прямо из адресной строки: выделите цифры после "/product/" и нажмите Ctrl+C (Windows) или Cmd+C (Mac).

Способ 2: Поиск ID в мобильном приложении Ozon

В официальном приложении Ozon для Android и iOS прямой просмотр ID затруднён, но есть обходные пути:

Метод 1: Через "Поделиться"

  1. Откройте карточку товара в приложении.
  2. Тапните на иконку "Поделиться" (стрелочка вверх).
  3. Выберите "Скопировать ссылку".
  4. Вставьте ссылку в любое текстовое поле (например, в заметки) — ID будет после /p/.

Метод 2: Через браузер на телефоне

Если первый способ не сработал (например, ссылка сокращённая), откройте Ozon через мобильный браузер (Chrome, Safari) в режиме "Для компьютера" (в меню браузера). Далее действуйте как в Способе 1 для ПК.

Почему в приложении нет прямого отображения ID?

Ozon скрывает технические идентификаторы в мобильной версии для упрощения интерфейса. Это часть стратегии по улучшению пользовательского опыта (UX), когда лишние цифры могут сбивать с толку неискушённых покупателей.

⚠️ Внимание: В приложении Ozon Seller для продавцов ID отображается в полной версии карточки товара (раздел "Основная информация"). Не путайте его с приложением для покупателей!

Способ 3: Массовое извлечение ID для продавцов

Если вам нужно получить ID для сотен или тысяч товаров (например, для загрузки в 1С или Google Merchant Center), ручные методы не подойдут. Используйте:

1. Экспорт через Личный кабинет Ozon Seller

  • 📊 Перейдите в раздел Товары → Список товаров.
  • 🔧 Нажмите "Экспортировать" и выберите формат XLSX или CSV.
  • 📂 В полученном файле будет колонка product_id.

2. API Ozon

Для автоматизации используйте метод /v2/product/info из документации API. Пример запроса:

GET https://api-seller.ozon.ru/v2/product/info?product_id=100784325

Headers: Client-Id: [ваш_client_id]


Api-Key: [ваш_api_key]

3. Парсеры и сервисы

Сервисы вроде Ozon Parser или DataCol позволяют собирать ID в полуавтоматическом режиме. Средняя стоимость — от 500 ₽/месяц.

Убедитесь, что у вас есть доступ к Ozon Seller|Проверьте лимиты API (10 000 запросов/день для стандартного тарифа)|Сохраните резервную копию текущих данных|Используйте фильтры, чтобы экспортировать только нужные товары-->

Способ 4: ID в чеках, накладных и email-уведомлениях

ID товара можно найти в документах, которые генерирует Ozon:

1. Электронный чек (для покупателей)

  • 📄 Откройте письмо с чеком от no-reply@ozon.ru.
  • 🔍 Найдите строку с названием товара — рядом может быть указан Артикул: 12345678.
  • ⚠️ Иногда в чеках указывается SKU продавца, а не ID Ozon. Уточняйте по URL (см. Способ 1).

2. Товарная накладная (для продавцов)

В документах на отгрузку (раздел Логистика → Документы в Ozon Seller) ID указан в колонке "Код товара Ozon".

3. Уведомления о заказах

В письмах о новых заказах (orders@ozon.ru) ID прописывается в таблице с позициями под названием товара.

💡

Если в документе указано два кода (например, "Артикул: ABC123" и "ID: 100784325"), для работы с Ozon всегда используйте второй — это и есть product_id.

Способ 5: Альтернативные методы (для продвинутых пользователей)

Если стандартные способы не сработали, попробуйте:

1. Просмотр исходного кода страницы

  1. Откройте карточку товара на ПК.
  2. Нажмите Ctrl+U (или ПКМ → "Просмотр кода страницы").
  3. Ищите строку "productId":100784325 (используйте поиск по странице — Ctrl+F).

2. Через инструменты разработчика

В браузере нажмите F12, перейдите на вкладку "Network", обновите страницу и найдите запрос product-card. В ответе будет поле id.

3. Боты для Telegram

Боты вроде @OzonParserBot или @OzonHelperBot могут извлекать ID по ссылке на товар. Достаточно отправить боту URL, и он вернёт все технические данные.

⚠️ Внимание: Использование парсеров и ботов может нарушать Пользовательское соглашение Ozon. Для массового сбора данных обязательно используйте официальный API.

Типичные ошибки и как их избежать

Ошибка 1: Путаница между ID и SKU

🔹 Признаки: Вы видите буквенно-цифровой код (например, IPH15-256-NT) вместо чисел.

🔹 Решение: SKU присваивает продавец, а ID — Ozon. Всегда проверяйте по URL (см. Способ 1).

Ошибка 2: Копирование ID из сокращённых ссылок

🔹 Признаки: В адресе видно ozon.ru/pr/abc123 вместо /product/100784325.

🔹 Решение: Перейдите по сокращённой ссылке — в конечном URL будет полный ID.

Ошибка 3: Использование ID удалённого товара

🔹 Признаки: При вводе ID в поиск или API возвращается ошибка 404.

🔹 Решение: Проверьте статус товара в Ozon Seller (раздел Товары → Архив). Удалённые позиции теряют свой ID.

💡

Если товар был удалён, но вам нужен его ID для возврата, попробуйте найти его в истории заказов (раздел "Мои заказы" → "Архив").

FAQ: Частые вопросы про ID товаров на Ozon

Можно ли по ID товара узнать, кто его продавец?

Нет, product_id относится только к карточке в каталоге Ozon и не содержит информации о продавце. Чтобы узнать поставщика, нужно:

  1. Перейти на страницу товара.
  2. Нажать на название магазина под кнопкой "Купить".

Для продавцов: в Ozon Seller используйте фильтр по offer_id, чтобы найти свои предложения.

Как найти ID товара, если он скрыт или заблокирован?

Если товар скрыт модератором, его ID можно извлечь:

  • Из письма от поддержки (обычно указывается в уведомлении о блокировке).
  • Через API (метод /v1/product/list с фильтром по статусу).
  • Из архива заказов, если товар ранее продавался.

⚠️ Если товар удалён окончательно, восстановить его ID невозможно — придётся создавать новую карточку.

Меняется ли ID товара при изменении цены или описания?

Нет, product_id присваивается один раз при создании карточки и остаётся неизменным, даже если:

  • 💰 Изменилась цена;
  • 📝 Обновилось описание или фото;
  • 📦 Поменялся состав комплектации;
  • 🏷️ Добавились или удалились метки (например, "Хит продаж").

Исключение: если товар был удалён и создан заново, он получит новый ID.

Как по ID товара найти его аналог на Wildberries?

Прямой привязки между ID на Ozon и Wildberries нет, но можно:

  1. Скачать фото товара с Ozon по его ID.
  2. Загрузить фото в поиск по изображению на Wildberries.
  3. Сравнить названия и характеристики вручную.

🤖 Для автоматизации используйте сервисы вроде EcomContent или PriceGuard, которые ищут аналоги по ключевым параметрам.

Можно ли использовать ID товара для парсинга цен?

Технически да, но:

  • ✅ Разрешено: отслеживать цены своих товаров через API.
  • ❌ Запрещено: массовый сбор цен конкурентов без согласия Ozon (нарушение п. 4.7 Пользовательского соглашения).

🔍 Альтернатива: используйте официальные инструменты анализа рынка, например, Ozon Trendy или SellerLab.

Почему Озон требует сертификаты и что будет без них

Маркетплейс Ozon строго контролирует легальность товаров на своей площадке — это требование не только внутренней политики, но и российского законодательства. Без действующих сертификатов или деклараций соответствия ваши карточки товаров могут быть скрыты от покупателей, а аккаунт — заблокирован. По статистике 2026 года, 37% новых продавцов получают предупреждения о недостающих документах в первые 2 недели работы.

Сертификаты подтверждают, что ваш товар соответствует ГОСТ, ТР ТС или другим нормативным актам. Например, для детских игрушек требуется декларация по ТР ТС 008/2011, а для электроники — сертификат по ТР ТС 004/2011. Озон автоматически проверяет наличие документов при модерации карточки, но даже если товар уже продаётся, система может запросить сертификаты позже — например, при жалобе покупателя или плановой проверке.

Важно: с 1 марта 2026 года Озон ужесточил правила для категорий Одежда и обувь, Косметика и Бытовая химия. Теперь для них требуются не только сертификаты на товар, но и документы на материалы (например, сертификат на ткань для одежды).

Какие документы нужны для Озон: полный список

Тип требуемого документа зависит от категории товара и его характеристик. Вот основные виды сертификатов, которые запрашивает Озон:

  • 📄 Декларация соответствия ТР ТС — для большинства непродовольственных товаров (одежда, обувь, мебель, электроника). Действует до 5 лет.
  • 🔒 Сертификат соответствия ТР ТС — для товаров повышенной опасности (детские товары, электроинструменты, автозапчасти). Обычно действует 1–3 года.
  • 🧪 Санитарно-эпидемиологическое заключение (СЭЗ) — для косметики, бытовой химии, товаров для детей.
  • 🌿 Фитосанитарный сертификат — для растительной продукции (например, чай, специи).
  • 📦 Документы на упаковку — если она контактирует с пищевыми продуктами (например, для контейнеров).

Как узнать, какой именно документ нужен для вашего товара? Воспользуйтесь официальным классификатором Озон или проверьте требования в личном кабинете при загрузке карточки. Например, для смартфонов потребуется сертификат ТР ТС 004/2011 + декларация на аккумулятор, а для детских сосок — сертификат ТР ТС 008/2011 + СЭЗ.

Способы добавления сертификата в личном кабинете Озон

Озон предоставляет три основных способа загрузки сертификатов. Выбор зависит от того, сколько товаров вы сертифицируете и насколько часто обновляете документы.

Способ Когда использовать Плюсы Минусы
Через карточку товара Для 1–5 товаров Быстро, не требует подготовки Неудобно для большого ассортимента
Массовая загрузка (Excel) Для 10+ товаров Экономит время, подходит для FBS Требует знания структуры файла
Через API Для автоматизации (100+ товаров) Интеграция с 1С, минимальные ручные действия Нужны навыки программирования

Рассмотрим каждый способ подробнее.

1. Загрузка через карточку товара (пошагово)

Это самый простой метод, подходящий для начинающих продавцов. Инструкция:

  1. Откройте личный кабинет Озон и перейдите в раздел Товары → Мои товары.
  2. Найдите нужный товар и нажмите на его название.
  3. В блоке Сертификаты кликните Добавить сертификат.
  4. Заполните поля:
    • 📌 Тип документа (декларация/сертификат)
    • 📅 Дата выдачи и срок действия
    • 🏢 Орган, выдавший документ (название и реквизиты)
    • 📑 Номер документа (указывайте точно, как в оригинале)
  • Прикрепите скан документа в формате PDF или JPEG (макс. 10 МБ).
  • Сохраните изменения и дождитесь модерации (обычно 1–3 часа).

    • Скан читаемый (нет размытости, все печати видны)|

    Номер документа совпадает с оригиналом|

    Срок действия не истёк|

    Формат файла — PDF/JPEG|

    Размер файла не превышает 10 МБ-->

    ⚠️ Внимание: Если вы продаёте товар по схеме FBO, сертификаты проверяются строже — модератор может запросить дополнительные документы (например, договор с поставщиком).

    2. Массовая загрузка через Excel

    Для продавцов с большим ассортиментом удобнее загружать сертификаты пакетом. Для этого:

    1. Скачайте шаблон Excel в разделе Товары → Импорт/Экспорт → Шаблоны (выберите "Шаблон для сертификатов").
    2. Заполните столбцы:
      • Product ID — уникальный идентификатор товара на Озон
      • Certificate Type — тип документа (DECLARATION или CERTIFICATE)
      • Certificate Number — номер сертификата
      • Issue Date и Expiry Date — даты в формате ДД.ММ.ГГГГ
      • Issuer — название органа сертификации
      • File URL — ссылка на загруженный скан (если файлы уже в облаке Озон)
  • Сохраните файл в формате .xlsx и загрузите через Товары → Импорт → Сертификаты.

    • 🔹 Совет: Если у вас много товаров с одинаковыми сертификатами (например, одна модель телефона в разных цветах), используйте групповую привязку. Для этого в шаблоне укажите один и тот же Certificate Number для нескольких Product ID.

    3. Автоматизация через API

    Для крупных продавцов (от 1000 SKU) оптимально использовать API Озон. Это позволяет:

    • 🔄 Автоматически обновлять истекающие сертификаты
    • 📊 Синхронизировать данные с 1С или другой ERP-системой
    • ⚡ Уменьшить время модерации за счёт предварительной валидации данных

    Пример запроса для загрузки сертификата через API:

    POST https://api-seller.ozon.ru/v1/product/certificate/upload

    Headers:
    

    - Client-Id: [ваш_client_id]
    

    - Api-Key: [ваш_api_key]
    

    Body:
    

    {
    

    "product_id": 12345678,
    

    "certificate_type": "DECLARATION",
    

    "certificate_number": "ТР ТС RU Д-RU.АБ12.В.00001",
    

    "issue_date": "01.01.2023",
    

    "expiry_date": "01.01.2028",
    

    "issuer": "ООО «Центр Сертификации»",
    

    "file_url": "https://yourstorage.com/cert123.pdf"
    

    }

    Подробную документацию по API можно найти здесь. Для работы потребуется получить доступ к Ozon Seller API в личном кабинете.

    💡

    Массовая загрузка и API экономят время, но требуют тщательной подготовки данных. Ошибки в Excel или JSON могут привести к отказу в модерации всей партии сертификатов.

    Типичные ошибки при загрузке сертификатов и как их избежать

    По данным поддержки Озон, 4 из 10 отказов в модерации сертификатов связаны с типичными ошибками. Вот самые распространённые:

    • 🔍 Нечитаемый скан — документ сфотографирован под углом, размыт или имеет блики. Решение: используйте сканер или специальные приложения (например, CamScanner).
    • ⏳ Просроченный сертификат — срок действия истёк, но продавец не обновил документ. Решение: настройте напоминания в календаре за 2 месяца до истечения.
    • 📝 Несовпадение данных — номер сертификата в личном кабинете не совпадает с номером в скане. Решение: проверяйте данные трижды перед сохранением.
    • 🏷️ Не тот тип документа — например, вместо декларации загружена справка от производителя. Решение: уточняйте требования для вашей категории в справке Озон.
    • 🌍 Иностранные сертификаты без перевода — документы на английском или китайском без нотариального перевода. Решение: закажите перевод у аккредитованного бюро.

    ⚠️ Внимание: Если Озон отклонил ваш сертификат, в личном кабинете появится уведомление с причиной. У вас есть 3 попытки исправить ошибку, после чего карточка товара будет заблокирована до ручной проверки модератором (срок — до 5 дней).

    Что делать, если Озон требует сертификат на товар, который его не требует?

    Иногда система ошибочно запрашивает документы для товаров, которые не подлежат обязательной сертификации (например, книги или канцтовары). В этом случае:

    1. Проверьте категорию товара — возможно, вы указали её неверно.

    2. Если категория верна, обратитесь в поддержку Озон через форму "Запрос на проверку сертификатов" (раздел Помощь → Обращения).

    3. Приложите скриншот из реестра Росаккредитации, подтверждающий, что ваш товар не требует сертификации.

    Обычно вопрос решается в течение 1–2 дней.

    Как проверить статус сертификата после загрузки

    После добавления сертификата его статус можно отследить в личном кабинете. Для этого:

    1. Перейдите в Товары → Мои товары.
    2. Найдите нужный товар и откройте его карточку.
    3. В блоке Сертификаты посмотрите на значок статуса:
      • ✅ Зелёная галочка — сертификат принят.
      • ⏳ Жёлтый восклицательный знак — документ на модерации.
      • ❌ Красный крестик — отклонён (причина будет указана при наведении).

    Если сертификат на модерации дольше 24 часов, напишите в поддержку с указанием:

    • ID товара (Product ID)
    • Номер сертификата
    • Дата и время загрузки

    🔹 Полезный лайфхак: Чтобы ускорить модерацию, загружайте сертификаты в рабочие дни с 10:00 до 16:00 по московскому времени — в это время работает основной штат модераторов.

    Что делать, если сертификат истёк или изменились требования

    Если срок действия вашего сертификата подходит к концу, Озон пришлёт уведомление за 30 дней до истечения. Вот алгоритм действий:

    1. Проверьте актуальность требований. Возможно, для вашей категории ввели новые стандарты (например, с 2026 года для мебели требуется декларация по ТР ЕАЭС 025/2012).
    2. Обратитесь в орган сертификации. Если сертификат ещё действителен, можно запросить протокол испытаний для продления. Если истёк — потребуется новая процедура сертификации.
    3. Обновите данные в Озон. Загрузите новый сертификат тем же способом, что и ранее (через карточку товара, Excel или API).
    4. Свяжитесь с менеджером Озон. Если товар временно недоступен для продажи из-за просроченного сертификата, менеджер может приостановить блокировку на 7 дней для устранения проблемы.

    ⚠️ Внимание: Если вы продаёте товар по модели FBS и его сертификат истёк, Озон может приостановить отгрузку со склада до устранения проблемы. Это приведёт к штрафам за невыполненные заказы.

    - Название товара

    - Номер сертификата

    - Дату истечения

    - Ссылку на скан

    Это поможет избежать просрочек и быстро обновлять данные.-->

    Частые вопросы о сертификатах на Озон

    Нужно ли загружать сертификат, если товар уже продаётся без него?

    Да, даже если товар уже опубликован, Озон может запросить сертификат позже — например, при жалобе покупателя или плановой проверке. Если документ не предоставить в течение 7 дней, карточка будет скрыта. Рекомендуем загружать сертификаты заранее для всех товаров, подлежащих обязательной сертификации.

    Можно ли использовать один сертификат для нескольких товаров?

    Да, если товары относятся к одной модели/серии и имеют одинаковые характеристики. Например, один сертификат на наушники Sony WH-1000XM5 подойдёт для всех цветовых вариантов (чёрный, серебристый, синий). Но если товары отличаются по техническим параметрам (например, разная мощность у пылесосов), потребуются отдельные сертификаты.

    Что делать, если потерян оригинал сертификата?

    Обратитесь в орган, выдавший документ, с заявлением о выдаче дубликата. Обычно это занимает 3–5 рабочих дней. Также можно запросить выписку из реестра Росаккредитации, но не все категории товаров поддерживают этот вариант.

    Как сертифицировать товар, если я дропшиппер и не являюсь производителем?

    В этом случае вам понадобится:

    1. Договор с поставщиком, где прописано, что он передаёт вам право на реализацию товара.
    2. Копия сертификата производителя (с нотариальным переводом, если документ на иностранном языке).
    3. Декларация о соответствии на имя вашей компании (её можно оформить на основе сертификата производителя).

    Озон принимает такие документы, но может запросить дополнительные подтверждения.

    Сколько стоит оформление сертификата для Озон?

    Стоимость зависит от категории товара и типа документа:

    • Декларация ТР ТС — от 5 000 до 20 000 ₽
    • Сертификат ТР ТС — от 15 000 до 50 000 ₽
    • СЭЗ (санитарно-эпидемиологическое заключение) — от 10 000 до 30 000 ₽

    Для удешевления можно оформить сертификат на серию товаров (например, на всю линейку смартфонов одной модели).

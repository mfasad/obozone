Где и как найти ID товара на Ozon: полное руководство для продавцов и покупателей

Идентификатор товара (ID) на Ozon — это уникальный цифровой или буквенно-цифровой код, который присваивается каждой позиции в каталоге. Без него невозможно отследить заказ, обновить информацию о товаре, настроить рекламу или решить спорную ситуацию с возвратом. Продавцы сталкиваются с необходимостью найти ID при работе с FBS/FBO-схемами, загрузке прайс-листов или интеграции через API. Покупателям же этот код может понадобиться для уточнения деталей заказа в поддержке или при оформлении возврата.

Проблема в том, что Ozon не всегда показывает ID на видном месте — его расположение зависит от типа устройства (ПК, смартфон), роли пользователя (продавец/покупатель) и даже версии приложения. В этой статье мы разберём все актуальные способы поиска ID — от очевидных до скрытых, — а также объясним, чем отличаются SKU, артикул продавца и идентификатор Ozon. Если вы тратите время на ручной поиск кода в каждом заказе, после прочтения сможете автоматизировать процесс или найти ID за 10 секунд.

Для удобства мы разделили инструкции по категориям: отдельно для продавцов (в личном кабинете Ozon Seller), для покупателей (в мобильном приложении и на сайте), а также для случаев, когда ID нужно извлечь из ссылки или API. Если ваш случай не подходит ни под одну категорию — проверьте раздел FAQ в конце статьи: там разобраны нестандартные ситуации, например, поиск ID для товаров, удалённых из каталога.

Почему важен ID товара на Ozon

На первый взгляд, идентификатор товара — это просто набор цифр. Но на практике он выполняет критически важные функции:

  • 🔗 Связь с API: Без ID невозможно отправить запрос на обновление цены, остатков или описания через Ozon API. Например, для изменения стоимости товара в запросе обязательно указывается параметр product_id.
  • 📦 Логистика FBS/FBO: При передаче товара на склад Ozon или оформлении возврата система требует указать ID для корректной привязки позиции к заказу.
  • 🛒 Реклама и промоакции: Чтобы добавить товар в кампанию Ozon Реклама или участие в распродаже (например, Чёрная пятница), нужно знать его уникальный код.
  • 🔍 Поиск дублей: Продавцы используют ID для проверки, не загружен ли тот же товар в каталог под другим названием (что запрещено правилами Ozon).

Для покупателей ID тоже полезен: с его помощью можно:

  • 📞 Быстро найти товар в поддержке, если возникли проблемы с заказом (например, пришла не та позиция).
  • 🔄 Отследить историю изменений цены или наличия по конкретному артикулу.
  • ⚖️ Оформить претензию на возврат, если продавец отказывается принимать товар без указания ID.

Важно: ID товара на Ozon не совпадает с артикулом производителя или штрихкодом. Это внутренний идентификатор маркетплейса, который генерируется автоматически при добавлении товара в каталог.

Где продавцы могут найти ID товара в личном кабинете Ozon Seller

Если вы продавец, то самый надёжный способ получить ID — через панель управления Ozon Seller. Рассмотрим все доступные варианты.

Способ 1: В карточке товара

Перейдите в раздел Товары → Мой каталог. Найдите нужную позицию в списке и кликните по её названию. В открывшейся карточке товара ID будет отображаться:

  • 🖥️ На десктопе — в строке браузера (в URL после /product/) и в блоке "Основная информация".
  • 📱 В мобильном приложении Ozon Seller — в верхней части экрана под названием товара (нажмите на три точки в правом верхнем углу, чтобы скопировать).

Пример URL с ID: 

https://seller.ozon.ru/products/product/123456789/edit

Здесь 123456789 — это и есть идентификатор товара.

Способ 2: В экспортном файле

Если товаров много, удобнее скачать полный список с ID через экспорт:

  1. Перейдите в Товары → Мой каталог.
  2. Нажмите кнопку Экспорт (в правом верхнем углу).
  3. Выберите формат XLSX или CSV и дождитесь формирования файла.
  4. В полученной таблице ID будет в колонке product_id.

Этот метод подходит для массовой обработки данных, например, при настройке автоматической выгрузки остатков через 1С или МойСклад.

💡

Если в экспортном файле нет колонки с ID, проверьте настройки экспорта: иногда её нужно добавить вручную в параметрах выгрузки.

Способ 3: Через API-запрос

Для продвинутых пользователей, работающих с Ozon API, ID можно получить программно. Например, запрос для получения списка товаров с идентификаторами: 

GET https://api-seller.ozon.ru/v2/product/list

Headers:


- Client-Id: {ваш_client_id}


- Api-Key: {ваш_api_key}

В ответе будет массив объектов, где каждому товару соответствует поле "id".

⚠️

Внимание: При работе с API убедитесь, что у вас есть действующий токен доступа. Если токен просрочен, запрос вернёт ошибку 401 Unauthorized. Обновите ключи в личном кабинете Ozon Seller в разделе Настройки → API.

Как покупателю найти ID товара в заказе или каталоге Ozon

Покупателям идентификатор товара может понадобиться реже, но в некоторых ситуациях (например, при оформлении возврата или спорной ситуации с продавцом) он становится критически важным. Рассмотрим, где его искать.

Способ 1: В мобильном приложении Ozon

Откройте приложение и выполните следующие шаги:

  1. Перейдите в раздел Заказы (значок коробки в нижнем меню).
  2. Выберите нужный заказ и нажмите на товар в списке.
  3. В открывшейся карточке товара тапните по трём точкам (⋮) в правом верхнем углу.
  4. В меню выберите Поделиться — в ссылке будет ID (цифры после /product/).

Если товар ещё не заказан, но вы хотите узнать его ID:

  1. Найдите товар через поиск.
  2. Откройте его карточку.
  3. Скопируйте ссылку через кнопку Поделиться (ID — это последняя часть URL).
Что делать, если в ссылке нет цифр?

Если в URL товара вместо цифр отображаются буквы или символы (например, /product/some-name-123/), это значит, что Ozon использует "человекочитаемый" формат. В этом случае ID можно найти в HTML-коде страницы:

  1. Откройте карточку товара в браузере (на ПК или через режим "Десктопная версия" на телефоне).
  2. Нажмите Ctrl+U (или выберите "Просмотр кода страницы" в меню браузера).
  3. Найдите строку с data-product-id=" — цифры в кавычках и есть ID.

Способ 2: На сайте Ozon (десктопная версия)

На компьютере процесс ещё проще:

  1. Откройте ozon.ru и найдите нужный товар.
  2. В адресной строке браузера будет URL вида: 
    https://www.ozon.ru/product/{название}-{id}/

    Например: https://www.ozon.ru/product/umnyy-chasy-xiaomi-mi-band-7-123456789/ — здесь ID это 123456789.

Если товар уже куплен:

  1. Перейдите в Личный кабинет → Мои заказы.
  2. Выберите заказ и кликните по названию товара.
  3. ID будет в URL страницы товара (как в примере выше).

⚠️

Внимание: В некоторых категориях (например, Одежда или Обувь) Ozon может использовать составные ID, где первая часть — код модели, а вторая — размер/цвет. В этом случае для возврата или обращения в поддержку нужен полный идентификатор, включая вариацию. Например: 123456789_42_black.

Разница между SKU, артикулом продавца и ID Ozon

Многие путают эти три понятия, что приводит к ошибкам при работе с заказами или API. Разберёмся, чем они отличаются:

Термин Что это? Пример Где используется?
ID Ozon Уникальный идентификатор товара в каталоге Ozon, присваивается автоматически. 123456789 API, ссылки на товар, внутренние процессы маркетплейса.

🔹 Когда нужен ID Ozon?

  • Для работы с Ozon API.
  • При оформлении возврата через личный кабинет.
  • Для добавления товара в рекламную кампанию.

🔹 Когда нужен SKU или артикул продавца?

  • Для поиска товара в вашей базе (например, в 1С или Excel).
  • При общении с вашей поддержкой (если проблема на стороне продавца).
  • Для загрузки прайс-листов с обновлениями.
💡

ID Ozon — это "паспорт" товара на маркетплейсе, а SKU/артикул — ваши внутренние метки. Никогда не путайте их при обращении в поддержку или настройке API!

Как найти ID товара, если он удалён или архивирован

Ситуация: товар больше не продаётся, но вам нужен его ID для отчётов, возвратов или разрешения споров. Вот что можно сделать:

Для продавцов

Если товар удалён из каталога, но ранее был в продаже:

  1. Перейдите в Товары → Архив в личном кабинете Ozon Seller.
  2. Найдите товар в списке архивированных.
  3. ID будет в колонке ID товара или в URL при открытии карточки.

Если товара нет в архиве:

  • 📧 Проверьте письма от Ozon — при удалении товара иногда приходит уведомление с его ID.
  • 📊 Экспортируйте исторические данные через Отчёты → История изменений (там могут сохраняться ID удалённых товаров).
  • 🔄 Если товар был в заказах, найдите его через Заказы → История и скопируйте ID оттуда.

Для покупателей

Если товар исчез из каталога, но был куплен ранее:

  1. Откройте историю заказов в личном кабинете.
  2. Найдите заказ с нужным товаром и кликните по его названию.
  3. Даже если товар больше не доступен для покупки, его карточка откроется, и ID будет в URL.

Если товар не был куплен, но нужен его ID (например, для сравнения или жалобы):

  • 🔍 Попробуйте найти его через поиск Google по запросу site:ozon.ru "название товара" — возможно, страница сохранилась в кэше.
  • 📱 Проверьте историю просмотров в мобильном приложении (раздел Избранное → История).

⚠️

Внимание: Если товар был удалён за нарушение правил Ozon (например, за продажу контрафактной продукции), его ID может быть заблокирован. В этом случае восстановить доступ к идентификатору невозможно — придётся создавать новую карточку товара.

Автоматизация поиска ID: инструменты и сервисы

Если вам регулярно нужно искать ID товаров (например, для интеграции с CRM или анализа конкурентов), ручные методы станут слишком трудоёмкими. Рассмотрим способы автоматизации:

1. Парсинг ссылок

Для массового извлечения ID из ссылок можно использовать:

  • 📊 Excel/Google Sheets: Функция =REGEXEXTRACT(A1; "/product/(\d+)") извлечёт ID из URL в ячейке A1.
  • 🤖 Python-скрипты: Библиотека re позволяет парсить ID из списка ссылок: 
    import re

    url = "https://www.ozon.ru/product/umnye-chasy-123456789/"
    

    product_id = re.search(r'/product/.*?-(\d+)', url).group(1)
    

    print(product_id)  # Выведет: 123456789

2. Расширения для браузера

Полезные расширения для Chrome и Firefox:

  • 🛠️ Ozon Helper: Показывает ID товара прямо на странице каталога (добавляет кнопку "Копировать ID").
  • 📋 Tampermonkey + пользовательский скрипт: Можно написать скрипт, который будет выделять ID на всех страницах Ozon.

3. Интеграция с 1С или CRM

Если вы используете 1С, МойСклад или Bitrix24, настройте автоматическую синхронизацию ID:

  • В 1С используйте обработку Обмен с Ozon (есть готовые решения на Инфостарт).
  • В Bitrix24 настройте вебхуки для получения ID через API.

🔹 Пример настройки в 1С:

  1. Установите модуль интеграции с Ozon (например, от Атол или Клеверенс).
  2. В настройках укажите Client-ID и API-Key из личного кабинета.
  3. Запустите выгрузку товаров — ID автоматически подтянутся в номенклатуру.
💡

Для массовой обработки ID удобно использовать Google Apps Script. Например, скрипт может ежедневно проверять новые товары в вашем каталоге и записывать их ID в таблицу.

Частые ошибки и как их избежать

При работе с ID товара на Ozon многие сталкиваются с типичными проблемами. Разберём самые распространённые:

Ошибка 1: Путают ID и SKU

🔴 Проблема: При обращении в поддержку или настройке API передают SKU вместо ID, из-за чего система не находит товар.

✅ Решение:

  • Всегда проверяйте, какой именно идентификатор запрашивается в инструкции.
  • В личном кабинете Ozon Seller ID отображается в колонке ID товара, а SKU — в Артикул продавца.

Ошибка 2: Копируют неполный ID

🔴 Проблема: В некоторых категориях ID состоит из нескольких частей (например, 123456789_42_black). Если скопировать только первую часть, система не распознает товар.

✅ Решение:

  • Всегда копируйте полный ID, включая вариации (размер, цвет).
  • В карточке товара полный ID отображается в блоке Характеристики под названием "Идентификатор товара".

Ошибка 3: ID не отображается в мобильном приложении

🔴 Проблема: В некоторых версиях приложения Ozon (особенно на Android) ID не видно в карточке товара.

✅ Решение:

  • Обновите приложение до последней версии.
  • Используйте браузерную версию сайта (в мобильном браузере запросите "Десктопную версию").
  • Скопируйте ссылку на товар и извлеките ID из URL.

Ошибка 4: Ошибка 404 при переходе по ID

🔴 Проблема: При переходе по ссылке вида ozon.ru/product/123456789 выдаётся ошибка "Страница не найдена".

✅ Решение:

  • Проверьте, не удалён ли товар из каталога.
  • Убедитесь, что ID скопирован полностью (иногда в конце есть символы _1, _2 и т.д.).
  • Если товар архивирован, попробуйте найти его через историю заказов.

⚠️

Внимание: Если вы продавец и столкнулись с ошибкой 403 Forbidden при работе с API и конкретным ID, это может означать, что товар заблокирован за нарушение правил Ozon. Проверьте уведомления в личном кабинете или обратитесь в поддержку.

FAQ: Ответы на частые вопросы

Можно ли найти ID товара по фотографии?

Нет, Ozon не предоставляет такой функции. Однако вы можете:

  1. Загрузить фото в Google Картинки или Yandex.Images, чтобы найти страницу товара.
  2. Использовать расширения для браузера вроде Search by Image, которые ищут аналогичные изображения на Ozon.

Если товар найдётся, его ID будет в URL страницы.

Что делать, если ID товара изменился?

Ozon крайне редко меняет ID товаров, но если это произошло (например, при переносе товара в другой каталог), старый идентификатор станет неактуальным. В этом случае:

  • Для продавцов: проверьте новый ID в личном кабинете или через API-запрос GET /v2/product/info.
  • Для покупателей: найдите товар заново в каталоге и скопируйте актуальный ID из URL.

Если товар был удалён и создан заново, его ID сбросится — это нормально.

Как найти ID товара в чеке или квитанции?

В бумажных чеках и электронных квитанциях от Ozon ID товара обычно не указывается. Вместо этого там может быть:

  • Артикул продавца (если продавец его указал).
  • Название товара (можно найти по нему в каталоге).
  • Штрихкод (если он есть на упаковке).

Чтобы получить ID:

  1. Найдите товар в каталоге Ozon по названию или штрихкоду.
  2. Скопируйте ID из URL или через кнопку "Поделиться".
Можно ли восстановить ID удалённого товара?

Да, но с оговорками:

  • Если товар в архиве (удалён продавцом), его ID доступен в личном кабинете Ozon Seller (раздел Товары → Архив).
  • Если товар удален администрацией за нарушение правил, его ID становится недоступным.
  • Если товар был в заказах, его ID можно найти в истории заказов (даже если сам товар удалён из каталога).

Если ни один из способов не работает, обратитесь в поддержку Ozon с просьбой предоставить ID удалённого товара (указывайте название, дату удаления и причину, по которой он нужен).

Как найти ID товара по названию без доступа к личному кабинету?

Если вы не продавец и не покупатель этого товара, но знаете его название:

  1. Откройте ozon.ru и введите название в поиск.
  2. Найдите нужный товар в выдаче и откройте его карточку.
  3. Скопируйте ID из URL (цифры после /product/).

Если товаров с похожим названием много, используйте фильтры (цена, бренд, характеристики), чтобы сузить поиск.

⚠️ Обратите внимание: если товар продаётся несколькими продавцами, у каждого может быть свой ID (даже если название и фотографии совпадают).

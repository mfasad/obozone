Как удалить товары с Ozon: все способы для продавцов в 2026 году

Удаление товаров с Ozon — казалось бы, простая процедура, но на практике она таит массу подводных камней. От ошибок при массовом удалении до неожиданных блокировок карточек — даже опытные продавцы сталкиваются с проблемами. Эта статья поможет разобраться, как корректно удалить товар с маркетплейса, избежать штрафов и сохранить репутацию магазина.

Мы рассмотрим все актуальные способы: через Личный кабинет Ozon Seller, с помощью API, а также альтернативные варианты для ситуаций, когда удаление невозможно. Особое внимание уделим различиям между схемами FBO и FBS, так как процесс удаления в них принципиально отличается. Если вы ищете, как удалить товар с Ozon без последствий для рейтинга, этот гайд для вас.

Почему нельзя просто удалить товар с Ozon?

Многие продавцы ошибочно думают, что удаление карточки товара — это как стирание файла с компьютера: нажал кнопку, и всё исчезло. На самом деле Ozon сохраняет историю всех действий, и некорректное удаление может привести к:

  • 📉 Падению позиции в поисковой выдаче (алгоритмы маркетплейса воспринимают удаление как признак ненадёжного товара).
  • 🚫 Блокировке возможности загружать аналогичные товары в будущем.
  • 💰 Штрафам за нарушение правил (например, если товар был в заказах покупателей).
  • ⏳ Задержке при повторной загрузке того же SKU (система может воспринять это как дубликат).

Кроме того, если товар был продан хотя бы раз, его карточка остаётся в архиве Ozon и может влиять на общий рейтинг магазина. Например, негативные отзывы на удалённый товар продолжают учитываться в метрике DSR (Detail Seller Rating).

Способ 1: Удаление товара через Личный кабинет Ozon Seller

Самый простой и безопасный метод — удаление вручную через интерфейс Ozon Seller. Он подходит для единичных товаров или небольших партий (до 50 позиций). Вот пошаговая инструкция:

  1. Авторизуйтесь в Личном кабинете Ozon Seller.
  2. Перейдите в раздел Товары → Мои товары.
  3. В строке поиска введите SKU или название товара, который нужно удалить.
  4. Нажмите на три точки (⋮) в правой части строки товара и выберите Удалить.
  5. Подтвердите действие в всплывающем окне.

Важно: если товар был продан хотя бы раз, система предложит архивировать его вместо удаления. Архивация скрывает товар от покупателей, но сохраняет историю продаж и отзывы. Полное удаление возможно только для товаров, которые никогда не продавались.

Способ 2: Массовое удаление товаров через Excel

Если нужно удалить сотни или тысячи товаров, ручной метод не подходит. Ozon позволяет делать это через загрузку Excel-файла. Алгоритм действий:

  1. В Личном кабинете перейдите в Товары → Импорт/Экспорт.
  2. Скачайте шаблон для массового удаления (файл products_delete.xlsx).
  3. В столбце A (Product ID) укажите идентификаторы товаров, которые нужно удалить. Их можно скопировать из раздела Мои товары (столбец ID).
  4. Сохраните файл в формате .xlsx (не .csv!).
  5. Загрузите файл обратно в систему через кнопку Импортировать.

Обратите внимание: массовое удаление обрабатывается не мгновенно. В пиковые часы (с 10:00 до 18:00 по МСК) процесс может занять до 2–3 часов. Если в файле будут ошибки (например, несуществующие Product ID), система вернёт отчёт с пометками.

Что делать, если массовое удаление зависло?

Если статус обработки файла долго висит на "В процессе" (более 4 часов), попробуйте:

1. Обновить страницу (иногда помогает сброс кеша браузера).

2. Скачать лог ошибок через Товары → История операций.

3. Разбить большой файл на части по 500 строк и загрузить их по отдельности.

4. Обратиться в поддержку с указанием ID операции (найдёте в истории импорта).

Ошибка при массовом удалении Причина Решение
Product not found Некорректный или устаревший Product ID Проверьте актуальность ID в разделе Мои товары
Product has active orders Товар есть в невыполненных заказах Дождитесь выполнения заказов или отмените их
Forbidden operation Товар участвует в акции или промокоде Сначала отключите участие в маркетинговых активностях
Limit exceeded Превышен лимит на массовые операции (10 000 товаров/день) Разбейте удаление на несколько дней

Способ 3: Удаление через API Ozon (для опытных пользователей)

Для автоматизации удаления товаров можно использовать Ozon API. Этот метод требует технических навыков, но позволяет интегрировать удаление в собственные системы управления товарами. Основные шаги:

  1. Получите доступ к API в разделе Настройки → API-ключи.
  2. Используйте метод POST /v2/products/delete для удаления.
  3. В теле запроса укажите массив product_id (максимум 100 штук за один запрос).

Пример запроса на Python:

import requests


url = "https://api-seller.ozon.ru/v2/products/delete"


headers = {


"Client-Id": "Ваш_Client_ID",


"Api-Key": "Ваш_API_Key",


"Content-Type": "application/json"


}


data = {


"product_id": [12345678, 87654321]  # Список ID товаров


}



response = requests.post(url, headers=headers, json=data)


print(response.json())

Ответ API будет содержать статус операции и список ошибок (если они есть). Например, код 400 с сообщением "product_has_orders" означает, что товар нельзя удалить из-за активных заказов.

💡

Перед массовым удалением через API всегда делайте тестовый запрос с 1–2 товарами. Это поможет избежать блокировки аккаунта за подозрительную активность.

Особенности удаления товаров на FBO и FBS

Процесс удаления зависит от схемы работы с Ozon:

  • 📦 FBO (Fulfillment by Ozon):
    • Товар физически хранится на складе Ozon.
    • При удалении нужно сначала отменить логистику через раздел Логистика → Заказы на склад.
    • Если товар уже на складе, его придётся вывести (оплатив логистику обратно).
  • 🚚 FBS (Fulfillment by Seller):
    • Товар хранится у продавца.
    • Удаление возможно в любой момент, если нет активных заказов.
    • Но если товар был в заказах последнюю неделю, система может заблокировать удаление.

Для FBO критично следить за остатками на складе. Если удалить товар, который физически есть на складе Ozon, но не отменить логистику, возникнет расхождение. Это может привести к штрафу за несоответствие остатков (до 5 000 ₽ за позицию).

💡

На FBO сначала отменяйте логистику, только потом удаляйте товар. На FBS достаточно проверки активных заказов.

Что делать, если товар не удаляется?

Иногда система блокирует удаление даже при отсутствии очевидных причин. Рассмотрим типичные сценарии и решения:

⚠️ Внимание: Если товар был в заказах последнюю неделю, Ozon может временно заблокировать его удаление для анализа возвратов. В этом случае подождите 7–10 дней и повторите попытку.
Проблема Возможная причина Решение
Кнопка "Удалить" неактивна Товар участвует в акции или промокоде Отключите участие в разделе Маркетинг → Акции
Ошибка "Товар в архиве" Товар уже архивирован, но не удалён Используйте массовое удаление через Excel
Ошибка "Превышен лимит" За день удалено более 10 000 товаров Разбейте удаление на несколько дней
Товар "завис" в статусе "Удаляется" Технический сбой на стороне Ozon Обратитесь в поддержку с указанием Product ID

Если ни один из методов не сработал, попробуйте скрыть товар вместо удаления:

  1. В разделе Мои товары найдите нужный товар.
  2. Нажмите ⋮ → Редактировать.
  3. В поле Видимость выберите Скрыть от покупателей.
  4. Сохраните изменения.

Скрытый товар не отображается в поиске, но остаётся в вашем каталоге и не влияет на рейтинг.

Альтернативы удалению: когда лучше архивировать или редактировать?

Удаление — не всегда оптимальное решение. В некоторых случаях лучше:

  • 📂 Архивировать товар:
    • Если планируете вернуть его в продажу позже (например, сезонный товар).
    • Если товар был продан хотя бы раз (сохраняется история отзывов).
  • ✏️ Редактировать карточку:
    • Если нужно изменить цену, описание или фото (например, при ребрендинге).
    • Если товар временно недоступен (установите остаток 0).
  • 🔄 Объединить с другим товаром:
    • Если есть дублирующие карточки (используйте инструмент Объединение товаров в Ozon Seller).

Архивация предпочтительнее удаления, если:

⚠️ Внимание: Архивированные товары продолжают учитываться в метрике DSR (Detail Seller Rating). Если на товаре много негативных отзывов, его удаление может улучшить общий рейтинг магазина.

Как архивировать товар правильно?

1. В разделе Мои товары найдите нужный товар.

2. Нажмите ⋮ → Архивировать.

3. Подтвердите действие.

4. Архивированные товары можно найти в фильтре Статус → Архив.

Чтобы вернуть товар из архива, нажмите Восстановить (но учтите, что его позиция в поиске сбросится).

Частые ошибки и как их избежать

Даже опытные продавцы допускают ошибки при удалении товаров. Вот самые распространённые:

  1. Удаление товара с активными заказами:

    Это приводит к автоматической отмене заказов и штрафу до 1 000 ₽ за каждый. Всегда проверяйте раздел Заказы → Активные перед удалением.

  2. Массовое удаление без резервной копии:

    Если случайно удалите не тот товар, восстановить его будет сложно. Всегда экспортируйте список товаров перед массовыми операциями.

  3. Игнорирование остатков на FBO:

    Удаление товара, который физически есть на складе Ozon, приводит к штрафу за несоответствие остатков.

  4. Удаление товаров с высоким рейтингом:

    Если товар имеет много положительных отзывов, его удаление может ухудшить общую метрику DSR.

Чтобы минимизировать риски:

  • 📌 Используйте тестовый аккаунт в Ozon Seller для отработки массовых операций.
  • 📊 Ведите журнал удалений (дату, SKU, причину) для отчётности.
  • 🔔 Настройте уведомления о новых заказах, чтобы не пропустить активные позиции.

FAQ: Ответы на частые вопросы

Можно ли удалить товар, который уже купили, но не доставили?

Нет. Если товар находится в статусе В пути или Готов к выдаче, его нельзя удалить до завершения заказа. Дождитесь, пока заказ будет закрыт (доставлен или отменён покупателем).

Сколько времени занимает удаление товара?

Единичный товар удаляется мгновенно. Массовое удаление через Excel или API может занять от 10 минут до 3 часов в зависимости от нагрузки на серверы Ozon.

Что будет, если удалить товар, который участвует в акции?

Система заблокирует удаление и вернёт ошибку Forbidden operation. Сначала отключите участие в акции в разделе Маркетинг → Акции, затем повторите попытку.

Можно ли восстановить удалённый товар?

Нет, удалённые товары восстановить нельзя. Можно только создать новую карточку с тем же SKU, но она будет восприниматься как новый товар (без истории продаж и отзывов).

Как удалить товар, если он дублируется?

Используйте инструмент Объединение товаров в разделе Товары → Дубли. Система автоматически найдёт похожие карточки и предложит объединить их. После объединения лишние карточки можно удалить.

Как продавцу на Озон посмотреть выручку по ПВЗ: все способы с примерами

Продавцы на Ozon часто сталкиваются с необходимостью анализировать выручку по конкретным пунктам выдачи заказов (ПВЗ). Это критично для оптимизации логистики, оценки рентабельности точек и принятия решений о расширении сети. Однако стандартные отчёты маркетплейса не всегда дают детализированную информацию по каждому ПВЗ — данные разбросаны по разным разделам, а некоторые показатели скрыты за непонятными формулировками.

В этой статье разберём все доступные способы проверки выручки по ПВЗ — от базовых инструментов личного кабинета до продвинутых методов с использованием Ozon API и сторонних сервисов. Особое внимание уделим нюансам, которые влияют на точность данных: например, как учитываются возвраты, комиссии и доход от дополнительных услуг (упаковка, сборка). Также раскроем малоизвестные фишки, которые помогут выгрузить данные в удобном для анализа формате.

Если вы только начинаете работать с ПВЗ или планируете открыть собственную точку, информация из этой статьи поможет избежать типичных ошибок при расчёте прибыли. Для опытных продавцов мы подготовили раздел с автоматизацией отчётности — это сэкономит часы ручной работы.

1. Базовый способ: выручка по ПВЗ в личном кабинете Ozon Seller

Самый простой метод — использовать стандартные отчёты в личном кабинете. Однако здесь есть подводные камни: данные могут дублироваться или не учитывать некоторые транзакции. Чтобы получить корректную выручку по ПВЗ, следуйте этой инструкции:

Перейдите в раздел Финансы → Отчёты → Выручка. В фильтрах выберите:

  • 📅 Период (например, последний месяц)
  • 📦 Тип доставки — «Пункты выдачи заказов (ПВЗ)»
  • 📍 Регион (если нужно сузить поиск)

Система покажет общую выручку по всем ПВЗ, но без детализации по конкретным точкам. Чтобы увидеть данные по каждому пункту, экспортируйте отчёт в Excel (кнопка «Экспорт» в правом верхнем углу). В выгруженном файле будет колонка ПВЗ с названиями точек — по ней можно сгруппировать данные и посчитать выручку для каждой.

💡

Если в отчёте нет колонки с названием ПВЗ, добавьте её через настройки экспорта. Для этого перед выгрузкой нажмите «Настроить колонки» и отметьте галочкой поле «Пункт выдачи».

Важно учитывать, что в этом отчёте не отображаются:

  • 🔄 Возвраты (их нужно проверять отдельно в разделе Возвраты)
  • 💳 Комиссии Ozon (они вычитаются из итоговой суммы)
  • 📦 Доход от дополнительных услуг (упаковка, сборка заказов)

2. Продвинутый анализ: отчёт «Доставка и логистика»

Для более детального анализа используйте отчёт Доставка и логистика → Отчёт по доставке. Здесь можно увидеть не только выручку, но и количество заказов, средний чек и даже время обработки по каждому ПВЗ.

Алгоритм действий:

  1. Выберите период и фильтр «ПВЗ».
  2. В результатах кликните на название интересующего ПВЗ.
  3. Система покажет детализированную статистику, включая чистую выручку (за минусом возвратов и комиссий).

Преимущество этого отчёта — возможность сравнить эффективность разных ПВЗ по ключевым метрикам:

Метрика Что показывает Зачем нужно
Конверсия в выдачу % заказов, которые клиенты забрали (а не отказались) Оценка надёжности ПВЗ
Среднее время хранения Сколько дней заказ лежит на ПВЗ до выдачи Оптимизация логистики
Доля возвратов % заказов, возвращённых с этого ПВЗ Выявление проблемных точек

Обратите внимание: в этом отчёте выручка показывается без учёта дохода от дополнительных услуг (упаковка, сборка). Эти данные нужно проверять отдельно в разделе Услуги → Дополнительные услуги.

3. Как учесть комиссии и дополнительные услуги

Чистая выручка по ПВЗ — это не просто сумма продаж. Из неё вычитаются:

  • 💰 Комиссия Ozon (от 5% до 15% в зависимости от категории)
  • 📦 Логистические расходы (если используете FBO)
  • 🔄 Стоимость обработки возвратов

Но к выручке могут добавляться доходы от:

  • 📦 Упаковки заказов (если ПВЗ предоставляет эту услугу)
  • 🛠️ Сборки сложных товаров (мебель, техника)
  • 📋 Печати документов для клиентов

Чтобы посчитать реальную прибыль по ПВЗ, используйте формулу: 



Чистая выручка = (Выручка от продаж - Комиссия Ozon - Логистика) + Доход от услуг - Расходы на аренду/зарплаты

Для автоматизации расчётов можно использовать шаблон в Google Sheets:

Шаблон для расчёта прибыли по ПВЗ

1. Создайте таблицу с колонками: "ПВЗ", "Выручка", "Комиссия", "Логистика", "Услуги", "Аренда", "ЗП", "Прибыль".

2. В колонке "Прибыль" используйте формулу:

=B2-C2-D2+E2-F2-G2

3. Добавьте условное форматирование: если прибыль < 0, ячейка красная.

💡

Без учёта дополнительных услуг выручка по ПВЗ может быть занижена на 10–30%. Всегда проверяйте раздел "Дополнительные услуги" в личном кабинете.

4. API Ozon: автоматическая выгрузка данных по ПВЗ

Для продавцов с большим количеством ПВЗ ручной анализ неэффективен. Решение — использовать Ozon Seller API для автоматической выгрузки данных. Это позволит:

  • 📊 Получать актуальную выручку по каждому ПВЗ в реальном времени
  • 🤖 Интегрировать данные с 1С, Excel или Power BI
  • 📈 Строить дашборды с визуализацией ключевых метрик

Основные endpoint’ы для работы с ПВЗ:

  • /v2/analytics/sales — выручка по заказам (можно фильтровать по ПВЗ)
  • /v1/analytics/delivery — данные по логистике
  • /v1/finance/transaction — финансовые транзакции

Пример запроса для получения выручки по конкретному ПВЗ (на Python): 



import requests



url = "https://api-seller.ozon.ru/v2/analytics/sales"


headers = {


"Client-Id": "Ваш Client-ID",


"Api-Key": "Ваш API-ключ"


}


params = {


"date_from": "2026-01-01",


"date_to": "2026-01-31",


"delivery_method_id": ["пвз"]  # Фильтр по ПВЗ


}



response = requests.get(url, headers=headers, params=params)


data = response.json()

Для новичков в API рекомендуем использовать готовые коннекторы:

  • 🔌 Ozon Connector для 1С
  • 📊 Power BI с шаблонами дашбордов
  • 📈 Google Data Studio (бесплатный вариант)
💡

Перед подключением API проверьте лимиты запросов в вашем тарифе. Для большого количества ПВЗ может потребоваться тариф "Профи" или "Эксперт".

5. Скрытые фишки: как выгрузить данные по ПВЗ в удобном формате

Многие продавцы не знают, что в Ozon Seller есть скрытые инструменты для работы с ПВЗ. Вот несколько лайфхаков:

Способ 1: Отчёт «Детализация по заказам»

Перейдите в Заказы → Архив заказов, экспортируйте данные в Excel и отфильтруйте по столбцу Тип доставки = ПВЗ. В результате вы получите список всех заказов с указанием конкретного ПВЗ, суммы и статуса.

Способ 2: Карта тепловой активности

В разделе Аналитика → Карта продаж можно включить слой «ПВЗ» и увидеть географическое распределение выручки. Это поможет выявить самые прибыльные и убыточные регионы.

Способ 3: Интеграция с Яндекс Картами

Экспортируйте список ПВЗ с координатами (их можно получить через API) и загрузите в Яндекс Карты. Это позволит визуализировать зоны покрытия и планировать открытие новых точек.

Как получить координаты ПВЗ

1. В отчёте "Доставка и логистика" экспортируйте данные с колонкой "Адрес ПВЗ".

2. Используйте сервис Geocoder (например, Яндекс Геокодер) для преобразования адресов в координаты.

3. Загрузите результат в Google My Maps или QGIS для анализа.

6. Типичные ошибки при анализе выручки ПВЗ

Даже опытные продавцы допускают ошибки, которые искажают реальную картину. Вот самые распространённые:

Ошибка 1: Игнорирование возвратов

Многие смотрят только на сумму продаж, забывая вычесть возвраты. Например, если по ПВЗ «Ленинская, 15» выручка составила 500 000 ₽, но возвраты — 80 000 ₽, реальная выручка — 420 000 ₽.

Ошибка 2: Неучёт сезонности

Выручка по ПВЗ может сильно варьироваться в зависимости от сезона. Например, точки в курортных городах приносят максимум дохода летом, а в декабре их выручка падает.

Ошибка 3: Смешивание данных по FBS и FBO

Если вы работаете и по модели FBS (склад Ozon), и по FBO (собственная логистика), разделяйте эти потоки. Логистические расходы и комиссии будут разными.

💡

Всегда анализируйте выручку по ПВЗ в динамике — сравнивайте текущий период с предыдущим месяцем/годом. Разовое значение не даёт полной картины.

⚠️ Внимание: Если выручка по ПВЗ стабильно падает, проверьте:
  • 📉 Изменились ли условия аренды точки?
  • 🚚 Не ухудшилось ли качество логистики (задержки, потери заказов)?
  • 🌍 Появились ли конкуренты поблизости?

7. Как автоматизировать мониторинг выручки по ПВЗ

Ручной анализ отнимает много времени, особенно если у вас сеть из 10+ точек. Автоматизировать процесс помогут:

1. Сервисы аналитики

  • 📊 Sellerboard — собирает данные по ПВЗ и строит отчёты
  • 📈 SplitMetrics — анализирует эффективность точек выдачи
  • 🤖 Peak — AI-инструмент для прогнозирования выручки

2. Собственные скрипты

Если у вас есть программист в команде, можно написать скрипт на Python, который будет:

  • 🔄 Ежедневно выгружать данные по ПВЗ через API
  • 📊 Строить графики в Google Sheets
  • 🔔 Отправлять уведомления при падении выручки

3. Интеграция с 1С

Для крупных продавцов оптимально настроить автоматический импорт данных из Ozon в 1С. Это позволит:

  • 💰 Вести учёт по каждому ПВЗ в бухгалтерии
  • 📑 Автоматически формировать отчёты для налоговой
  • 📊 Сравнивать выручку с расходами на аренду и зарплаты
💡

Для автоматизации можно использовать готовые решения, например, МойСклад или Атолл Онлайн. Они поддерживают интеграцию с Ozon и умеют разделять данные по ПВЗ.

FAQ: Частые вопросы о выручке ПВЗ на Ozon

Можно ли увидеть выручку по конкретному ПВЗ за сегодня?

Нет, данные в личном кабинете обновляются с задержкой в 1–2 дня. Для оперативного мониторинга используйте Ozon API или подключите сервис аналитики типа Sellerboard.

Почему выручка по ПВЗ в отчёте не совпадает с суммой на счёту?

Это происходит из-за:

  • 💳 Удержанных комиссий (они списываются позже)
  • 🔄 Возвратов, которые ещё не обработали
  • 📅 Задержки в обновлении данных (до 48 часов)

Сверяйте данные с разделом Финансы → Транзакции.

Как посчитать рентабельность ПВЗ?

Используйте формулу: 



Рентабельность (%) = (Чистая прибыль / Расходы на ПВЗ) × 100

Где расходы на ПВЗ включают аренду, зарплаты, коммунальные платежи и логистику.

Можно ли сравнить выручку своего ПВЗ с конкурентами?

Прямых данных по конкурентам Ozon не предоставляет. Но можно использовать косвенные методы:

  • 📊 Анализ трафика в регионе через Яндекс.Радар
  • 🛒 Мониторинг цен и ассортимента конкурентов на ПВЗ
  • 🗺️ Карты тепловой активности (показывают плотность заказов)
Что делать, если выручка по ПВЗ резко упала?

Проверьте:

  • 📉 Изменились ли условия работы с Ozon (комиссии, тарифы)?
  • 🚛 Не было ли проблем с логистикой (задержки, потери заказов)?
  • 🌍 Появились ли новые ПВЗ конкурентов поблизости?
  • 📢 Не ухудшилось ли качество обслуживания (отзывы клиентов)?

Если проблема неясна, запросите детализированный отчёт в поддержке Ozon через тикет.

Как изменить категорию товара на Ozon Seller: 3 рабочих способа + частые ошибки

Неправильно выбранная категория на Ozon — одна из главных причин низкой видимости товара, штрафов за несоответствие или даже блокировки карточки. По статистике платформы, 38% новых продавцов сталкиваются с проблемами из-за ошибок категоризации в первые 3 месяца работы. При этом изменить категорию после публикации товара можно далеко не всегда: система накладывает ограничения в зависимости от статуса карточки, типа товара и даже истории продаж.

В этой статье разберём все доступные способы смены категории — от ручного редактирования в личном кабинете до массового импорта через API, а также расскажем, как избежать типичных ошибок, из-за которых Ozon может отклонить ваши правки. Особое внимание уделим нюансам для товаров с вариациями, комплектами и FBS/FBO-схемам, где правила категоризации отличаются.

Если вы только начинаете работать с Ozon Seller, важно понимать: категория влияет не только на видимость в поиске, но и на комиссию маркетплейса, доступные способы доставки и даже требования к упаковке. Например, товары категории «Электроника» проверяются строже, чем «Товары для дома», а для категории «Детские товары» требуются дополнительные сертификаты.

1. Когда можно и нельзя менять категорию товара на Ozon

Платформа разрешает редактировать категорию не для всех товаров. Основные ограничения связаны со статусом карточки и типом товара:

  • 🔹 Новые товары (неопубликованные): категорию можно менять без ограничений до первой публикации.
  • 🔹 Опубликованные товары без продаж: разрешено изменять категорию, но с проверкой модератором (срок — до 72 часов).
  • 🔹 Товары с продажами: категорию поменять нельзя, кроме случаев, когда новая категория входит в список «родственных» (например, «Смартфоны» → «Аксессуары для смартфонов»).
  • 🔹 Товары в архиве: категорию можно редактировать, но после возврата в продажу карточка пройдёт повторную модерацию.

Исключение составляют товары, попадающие под специальные программы (например, «Ozon Premium» или «Экотовары»). Для них смена категории возможна только через поддержку с обоснованием.

⚠️ Внимание: Если вы измените категорию для товара с действующими заказами, Ozon может аннулировать невыполненные продажи. Перед редактированием проверьте раздел «Заказы → В работе».

Также учитывайте, что некоторые категории имеют дополнительные требования. Например, для «Продукты питания» потребуется указать срок годности, а для «Одежды» — размерную сетку. Если новая категория требует атрибутов, которых нет в текущей карточке, система запросит их заполнить.

2. Способ 1: Изменение категории через личный кабинет Ozon Seller

Самый простой метод — редактирование напрямую в интерфейсе. Подходит для единичных товаров или небольших каталогов (до 50 позиций).

Пошаговая инструкция:

  1. Перейдите в раздел Товары → Список товаров.
  2. Найдите нужный товар по артикулу или названию.
  3. Нажмите на три точки (⋮) в строке товара и выберите «Редактировать».
  4. В блоке «Категория» нажмите «Изменить».
  5. Выберите новую категорию из дерева или воспользуйтесь поиском.
  6. Подтвердите изменения и дождитесь модерации (от 1 часа до 3 дней).

Если кнопка «Изменить» неактивна, проверьте:

  • 🔸 Статус товара (должен быть «Черновик» или «На модерации» для новых карточек).
  • 🔸 Наличие продаж (если товар уже продавался, смена категории может быть заблокирована).
  • 🔸 Тип товара (для FBO-товаров правила строже, чем для FBS).

3. Способ 2: Массовое изменение категории через Excel/CSV

Для крупных каталогов (от 50 товаров) удобнее использовать импорт через файл. Этот метод позволяет изменить категорию для нескольких позиций одновременно, но требует аккуратности: ошибки в файле приведут к отклонению всех правок.

Алгоритм действий:

  1. Скачайте шаблон для импорта в разделе Товары → Импорт/Экспорт → Скачать шаблон.
  2. В колонке Product ID укажите идентификаторы товаров (не путайте с артикулом!).
  3. В колонке Category ID пропишите числовой ID новой категории (найти его можно в справочнике категорий Ozon).
  4. Заполните обязательные атрибуты новой категории (если они отличаются от текущих).
  5. Сохраните файл в формате .csv (кодировка UTF-8).
  6. Загрузите файл в раздел Импорт/Экспорт → Импорт товаров.

Важные нюансы:

  • 📌 ID категории — это не её название! Например, для «Смартфоны» ID может быть 12345, а для «Чехлы для смартфонов» — 67890.
  • 📌 Если в файле указаны товары с продажами, их категории не изменятся (система проигнорирует эти строки).
  • 📌 Для товаров с вариациями (например, разные цвета одной модели) категорию нужно менять для каждой вариации отдельно.
💡

Перед массовым импортом экспортируйте текущий каталог и сохраните резервную копию. Если что-то пойдёт не так, вы сможете откатить изменения.

Если после импорта часть товаров не обновилась, проверьте лог ошибок в разделе Импорт/Экспорт → История импортов. Частые причины отклонения:

  • 🚫 Неверный Category ID.
  • 🚫 Отсутствуют обязательные атрибуты новой категории.
  • 🚫 Товар участвует в акции или имеет активные заказы.

4. Способ 3: Изменение категории через API Ozon Seller

Для автоматизации или интеграции с 1C/ERP-системами используйте API маркетплейса. Этот метод требует технических навыков, но позволяет гибко управлять категориями в больших объёмах.

Основные этапы:

  1. Получите Client-ID и API-ключ в настройках личного кабинета (Интеграции → API).
  2. Используйте метод POST /v2/product/info/update для изменения категории.
  3. В теле запроса укажите: 
    {

    "items": [
    

    {
    

    "offer_id": "VASH_PRODUCT_ID",
    

    "category_id": NOVYI_CATEGORY_ID
    

    }
    

    ]
    

    }

  4. Обработайте ответ сервера: статус 200 означает успешное обновление, 400 или 403 — ошибку.

Пример запроса на Python (с использованием библиотеки requests): 

import requests


url = "https://api-seller.ozon.ru/v2/product/info/update"


headers = {


"Client-Id": "VASH_CLIENT_ID",


"Api-Key": "VASH_API_KEY",


"Content-Type": "application/json"


}


payload = {


"items": [


{


"offer_id": "12345678",


"category_id": 98765


}


]


}



response = requests.post(url, headers=headers, json=payload)


print(response.json())

Ограничения API:

  • 🛑 Лимит запросов: 1000 в час (превышение ведёт к блоку на 1 день).
  • 🛑 Для товаров с продажами категорию изменить нельзя (вернётся ошибка "category_change_forbidden").
  • 🛑 Новая категория должна быть совместима с текущими атрибутами товара.
Как проверить совместимость категорий через API?

Используйте метод GET /v1/category/tree, чтобы получить дерево категорий, и сверьте attributes текущей и новой категории. Если обязательные атрибуты не совпадают, запрос на смену категории будет отклонён.

5. Частые ошибки при смене категории и как их избежать

Даже опытные продавцы сталкиваются с проблемами при изменении категорий. Вот TOP-5 ошибок и способы их решения:

Ошибка Причина Решение
Категория не меняется для товара с продажами Ozon блокирует редактирование для товаров с историей продаж Создайте новую карточку в нужной категории и архивируйте старую
Ошибка «Недопустимый Category ID» Указан несуществующий или устаревший ID категории Проверьте актуальный Category ID в справочнике Ozon

Модерация отклоняет изменение категории Несоответствие товара новой категории (например, наушники в категории Одежда) Выберите родственную категорию или уточните характеристики товара

Ещё одна распространённая проблема — потеря позиций в поиске после смены категории. Это связано с тем, что Ozon сбрасывает историю показа карточки при изменении категории. Чтобы минимизировать потери:

  • 📉 Заранее настройте релевантные ключевые слова для новой категории.
  • 📉 Используйте кросспромо (указывайте старую категорию в атрибутах новой).
  • 📉 Обновите title и description под запросы новой категории.
⚠️ Внимание: Если вы часто меняете категории (более 3 раз за месяц), Ozon может расценить это как манипуляцию поисковым алгоритмом и понизить приоритет ваших товаров.

6. Особенности смены категории для FBS и FBO

Тип логистики (FBS или FBO) влияет на правила категоризации. Основные различия:

  • 📦 FBS (доставка через Ozon):
    • Категорию можно менять только для товаров на складе Ozon (не в пути к покупателю).
    • Для некоторых категорий (например, «Крупная бытовая техника») требуется согласование с логистическим центром.
    • Изменение категории может повлечь пересчёт тарифов хранения.
  • 🚚 FBO (самостоятельная доставка):
    • Больше свободы в смене категорий, но строже проверка на соответствие.
    • Для товаров с гарантией (например, электроника) категория должна совпадать с данными в ГИС Маркировка.
    • При смене категории может потребоваться повторная сертификация (например, для «Детских товаров»).

Для FBS-товаров перед изменением категории проверьте:

  1. Наличие товара на складе (Товары → Остатки).
  2. Совместимость новой категории с габаритно-весовыми характеристиками (например, категория «Мебель» требует указания веса и объёма).
  3. Отсутствие блокировок со стороны логистики (проверяется в разделе Логистика → Проблемы).

Например, переводить смартфон из категории «Электроника» в «Аксессуары» нельзя — это приведёт к аннулированию гарантии для покупателей.

7. Как выбрать правильную категорию: советы по SEO и конверсии

Выбор категории влияет не только на модерацию, но и на видимость товара в поиске и каталоге. Вот критерии оптимальной категории:

  • 🎯 Релевантность: категория должна точно отражать назначение товара. Например, беспроводные наушники относятся к «Аудиотехника», а не к «Аксессуары для телефонов».
  • 🎯 Конкурентность: в некоторых категориях (например, «Смартфоны») высокая конкуренция, и новичкам сложно пробиться в топ. Рассмотрите нишевые подкатегории.
  • 🎯 Спрос: проверьте количество поисковых запросов в категории через Ozon Trend или Wordstat.
  • 🎯 Комиссия: в некоторых категориях (например, «Ювелирные изделия») комиссия Ozon выше средней.

Инструменты для анализа категорий:

  • 🔍 Ozon Trend — показывает динамику спроса по категориям.
  • 🔍 Метрика Ozon — анализирует трафик и конверсию в разных категориях.
  • 🔍 Справочник категорий Ozon — содержит актуальные Category ID и требования к атрибутам.

Пример: если вы продаёте умные часы, сравните две категории:

  • «Электроника → Умные часы» — высокая конкуренция, но большой трафик.
  • «Спорт и отдых → Фитнес-браслеты» — меньше конкуренции, но и меньший спрос.
💡

Перед сменой категории проанализируйте топ-10 товаров в новой категории по ключевым метрикам: цена, рейтинг, количество отзывов. Это поможет адаптировать вашу карточку под ожидания покупателей.

8. Что делать, если Ozon не позволяет поменять категорию

Если все способы исчерпаны, а категорию изменить не удаётся, попробуйте эти альтернативные решения:

  • 🔄 Создайте новую карточку в нужной категории и перенесите туда:
    • Фото и описание (с доработкой под новую категорию).
    • Остатки (через Товары → Остатки).
    • Отзывы (если они есть, обратитесь в поддержку с просьбой перенести).
  • 📧 Напишите в поддержку с обоснованием:
    • Укажите Product ID и текущую/желаемую категорию.
    • Приложите скриншоты товара, подтверждающие его принадлежность к новой категории.
    • Ссылку на аналогичные товары в нужной категории (если есть).
  • 🛠 Используйте «родственные» категории:
    • Например, для чехла для телефона можно выбрать «Аксессуары для смартфонов» вместо «Электроника».
    • Проверьте допустимые переходы в документации Ozon.

Если категорию нельзя изменить из-за продаж, а товар больше не актуален в текущей категории, рассмотрите варианты:

  • 💡 Архивируйте старую карточку и создайте новую.
  • 💡 Добавьте товар в комплект с другим товаром подходящей категории.
  • 💡 Продайте остатки по акции и снимите товар с продажи.
⚠️ Внимание: При создании новой карточки вместо старой учтите, что история продаж и отзывы не переносятся автоматически. Это может временно снизить конверсию.

Ответы на частые вопросы

Можно ли поменять категорию для товара, который уже купили?

Нет, для товаров с историей продаж смена категории заблокирована. Исключение — переход в «родственную» категорию (например, «Ноутбуки» → «Компьютерная периферия»), но это требует согласования с поддержкой.

Сколько времени занимает модерация после смены категории?

Срок зависит от загруженности системы:

  • Для новых товаров — от 1 часа до 1 дня.
  • Для опубликованных товаров — до 3 дней.
  • Для товаров с высоким рейтингом (4.5+) модерация может затянуться до 5 дней.

Отслеживайте статус в разделе Товары → На модерации.

Что будет, если указать неверную категорию?

Ozon может:

  • Отклонить карточку на модерации.
  • Понизить товар в поисковой выдаче.
  • Наложить штраф за несоответствие (от 500 до 5000 ₽ в зависимости от категории).
  • Заблокировать карточку, если категория противоречит ассортиментной матрице.

Например, если вы укажете игрушку в категории «Бытовая химия», карточка будет удалена без возможности восстановления.

Как узнать, какие категории являются «родственными»?

Список допустимых переходов между категориями не публикуется, но вы можете:

  • Просмотреть аналогичные товары в каталоге Ozon и их категории.
  • Обратиться в поддержку с запросом о проверке совместимости категорий.
  • Использовать метод GET /v1/info/categories/compatible в API (требуются права доступа).

Например, для категории «Смартфоны» родственными считаются «Аксессуары для смартфонов» и «Гарнитуры», но не «Ноутбуки».

Нужно ли менять категорию, если товар подходит к нескольким?

Выбирайте категорию, которая:

  • Максимально точно описывает основное назначение товара.
  • Имеет наибольший спрос среди вашей целевой аудитории.
  • Содержит меньше конкурентов (проверяйте через Ozon Trend).

Например, умные часы с функцией фитнес-трекера лучше разместить в категории «Умные часы», а не «Фитнес-браслеты», если основная аудитория ищет именно часы.

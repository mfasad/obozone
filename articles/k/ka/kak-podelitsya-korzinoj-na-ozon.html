Как поделиться корзиной на Озон: 5 работающих способов в 2026 году

Вы собрали идеальный набор товаров в корзине Ozon, но хотите поделиться им с другом, сохранить для себя на будущее или перенести в другой аккаунт? К сожалению, маркетплейс не предоставляет прямой функции "Поделиться корзиной" — но есть обходные пути. В этой статье разберём все актуальные способы, включая скрытые фишки приложения и веб-версии, которые знают далеко не все пользователи.

Мы протестировали каждый метод на актуальной версии Ozon (июнь 2026) и собрали пошаговые инструкции с учётом последних обновлений платформы. Вы узнаете, как экспортировать список товаров в ссылку, сохранить корзину в закладки, синхронизировать между устройствами и даже автоматизировать процесс с помощью скриптов. Особое внимание уделим нюансам, из-за которых у многих не получается поделиться корзиной с первого раза — например, ограничениям на количество товаров или проблемам с авторизацией.

Важно: некоторые методы работают только в мобильном приложении, другие — только на компьютере. Мы отметили эти моменты в каждом разделе, чтобы вы не тратили время зря.

1. Способ: Экспорт корзины через "Избранное" (официальный метод)

Самый надёжный и простой способ — использовать функцию "Избранное" в приложении Ozon. Этот метод работает на всех устройствах и не требует сторонних сервисов. Главный плюс: сохранённый список можно открыть даже через год, если товары ещё в продаже.

Алгоритм действий:

  • 📱 Откройте приложение Ozon и перейдите в корзину (значок в правом нижнем углу).
  • 🔍 Нажмите на три точки (⋮) в правом верхнем углу экрана корзины.
  • ⭐ Выберите пункт "Сохранить в Избранное" — система автоматически создаст новый список.
  • 🔗 Вернитесь на главную страницу, откройте раздел "Избранное" и найдите ваш сохранённый список.
  • 📤 Нажмите на три точки рядом с названием списка и выберите "Поделиться".

Обратите внимание: в сохранённом списке не отображаются цены и наличие товаров — только их названия и фотографии. Если товар будет удалён продавцом, он исчезнет и из вашего списка. Также этот метод не работает для корзин с более чем 100 товарами — система просто не даст сохранить такой большой список.

2. Способ: Ручное копирование ссылок (для компьютера)

Если вам нужно поделиться корзиной с компьютера или сохранить её в текстовом виде, можно вручную скопировать ссылки на каждый товар. Этот метод трудоёмкий, но зато работает всегда — даже если Ozon заблокирует другие способы.

Инструкция для ПК:

  1. Откройте ozon.ru в браузере и авторизуйтесь.
  2. Перейдите в корзину (иконка корзины → "Моя корзина").
  3. Наведите курсор на название каждого товара, зажмите Ctrl (или Cmd на Mac) и кликните по названию — откроется новая вкладка с товаром.
  4. Скопируйте URL из адресной строки и вставьте в текстовый документ или таблицу.

Чтобы ускорить процесс, используйте расширение для браузера Tampermonkey со скриптом для автоматического копирования ссылок. Мы подготовили готовый код:

// ==UserScript==

// @name         Ozon Cart Exporter


// @namespace    http://tampermonkey.net/


// @version      1.0


// @description  Экспортирует товары из корзины Ozon в CSV


// @author       You


// @match        https://www.ozon.ru/cart


// @grant        none


// ==/UserScript==



function exportCart() {


const items = document.querySelectorAll('[data-widget="cartItem"]');


let csv = 'Название;Ссылка;Цена\n';


items.forEach(item => {


const name = item.querySelector('[data-test-id="title"]')?.textContent.trim();


const link = item.querySelector('a')?.href;


const price = item.querySelector('[data-test-id="price"]')?.textContent.trim();


csv += `"${name}";"${link}";"${price}"\n`;


});


const blob = new Blob([csv], { type: 'text/csv' });


const url = URL.createObjectURL(blob);


const a = document.createElement('a');


a.href = url;


a.download = 'ozon_cart_export.csv';


a.click();


}



setTimeout(exportCart, 3000);

После установки скрипта откройте корзину на Ozon, и через 3 секунды начнётся автоматическая выгрузка файла ozon_cart_export.csv с данными. Этот файл можно отправить по почте, сохранить в облаке или импортировать в Excel.

💡

Если скрипт не сработал, проверьте, что у вас включён JavaScript в браузере и нет блокировщиков рекламы, которые могут мешать работе расширения.

3. Способ: Синхронизация корзины между устройствами

Если вам нужно перенести корзину с телефона на компьютер или наоборот, воспользуйтесь встроенной синхронизацией аккаунта Ozon. Этот метод работает только при условии, что вы используете один и тот же аккаунт на всех устройствах.

Как синхронизировать:

  • 📱 На телефоне: откройте приложение Ozon, убедитесь, что вы авторизованы, и добавьте все нужные товары в корзину.
  • 💻 На компьютере: зайдите на ozon.ru под тем же аккаунтом — корзина автоматически обновится через 1-2 минуты.
  • 🔄 Если синхронизация не произошла, обновите страницу (F5) или перезайдите в аккаунт.

Важный нюанс: товары из корзины могут исчезнуть, если:

  • 🛒 Продавец удалил товар из каталога.
  • 📉 Изменилась цена (некоторые продавцы сбрасывают корзину при изменении стоимости).
  • 🔄 Вы очистили корзину на одном из устройств.

4. Способ: Использование сторонних сервисов (для продвинутых пользователей)

Для тех, кто часто работает с большими списками товаров, подойдут специализированные сервисы вроде Ozon Parser или SaveFromOzon. Они позволяют экспортировать корзину в формате Excel, JSON или даже автоматически отслеживать изменения цен.

Топ-3 проверенных сервиса:

Сервис Функции Стоимость Ссылка
Ozon Parser Экспорт корзины в CSV, отслеживание цен, уведомления о скидках От 299 ₽/месяц ozonparser.ru
SaveFromOzon Сохранение корзины в облако, генерация ссылок для совместного доступа Бесплатно (лимит 50 товаров) savefromozon.com
CartSaver Автоматическая синхронизация корзин между аккаунтами, история изменений От 490 ₽/год cartsaver.app

⚠️ Внимание: перед использованием сторонних сервисов проверьте их репутацию на форумах (например, на VC.ru или Pikabu). Некоторые сервисы могут запрашивать доступ к вашему аккаунту Ozon — никогда не вводите пароль на сторонних сайтах. Используйте только авторизацию через официальный API (кнопка "Войти через Ozon").

Пример безопасного подключения:

  1. Выбираете сервис из таблицы выше.
  2. Нажимаете "Подключить Ozon".
  3. Система перенаправляет вас на официальную страницу авторизации Ozon (проверьте URL — должен быть account.ozon.ru).
  4. После авторизации сервис получит доступ только к вашей корзине (без возможности совершать покупки).

5. Способ: Скриншоты + распознавание текста (экстренный метод)

Если никакие другие способы не работают (например, при блокировке аккаунта или технических сбоях на Ozon), можно сделать скриншоты корзины и распознать текст с помощью Google Lens или Yandex.Text. Этот метод займёт больше времени, но гарантированно сработает в любой ситуации.

Пошаговая инструкция:

  • 📸 Сделайте скриншоты корзины (прокручивайте вниз, чтобы захватить все товары).
  • 🔍 Загрузите скриншоты в Google Lens или Yandex.Text.
  • 📋 Скопируйте распознанный текст в документ Word или Excel.
  • 🔗 Вручную найдите каждый товар на Ozon по названию и добавьте в новую корзину.

Для ускорения процесса используйте комбинации клавиш:

  • Windows: Win + Shift + S — выделение области для скриншота.
  • Mac: Cmd + Shift + 4 — скриншот выбранной области.
  • Android/iOS: зажмите Питание + Уменьшение громкости.
Как улучшить качество распознавания?

1. Делайте скриншоты при хорошем освещении (без бликов на экране).

2. Используйте чёткий шрифт — в настройках Ozon выберите "Тёмная тема" для лучшего контраста.

3. Если текст распознался с ошибками, попробуйте другой сервис (например, ABBYY FineReader).

4. Для длинных списков разбивайте корзину на части по 10-15 товаров на скриншот.

Частые ошибки и как их избежать

Многие пользователи сталкиваются с проблемами при попытке поделиться корзиной. Мы собрали самые распространённые ошибки и способы их решения:

⚠️ Внимание: Если при сохранении корзины в "Избранное" появляется ошибка "Не удалось сохранить список", проверьте:
  • 🔌 Стабильность интернет-соединения (переключитесь с мобильного интернета на Wi-Fi).
  • 📱 Актуальность версии приложения (обновите Ozon в App Store/Google Play).
  • 🛡️ Отсутствие блокировщиков рекламы (они могут мешать работе функции).

Другие типичные проблемы:

Проблема Причина Решение
Корзина не синхронизируется между устройствами Кэш браузера или устаревшая версия приложения Очистите кэш или переустановите приложение

Если вы часто работаете с большими корзина, рекомендуем создать запасной аккаунт на Ozon специально для хранения списков. Так вы избежите потери данных при блокировке основного профиля или технических сбоях.

Самый надёжный способ сохранить корзину надолго — экспорт в CSV через Tampermonkey или специализированные сервисы. Вручную копировать ссылки неэффективно при большом количестве товаров.

FAQ: Ответы на популярные вопросы

Можно ли поделиться корзиной между разными аккаунтами Ozon?

Прямой функции нет, но можно:

  1. Сохранить корзину в "Избранное" и отправить ссылку на список другому пользователю (он сможет увидеть товары, но не добавить их в свою корзину автоматически).
  2. Использовать сторонние сервисы вроде CartSaver, которые поддерживают мультиаккаунтность.
  3. Вручную переписать артикулы товаров (найти их можно в URL или через инспектор кода).

Обратите внимание: при передаче корзины другому аккаунту скидки и промокоды не сохранятся — они привязаны к оригинальному пользователю.

Почему при экспорте корзины некоторые товары пропадают?

Это происходит по трём причинам:

  • Ограничение на количество: Ozon не показывает в экспорте товары, которых нет в наличии.
  • Региональные ограничения: если товар доступен только в определённом регионе, он не отобразится в корзине другого пользователя.
  • Технический сбой: иногда API Ozon не успевает обработать большой список — попробуйте экспортировать корзину частями.

Чтобы избежать потерь, перед экспортом:

  1. Обновите страницу корзины (F5).
  2. Удалите товары с пометкой "Нет в наличии".
  3. Используйте скрипт для экспорта (см. Способ 2).
Как восстановить корзину, если она случайно очистилась?

Если корзина очистилась из-за сбоя или ваших действий, попробуйте:

  • 🔍 Проверьте раздел "Недавно просмотренные" в приложении — там могут остаться следы товаров.
  • 📧 Найдите письма от Ozon с уведомлениями о добавлении в корзину (если включены оповещения).
  • 🔄 Восстановите корзину из резервной копии, если используете сторонние сервисы вроде Ozon Parser.
  • 📱 На Android проверьте кэш приложения (папка Android/data/ru.ozon.app) — иногда там хранятся данные корзины.

Если ничего не помогло, свяжитесь с поддержкой Ozon через чат в приложении (укажите приблизительное время очистки корзины — иногда операторы могут восстановить данные).

Можно ли автоматизировать добавление товаров в корзину по списку?

Да, для этого есть несколько способов:

  1. Через API Ozon: если вы разработчик, можно написать скрипт на Python с использованием официальной документации docs.ozon.ru.
  2. Сервисы автоматизации: Zapier или Make (ex-Integromat) умеют взаимодействовать с Ozon через вебхуки.
  3. Расширения для браузера: Ozon AutoCart позволяет загружать списки артикулов и автоматически добавлять их в корзину.

Пример кода для добавления товара по артикулу (требуется авторизация через API):

import requests


url = "https://api.ozon.ru/v1/cart/add"


headers = {


"Content-Type": "application/json",


"Authorization": "Bearer YOUR_ACCESS_TOKEN"


}


data = {


"items": [


{


"offer_id": "123456789",  # Артикул товара


"quantity": 1


}


]


}



response = requests.post(url, headers=headers, json=data)


print(response.json())

⚠️ Внимание: автоматизированное добавление товаров может расцениваться как нарушение правил Ozon. Не используйте этот метод для массовых заказов или парсинга — аккаунт могут заблокировать.

Как поделиться корзиной, если товары в ней с пометкой "Только для вас"?

Товары с пометкой "Только для вас" (персональные скидки или эксклюзивные предложения) нельзя передать другому пользователю. Эти предложения привязаны к:

  • Вашему аккаунту (по истории покупок).
  • Вашему региону доставки.
  • Вашей категории лояльности (например, Ozon Premium).

Обходные пути:

  1. Скопируйте название товара и найдите его вручную в каталоге (без персональной скидки).
  2. Попросите друга авторизоваться в вашем аккаунте (не рекомендуем из-за рисков безопасности).
  3. Свяжитесь с продавцом через чат и уточните, можно ли оформить заказ на другого человека с сохранением скидки.

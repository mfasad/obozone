Как полностью удалить rich-контент с карточки товара на Ozon: все рабочие методы

Rich-контент на Ozon — это интерактивные элементы карточки товара, которые должны повышать конверсию: видеообзоры, 3D-модели, гифки, сравнительные таблицы или блоки с часто задаваемыми вопросами. Однако не всегда такой контент работает на пользу продажам. Иногда он тормозит загрузку страницы, отвлекает покупателей от ключевой информации или просто устарел. В этой статье разберём, как полностью удалить rich-контент с карточки товара на Ozon — как через личный кабинет продавца, так и программными методами (API, массовое редактирование).

Важно понимать: процедура удаления зависит от того, как именно был добавлен rich-контент — вручную через интерфейс Ozon Seller, автоматически через фид или с помощью партнёрских сервисов (например, RichContent Pro или ContentMonster). Мы рассмотрим все сценарии, включая нюансы для товаров на FBS и FBO, а также объясним, почему иногда контент "возвращается" после удаления.

Что такое rich-контент на Ozon и зачем его удалять

Rich-контент (от англ. rich content — "обогащённый контент") — это любой мультимедийный или интерактивный элемент на карточке товара, выходящий за рамки стандартных фотографий и текстового описания. На Ozon к нему относятся:

  • 🎥 Видеообзоры (загруженные продавцом или сгенерированные Ozon автоматически)
  • 🔄 3D-модели (вращаемые изображения товара)
  • 📊 Сравнительные таблицы (характеристики в виде интерактивных блоков)
  • ❓ FAQ-блоки (вопросы и ответы, разворачиваемые покупателем)
  • 🎯 Гифки и анимации (демонстрация использования товара)
  • 📌 Прикреплённые документы (PDF-инструкции, сертификаты)

Причины для удаления rich-контента:

  • 🐢 Тормозит загрузку страницы — особенно актуально для мобильных пользователей с медленным интернетом. По статистике Ozon, карточки с видео грузятся на 1.5–2 секунды дольше, что увеличивает отказы.
  • 💸 Не оправдывает вложения — если конверсия с rich-контентом не выросла (или даже упала) после A/B-тестов.
  • 🔄 Устаревшие данные — например, видео с прошлогодней моделью товара или неактуальные характеристики.
  • 🚫 Нарушение правил Ozon — если контент содержит запрещённую информацию (логотипы конкурентов, неверные сравнения).
⚠️ Внимание: Удаление rich-контента может временно снизить позицию товара в поисковой выдаче Ozon, если алгоритмы учитывали его как фактор ранжирования. Однако через 1–2 недели позиции обычно восстанавливаются.

Способ 1: Удаление rich-контента через личный кабинет Ozon Seller

Самый простой метод — ручное удаление через интерфейс продавца. Подходит для единичных товаров или небольших каталогов. Инструкция актуальна для версии Ozon Seller 2026 года.

Шаги:

  1. Авторизуйтесь в Ozon Seller и перейдите в раздел Товары → Мои товары.
  2. Найдите нужный товар по артикулу или названию (используйте фильтр по статусу "Активен").
  3. Кликните на название товара, чтобы открыть его карточку.
  4. Прокрутите до блока Rich-контент (обычно расположен под описанием и характеристиками).
  5. Нажмите на иконку корзины (🗑️) рядом с каждым элементом, который хотите удалить. Для видео и 3D-моделей может потребоваться подтверждение.
  6. Сохраните изменения кнопкой Применить внизу страницы.

Обратите внимание:

  • 🔍 Если кнопки удаления нет, значит rich-контент был добавлен через фид или API — его нужно убирать другими способами (см. следующие разделы).
  • ⏳ Изменения применяются не мгновенно: на синхронизацию с фронтендом Ozon уходит до 24 часов.
  • 📌 Видео, загруженные Ozon автоматически (например, из ревью покупателей), удалить нельзя — только через поддержку.

Способ 2: Массовое удаление через фид (для большого каталога)

Если rich-контент был добавлен через фид данных (XML, YML, Excel), его можно убрать массово, обновив фид и загрузив его повторно. Этот метод подходит для каталогов от 50+ товаров.

Алгоритм:

  1. Скачайте актуальный фид из Ozon Seller → Товары → Импорт/Экспорт → Экспорт товаров.
  2. Откройте файл в редакторе (например, Excel или Notepad++) и найдите колонки, связанные с rich-контентом:
    • rich_content_video
    • rich_content_3d
    • rich_content_faq
    • rich_content_images
  • Удалите содержимое этих колонок или заполните их значением NULL.
  • Сохраните файл в исходном формате (например, .xlsx или .xml).
  • Загрузите обновлённый фид обратно через Импорт товаров.

    • Важные нюансы:

    • 🔄 Если rich-контент был добавлен не через фид, а вручную, этот метод не сработает — данные из фида не перезапишут ручные настройки.
    • 📂 Для товаров на FBO (доставка Ozon) массовое удаление может занять до 48 часов.
    • 🛑 Если после загрузки фида rich-контент остался, проверьте, не блокирует ли его партнёрский сервис (например, ContentMonster может автоматически восстанавливать контент).

    Способ 3: Удаление через API Ozon (для технически подкованных)

    Если вы используете API Ozon для управления каталогом, rich-контент можно удалить программно. Этот метод требует знаний работы с JSON и HTTP-запросами, но позволяет автоматизировать процесс для тысяч товаров.

    Основные шаги:

    1. Получите API-key и Client-ID в разделе Настройки → Интеграции → API.
    2. Используйте метод POST /v2/product/info/update для обновления данных товара.
    3. В теле запроса передайте параметры rich-контента со значением null: 
      {

      "items": [
      

      {
      

      "offer_id": "12345678", // Ваш артикул
      

      "rich_content": {
      

      "video": null,
      

      "3d_model": null,
      

      "faq": null
      

      }
      

      }
      

      ]
      

      }

    4. Отправьте запрос и проверьте ответ сервера. Успешный статус: 200 OK.

    Пример на Python с использованием библиотеки requests:

    import requests


    url = "https://api-seller.ozon.ru/v2/product/info/update"
    

    headers = {
    

    "Client-Id": "ваш_client_id",
    

    "Api-Key": "ваш_api_key",
    

    "Content-Type": "application/json"
    

    }
    

    data = {
    

    "items": [{
    

    "offer_id": "12345678",
    

    "rich_content": {
    

    "video": None,
    

    "3d_model": None
    

    }
    

    }]
    

    }
    


    response = requests.post(url, headers=headers, json=data)
    

    print(response.json())

    ⚠️ Внимание: При работе с API Ozon действуют лимиты на количество запросов: до 1000 в час для стандартного тарифа. Превышение лимита ведёт к блокировке на 1 час.
    💡

    Перед массовым удалением через API протестируйте запрос на 1–2 товарах. Некоторые типы rich-контента (например, 3D-модели) могут требовать дополнительной валидации со стороны Ozon.

    Способ 4: Обращение в поддержку Ozon (если ничего не помогает)

    Если rich-контент не удаляется ни одним из перечисленных способов, остаётся обратиться в службу поддержки Ozon. Это актуально для:

    • 🔒 Контента, добавленного Ozon автоматически (например, видео из отзывов).
    • 🔄 Rich-контента, который "возвращается" после удаления (часто бывает при интеграции с партнёрскими сервисами).
    • 📛 Товаров, заблокированных за нарушение правил размещения контента.

    Как правильно составить запрос:

    1. Перейдите в Помощь → Написать в поддержку.
    2. Выберите тему: Работа с каталогом → Проблемы с контентом.
    3. Укажите:
      • Артикул (offer_id) товара.
      • Тип rich-контента, который нужно удалить.
      • Способы, которые вы уже пробовали (чтобы не получить шаблонный ответ).
      • Скриншот проблемы (если контент отображается некорректно).
  • В поле "Дополнительная информация" уточните, что просите полностью удалить контент, а не скрыть его.

    • Сроки решения:

    • ⏳ Стандартный запрос рассматривается до 48 часов.
    • ⚡ Срочные обращения (через кнопку "Это срочно!") — до 12 часов.
    • 📧 Ответ приходит на email, привязанный к аккаунту Ozon Seller.

    Почему rich-контент может "возвращаться" после удаления

    Частая проблема: продавец удаляет rich-контент, но через несколько дней (или даже часов) он появляется снова. Причины:

    1. Автоматическое восстановление партнёрскими сервисами:

      Сервисы вроде ContentMonster, RichContent Pro или SellerTools могут синхронизировать контент по расписанию. Проверьте настройки интеграции и отключите автоматическое обновление rich-контента.

    2. Действия Ozon:

      Ozon может добавлять видео из отзывов покупателей или 3D-модели из базы поставщиков. Такой контент удаляется только через поддержку.

    3. Ошибки кэширования:

      Иногда старые данные сохраняются в кэше Ozon. Попробуйте очистить кэш браузера или проверьте карточку в режиме инкогнито.

    4. Конфликт фидов:

      Если вы загружаете данные из нескольких источников (например, одновременно через 1C и Excel), один фид может перезаписывать другой.

    Как предотвратить возврат контента:

    • 🔌 Отключите все сторонние сервисы, которые имеют доступ к вашему каталогу на Ozon.
    • 📋 Ведите журнал изменений: фиксируйте, когда и какой контент был удалён.
    • 🔍 Регулярно проверяйте карточки товаров на наличие "лишнего" контента (можно настроить уведомления через Ozon API).
    💡

    Если rich-контент возвращается в течение суток после удаления, проблема почти всегда кроется в сторонних сервисах или конфликтах фидов. Начните с аудита интеграций.

    Даже опытные продавцы допускают ошибки, которые мешают корректному удалению rich-контента. Рассмотрим самые распространённые:

    Ошибка Последствия Как исправить

    Использование устаревшей версии API Запрос игнорируется или возвращает ошибку 400 Bad Request Обновите документацию API на официальном сайте

    Удаление контента во время пиковой нагрузки (акции, распродажи) Задержки в обработке до 72 часов Планируйте изменения на ночное время или выходные

    Дополнительные рекомендации:

    • 📅 Синхронизируйте действия: если удаляете rich-контент перед запуском рекламной кампании, сделайте это за 3–5 дней, чтобы избежать сбоев.
    • 📊 Анализируйте метрики: после удаления отслеживайте конверсию и отказы — иногда rich-контент маскирует другие проблемы карточки (например, плохие фото).
    • 🔧 Тестируйте на копии: перед массовым удалением проверьте процесс на 1–2 товарах с низким трафиком.

    FAQ: Ответы на частые вопросы

    Могу ли я удалить rich-контент с карточки товара, если он добавлен поставщиком?

    Нет, если rich-контент загружен поставщиком (например, брендом), вы не можете удалить его самостоятельно. Обратитесь к поставщику с просьбой обновить данные или запросите удаление через поддержку Ozon, приложив подтверждение согласия бренда (например, письмо на фирменном бланке).

    Сколько времени занимает удаление rich-контента через фид?

    Срок зависит от типа контента и модели сотрудничества:

    • FBS: 6–24 часа.
    • FBO: до 48 часов (из-за дополнительной модерации).
    • 3D-модели: до 72 часов (требуют проверки на соответствие техническим требованиям).

    Если изменения не применены за указанное время, проверьте статус обработки фида в разделе Импорт/Экспорт → История загрузок.

    Можно ли удалить rich-контент с мобильного приложения Ozon Seller?

    На данный момент (2026 год) мобильное приложение Ozon Seller не поддерживает редактирование rich-контента. Вы можете только просматривать его. Для удаления используйте десктопную версию или API.

    Что делать, если после удаления rich-контента упали продажи?

    Снижение продаж после удаления rich-контента может быть временным (алгоритмы Ozon переиндексируют карточку) или сигнализировать о более глубоких проблемах:

    1. Проверьте конверсию по трафику: если посещаемость осталась прежней, а продажи упали, возможно, rich-контент действительно влиял на решение покупателей.
    2. Проанализируйте отзывы:maybe покупатели упоминали, что видео или 3D-модель помогли им сделать выбор.
    3. Верните часть контента (например, оставьте только видео без 3D-модели) и отслеживайте динамику.

    Если продажи не восстанавливаются в течение 2 недель, рассмотрите возможность создания нового rich-контента с учётом аналитики.

    Как узнать, кто добавил rich-контент на мою карточку — я или Ozon?

    Проверьте источник контента:

    1. Откройте карточку товара в Ozon Seller и найдите блок Rich-контент.
    2. Наведите курсор на элемент (видео, 3D-модель). Если появится подсказка Добавлено автоматически или Источник: Ozon, контент добавлен платформой.
    3. Если подсказки нет или указан ваш логи, контент загружен вами или интегрированным сервисом.

    Для 100% уверенности запросите лог изменений у поддержки, указав артикул товара и тип контента.

Где искать номер селлера на Ozon: все способы для продавцов в 2026 году

Вы зарегистрировались как продавец на Ozon, но не можете найти свой уникальный номер селлера? Этот идентификатор — ключ к взаимодействию с маркетплейсом: без него невозможно оформить возвраты, связаться с поддержкой или подтвердить права на аккаунт. Проблема в том, что платформа не всегда показывает ID на видном месте, а в документах он может быть замаскирован под другими кодами.

В этой статье мы разберём все актуальные способы, как узнать номер селлера на Ozon в 2026 году — от очевидных (личный кабинет) до малоизвестных (API-отчёты). Также выясним, чем отличается Seller ID от Client ID, и почему некоторые продавцы путают их с номером договора. Если вы только начинаете работать с маркетплейсом или столкнулись с блокировкой аккаунта — сохраните эту инструкцию в закладки.

1. Что такое номер селлера на Ozon и зачем он нужен

Номер селлера (или Seller ID) — это уникальный цифровой идентификатор, который Ozon присваивает каждому продавцу при регистрации. Он состоит из 10–12 цифр и используется:

  • 🔹 Для авторизации в API Ozon (если вы подключаете автоматизацию через 1С, МойСклад или другие сервисы).
  • 🔹 При обращении в службу поддержки — без него операторы не смогут найти ваш аккаунт.
  • 🔹 Для подтверждения прав на аккаунт при восстановлении доступа или смене email.
  • 🔹 В договорах и финансовых документах (например, в актах сверки или счетах на комиссию).

Важно не путать Seller ID с другими кодами:

Тип кода Пример Где используется
Seller ID 1234567890 Уникальный номер продавца на платформе
Client ID CL12345678 Идентификатор для API (начинается с CL)
Номер договора ДОГ-2026-00012345 Юридический документ, не заменяет Seller ID
SKU 123456789-1 Идентификатор товара, не связан с аккаунтом

Обратите внимание: с 2023 года Ozon скрыл Seller ID из публичного профиля магазина, поэтому найти его через карточку товара (как это было раньше) невозможно. Теперь идентификатор доступен только в закрытых разделах личного кабинета или документах.

2. Способ 1: как найти Seller ID в личном кабинете Ozon Seller

Самый надёжный метод — проверить данные в веб-версии личного кабинета. Следуйте пошаговой инструкции:

  1. Авторизуйтесь на сайте seller.ozon.ru под своим аккаунтом.
  2. В правом верхнем углу кликните на иконку профиля (аватар или первую букву имени).
  3. В выпадающем меню выберите пункт Настройки.
  4. В разделе Общая информация найдите блок «Идентификатор продавца» — это и есть ваш Seller ID.

Если блок с идентификатором отсутствует:

  • 🔄 Обновите страницу (F5 или Ctrl+R).
  • 🛠️ Попробуйте другой браузер (рекомендуем Google Chrome или Mozilla Firefox).
  • 📧 Убедитесь, что вы вошли под правильным email (некоторые продавцы регистрируют несколько аккаунтов).

Авторизоваться на seller.ozon.ru|Открыть настройки профиля|Найти блок "Идентификатор продавца"|Скопировать 10–12 цифр без пробелов-->

⚠️ Внимание: Если в настройках вместо цифрового кода отображается строка Не указано, это может означать, что ваш аккаунт ещё не прошёл верификацию или находится в статусе «На модерации». В этом случае свяжитесь с поддержкой через чат в личном кабинете.

3. Способ 2: поиск номера селлера в мобильном приложении Ozon Seller

Мобильное приложение Ozon Seller (доступно для Android и iOS) дублирует основные функции веб-версии, но интерфейс отличается. Чтобы найти Seller ID:

  1. Откройте приложение и авторизуйтесь.
  2. Тапните на иконку профиля в нижнем меню (справа).
  3. Прокрутите вниз до блока Информация о продавце.
  4. Найдите строку «Идентификатор» — здесь будет ваш номер.

Если в приложении нет этого раздела:

  • 📱 Обновите приложение до последней версии в App Store или Google Play.
  • 🔄 Выйдите из аккаунта и войдите заново.
  • 💬 Напишите в чат поддержки прямо из приложения (кнопка Помощь в меню).

Важно: В старых версиях приложения (до 2023 года) номер селлера мог отображаться в разделе Финансы → Выплаты. Сейчас этот путь неактуален.

💡

Сфотографируйте или скопируйте Seller ID в безопасное место (например, в Google Keep или 1Password). При блокировке аккаунта этот номер поможет быстрее восстановить доступ.

4. Способ 3: где искать Seller ID в договоре с Ozon

Если у вас есть доступ к документам, заключённым с маркетплейсом, номер селлера можно найти в:

  • 📄 Договоре оферты (обычно в шапке документа или разделе «Реквизиты сторон»).
  • 📊 Актах сверки (в верхней части, рядом с названием вашего магазина).
  • 💰 Счетах на комиссию (в поле «Продавец» или «Исполнитель»).

Пример, как может выглядеть строка в договоре:

Продавец: ИП Иванов Иван Иванович, Seller ID: 1234567890, ИНН 123456789012

Если в документах нет явного упоминания Seller ID, ищите:

  • 🔢 Любой 10–12-значный цифровой код рядом с названием вашей компании.
  • 🏷️ Блоки с пометками ID, Код продавца или Номер партнёра.
⚠️ Внимание: В некоторых шаблонах договоров Ozon указывает только Client ID (например, CL12345678). Этот код не подходит для обращения в поддержку или работы с API! Вам нужен именно Seller ID — чиселой идентификатор без букв.

5. Способ 4: получение Seller ID через API или отчёты

Если вы используете автоматизацию (например, выгружаете заказы через API Ozon или подключены к 1С), номер селлера можно найти в:

  • 📥 Отчётах по заказам (файлы orders.csv или orders.xlsx в разделе Аналитика → Отчёты).
  • 🔌 Настройках API (путь: Настройки → Интеграции → API).
  • 📦 Логах интеграций (если вы используете МойСклад, Bitrix24 или другие CRM).

Пример строки из отчёта по заказам:

"order_id":123456789,"seller_id":9876543210,"status":"delivered"

Если вы не знаете, как работать с API:

  1. Перейдите в Настройки → Интеграции → API.
  2. Скопируйте Client ID (начинается с CL) и API Key.
  3. Обратитесь к разработчику вашей CRM или используйте официальную документацию Ozon API для тестового запроса.
Как выглядит тестовый запрос для получения Seller ID?

Отправьте GET-запрос на эндпоинт https://api-seller.ozon.ru/v1/analytics/data с заголовком Client-ID: CL12345678 и Api-Key: ваш_ключ. В ответе будет поле "seller_id" с нужным номером.

6. Способ 5: запрос Seller ID через поддержку Ozon

Если ни один из способов не сработал, обратитесь в службу поддержки. Для этого:

  1. В личном кабинете перейдите в раздел Помощь (значок ? в правом верхнем углу).
  2. Выберите тему «Вопросы по аккаунту».
  3. В сообщении укажите:
    • 📌 Название вашего магазина.
    • 📧 Email, привязанный к аккаунту.
    • 📄 Номер договора (если есть).
  • Попросите предоставить «идентификатор продавца (Seller ID)».

    • Пример текста для поддержки:

    Здравствуйте! Не могу найти номер селлера (Seller ID) в личном кабинете. Магазин: "НазваниеМагазина", email: example@mail.ru. Пожалуйста, сообщите мой идентификатор для работы с API. Спасибо!
    ⚠️ Внимание: Операторы поддержки Ozon не имеют права сообщать Seller ID по телефону или в соцсетях. Запрос должен идти только через официальный чат в личном кабинете с привязанного email.
    💡

    Если поддержка просит подтвердить личность, прикрепите скан паспорта или выписку из ЕГРИП/ЕГРЮЛ. Без этого данные не предоставят из-за политики безопасности.

    7. Частые ошибки при поиске Seller ID и как их избежать

    Многие продавцы путают Seller ID с другими кодами или допускают ошибки при его поиске. Рассмотрим типичные случаи:

    Ошибка Последствия Как исправить
    Путают Seller ID с Client ID API не работает, поддержка не распознаёт аккаунт Ищите 10–12 цифр, а не код с буквами CL
    Копируют номер с пробелами или тире Система не принимает идентификатор Удалите все символы, кроме цифр
    Ищут ID в публичном профиле магазина Номер скрыт для покупателей с 2023 года Используйте методы из этой статьи
    Пробуют найти ID в мобильном приложении Ozon (для покупателей) В приложении покупателя этой информации нет Установите Ozon Seller для продавцов

    Совет для новых продавцов: Если вы ещё не заключили договор с Ozon, номер селлера будет присвоен только после подписания оферты и прохождения верификации. До этого момента в личном кабинете вместо ID может отображаться статус На модерации.

    8. FAQ: ответы на частые вопросы о Seller ID

    Можно ли узнать Seller ID другого продавца?

    Нет. С 2023 года Ozon скрыл идентификаторы продавцов из публичного доступа. Раньше номер можно было найти в URL карточки товара (например, https://www.ozon.ru/product/...?seller=1234567890), но теперь этот параметр удалён.

    Что делать, если Seller ID изменился?

    Номер селлера присваивается один раз и никогда не меняется. Если в личном кабинете отображается другой ID, это может означать:

    • Вы вошли в другой аккаунт (например, тестовый).
    • Ваш магазин был перенесён на другой юридический статус (ИП → ООО), и создан новый аккаунт.

    Уточните детали в поддержке.

    Можно ли использовать Seller ID для входа в личный кабинет?

    Нет. Для авторизации нужны email и пароль (или код из SMS). Seller ID используется только для идентификации аккаунта в API, документах и обращениях в поддержку.

    Где хранить Seller ID, чтобы не потерять?

    Рекомендуем сохранять номер в:

    • 🔐 Менеджере паролей (1Password, Keeper).
    • 📄 Облачном документе (Google Docs, Notion) с ограниченным доступом.
    • 📱 Заметках телефона (в зашифрованной папке).

    Не храните ID в открытых чатах или пересылках!

    Что делать, если привязанный к Seller ID email утрачен?

    Если вы потеряли доступ к email, привязанному к аккаунту Ozon Seller:

    1. Свяжитесь с поддержкой через чат в личном кабинете (если есть доступ).
    2. Приложите скан паспорта и договора с Ozon.
    3. Укажите Seller ID (если помните) для ускорения проверки.

    Восстановление может занять до 5 рабочих дней.

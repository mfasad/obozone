Где искать код Озон в личном кабинете: полное руководство с картинками

Вы ищете, где найти код Озон в личном кабинете, но не знаете, с какой стороны подойти к этой задаче? Не удивляйтесь — этот вопрос возникает у сотен пользователей ежедневно. Код может понадобиться для подтверждения личности, восстановления доступа, получения скидок или даже для работы с API, если вы продавец. Проблема в том, что Ozon не всегда размещает его на видном месте, а интерфейс личного кабинета регулярно обновляется.

В этой статье мы разберём все возможные сценарии: где искать код в мобильном приложении и на десктопной версии, чем отличается код для покупателей и продавцов, и что делать, если его нет вообще. А ещё — ответим на частые вопросы, например, можно ли использовать один и тот же код несколько раз или как его сбросить. Не теряйте время на бессмысленные клики по меню: следуйте нашей пошаговой инструкции.

Что такое код Озон и зачем он нужен

Прежде чем искать код, важно понять, о каком именно коде идёт речь. На Ozon существует несколько типов кодов, и их легко перепутать:

  • 🔑 Код подтверждения — одноразовый пароль для входа или восстановления аккаунта (приходит в SMS или на почту).
  • 📦 Код товара (артикул) — уникальный идентификатор продукта, нужный продавцам для загрузки каталога.
  • 🔧 Код API — ключ для интеграции с маркетплейсом (только для продавцов).
  • 💳 Код скидки/промокод — активационный купон для покупателей.
  • 🆔 Код личного кабинета — статический идентификатор вашего аккаунта (именно его чаще всего ищут).

В 90% случаев пользователи имеют в виду последний вариант — уникальный код личного кабинета. Он может понадобиться для:

  • 📞 Подтверждения личности при обращении в поддержку.
  • 🔄 Восстановления доступа, если вы потеряли доступ к почте или телефону.
  • 🤝 Передачи прав на аккаунт (например, при продаже бизнеса на Ozon).
  • 📊 Подключения аналитических сервисов (например, Ozon Statistics или МойСклад).

Если вы продавец, то дополнительно может понадобиться код API для автоматической выгрузки товаров или управления заказами через сторонние программы. Об этом мы расскажем отдельно в разделе для бизнес-аккаунтов.

Где найти код Озон в личном кабинете покупателя (пошаговая инструкция)

Если вы обычный покупатель, ваш код спрятан в настройках профиля. Вот точный путь, как его найти:

  1. Откройте Ozon и авторизуйтесь.
  2. Кликните на иконку профиля в правом верхнем углу (на десктопе) или нажмите на аватарку внизу экрана (в мобильном приложении).
  3. Выберите пункт Личные данные (на некоторых версиях — Настройки профиля).
  4. Прокрутите страницу вниз до блока Информация об аккаунте.
  5. Найдите строку Ваш идентификатор (ID) или Код клиента — это и есть нужный код.

В мобильном приложении путь немного другой:

  1. Тапните по иконке профиля внизу экрана.
  2. Выберите Настройки → Личные данные.
  3. Прокрутите до раздела Данные аккаунта — код будет указан под вашим именем.

Если вы не видите код, возможно, ваш аккаунт не полностью верифицирован. В этом случае:

Пройдите верификацию по телефону|Подтвердите email|Добавьте паспортные данные (если требуется)|Обновите приложение до последней версии-->

Важно! Код личного кабинета покупателя — это не то же самое, что номер заказа или артикул товара. Его нельзя использовать для отслеживания посылки или получения скидки.

Как выглядит код Озон для покупателя?

Обычно это комбинация из 8–12 цифр или буквенно-цифровой набор (например, UZ12345678 или 9876543210). Иногда код может начинаться с префикса CL- (от слова "client").

Где искать код Озон в личном кабинете продавца (Seller)

Для продавцов процесс сложнее, так как им может понадобиться не один, а несколько кодов. Разберёмся по порядку.

1. Код личного кабинета (идентификатор продавца)

Этот код нужен для подтверждения прав на аккаунт при обращении в поддержку или передаче бизнеса. Найти его можно так:

  1. Зайдите в Ozon Seller.
  2. В верхнем меню выберите Профиль → Настройки компании.
  3. Прокрутите до блока Основная информация — там будет строка ID продавца или Код партнёра.

2. Код API (для автоматизации)

Если вы подключаете Ozon API для интеграции с 1С, МойСклад или другими сервисами, понадобится отдельный ключ. Вот как его получить:

  1. Перейдите в Настройки → API и интеграции.
  2. Нажмите Создать новый ключ API.
  3. Заполните форму (укажите название ключа и выберите права доступа).
  4. Скопируйте сгенерированный код — он отобразится только один раз!

Предупреждение: Ключ API — это как пароль от вашего бизнеса. Никому его не передавайте, даже сотрудникам Ozon.

💡

Сохраните ключ API в защищённом менеджере паролей (например, 1Password или KeePass). Если потеряете, придётся создавать новый, а старый перестанет работать.

3. Код для подключения к FBS/FBO

Если вы работаете по схеме FBS (склад Ozon) или FBO (самостоятельная доставка), может понадобиться дополнительный код для подключения склада. Его выдаёт менеджер Ozon после заключения договора. Искать в личном кабинете бесполезно — его присылают на email.

Тип кода Где искать Для чего нужен
ID продавца Настройки компании → Основная информация Подтверждение аккаунта, обращение в поддержку
Ключ API Настройки → API и интеграции Автоматизация загрузки товаров, управления заказами
Код FBS/FBO Письмо от менеджера Ozon Подключение к логистической системе
Код верификации SMS/email Вход в аккаунт, смена пароля
💡

Ключ API и ID продавца — это разные вещи. Первый нужен для автоматизации, второй — для идентификации вашего аккаунта.

Что делать, если кода Озон нет в личном кабинете

Иногда код может отсутствовать по следующим причинам:

  • 🔄 Ваш аккаунт не верифицирован (не подтверждён телефон или email).
  • 📱 Вы используете устаревшую версию приложения или браузера.
  • 🚫 Ваш аккаунт заблокирован или находится на модерации.
  • 🔐 Вы ищете не в том разделе (например, путаете код личного кабинета с промокодом).

Вот что можно сделать:

  1. Обновите приложение или браузер до последней версии.
  2. Пройдите верификацию:
    • Подтвердите email (проверьте папку "Спам").
    • Привяжите телефон (если не сделано).
    • Загрузите скан паспорта (если требуется для продавцов).
  • Проверьте другой браузер — иногда блокировщики рекламы или расширения мешают отображению кода.
  • Обратитесь в поддержку:
    • Для покупателей: справка Ozon.
    • Для продавцов: техническая поддержка Ozon Seller.
    ⚠️ Внимание! Если вам предлагают "купить код Озон" или "взломать аккаунт" — это мошенники. Код генерируется автоматически и не продаётся.

    Если вы продавец и не видите код API, убедитесь, что у вас:

    • 📌 Подключён тариф, поддерживающий API (не все тарифы это позволяют).
    • 📌 Нет задолженностей перед Ozon.
    • 📌 Аккаунт не находится на модерации.

    Можно ли использовать один код Озон несколько раз

    Это зависит от типа кода:

    • ✅ Код личного кабинета (ID) — постоянный. Можно использовать сколько угодно раз для подтверждения личности.
    • ❌ Код подтверждения (SMS/email) — одноразовый. Действует 5–10 минут.
    • ⚠️ Ключ API — постоянный, но его можно отозвать в настройках. После отзыва старый ключ перестаёт работать.
    • 🔄 Код FBS/FBO — действует до смены логистической схемы.

    Если речь идёт о коде личного кабинета (например, CL-12345678), то да, его можно использовать многократно. Но есть нюансы:

    • При передаче аккаунта другому лицу код может быть сброшен (например, при смене email или телефона).
    • Если аккаунт заблокирован, код становится недействительным.
    • Для некоторых операций (например, подключение к Ozon Банку) может потребоваться свежий код подтверждения.

    Важно! Никогда не делитесь кодом личного кабинета с посторонними, даже если они представляются сотрудниками Ozon. Для решения любых вопросов достаточно указать код в личном сообщении поддержке — передавать его по телефону или в соцсетях небезопасно.

    Как сбросить или изменить код Озон

    Не все коды можно сбросить самостоятельно. Разберёмся по порядку:

    1. Код личного кабинета (ID)

    Этот код нельзя изменить — он присваивается аккаунту навсегда. Однако если вы:

    • Сменили email или телефон,
    • Восстановили аккаунт после блокировки,
    • Передали права на аккаунт другому лицу,

    то старый код может стать неактуальным. В этом случае новый код сгенерируется автоматически и отобразится в настройках профиля.

    2. Ключ API

    Ключ API можно сбросить вручную:

    1. Перейдите в Настройки → API и интеграции.
    2. Найдите активный ключ и нажмите Отозвать.
    3. Создайте новый ключ по инструкции из раздела выше.

    Предупреждение: После отзыва старого ключа все интеграции, использующие его, перестанут работать. Убедитесь, что у вас есть резервная копия данных.

    💡

    Перед сбросом ключа API проверьте, какие сервисы его используют. Например, если у вас подключён МойСклад, после сброса может понадобиться повторная настройка.

    3. Код подтверждения (SMS/email)

    Этот код сбрасывается автоматически после использования или по истечении времени (обычно 5–10 минут). Если вы не получили код:

    • Проверьте папку "Спам" в почте.
    • Убедитесь, что номер телефона верный.
    • Попробуйте запросить код повторно (кнопка Отправить код ещё раз).

    Если код не приходит вообще, возможно, ваш аккаунт заблокирован или есть технические проблемы на стороне Ozon. В этом случае обратитесь в поддержку.

    FAQ: Частые вопросы о коде Озон

    Можно ли узнать код Озон без входа в аккаунт?

    Нет. Код личного кабинета отображается только после авторизации. Если вы потеряли доступ к аккаунту, сначала восстановите его через email или телефон, а затем уже смотрите код в настройках.

    Чем отличается код Озон для покупателя и продавца?

    Это разные идентификаторы:

    • У покупателя код начинается с CL- (client) или состоит из цифр.
    • У продавца код обычно длиннее и может содержать префикс SELLER- или PARTNER-.

    Они не взаимозаменяемы и используются в разных системах.

    Можно ли использовать код Озон для входа на сайт?

    Нет. Код личного кабинета — это идентификатор, а не логин или пароль. Для входа нужны email/телефон + пароль или код подтверждения из SMS.

    Где взять код Озон для возврата товара?

    Для возврата нужен не код личного кабинета, а номер заказа или артикул товара. Их можно найти:

    • В истории заказов (Мои заказы).
    • В email-уведомлении о покупке.
    • На чеке или упаковке товара.

    Код личного кабинета для возврата не требуется.

    Что делать, если код Озон украли или он попал к мошенникам?

    Срочно:

    1. Смените пароль от аккаунта.
    2. Отзовите все активные ключи API (если вы продавец).
    3. Обратитесь в поддержку Ozon с просьбой заблокировать старый код и сгенерировать новый.
    4. Проверьте историю заказов и платежей на подозрительную активность.

    Код личного кабинета сам по себе не даёт доступ к аккаунту, но в сочетании с другими данными может быть использован для мошенничества.

Где на Ozon посмотреть сумму выкупа: 5 способов для продавцов и партнёров

Сумма выкупа на Ozon — это ключевой показатель для продавцов, работающих по моделям FBS (Fulfillment by Seller) и FBO (Fulfillment by Ozon). Она отражает реальную стоимость товара после всех удержаний маркетплейса: комиссий, логистических расходов, скидок и промокодов. Без точного понимания, где и как посмотреть эту сумму, сложно оценивать рентабельность бизнеса, планировать закупки или анализировать эффективность акций.

Проблема в том, что интерфейс Ozon Seller регулярно обновляется, а информация о выкупе «размазана» по разным разделам: отчётам, карточкам заказов и даже API. В этой статье мы собрали все актуальные способы проверки суммы выкупа на 2026 год, включая скрытые фишки (например, как экспортировать данные для анализа в Excel) и нюансы для разных схем работы. Если вы продавец, логист или аналитик — здесь найдёте ответы на свои вопросы.

1. Что такое сумма выкупа на Ozon и почему её важно отслеживать

Сумма выкупа — это чистый доход продавца от продажи товара после всех вычетов Ozon. Она рассчитывается по формуле:

Сумма выкупа = Цена товара для покупателя – Комиссия Ozon – Стоимость доставки (если по FBO) – Скидки/промокоды – Возвраты

Например, если покупатель оплатил товар за 2 500 ₽, а комиссия составила 15% (375 ₽), доставка по FBO — 200 ₽, и был применён промокод на 10% (250 ₽), то сумма выкупа составит:

2 500 – 375 – 200 – 250 = 1 675 ₽

Почему это критично отслеживать:

  • 📊 Анализ рентабельности: без точных данных невозможно понять, приносит ли товар прибыль после всех расходов.
  • 📦 Планирование закупок: сумма выкупа помогает рассчитать, сколько можно потратить на заказ новой партии.
  • 🔄 Контроль возвратов: если сумма выкупа отрицательная (например, из-за высоких комиссий на возвраты), это сигнал пересмотреть условия работы.
  • 📈 Оценка эффективности акций: сравнивая суммы выкупа до и после промо, вы поймёте, оправданы ли скидки.
💡

Если сумма выкупа постоянно ниже себестоимости товара, проверьте настройки комиссий в Личном кабинете → Тарифы. Возможно, вы платите за premium-размещение или дополнительные услуги, которые не окупаются.

2. Где посмотреть сумму выкупа в личном кабинете Ozon Seller (веб-версия)

Самый очевидный способ — через веб-версию личного кабинета. Инструкция актуальна для интерфейса 2026 года:

  1. Авторизуйтесь на сайте seller.ozon.ru.
  2. Перейдите в раздел Заказы (в верхнем меню).
  3. Выберите нужный заказ из списка. Сумма выкупа отображается в колонке Выкуп, ₽.
  4. Для детального разбора кликните по номеру заказа. В открывшейся карточке сумма выкупа будет в блоке Финансы.

Если колонки Выкуп, ₽ нет в таблице заказов:

  • 🔧 Нажмите на иконку шестерёнки (Настройки таблицы) в правом верхнем углу.
  • 📋 В списке доступных колонок найдите Выкуп, ₽ и добавьте её в отображение.
  • 🔄 Сохраните настройки — теперь сумма будет видна для всех заказов.
Как экспортировать данные по выкупу в Excel?

Чтобы выгрузить отчёт с суммами выкупа, перейдите в Аналитика → Отчёты → Финансовые отчёты. Выберите период, добавьте колонку Сумма выкупа и нажмите Экспорт в Excel. В файле будут данные по каждому заказу, включая детализацию удержаний.

Обратите внимание: в карточке заказа сумма выкупа может отличаться от первоначальной из-за:

  • 🔙 Возвратов: если покупатель вернул товар, сумма выкупа уменьшится на стоимость возврата и комиссию Ozon за обработку.
  • 🎁 Подарков: если товар был подарен (например, по акции «Купи 2, получи 3-й в подарок»), сумма выкупа за подарок будет 0 ₽.
  • 📉 Корректировок: иногда Ozon вносит изменения retroactively (например, если была ошибка в расчёте комиссии).

3. Как проверить сумму выкупа в мобильном приложении Ozon Seller

Мобильное приложение Ozon Seller (доступно для iOS и Android) дублирует основные функции веб-версии, но с некоторыми ограничениями. Чтобы найти сумму выкупа:

  1. Откройте приложение и авторизуйтесь.
  2. Перейдите во вкладку Заказы (значок коробки в нижнем меню).
  3. Выберите нужный заказ. Сумма выкупа отображается в блоке Финансы под статусом заказа.
  4. Для детализации тапните по строке Подробнее о финансах.

Особенности мобильной версии:

  • 📱 Ограниченная аналитика: в приложении нельзя экспортировать данные или настроить отображение колонок, как в веб-версии.
  • 🔄 Задержка обновлений: иногда суммы в мобильном приложении обновляются позже, чем на сайте (до 24 часов).
  • 📊 Нет сводных отчётов: чтобы увидеть сумму выкупа по всем заказам за период, придётся пользоваться веб-версией.

4. Просмотр суммы выкупа через API Ozon (для продвинутых пользователей)

Если вам нужно автоматизировать сбор данных (например, для интеграции с 1С, Google Sheets или собственной CRM), можно использовать API Ozon. Для этого:

  1. Получите доступ к API в разделе Настройки → Интеграции → API-ключи.
  2. Используйте метод POST /v2/finance/transaction/list для получения финансовых транзакций.
  3. В ответе ищите поле payout — это и есть сумма выкупа.

Пример запроса (на Python с использованием библиотеки requests):

import requests


url = "https://api-seller.ozon.ru/v2/finance/transaction/list"


headers = {


"Client-Id": "Ваш_Client_ID",


"Api-Key": "Ваш_API_ключ",


"Content-Type": "application/json"


}


payload = {


"filter": {


"date": {


"from": "2026-01-01T00:00:00Z",


"to": "2026-01-31T23:59:59Z"


}


},


"limit": 1000


}



response = requests.post(url, headers=headers, json=payload)


data = response.json()


for transaction in data["result"]["operations"]:


print(f"Заказ: {transaction['order_id']}, Выкуп: {transaction['payout']}")

Что можно сделать через API:

  • 🤖 Автоматический сбор данных: настроить скрипт, который будет ежедневно выгружать суммы выкупа в Google Таблицы.
  • 📈 Интеграция с аналитикой: передавать данные в Power BI, Tableau или другие системы для визуализации.
  • 🔔 Уведомления о проблемах: настроить алерты, если сумма выкупа по заказу ниже порогового значения.
💡

API Ozon позволяет получить данные по сумме выкупа в реальном времени, но требует технических навыков. Если вы не программист, используйте веб-версию или мобильное приложение.

Ограничения API:

  • 🔒 Лимиты запросов: бесплатный тариф позволяет делать до 100 запросов в минуту.
  • 📅 Ограничение по датам: за один запрос можно получить данные максимум за 30 дней.
  • 🔧 Сложность настройки: для работы с API нужны знания JSON и основы программирования.

  • 📊 Сгруппировать данные: по товарам, категориям, регионам или типам доставки (FBS/FBO).

Пошаговая инструкция:

6. Частые проблемы с суммой выкупа и как их решить

Иногда сумма выкупа может казаться некорректной. Рассмотрим типичные ситуации и способы их решения:

🔴 Проблема 1: Сумма выкупа отрицательная

Это происходит, если:

  • 🔙 Возврат товара: Ozon удерживает комиссию за обработку возврата (до 300 ₽ за единицу).
  • 🎁 Подарки по акции: если товар был отдан бесплатно, сумма выкупа будет 0 ₽ или отрицательной (из-за комиссий).
  • 📦 Логистические ошибки: например, если вес товара был указан неверно, и Ozon доначислил стоимость доставки.

🔴 Проблема 2: Сумма выкупа не совпадает с расчётами

Проверьте:

  • 💰 Комиссию: она может меняться в зависимости от категории товара (от 5% до 25%).
  • 🚚 Тарифы доставки: для FBO они фиксированные, для FBS — зависят от веса и габаритов.
  • 🎫 Промокоды: если покупатель использовал купон, сумма выкупа уменьшится на его номинал.

🔴 Проблема 3: Сумма выкупа не отображается

Возможные причины:

  • 🔄 Задержка обработки: данные могут обновляться до 48 часов после доставки заказа.
  • 🔒 Ограничения доступа: если у вас роль «Менеджер», а не «Владелец», некоторые финансовые данные могут быть скрыты.
  • 📵 Технический сбой: попробуйте очистить кэш браузера или использовать другой устройство.
⚠️ Внимание: Если сумма выкупа не появляется более 3 дней после доставки заказа, напишите в поддержку Ozon с указанием номера заказа и скриншотом проблемы. В 90% случаев это решается в течение суток.

7. Как увеличить сумму выкупа: практические советы

Если сумма выкупа слишком мала, попробуйте эти методы:

  • 📈 Повысьте наценку: проанализируйте конкурентов и увеличьте цену на 5–10%, если это не повредит продажам.
  • 🚀 Оптимизируйте логистику: перейдите на FBS, если доставка по FBO съедает большую часть прибыли.
  • 🎯 Участвуйте в акциях выборочно: не все промо окупаются — рассчитайте, как скидка повлияет на сумму выкупа.
  • 📦 Снизьте вес и габариты упаковки: это уменьшит стоимость доставки по FBO.
  • 🔄 Сократите количество возвратов: улучшите описания товаров и фотографии, чтобы покупатели точно понимали, что покупают.

Пример расчёта:

Допустим, вы продаёте товар за 3 000 ₽ с комиссией 15% (450 ₽) и доставкой FBO за 200 ₽. Сумма выкупа:

3 000 – 450 – 200 = 2 350 ₽

Если вы повысите цену до 3 200 ₽ и оптимизируете упаковку, сократив доставку до 150 ₽, сумма выкупа вырастет:

3 200 – 480 (15%) – 150 = 2 570 ₽

Прирост: +220 ₽ с одного заказа.

💡

Даже небольшое увеличение наценки (на 3–5%) может значительно повысить сумму выкупа, если продажи остаются на том же уровне.

FAQ: Ответы на частые вопросы о сумме выкупа на Ozon

🔍 Как часто обновляется сумма выкупа в личном кабинете?

Сумма выкупа обновляется в реальном времени после доставки заказа покупателю. Однако в редких случаях (например, при возвратах или корректировках) данные могут обновляться с задержкой до 48 часов. Если сумма не появилась через 2 дня — обратитесь в поддержку.

💰 Почему сумма выкупа меньше, чем цена товара?

Это нормально: сумма выкупа — это цена за вычетом всех комиссий и расходов. Например, если товар стоит 5 000 ₽, а комиссия Ozon — 15% (750 ₽), доставка FBO — 300 ₽, то сумма выкупа составит 4 000 – 750 – 300 = 3 950 ₽. Также на сумму влияют скидки, промокоды и возвраты.

📊 Можно ли увидеть сумму выкупа по всем заказам за месяц?

Да, для этого перейдите в раздел Аналитика → Финансовые отчёты → Отчёт по заказам. Укажите нужный период (например, последний месяц) и добавьте колонку Сумма выкупа. Отчёт можно экспортировать в Excel для дальнейшего анализа.

🔄 Что делать, если сумма выкупа отрицательная?

Отрицательная сумма выкупа означает, что расходы (комиссии, доставка, возвраты) превысили доход от продажи. Чтобы исправить ситуацию:

  1. Проверьте себестоимость товара — возможно, она слишком высока.
  2. Проанализируйте комиссии: для некоторых категорий они достигают 25%.
  3. Оцените эффективность акций: если скидки съедают всю прибыль, откажитесь от них.
  4. Рассмотрите переход на FBS, если доставка по FBO слишком дорогая.
📱 Почему в мобильном приложении сумма выкупа отличается от веб-версии?

Это может быть связано с:

  • Кэшированием данных: приложение не всегда обновляет информацию в реальном времени.
  • Ошибкой синхронизации: попробуйте выйти и заново войти в аккаунт.
  • Разными ролями доступа: если в веб-версии у вас права администратора, а в мобильном — менеджера, некоторые данные могут отличаться.

Если разница сохраняется более суток, напишите в поддержку Ozon с указанием номера заказа и скриншотами из обоих источников.

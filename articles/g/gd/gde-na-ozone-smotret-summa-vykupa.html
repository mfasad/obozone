Где на Озоне посмотреть сумму выкупа заказа: все способы для продавцов

Сумма выкупа на Ozon — это фактическая стоимость, которую покупатель заплатил за товар после применения всех скидок, промокодов и учёта комиссий маркетплейса. Для продавцов этот показатель критичен: от него зависит прибыль, аналитика продаж и планирование закупок. Однако найти эту цифру в интерфейсе платформы не всегда интуитивно — особенно для новичков или при работе с большим количеством заказов.

В этой статье разберём все официальные способы проверки суммы выкупа на Ozon, включая скрытые функции личного кабинета, мобильного приложения и API. Также объясним, почему цифры в отчётах могут отличаться от ожидаемых, и как избежать ошибок при расчёте чистой прибыли. Информация актуальна для схем FBS и FBO на 2026 год.

1. Где искать сумму выкупа в личном кабинете Ozon Seller

Основной источник данных — веб-версия личного кабинета продавца. Здесь сумма выкупа отображается в нескольких разделах, но с разной степенью детализации. Рассмотрим каждый из них.

Самый быстрый способ — перейти в раздел Заказы → Все заказы. В таблице с заказами сумма выкупа отображается в колонке «Сумма выкупа» (может называться «Итоговая сумма» в зависимости от версии интерфейса). Если колонки нет — нажмите на иконку шестерёнки в правом верхнем углу таблицы и добавьте её вручную.

  • 📊 Раздел «Заказы»: колонка «Сумма выкупа» (по умолчанию скрыта — нужно включить в настройках отображения).
  • 📄 Карточка заказа: блок «Финансы» → строка «Сумма выкупа» (с разбивкой по товарам, если в заказе несколько позиций).
  • 📈 Отчёты: в отчёте «Продажи» (раздел Аналитика → Отчёты → Продажи) сумма выкупа указана в колонке «Выкуп (₽)».

Важно: в карточке заказа сумма выкупа может отличаться от той, что видна в таблице заказов, если к заказу были применены ретроактивные скидки (например, по программе Ozon Premium или акциям с отсроченным бонусом). В этом случае актуальные данные всегда берутся из блока «Финансы» внутри карточки.

💡

Если сумма выкупа в отчёте «Продажи» не совпадает с данными в карточке заказа, обновите страницу через 10–15 минут. Иногда происходит задержка синхронизации между системами Ozon.

2. Как проверить сумму выкупа в мобильном приложении Ozon Seller

Мобильное приложение Ozon Seller (доступно для iOS и Android) дублирует основные функции веб-версии, но с упрощённым интерфейсом. Чтобы найти сумму выкупа:

  1. Откройте приложение и авторизуйтесь.
  2. Перейдите в раздел Заказы (значок коробки в нижнем меню).
  3. Выберите нужный заказ из списка.
  4. Прокрутите экран до блока «Финансовая информация» — здесь будет строка «Сумма выкупа».

В мобильной версии сумма выкупа не отображается в общем списке заказов — её можно увидеть только внутри карточки конкретного заказа. Это ограничение интерфейса, которое может создать неудобства при массовой проверке. Решение: используйте веб-версию для анализа большого количества заказов или экспортируйте данные в Excel.

3. Экспорт данных: как выгрузить суммы выкупа в Excel

Если вам нужно проанализировать суммы выкупа по большому количеству заказов (например, для расчёта рентабельности или налоговой отчётности), ручной просмотр каждого заказа неэффективен. В этом случае поможет экспорт данных:

  1. В личном кабинете перейдите в Аналитика → Отчёты → Продажи.
  2. Настройте фильтры (период, статус заказов, склад и т. д.).
  3. Нажмите кнопку Экспортировать (в правом верхнем углу) и выберите формат Excel.
  4. В полученном файле сумма выкупа будет в колонке «Выкуп (₽)».

Обратите внимание: в экспортируемом отчёте сумма выкупа указана без учёта комиссий Ozon. Чтобы посчитать чистую прибыль, нужно вычесть:

  • 💰 Комиссию маркетплейса (от 5% до 15% в зависимости от категории).
  • 📦 Стоимость логистики (для FBS — тарифы хранения и доставки).
  • 🔄 Возвраты и штрафы (если были).
Как посчитать чистую прибыль по сумме выкупа?

Чистая прибыль = Сумма выкупа − Комиссия Ozon − Логистика − Возвраты/штрафы.

Пример: если сумма выкупа 1 500 ₽, комиссия 10% (150 ₽), логистика FBS 200 ₽, то чистая прибыль = 1 500 − 150 − 200 = 1 150 ₽.

4. Проверка суммы выкупа через API Ozon

Для продавцов с большим объёмом заказов или автоматизированными системами учёта удобно использовать API Ozon. Через него можно получать суммы выкупа в реальном времени и интегрировать данные с 1С, CRM или собственными дашбордами.

Основные методы API для работы с суммами выкупа:

  • 🔗 /v2/analytics/sales — отчёт по продажам с разбивкой по заказам.
  • 📦 /v2/posting/fbo/list — список заказов FBO с финансовой информацией.
  • 📊 /v1/finance/transaction/totals — сводка по выплатам (включает суммы выкупа за вычетом комиссий).

Пример запроса для получения суммы выкупа по конкретному заказу (на языке Python):

import requests


url = "https://api-seller.ozon.ru/v2/posting/fbo/list"


headers = {


"Client-Id": "Ваш Client-ID",


"Api-Key": "Ваш API-ключ",


"Content-Type": "application/json"


}


payload = {


"limit": 1,


"order_id": 123456789  # ID заказа


}



response = requests.post(url, headers=headers, json=payload)


print(response.json()["result"]["postings"][0]["financial_data"]["products"][0]["payout"])

В ответе API сумма выкупа может передаваться в разных полях в зависимости от схемы:

  • Для FBO: financial_data.products[].payout.
  • Для FBS: financial_data.products[].price (до вычета логистики).
💡

API Ozon позволяет получать суммы выкупа в автоматическом режиме, но требует настройки интеграции. Для небольших продавцов проще использовать экспорт в Excel.

5. Почему сумма выкупа может отличаться от цены товара

Многие продавцы сталкиваются с ситуацией, когда сумма выкупа в отчётах Ozon не совпадает с указанной ценой товара. Это нормальная практика, и вот основные причины таких расхождений:

  • 🎁 Скидки и промокоды: покупатель мог применить купон, акцию «Кэшбэк» или скидку по программе Ozon Premium.
  • 🔄 Ретроактивные бонусы: некоторые скидки (например, «Купи 2 по цене 1») применяются после оплаты заказа.
  • 💳 Округление: Ozon округляет копейки в пользу покупателя (например, 999,99 ₽ становятся 999 ₽).
  • 📉 Динамическое ценообразование: если вы используете автоскидки или участвуете в акциях типа «Товар дня», цена могла измениться между моментом размещения заказа и выкупом.

Чтобы избежать путаницы, всегда сверяйте сумму выкупа с данными в блоке «Финансы» внутри карточки заказа. Там отображается финальная сумма с учётом всех корректировок, включая ретроактивные скидки.

6. Частые ошибки при работе с суммами выкупа

Даже опытные продавцы иногда допускают ошибки при анализе сумм выкупа. Вот самые распространённые из них и как их избежать:

⚠️ Внимание: Если вы продаёте товары по схеме FBS, сумма выкупа в отчётах не включает стоимость логистики. Не путайте её с чистой прибылью!
  • 🔍 Игнорирование ретроактивных скидок: некоторые акции (например, «Подари скидку другу») применяются после оплаты заказа. Если не учитывать их, реальная сумма выкупа будет занижена.
  • 📅 Несовпадение дат: в отчёте «Продажи» сумма выкупа фиксируется на дату оплаты заказа, а не доставки. Это важно для анализа динамики.
  • 💱 Валюта: если вы продаёте на Ozon Global, сумма выкупа может быть указана в иностранной валюте (доллары, тенге). Внимательно смотрите на значок валюты в отчётах.

Ещё одна типичная ошибка — сравнение суммы выкупа с рекомендованной розничной ценой (РРЦ). На Ozon покупатель часто платит меньше РРЦ из-за скидок маркетплейса, поэтому ориентироваться нужно именно на фактический выкуп, а не на заданную цену.

⚠️ Внимание: В отчёте «Финансы» сумма выкупа уже за вычетом комиссий Ozon, а в отчёте «Продажи» — нет. Не путайте эти документы при расчёте прибыли!

7. Как сумма выкупа влияет на выплаты продавцу

Сумма выкупа — это не то же самое, что сумма, которую вы получите на счёт. На финальную выплату влияют:

  1. Комиссия Ozon (от 5% до 20% в зависимости от категории товара).
  2. Логистические расходы (для FBS: хранение, упаковка, доставка).
  3. Штрафы и возвраты (если товар был возвращён или нарушены условия хранения).
  4. Налоги (НДС, если вы работаете с ним).

Формула расчёта выплаты:

Сумма к выплате = Сумма выкупа − Комиссия − Логистика − Штрафы/возвраты − Налоги

Пример: если сумма выкупа составила 2 000 ₽, комиссия 10% (200 ₽), логистика FBS 300 ₽, то на счёт вы получите 2 000 − 200 − 300 = 1 500 ₽ (до вычета налогов).

Чтобы увидеть детализацию выплат, перейдите в раздел Финансы → Выплаты и выберите интересующий период. Здесь будет таблица с разбивкой по каждому заказу и указанием всех удержаний.

FAQ: Ответы на частые вопросы о сумме выкупа

Можно ли увидеть сумму выкупа до того, как покупатель оплатил заказ?

Нет, сумма выкупа формируется только после оплаты заказа покупателем. До этого момента в личном кабинете отображается предварительная цена, которая может измениться из-за скидок или промокодов.

Почему сумма выкупа в отчёте «Продажи» и «Финансы» разная?

В отчёте «Продажи» указана полная сумма выкупа (до вычета комиссий), а в «Финансах» — сумма к выплате (уже за минусом всех удержаний). Для анализа продаж используйте первый отчёт, для расчёта прибыли — второй.

Как узнать сумму выкупа по возвращённому заказу?

В карточке возвращённого заказа сумма выкупа будет обнулена или отмечена как «Возврат». Чтобы увидеть изначальную сумму, проверьте историю изменений в блоке «Финансы» или экспортируйте отчёт по продажам за нужный период.

Можно ли оспорить сумму выкупа, если она кажется неверной?

Да, если вы обнаружили ошибку (например, неверно применённая скидка), обратитесь в поддержку Ozon с указанием номера заказа и причины спорной ситуации. Приложите скриншоты из личного кабинета.

Где посмотреть сумму выкупа по заказам, оформленным через Ozon Global?

Для заказов с Ozon Global сумма выкупа отображается в той же карточке заказа, но может быть указана в иностранной валюте. Чтобы перевести её в рубли, используйте курс Ozon на дату оплаты (он указан в блоке «Финансы»).

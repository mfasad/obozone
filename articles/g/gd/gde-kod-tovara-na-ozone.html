Где искать код товара на Ozon: подробное руководство с картинками и примерами

Вы пытаетесь найти код товара на Ozon, но не знаете, где именно его искать? Эта проблема знакома и покупателям, и продавцам. Код товара (он же артикул, SKU или ID) нужен для возвратов, жалоб на продавца, синхронизации с 1С, заказа рекламы или просто чтобы уточнить детали у поддержки. Но маркетплейс не всегда показывает его на видном месте — иногда приходится копать глубже.

В этой статье мы разберём все возможные места, где может скрываться код товара на Ozon — от очевидных (карточка товара, чек) до неочевидных (API, Excel-отчёты для продавцов). А ещё объясним, чем отличаются артикул продавца (SKU), ID товара в системе Ozon и штрихкод, чтобы вы не путались в терминах. Если вы торопитесь — воспользуйтесь быстрой навигацией по разделам.

Быстрая навигация: где искать код товара?

Вот краткий чек-лист, где можно найти код товара на Ozon в зависимости от вашей роли:

  • 🛒 Покупателям: в карточке товара (под ценой), в чеке (электронном или бумажном), в истории заказов, в URL-адресе страницы.
  • 📦 Продавцам: в личном кабинете (Товары → Каталог), в Excel-отчётах, через API, в штрихкодах на упаковке.
  • 🔍 Универсально: в ссылке на товар (цифры после /product/ или /p/).

Если вам нужно срочно найти код — прокрутите до нужного раздела. Если хотите разобраться подробно — читайте статью полностью.

1. Код товара в карточке на сайте Ozon (для покупателей)

Самый простой способ — посмотреть код прямо в карточке товара. Он может называться по-разному:

  • 🏷️ Артикул — уникальный код, который присваивает продавец (например, SKU12345 или ART-2026-XL).
  • 🆔 ID товара — внутренний идентификатор Ozon (например, 312846754).
  • 📌 Код для поиска — иногда дублирует артикул или ID.

Где именно искать:

  1. Откройте карточку товара на сайте или в приложении Ozon.
  2. Прокрутите вниз до блока "Характеристики" или "Детали".
  3. Найдите строки Артикул, Код товара или ID. Если их нет — смотрите следующий способ.

Пример того, как это выглядит на практике:

Тип кода Пример Где отображается
Артикул продавца (SKU) MB-PRO-2026-BLK В характеристиках под блоком "О товаре"
ID товара Ozon 298745613 В URL-адресе или в чеке
Штрихкод (EAN) 4607002487135 На упаковке или в характеристиках

Важно! Не все продавцы заполняют поле "Артикул". Если его нет — используйте ID товара из URL (об этом ниже).

💡

Если вы не видите код в карточке, попробуйте открыть её в браузере на компьютере — в мобильном приложении некоторые данные могут быть скрыты.

2. Как узнать код товара из URL-адреса страницы

Каждому товару на Ozon присваивается уникальный числовой идентификатор, который отображается в ссылке. Это самый надёжный способ найти ID товара, даже если продавец не указал артикул.

Инструкция:

  1. Откройте карточку товара в браузере (не в приложении!).
  2. Посмотрите на адресную строку. Ссылка будет выглядеть так: 
    https://www.ozon.ru/product/nazvanie-tovara-123456789/

    или так:

    https://www.ozon.ru/p/context/nazvanie-tovara/987654321/
  3. Цифры в конце (выделены жирным) — это и есть ID товара в системе Ozon.

Этот ID пригодится для:

  • 🔄 Обмена или возврата (указывайте его в обращении в поддержку).
  • 📊 Аналитики продаж (если вы продавец).
  • 🛠️ Интеграции с CRM или 1С.

Пример: В ссылке https://www.ozon.ru/product/umnye-chasy-xiaomi-mi-band-8-pro-123456789/ код товара — 123456789.

Что делать, если в URL нет цифр?

Иногда Ozon использует "человекопонятные" ссылки без ID (например, /product/iphone-15-pro-max/). В этом случае откройте страницу, нажмите Ctrl+U (просмотр кода), найдите строку "productId" — там будет нужный идентификатор.

3. Где искать код товара в чеке (электронном или бумажном)

Если вы уже купили товар, код можно найти в чеке — как в электронном (приходит на почту), так и в бумажном (если забирали в ПВЗ).

Вот где именно смотреть:

  • 📄 Бумажный чек: обычно код указан рядом с названием товара или в строке "Артикул".
  • 📧 Электронный чек (PDF): ищите столбец Код товара или ID в таблице с покупками.
  • 🛒 Чек в приложении Ozon: откройте заказ → "Чек" → прокрутите до списка товаров.

Внимание: в чеке может быть указан как артикул продавца (SKU), так и ID Ozon. Для возврата или жалобы лучше использовать ID — он универсальный.

Пример чека с кодом:

Заказ №56789 от 10.05.2026



1. Умные часы Xiaomi Mi Band 8   |  3 490 ₽


Артикул: XMB8-BLACK           |  ID: 298745613
💡

Если в чеке нет кода — проверьте email с подтверждением заказа. Там часто дублируется информация.

4. Поиск кода товара в личном кабинете продавца (для seler.ozon.ru)

Если вы продавец, у вас есть несколько способов найти код товара в личном кабинете Ozon Seller:

Способ 1: Через каталог товаров

  1. Перейдите в раздел Товары → Каталог.
  2. Найдите нужный товар в списке.
  3. В столбце Артикул или ID будет его код.
  4. Если нужно скопировать ID системы Ozon — кликните на товар, и в URL появится его номер (например, /product/123456789/edit).

Способ 2: В Excel-отчётах

Скачайте отчёт по товарам (Аналитика → Отчёты → Товары) и ищите столбцы:

  • sku — артикул продавца.
  • product_id — ID товара в Ozon.
  • offer_id — идентификатор торгового предложения (если у товара несколько вариантов).

Пример строки из отчёта:

sku product_id Название
WATCH-XM8-BLK 298745613 Умные часы Xiaomi Mi Band 8 Black

Способ 3: Через API (для разработчиков)

Если вы используете API Ozon, код товара можно получить через запросы:

  • 🔗 GET /v2/product/info — возвращает данные по product_id.
  • 🔗 GET /v1/product/list — список товаров с их SKU и ID.

Пример ответа API:

{

"result": {


"items": [


{


"product_id": 298745613,


"sku": "WATCH-XM8-BLK",


"name": "Умные часы Xiaomi Mi Band 8 Black"


}


]


}


}

5. Разница между артикулом (SKU), ID Ozon и штрихкодом

Многие путают эти понятия, но они означают разное:

Термин Кто присваивает Пример Где используется
Артикул (SKU) Продавец MB8-PRO-BLACK-2026 Для внутреннего учёта, синхронизации с 1С
ID товара Ozon Маркетплейс Ozon 298745613 Для возвратов, жалоб, API-запросов
Штрихкод (EAN/UPC) Производитель 4607002487135 Для сканирования на складе, в магазинах
Offer ID Ozon 123456789-1 Для товаров с вариантами (цвет, размер)

🔹 SKU — ваш внутренний код. Может совпадать с артикулом производителя или быть уникальным для вашего магазина.

🔹 ID Ozon — универсальный идентификатор товара на маркетплейсе. Он не меняется, даже если вы обновите название или цену.

🔹 Штрихкод — международный стандарт. Его можно просканировать в любом магазине, но на Ozon он не всегда совпадает с ID.

💡

Если вы продавец и загружаете товары через Excel, используйте столбец sku для своего артикула и product_id для ссылки на товар в Ozon.

6. Частые проблемы и как их решить

Иногда код товара найти не получается. Вот типичные ситуации и их решения:

⚠️ Внимание! Если вы продавец и не видите код товара в личном кабинете, проверьте:
  • 🔍 Не удалён ли товар из каталога (архивированные товары не отображаются в основном списке).
  • 🔄 Не изменяли ли вы недавно название или категорию — иногда это сбрасывает видимость кода.
  • 📧 Не пришёл ли вам email от Ozon с уведомлением об ошибке в карточке товара.

Проблема 1: В карточке товара нет поля "Артикул".

Решение: Используйте ID из URL или обратитесь к продавцу через кнопку "Задать вопрос" в карточке.

Проблема 2: В чеке указан только штрихкод, а нужна ID Ozon.

Решение: Найдите товар на сайте по названию и скопируйте ID из ссылки.

Проблема 3: При интеграции с 1С выдаёт ошибку "Неверный SKU".

Решение: Проверьте, что в 1С указан тот же артикул, что и в личном кабинете Ozon (включая регистр и символы).

Что делать, если код товара изменился?

Ozon иногда обновляет ID товаров при смене категории или переносе между аккаунтами. В этом случае старый ID станет неактуальным. Чтобы найти новый, воспользуйтесь поиском по названию в личном кабинете или свяжитесь с поддержкой.

7. FAQ: Ответы на частые вопросы

🔍 Как узнать код товара, если я купил его давно и не сохранял чек?

Откройте историю заказов в личном кабинете Ozon (Мои заказы), найдите нужный заказ и кликните на название товара. В открывшейся карточке будет ID в URL или в характеристиках. Если товара больше нет в продаже — запросите данные у поддержки через форму обратной связи.

🛒 Можно ли найти код товара по фото?

Нет, Ozon не поддерживает поиск по изображению. Но вы можете:

  1. Сделать скриншот этикетки с штрихкодом и просканировать его через приложение-сканер (например, Barcode Scanner).
  2. Вручную ввести название товара с картинки в поиск Ozon.
📦 Я продавец. Как массово получить коды всех своих товаров?

Используйте один из способов:

  • Скачайте Excel-отчёт в разделе Аналитика → Отчёты → Товары.
  • Воспользуйтесь API-запросом GET /v1/product/list (нужны права доступа).
  • Используйте сервисы интеграции типа МойСклад, 1С или RetailCRM, если они подключены к вашему аккаунту Ozon.
🔄 Можно ли по коду товара узнать, кто продавец?

Нет, ID товара или артикул не содержат информации о продавце. Но вы можете:

  1. Открыть карточку товара по ID (добавив его в URL: ozon.ru/product/ID).
  2. Посмотреть название магазина под кнопкой "Купить".
  3. Если товар уже куплен — имя продавца будет в чеке или истории заказов.
🤖 Как автоматизировать получение кодов товаров для большого каталога?

Для автоматизации подойдут:

  • 🤖 API Ozon: напишите скрипт на Python или PHP, который будет pullingить данные по /v1/product/list.
  • 📊 Google Sheets + Apps Script: настройте импорт данных из API в таблицу.
  • 🔄 Сервисы интеграции: AltoBI, RetailCRM или МойСклад имеют готовые коннекторы к Ozon.

Пример кода для API-запроса:

import requests


url = "https://api-seller.ozon.ru/v1/product/list"


headers = {


"Client-Id": "Ваш_Client_ID",


"Api-Key": "Ваш_API_ключ"


}


response = requests.get(url, headers=headers)


products = response.json()["result"]["items"]


for product in products:


print(f"ID: {product['product_id']}, SKU: {product['sku']}")

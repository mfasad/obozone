Где взять ID товара на Ozon: все способы для покупателей и продавцов

Нужно найти ID товара на Ozon, но не знаете, где его искать? Этот уникальный числовой код требуется для работы с API, настройки рекламы, обработки заказов или решения спорных ситуаций. Без него невозможно точно идентифицировать продукт в системе маркетплейса — будь то ваш собственный товар (если вы продавец) или чужой (если вы покупатель и хотите оставить отзыв или жалобу).

В этой статье мы разберём все актуальные способы получения ID — от очевидных (через карточку товара) до скрытых (через код страницы или API). Особое внимание уделим нюансам для мобильного приложения и десктопной версии, а также объясним, почему иногда ID товара и ID предложения (offer_id) — это разные вещи. Если вы торопитесь, воспользуйтесь быстрым гидом в конце статьи.

1. Что такое ID товара на Ozon и зачем он нужен

ID товара (или product_id) — это уникальный числовой идентификатор, который Ozon присваивает каждому продукту в своём каталоге. Он отличается от:

  • 📦 Артикула продавца (SKU) — внутреннего кода, который вы назначаете сами;
  • 🏷️ Штрихкода (EAN, UPC) — международного стандарта маркировки;
  • 🔄 ID предложения (offer_id) — идентификатора конкретного варианта товара (например, цвета или размера).

Без product_id вы не сможете:

  • 🔗 Настроить рекламные кампании в Ozon Рекламе;
  • 📊 Экспортировать данные по товару через API;
  • 🛠️ Решить спор с поддержкой (например, при блокировке карточки);
  • 📦 Объединить дублирующиеся товары в каталоге.
⚠️ Внимание: Если вы продавец и работаете по модели FBS, ID товара потребуется для формирования этикеток и передачи информации логистическому оператору. Без него груз не примут на складе Ozon.

На практике product_id выглядит как число от 6 до 10 цифр (например, 12345678). Его можно найти в URL карточки товара, в личном кабинете продавца или через инструменты разработчика в браузере. Далее разберём каждый способ подробно.

2. Способ 1: ID товара в адресной строке браузера

Самый простой метод — извлечь идентификатор прямо из ссылки на товар. Он работает как для покупателей, так и для продавцов, если у вас есть доступ к публичной карточке.

Инструкция:

  1. Откройте карточку товара на ozon.ru;
  2. Посмотрите на адресную строку браузера. URL будет выглядеть так:
    https://www.ozon.ru/product/{тут_ваш_id}-{название-товара}/
  3. Скопируйте цифры перед первым дефисом (см. пример ниже).
Пример URL ID товара
https://www.ozon.ru/product/smartfon-apple-iphone-15-pro-128gb-naturalnyy-titan-1723456789/ 1723456789
https://www.ozon.ru/product/umnye-chasy-amazfit-gtr-4-47-mm-chernyy-101234567/ 101234567
https://www.ozon.ru/product/kniga-haruki-murakami-norvezhskiy-les-789012345/ 789012345

Если в URL нет цифр (например, ozon.ru/product/naimenovanie-tovara/), значит, вы попали на некорректную ссылку или товар ещё не добавлен в каталог. Попробуйте обновить страницу или найти продукт через поиск.

💡

Если вы продавец и не можете найти свой товар через поиск Ozon, проверьте статус модерации в личном кабинете. Возможно, карточка ещё не опубликована или заблокирована.

3. Способ 2: Через код страницы (для опытных пользователей)

Если ID не отображается в URL (например, из-за редиректов или технических работ на сайте), его можно извлечь из HTML-кода страницы. Этот метод подходит для десктопной версии браузера.

Пошаговая инструкция:

  1. Откройте карточку товара на Ozon;
  2. Нажмите правой кнопкой мыши на свободное место страницы и выберите Просмотреть код (или Inspect в английской версии);
  3. В открывшемся окне инструментов разработчика нажмите Ctrl + F (Windows) или Cmd + F (Mac) и введите productId;
  4. Найдите строку, где указано "productId": 123456789 — это и есть нужный идентификатор.

Альтернативный вариант — искать по data-product-id в атрибутах элементов. Например:

<div data-product-id="123456789">
⚠️ Внимание: Ozon регулярно обновляет структуру кода страниц. Если вы не нашли productId через поиск, попробуйте искать itemId или offerId — иногда идентификатор скрыт под другими названиями.
Что делать, если в коде нет productId?

Если поиск по productId не дал результатов, попробуйте:

1. Обновить страницу (иногда данные подгружаются динамически).

2. Посмотреть вкладку Network → XHR в инструментах разработчика и найти запрос с данными товара (обычно содержит /api/commerce).

3. Использовать расширение для браузера, например JSON Viewer, чтобы проанализировать ответы сервера.

4. Способ 3: ID товара в мобильном приложении Ozon

В официальном приложении Ozon для Android и iOS нет прямого отображения ID в интерфейсе, но его можно получить двумя обходными путями:

Метод 1: Поделиться ссылкой

  1. Откройте карточку товара в приложении;
  2. Нажмите на иконку Поделиться (обычно в правом верхнем углу);
  3. Выберите любой мессенджер (например, Telegram или WhatsApp);
  4. Скопируйте отправленную ссылку — в ней будет product_id (как в способе 1).

Метод 2: Через браузерную версию

Если первый метод не сработал (например, ссылка сокращённая), откройте карточку товара в приложении, затем:

  1. Нажмите на три точки (⋮) в верхнем правом углу;
  2. Выберите Открыть в браузере;
  3. В адресной строке браузера появится полный URL с product_id.

5. Способ 4: ID товара в личном кабинете продавца

Если вы продавец, самый надёжный способ найти product_id — через панель управления Ozon Seller. Там идентификатор отображается в нескольких разделах:

Раздел 1: «Мои товары»

  1. Перейдите в Каталог → Мои товары;
  2. Найдите нужный товар в списке;
  3. ID будет в колонке ID товара (если её нет, нажмите Настроить колонки и добавьте этот параметр).

Раздел 2: Экспорт данных

Если товаров много, удобнее выгрузить список в Excel:

  1. В разделе Мои товары нажмите Экспортировать;
  2. Выберите формат XLSX и нужные колонки (обязательно отметьте ID товара);
  3. В полученном файле будет столбец с идентификаторами.

Раздел 3: Карточка товара

При редактировании товара ID отображается в верхней части экрана рядом с названием:

6. Способ 5: Через API Ozon (для разработчиков)

Если вам нужно получить ID программно (например, для интеграции с 1С или другой системой учёта), используйте Ozon API. Для этого потребуется:

  • 🔑 Client-ID и API-ключ (генерируются в личном кабинете продавца);
  • 📡 Доступ к endpoint’у /v2/product/info или /v1/product/list.

Пример запроса для получения информации о товаре (включая product_id):

GET https://api-seller.ozon.ru/v2/product/info?product_id=123456789

Headers:


Client-Id: {ваш_client_id}


Api-Key: {ваш_api_ключ}

Ответ будет в формате JSON:

{

"product_id": 123456789,


"name": "Название товара",


"offer_id": "123456789_1001",


...


}

⚠️ Внимание: При работе с API учитывайте лимиты запросов (обычно 1000 запросов в минуту для продавцов с высоким рейтингом). Превышение лимита ведёт к временной блокировке доступа.

Если вы не разработчик, но нужно автоматизировать получение ID, воспользуйтесь готовыми решениями:

  • 🤖 Ozon Helper (расширение для браузера);
  • 📊 МойСклад или 1С с модулем интеграции Ozon;
  • 🔧 Zapier для связки Ozon с другими сервисами.

7. Частые ошибки и как их избежать

При поиске ID товара пользователи часто сталкиваются с типичными проблемами:

Ошибка Причина Решение
В URL нет цифр Товар новый или не прошёл модерацию Проверьте статус в личном кабинете продавца
ID в коде страницы не совпадает с URL На странице отображается offer_id, а не product_id Ищите параметр productId (см. Способ 3)
Приложение не показывает ID Ограничение мобильной версии Откройте товар в браузере (Способ 4)
API возвращает ошибку 404 Неверный product_id или отсутствует доступ Проверьте корректность ID и права API-ключа

Ещё одна распространённая путаница — разница между product_id и offer_id:

  • product_id — идентификатор самой карточки товара (например, iPhone 15 Pro);
  • offer_id — идентификатор конкретного предложения (например, iPhone 15 Pro 128GB Natural Titanium).

Если вы продавец и работаете с вариативными товарами (разные цвета/размеры), для рекламы или API может потребоваться именно offer_id. Его можно найти в личном кабинете в разделе Мои товары → Вариации.

💡

ID товара (product_id) и ID предложения (offer_id) — это разные сущности. Для большинства операций нужен product_id, но при работе с вариациями (цвета, размеры) потребуется offer_id.

8. Быстрый гид: как найти ID за 10 секунд

Нет времени читать всю статью? Вот сверхкраткая инструкция:

  1. Для покупателей: Откройте товар на Ozon → скопируйте цифры из URL перед первым дефисом.
  2. Для продавцов: Зайдите в Личный кабинет → Мои товары → найдите колонку ID товара.
  3. Если не видно в URL: Нажмите Просмотреть код (F12) → найдите productId.
  4. В мобильном приложении: Поделитесь ссылкой на товар → скопируйте её и извлеките ID.

Если ничего не помогает, воспользуйтесь поддержкой Ozon — пришлите им ссылку на товар и попросите помочь с идентификатором.

FAQ: Частые вопросы про ID товара на Ozon

Можно ли изменить ID товара после создания карточки?

Нет, product_id присваивается автоматически при создании товара и не подлежит редактированию. Если вам нужно "обновить" идентификатор (например, при дублировании карточки), создайте новый товар — ему будет присвоен другой ID.

Где взять ID товара, если карточка заблокирована?

Если товар заблокирован модератором, его ID всё равно отображается в личном кабинете продавца в разделе Мои товары (даже если карточка не видна покупателям). Также можно найти идентификатор в письме от поддержки Ozon с причиной блокировки.

Чем отличается product_id от sku?

product_id — это внутренний идентификатор Ozon, а SKU (или article) — ваш собственный артикул товара. SKU вы назначаете сами (например, IPHONE15-128-BLACK), а product_id генерирует маркетплейс. Оба кода могут использоваться для поиска товара, но в API Ozon работает только product_id.

Можно ли по ID товара узнать, кто его продавец?

Нет, product_id относится только к карточке товара, а не к продавцу. Чтобы найти информацию о продавце, откройте карточку товара и посмотрите в блоке Продавец (или используйте API-метод /v1/product/info/seller с указанием offer_id).

Как найти ID товара по штрихкоду?

Прямого способа получить product_id по штрихкоду (EAN/UPC) нет, но можно:

  1. Найти товар на Ozon по штрихкоду через поиск;
  2. Открыть его карточку и извлечь ID из URL (Способ 1).

Если товар не найден, значит, он ещё не добавлен в каталог Ozon или штрихкод неверный.

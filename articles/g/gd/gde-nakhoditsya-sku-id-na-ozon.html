Где искать SKU ID на Озон: 5 способов найти артикул товара в 2026 году

Вы загрузили товар на Ozon, но не можете найти его SKU ID? Этот уникальный идентификатор нужен для редактирования карточки, анализа продаж, настройки рекламы или интеграции с 1С. Без него невозможно работать с API, выгружать отчёты или решать проблемы с модерацией. В этой статье мы покажем 5 проверенных способов, как найти SKU на Озон — от очевидных до скрытых.

Многие продавцы путают SKU ID с артикулом производителя или штрихкодом. На Ozon это внутренний номер товара, который присваивается автоматически при загрузке. Он выглядит как набор цифр (например, 2345678912) и никогда не повторяется. Если вы работаете с большим каталогом, знание, где быстро найти этот идентификатор, сэкономит часы времени.

1. Где находится SKU ID в личном кабинете Ozon (основной способ)

Самый быстрый способ найти SKU — через список товаров в личном кабинете. Этот метод работает для всех типов аккаунтов (FBS, FBO, Ozon Global). Вот пошаговая инструкция:

  1. Авторизуйтесь на сайте seller.ozon.ru.
  2. Перейдите в раздел Товары → Мои товары.
  3. В таблице с товарами найдите колонку "Артикул" (она может быть скрыта — включите её через настройки отображения).
  4. Именно здесь и будет SKU ID вашего товара (пример: 1234567890).

Если колонка отсутствует:

  • 🔧 Нажмите на иконку шестерёнки (Настройки таблицы) в правом верхнем углу.
  • 📋 В списке доступных колонок найдите "Артикул" и перетащите её в активные.
  • 🔄 Обновите страницу — колонка появится.

⚠️ Внимание: На мобильной версии сайта колонка со SKU может не отображаться даже после настройки. Используйте десктопную версию или приложение Ozon Seller.

Открыть раздел "Мои товары"|Включить колонку "Артикул" в настройках|Найти нужный товар по названию или фильтру|Скопировать SKU из колонки-->

2. Как узнать SKU ID через карточку товара

Если вам нужен SKU для одного конкретного товара, проще открыть его карточку. Этот метод удобен, когда вы работаете с модерацией или редактированием:

  1. В разделе Товары → Мои товары кликните по названию нужного товара.
  2. В верхней части карточки найдите блок "Основная информация".
  3. Поле "Артикул" (или "SKU") содержит искомый идентификатор.

Пример расположения:



Основная информация



Название: [Название товара]


Артикул: 9876543210  ← это и есть SKU ID


Категория: [Категория]

💡 Полезный совет: Если вы часто работаете с SKU, добавьте карточку товара в избранное (звёздочка в правом верхнем углу). Так вы сможете быстро находить её через раздел Избранное.

⚠️ Внимание: В карточках товаров, загруженных через Ozon Global, SKU может называться "Vendor Code". Не путайте его с артикулом производителя!

Чем отличается SKU от артикула производителя?

SKU (Stock Keeping Unit) — это внутренний идентификатор Ozon, который присваивается автоматически при загрузке товара. Он уникален для каждого предложения на площадке. Артикул производителя — это номер, который присваивает бренд своему товару (например, Apple для iPhone 15 использует артикул MQ123). На Ozon оба этих номера могут совпадать, но чаще всего SKU генерируется площадкой независимо от производителя.

3. Поиск SKU ID в отчётах Ozon (для массовой выгрузки)

Если вам нужны SKU для всех товаров сразу (например, для интеграции с 1С или анализа продаж), используйте отчёты. Этот метод подходит для продавцов с большим каталогом (100+ позиций).

Инструкция:

  1. Перейдите в раздел Аналитика → Отчёты → Товары.
  2. Выберите отчёт "Список товаров".
  3. Настройте фильтры (даты, статус товара и т. д.) или оставьте их пустыми для выгрузки всех позиций.
  4. Нажмите Сформировать отчёт и дождитесь генерации файла (обычно 1–5 минут).
  5. Скачайте отчёт в формате .xlsx или .csv.

В полученном файле SKU будет в колонке "Артикул" (или "SKU"). Пример структуры отчёта:

Название товара Артикул (SKU) Категория Статус
Смартфон Xiaomi Redmi Note 12 6/128GB 1122334455 Электроника → Смартфоны Активен
Наушники Sony WH-1000XM5 6677889900 Электроника → Аудиотехника На модерации
Умные часы Apple Watch Series 9 9988776655 Электроника → Умные часы Заблокирован

📌 Ключевой вывод: Отчёты — самый надёжный способ получить SKU для всех товаров сразу. Их можно обновлять ежедневно и использовать для автоматизации (например, через Google Sheets или Excel).

4. Как найти SKU ID в API Ozon (для разработчиков)

Если вы используете API Ozon для управления товарами, SKU можно получить через специальные запросы. Этот метод подходит для продавцов, которые автоматизируют работу с каталогом.

Пример запроса для получения списка товаров со SKU:

GET https://api-seller.ozon.ru/v1/product/list

Headers:


- Client-Id: [Ваш Client-ID]


- Api-Key: [Ваш API-ключ]

В ответе вы получите JSON-массив, где для каждого товара будет поле "sku":

{

"result": {


"items": [


{


"sku": 1234567890,


"name": "Ноутбук Lenovo IdeaPad 5",


"status": "active"


}


]


}


}

⚠️ Внимание: Для работы с API нужен Client-ID и API-ключ, которые выдаются в личном кабинете (Настройки → API). Без них запросы не будут работать. Также учитывайте лимиты запросов (обычно 1000 запросов в час).

💡 Полезный совет: Если вы не разбираетесь в API, но хотите автоматизировать работу со SKU, используйте готовые сервисы вроде MoySklad, 1С:Управление торговлей или Alto. Они умеют синхронизироваться с Ozon и pulls SKU автоматически.

5. Где искать SKU в мобильном приложении Ozon Seller

Если вы управляете магазином через мобильное приложение Ozon Seller, найти SKU можно так:

  1. Откройте приложение и авторизуйтесь.
  2. Перейдите в раздел Товары (значок коробки в нижнем меню).
  3. Найдите нужный товар в списке и тапните по нему.
  4. В открывшейся карточке прокрутите вниз до блока "Детали".
  5. Здесь будет строка "Артикул" — это и есть SKU.

📱 Особенности мобильной версии:

  • 🔍 В приложении нет возможности массово выгрузить SKU (только по одному товару).
  • 📋 Колонку со SKU нельзя добавить в основной список товаров (в отличие от десктопной версии).
  • 🔄 Если SKU не отображается, обновите приложение до последней версии.

⚠️ Внимание: В старых версиях приложения (до 2023 года) SKU мог отображаться в другом блоке — "Техническая информация". Если вы не видите артикул, проверьте все вкладки карточки товара.

6. Что делать, если SKU ID не отображается или потерялся

Иногда SKU может "исчезнуть" из интерфейса. Вот 3 причины и способы их решения:

Проблема Причина Решение
SKU не показывается в списке товаров Колонка скрыта в настройках Включите отображение колонки "Артикул" (см. раздел 1)
SKU отсутствует в карточке товара Товар не прошёл модерацию или удалён Проверьте статус товара в разделе Товары → Архив
SKU есть, но он не совпадает с тем, что в 1С При загрузке был указан другой артикул Используйте внешний SKU (артикул производителя) для синхронизации

Если SKU полностью потерян (например, после сбоя или переноса товара), восстановить его можно только через:

  • 📊 Отчёт "Список товаров" (см. раздел 3).
  • 📧 Обращение в поддержку (укажите название товара и приблизительную дату загрузки).
  • 🔧 API-запрос (если у вас есть доступ к API).

💡 Полезный совет: Чтобы не терять SKU, ведите собственную базу данных (например, в Google Sheets или Excel). Туда можно выгружать все артикулы раз в неделю и обновлять при добавлении новых товаров.

7. Лайфхаки для работы со SKU на Ozon

Работа с SKU может быть проще, если знать эти 5 приёмов:

  1. Используйте внешний SKU.

    При загрузке товара можно указать свой артикул (например, из 1С). Тогда Ozon сохранит его как vendor_code, и вам не придётся запоминать внутренний SKU площадки.

  2. Синхронизируйте с 1С автоматически.

    Настройте интеграцию через МойСклад, Альт Софт или Ozon API, чтобы SKU обновлялись в реальном времени.

  3. Добавляйте SKU в название файла.

    При выгрузке фотографий или описаний называйте файлы по шаблону: SKU_название.jpg (пример: 1234567890_iphone15_front.jpg).

  4. Используйте фильтры по SKU.

    В отчётах и личном кабинете можно фильтровать товары по артикулу — это ускорит поиск.

  5. Проверяйте SKU при дублировании товаров.

    Если вы копируете карточку, Ozon присваивает новый SKU. Старый артикул останется только в архиве.

⚠️ Внимание: Никогда не изменяйте SKU вручную после загрузки товара! Это может привести к разрыву связей в отчётах, рекламных кампаниях и API. Если нужно поменять артикул, создайте новый товар и архивируйте старый.

💡

SKU — это "паспорт" вашего товара на Ozon. Его потеря или ошибка в указании может привести к проблемам с продажами, возвратами и аналитикой. Всегда проверяйте артикулы при загрузке и синхронизации!

FAQ: Частые вопросы про SKU на Ozon

Можно ли изменить SKU после загрузки товара?

Нет, SKU на Ozon присваивается автоматически и не редактируется. Если нужно поменять артикул, создайте новый товар с правильным SKU и архивируйте старый. Исключение — внешний SKU (vendor_code), который можно указать при загрузке.

Чем отличается SKU от штрихкода?

SKU — это внутренний идентификатор Ozon, а штрихкод (EAN, UPC) — международный стандарт для розничной торговли. На площадке штрихкод используется для печати этикеток, а SKU — для управления товарами в личном кабинете.

Как найти SKU для товара, который ещё не прошёл модерацию?

SKU присваивается товару мгновенно после загрузки, даже если он на модерации. Найдите его в разделе Товары → На модерации или через отчёт "Список товаров" (установите фильтр по статусу).

Можно ли использовать один SKU для разных товаров?

Нет, каждый SKU на Ozon уникален. Если вы попробуете загрузить два товара с одинаковым артикулом, система выдаст ошибку. Для вариаций (например, разных цветов одной модели) используйте группировку товаров.

Где взять SKU для товара, который я хочу загрузить?

SKU генерируется автоматически после загрузки. Если вы только планируете добавить товар, у него ещё нет SKU. Вы можете указать свой артикул (vendor_code) в файле импорта, но внутренний SKU появится только после публикации.

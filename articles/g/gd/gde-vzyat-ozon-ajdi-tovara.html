Как узнать Ozon ID товара: все способы для продавцов и партнёров

Каждый товар на маркетплейсе Ozon имеет уникальный идентификатор — Ozon ID (или SKU, артикул Озон). Этот номер нужен для работы с каталогом, оформления заказов, возвратов и интеграции через API. Без него невозможно отследить товар в системе, обновить карточку или решить спорные ситуации с поддержкой. Но где его взять, если вы только начинаете работать с платформой или потеряли данные?

В этой статье мы разберём все официальные и альтернативные способы получения Ozon ID, включая ручной поиск в личном кабинете, выгрузку через Excel, использование API и даже парсинг страниц товаров. Особое внимание уделим нюансам для схем FBS и FBO, а также типичным ошибкам, из-за которых продавцы теряют доступ к идентификаторам. Если вы ищете конкретный метод — воспользуйтесь оглавлением ниже.

1. Что такое Ozon ID и зачем он нужен

Ozon ID (или product_id в API) — это уникальный числовой код, который маркетплейс присваивает каждому товару в каталоге. Он отличается от артикула продавца (vendor_code) и штрихкода (barcode). Основные функции идентификатора:

  • 🔗 Связь с системой Озон: по ID платформа определяет, какой именно товар вы имеете в виду при загрузке цен, остатков или характеристик.
  • 📦 Логистика: используется в документах для FBS (склад Озон) и FBO (самовывоз), чтобы не путать похожие позиции.
  • 🔄 Обновления: без ID невозможно изменить карточку товара через Excel или API.
  • 🛠️ Техническая поддержка: при обращении в службу помощи вам потребуется указать этот номер для быстрого поиска проблемы.

Важно понимать, что Ozon ID присваивается автоматически при создании карточки товара. Вы не можете задать его самостоятельно (в отличие от артикула продавца). Если товар ещё не загружен на маркетплейс, у него нет ID — его нужно сначала добавить в каталог.

⚠️ Внимание: Не путайте Ozon ID (уникальный для каждого товара) с offer_id (уникальный для каждого предложения от продавца). Например, у одного товара (скажем, iPhone 15) может быть несколько offer_id от разных продавцов, но один product_id.

2. Способ 1: Поиск Ozon ID в личном кабинете продавца

Самый простой способ — найти идентификатор в интерфейсе Ozon Seller. Для этого:

  1. Авторизуйтесь в личном кабинете продавца.
  2. Перейдите в раздел Товары → Мой каталог.
  3. Найдите нужный товар через поиск или фильтры.
  4. Кликните по названию товара, чтобы открыть его карточку.

В открывшейся карточке Ozon ID будет отображаться в нескольких местах:

  • 📌 В URL-адресе страницы: после /product/ (например, https://seller.ozon.ru/products/12345678 — где 12345678 и есть ID).
  • 📌 В блоке "Основная информация" под названием товара (поле ID товара на Ozon).
  • 📌 В коде для встраивания (если прокрутить страницу вниз до раздела Витрина).

Если товар ещё не опубликован (статус Черновик или На модерации), ID всё равно будет присвоен, но может измениться после одобрения. В этом случае лучше дождаться публикации, чтобы использовать актуальный номер.

💡

Скопируйте ID товара в буфер обмена прямо из URL-адреса — это быстрее, чем искать его в интерфейсе. Просто выделите цифры после"/product/" и нажмите Ctrl+C (или CMD+C на Mac).

3. Способ 2: Выгрузка Ozon ID через Excel

Если вам нужны идентификаторы для большого количества товаров (например, для массового обновления цен), ручной поиск займёт слишком много времени. В этом случае удобнее использовать выгрузку каталога в Excel:

  1. В личном кабинете перейдите в Товары → Мой каталог.
  2. Нажмите кнопку Выгрузить (справа над списком товаров).
  3. Выберите формат Excel (.xlsx) и дождитесь формирования файла.
  4. В полученной таблице ищите колонку product_id — это и есть Ozon ID.

Обратите внимание на нюансы:

  • 📊 В выгрузке могут быть дополнительные колонки: offer_id (ваше предложение), vendor_code (ваш артикул), barcode (штрихкод). Не путайте их!
  • 🔍 Если товар не опубликован, в колонке product_id может стоять 0 или пустое значение.
  • ⏳ Выгрузка большого каталога (10 000+ товаров) может занять до 30 минут.

4. Способ 3: Получение Ozon ID через API

Для автоматизации работы с каталогом (например, синхронизации с 1С или другим ПО) удобно использовать API Ozon. Чтобы получить Ozon ID через API, выполните следующие шаги:

  1. Получите client-id и api-key в настройках личного кабинета (Настройки → API).
  2. Используйте метод POST /v2/product/info/list для получения данных о товарах.
  3. В теле запроса укажите массив offer_id или vendor_code ваших товаров.

Пример запроса на Python с использованием библиотеки requests:

import requests


url ="https://api-seller.ozon.ru/v2/product/info/list"


headers = {


"Client-Id":"ваш_client-id",


"Api-Key":"ваш_api-key",


"Content-Type":"application/json"


}


payload = {


"offer_id": ["123456","789012"], # ваши offer_id


"vendor_code": ["ART001","ART002"] # или ваши артикулы


}



response = requests.post(url, headers=headers, json=payload)


data = response.json



Ozon ID будет в поле"product_id" для каждого товара


for item in data["result"]["items"]:


print(f"Ozon ID: {item['product_id']}, Название: {item['name']}")

В ответе API вы получите массив товаров с полем product_id — это и есть нужный идентификатор. Этот метод подходит для крупных продавцов, которые управляют тысячами позиций.

⚠️ Внимание: При работе с API Озон устанавливает лимиты на количество запросов (обычно 100 запросов в минуту). Превышение лимита приведёт к блокировке на 5 минут. Используйте паузы между запросами или кэшируйте данные.
Как увеличить лимиты API на Ozon?

Для повышения лимитов нужно обратиться в поддержку с обоснованием (например,"у нас 50 000 товаров, текущих лимитов недостаточно"). Озон может увеличить количество запросов до 1000 в минуту для проверенных продавцов.

5. Способ 4: Парсинг Ozon ID со страницы товара

Если у вас нет доступа к личному кабинету или API, но есть ссылка на товар в публичном каталоге Ozon, можно извлечь Ozon ID прямо из URL или HTML-кода страницы. Этот метод подходит для разовых проверок.

Вариант 1: Из URL

Откройте страницу товара на Ozon (например, https://www.ozon.ru/product/telefon-apple-iphone-15-128-gb-black-12345678). Цифры в конце адреса (после последнего /) — это и есть Ozon ID.

Вариант 2: Через код страницы

  1. Откройте страницу товара в браузере (например, Chrome).
  2. Нажмите Ctrl+U (или CMD+Option+U на Mac), чтобы открыть исходный код.
  3. Найдите строку с "productId" (используйте поиск по странице — Ctrl+F).
  4. Скопируйте значение рядом с "productId".

Пример фрагмента кода, где можно найти ID:

<script type="application/ld+json">

{


"@context":"https://schema.org",


"@type":"Product",


"sku":"12345678", <-- Это Ozon ID


"name":"Телефон Apple iPhone 15 128GB Black",



}


</script>

Этот способ работает даже для товаров других продавцов, но учитывайте, что парсинг может нарушать правила Озон при массовом сборе данных. Для личного использования риски минимальны.

6. Способ 5: Обращение в поддержку Ozon

Если ни один из перечисленных методов не сработал (например, товар удалён, но нужен его ID для возврата), можно запросить идентификатор через службу поддержки. Для этого:

  1. Перейдите в Помощь → Обратиться в поддержку в личном кабинете.
  2. Выберите тему Товары и каталог.
  3. Укажите в сообщении:
    • 📌 Название товара (точное совпадение).
    • 📌 Ваш vendor_code или offer_id (если есть).
    • 📌 Причину запроса (например,"нужен для оформления возврата").
  • Прикрепите скриншоты (если есть).

    • Обычно поддержка отвечает в течение 24 часов, но в пиковые периоды (например, перед"Чёрной пятницей") ожидание может затянуться до 3–5 дней. Чтобы ускорить процесс:

    • 📞 Укажите в теме тикеты ключевые слова: "Срочно: запрос Ozon ID для возврата".
    • 📎 Прикрепите доказательства (например, скриншот заказа с проблемным товаром).
    • 🔄 Если ответ задерживается, напишите повторное сообщение в тот же тикет (не создавайте новый!).
    ⚠️ Внимание: Поддержка Ozon не выдаёт Ozon ID для товаров других продавцов — только для ваших позиций. Если вам нужен ID чужого товара (например, для анализа конкурентов), используйте методы из раздела про парсинг.

    7. Типичные ошибки при работе с Ozon ID

    Даже опытные продавцы иногда сталкиваются с проблемами, связанными с идентификаторами товаров. Вот наиболее распространённые ошибки и как их избежать:

    Ошибка Причина Решение
    ID не найден в выгрузке Excel Товар не опубликован или удалён Проверьте статус товара в личном кабинете. Если Удалён, восстановите его через архив.
    API возвращает ошибку 404 Неверный offer_id или vendor_code Убедитесь, что передаёте корректные данные. Сверьтесь с выгрузкой из Excel.
    ID в URL и в личном кабинете не совпадают Товар был перенесён или объединён с другой карточкой Используйте актуальный ID из личного кабинета. Старые ссылки могут вести на несуществующие страницы.
    Поддержка отказывается выдавать ID Товар принадлежит другому продавцу Уточните, что вам нужен ID для вашего предложения (offer_id), а не чужого товара.

    Ещё одна частая проблема — изменение Ozon ID после модерации. Это происходит, если:

    • 🔄 Товар был объединён с другой карточкой (дубль).
    • 📝 Вы изменили ключевые характеристики (например, бренд или категорию).
    • 🚫 Товар был удалён и создан заново.

    В таких случаях старый ID становится неактуальным. Всегда проверяйте актуальность идентификатора перед использованием, особенно если товар недавно редактировался.

    💡

    Если товар был удалён, его Ozon ID нельзя восстановить — после повторной загрузки ему будет присвоен новый идентификатор.

    8. Часто задаваемые вопросы (FAQ)

    Можно ли использовать Ozon ID другого продавца для своего товара?

    Нет, это приведёт к ошибке при загрузке. Каждый товар на Ozon имеет уникальный Ozon ID, даже если это аналогичные позиции от разных продавцов. Если вы пытаетесь"привязаться" к чужой карточке, используйте механизм объединения товаров через поддержку.

    Как найти Ozon ID для товара, который ещё не загружен на маркетплейс?

    У незагруженного товара нет Ozon ID — он присваивается только после создания карточки в каталоге. Если вам нужен идентификатор для нового товара, сначала добавьте его в личном кабинете, затем используйте один из методов из этой статьи.

    Что делать, если Ozon ID изменился после модерации?

    Обновите идентификатор во всех своих системах (1С, Excel, API-интеграции). Старый ID станет неактуальным, и попытки использовать его приведут к ошибкам. Чтобы отслеживать такие изменения, регулярно выгружайте каталог в Excel и сравнивайте product_id с предыдущей версией.

    Можно ли по Ozon ID узнать, какой продавец добавил товар?

    Нет, Ozon ID относится к товару в каталоге, а не к конкретному продавцу. Чтобы найти продавца, используйте offer_id (идентификатор предложения) или проверяйте страницу товара на Ozon — там отображаются все продавцы, у которых есть это предложение.

    Как массово получить Ozon ID для всех своих товаров?

    Самый эффективный способ — выгрузка через Excel (раздел 3) или API (раздел 4). Если товаров больше 100 000, разбейте выгрузку на части или используйте скрипты для автоматического сбора данных. Для парсинга больших объёмов рекомендуем согласовать это с поддержкой, чтобы избежать блокировки.

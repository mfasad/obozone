Как узнать код продавца на Ozon: все способы для продавцов и партнёров

Код продавца на Ozon — это уникальный идентификатор, который присваивается каждому партнёру маркетплейса при регистрации. Он нужен для интеграции с API, оформления возвратов, работы с FBS/FBO, а также при взаимодействии с поддержкой или логистическими партнёрами. Без этого кода невозможно подтвердить ваш статус продавца в системе, поэтому его потеря или невозможность найти может заблокировать ключевые операции.

В этой статье мы разберём все официальные и альтернативные способы, как узнать код продавца на Ozon — от личного кабинета до технической документации. Также вы узнаете, чем отличается seller_id от client_id, где искать код в мобильном приложении, и что делать, если он не отображается в стандартных разделах. Информация актуальна для всех схем работы: FBS, FBO, DBS и RBS.

Что такое код продавца на Ozon и зачем он нужен

Код продавца (или seller_id) — это числовой идентификатор, который Ozon присваивает каждому зарегистрированному партнёру. Он используется:

  • 🔹 Для авторизации в API при интеграции с CRM, 1С или другими системами управления товарами.
  • 🔹 При оформлении возвратов и обменов — код указывается в заявках для подтверждения права на обработку.
  • 🔹 В логистических документах (накладные, акты приёмки) для FBS/FBO.
  • 🔹 Для подключения к программам лояльности (например, Ozon Premium или Ozon Карта).
  • 🔹 В переписке с службой поддержки — без кода продавца операторы не смогут идентифицировать ваш аккаунт.

Важно не путать seller_id с другими идентификаторами:

  • 📌 client_id — используется для авторизации в API (выдаётся при создании приложения в Ozon Seller Lab).
  • 📌 warehouse_id — идентификатор склада (нужен для FBS).
  • 📌 product_id — уникальный номер товара в каталоге Ozon.
⚠️ Внимание: Если вы работаете по модели FBO (самостоятельная доставка), код продавца может потребоваться для генерации этикеток и интеграции с транспортными компаниями. Без него вы не сможете формировать корректные данные для СДЭК, Boxberry или Почты России.

Способ 1: Как найти код продавца в личном кабинете Ozon Seller

Самый простой и быстрый способ — посмотреть код непосредственно в личном кабинете. Для этого:

  1. Авторизуйтесь на сайте seller.ozon.ru.
  2. Перейдите в раздел Профиль → Настройки компании.
  3. В блоке "Основная информация" найдите строку ID продавца или seller_id.

Если раздел Настройки компании не отображается, проверьте:

  • 🔸 Права вашего аккаунта (возможно, у вас роль "Менеджер" без доступа к настройкам).
  • 🔸 Регион кабинета (иногда интерфейс отличается для Ozon Global).
  • 🔸 Версию браузера (рекомендуется Google Chrome или Mozilla Firefox последней версии).

Если код отсутствует в настройках, попробуйте альтернативный путь:

  1. Откройте раздел Финансы → Выплаты.
  2. Скачайте любой отчёт по выплатам (например, за последний месяц).
  3. В названии файла или внутри документа будет указан seller_id.
⚠️ Внимание: В мобильном приложении Ozon Seller код продавца может отображаться не во всех версиях. Если вы не видите его в настройках, используйте десктопную версию кабинета.

Способ 2: Поиск кода в договоре с Ozon

Код продавца всегда указывается в договоре оферты, который вы заключаете с Ozon при регистрации. Чтобы его найти:

  1. Перейдите в раздел Документы → Договоры в личном кабинете.
  2. Скачайте актуальную версию договора (обычно файл в формате .pdf).
  3. Ищите строки: "Продавец (seller_id: XXXXX)" или "Уникальный идентификатор: XXXXX".

Если договор отсутствует в кабинете:

  • 📄 Запросите его у службы поддержки через чат или форму обратной связи.
  • 📄 Проверьте электронную почту, которую указывали при регистрации — на неё мог приходить скан договора.
Что делать, если в договоре нет seller_id?

В некоторых старых версиях договоров код продавца мог обозначаться как "ID партнёра" или "Код компании". Также проверьте приложения к договору — иногда идентификатор указывается в технических спецификациях.

Пример, как код может выглядеть в договоре:



1.1. Продавец: ООО "Ваша Компания", seller_id: 12345678, ИНН 1234567890.

Способ 3: Узнать seller_id через API Ozon

Если вы уже подключены к API Ozon, код продавца можно получить программно. Для этого:

  1. Авторизуйтесь в Ozon Seller Lab.
  2. Перейдите в раздел API → Мои приложения.
  3. Выберите активное приложение и посмотрите параметр seller_id в его настройках.

Также код можно получить через запрос к API. Пример на Python:



import requests



url = "https://api-seller.ozon.ru/v1/analytics/data"


headers = {


"Client-Id": "ваш_client_id",


"Api-Key": "ваш_api_key"


}


response = requests.get(url, headers=headers)


print(response.json()["seller_id"])  # Выведет ваш seller_id

Если вы не работаете с API, но планируете интеграцию, код продавца потребуется для:

  • 🔧 Регистрации нового приложения в Ozon Seller Lab.
  • 🔧 Настройки вебхуков для уведомлений о заказах.
  • 🔧 Подключения к Ozon Global (для продаж за рубеж).
💡

Если вы передаёте разработчику доступ к API, никогда не отправляйте seller_id, client_id и api_key в одном сообщении. Используйте отдельные каналы связи для каждого параметра.

Способ 4: Где посмотреть код в мобильном приложении Ozon Seller

В официальном приложении Ozon Seller (доступно для Android и iOS) код продавца можно найти следующим образом:

  1. Откройте приложение и авторизуйтесь.
  2. Тапните на иконку профиля в правом нижнем углу.
  3. Выберите Настройки → Информация о компании.
  4. Прокрутите вниз до блока "Данные для интеграции" — там будет seller_id.

Если в вашей версии приложения этого раздела нет:

  • 📱 Обновите приложение до последней версии в App Store или Google Play.
  • 📱 Попробуйте открыть кабинет через мобильный браузер (сайт seller.ozon.ru).
  • 📱 Напишите в поддержку через чат в приложении с просьбой предоставить seller_id.
⚠️ Внимание: В приложении Ozon для покупателей (не Ozon Seller!) код продавца посмотреть невозможно. Там отображаются только данные о товарах и заказах, но не о продавце.

Способ 5: Альтернативные источники (письма, поддержка, партнёры)

Если ни один из перечисленных способов не сработал, попробуйте следующие варианты:

Источник Где искать Пример данных
Письма от Ozon Почта, указанная при регистрации (тема: "Ваш аккаунт продавца") "Ваш seller_id: 98765432"
Чат с поддержкой Личный кабинет → "Помощь" → "Написать в поддержку" "Здравствуйте! Подскажите, пожалуйста, мой seller_id?"
Логистические документы Накладные от Ozon FBS или транспортных компаний "Продавец: 12345 (seller_id)"
Партнёрские сервисы Личный кабинет в МойСклад, 1С, RetailCRM Настройки интеграции с Ozon

Служба поддержки Ozon обязана предоставить seller_id по запросу в течение 24 часов. Если ответ задерживается, напишите повторно или позвоните по телефону горячей линии.

Частые ошибки и проблемы при поиске кода продавца

Даже опытные продавцы иногда сталкиваются с трудностями при поиске seller_id. Рассмотрим типичные ситуации и решения:

  • 🚫 Код не отображается в настройках → Проверьте роль аккаунта (должна быть "Администратор") или очистите кэш браузера.
  • 🚫 В договоре старый seller_id → Если вы меняли юридическое лицо, код мог поменяться. Запросите актуальный в поддержке.
  • 🚫 API возвращает ошибку 403 → Проверьте, что используете правильный client_id и api_key для вашего seller_id.
  • 🚫 В мобильном приложении нет раздела с кодом → Обновите приложение или используйте десктопную версию.

Если вы подозреваете, что ваш seller_id скомпрометирован (например, после утечки данных или смены команды разработчиков), немедленно:

  1. Смените api_key в Ozon Seller Lab.
  2. Отзовите все активные токены доступа.
  3. Напишите в поддержку с просьбой проверить активность по вашему seller_id.
⚠️ Внимание: Никогда не публикуйте свой seller_id в открытых источниках (форумы, чаты, социальные сети). Мошенники могут использовать его для фишинга или несанкционированного доступа к вашему аккаунту.

FAQ: Ответы на частые вопросы о коде продавца Ozon

Можно ли изменить seller_id?

Нет, seller_id присваивается один раз при регистрации и не подлежит изменению. Если вы сменили юридическое лицо, вам будет присвоен новый код.

Чем отличается seller_id от client_id?

seller_id — это идентификатор вашей компании на Ozon, а client_id — ключ для авторизации в API (выдаётся при создании приложения в Ozon Seller Lab). Для работы с API нужны оба параметра.

Где взять seller_id для Ozon Global?

Для Ozon Global используется тот же seller_id, что и для российского маркетплейса. Дополнительно может потребоваться warehouse_id для иностранного склада.

Можно ли узнать seller_id другого продавца?

Нет, код продавца — это конфиденциальная информация. Вы можете увидеть только свой seller_id в личном кабинете. Данные других продавцов не разглашаются.

Что делать, если seller_id не подходит для API?

Проверьте, что вы используете правильный client_id и api_key для вашего seller_id. Если ошибка сохраняется, обратитесь в поддержку с логами запросов.

Где на Озоне брать артикул товара: пошаговое руководство для продавцов и покупателей

  • 1. Где посмотреть артикул в карточке товара (для покупателей)

  • 5. Получение артикула через API Ozon (для продавцов)

Артикул на Ozon — это уникальный идентификатор товара, который используется для поиска, заказа, отслеживания и синхронизации данных между продавцом и маркетплейсом. Без него невозможно корректно загрузить товар в каталог, оформить возврат или решить спорную ситуацию с поддержкой. Но где именно его искать? Ответ зависит от вашей роли: покупатель или продавец — и от того, какой тип артикула вам нужен.

Для продавцов артикул (sku или offer_id) критичен при работе с FBS/FBO, загрузке товаров через Excel или API. Для покупателей он может понадобиться при оформлении возврата, поиске аналогов или уточнении характеристик. В этой статье разберём все актуальные способы получения артикула — от очевидных до скрытых.

Важно: на Ozon используется несколько типов артикулов:

  • 🔹 Артикул продавца (sku) — внутренний идентификатор товара в вашем ассортименте.
  • 🔹 Артикул Ozon (offer_id) — уникальный номер товара в системе маркетплейса.
  • 🔹 Артикул производителя — заводской код (например, iPhone 15 Pro имеет артикул MQ3G3).

Каждый из них можно найти разными способами — читайте дальше.

1. Где посмотреть артикул в карточке товара (для покупателей)

Если вы покупатель и хотите узнать артикул товара на Ozon, самый простой способ — заглянуть в его карточку. Однако не все артикулы отображаются одинаково: это зависит от типа товара и настроек продавца.

Для большинства товаров артикул производителя (если он есть) указан в блоке "Характеристики". Например, для смартфонов Samsung Galaxy S23 там будет строка Артикул: SM-S911B/DS. Но что делать, если этого поля нет?

В этом случае попробуйте следующие шаги:

  1. Откройте карточку товара на Ozon.
  2. Прокрутите страницу до блока "Информация о товаре" (обычно под описанием).
  3. Нажмите на ссылку "Все характеристики" — иногда артикул скрыт там.
  4. Если артикула нет, скопируйте URL страницы — в нём может содержаться offer_id (например, .../product/123456789, где 123456789 — это и есть идентификатор).
⚠️ Внимание: Не путайте артикул производителя с offer_id! Первый присваивает бренд (например, Xiaomi), второй — сам Ozon. Для возврата или обращения в поддержку обычно нужен именно offer_id.

2. Как продавцу найти артикул в личном кабинете Ozon Seller

Для продавцов самый надёжный способ получить артикул — через личный кабинет Ozon Seller. Здесь доступны оба типа идентификаторов: ваш внутренний sku и системный offer_id.

Инструкция по поиску:

  • 📌 Перейдите в раздел "Товары" → "Мой ассортимент".
  • 🔍 В поисковой строке введите название или часть названия товара.
  • 📋 В таблице с результатами найдите колонки:
    • Артикул продавца (SKU) — ваш внутренний идентификатор.
    • ID предложения (Offer ID) — уникальный номер Ozon.
  • 📤 Если нужно экспортировать все артикулы сразу, нажмите "Экспорт" → "Excel" — в файле будут обе колонки.

Если товара нет в ассортименте (например, он ещё не загружен), артикул можно найти в истории заказов:

  1. Перейдите в "Заказы" → "История заказов".
  2. Откройте любой заказ с нужным товаром.
  3. В строке товара будет указан Offer ID (кликните на него, чтобы скопировать).

3. Артикул в URL-адресе товара: скрытый идентификатор

Мало кто знает, но в ссылке на товар на Ozon зашит его уникальный offer_id. Это работает как для покупателей, так и для продавцов — главное, иметь прямую ссылку на карточку.

Разберём на примере. Возьмём произвольный товар: 

https://www.ozon.ru/product/telefon-apple-iphone-15-pro-128gb-naturalnyy-titan-172345678/

В этом URL последняя часть (172345678) — это и есть offer_id. Его можно использовать для:

  • 🔄 Быстрого поиска товара в личном кабинете (вставьте в строку поиска в Ozon Seller).
  • 📦 Оформления возврата через поддержку (укажите этот номер).
  • 🔗 Ссылки на товар в чате с менеджером.

Обратите внимание: если в конце URL есть параметры (например, ?asb=123...), их нужно удалить. Чистый offer_id — это только цифры до слеша или знака вопроса.

⚠️ Внимание: Некоторые продавцы используют псевдонимы ссылок (например, через сервисы сокращения). В этом случае offer_id в URL может отсутствовать — проверяйте исходный адрес.

4. Экспорт артикулов через Excel: массовое получение данных

Если вам нужно получить артикулы для сотен или тысяч товаров (например, для синхронизации с 1С или МойСклад), ручной поиск не подходит. В этом случае используйте экспорт данных из Ozon Seller.

Пошаговая инструкция:

  1. В личном кабинете перейдите в "Товары" → "Мой ассортимент".
  2. Нажмите кнопку "Экспорт" (в правом верхнем углу).
  3. Выберите формат Excel (.xlsx).
  4. В полученном файле будут колонки:
    • Артикул продавца (SKU) — ваш внутренний идентификатор.
    • ID предложения (Offer ID) — системный номер Ozon.
    • Артикул производителя — если он был указан при загрузке.

Если экспортированный файл не содержит нужных колонок:

  • 🔄 Обновите данные в личном кабинете (иногда требуется синхронизация).
  • 📊 Проверьте настройки экспорта — возможно, нужные поля отключены.
  • 📧 Обратитесь в поддержку Ozon с запросом на полный экспорт (указывайте причину, например, "нужно для интеграции с 1С").
💡

Если вам нужны артикулы только для определенной категории товаров, перед экспортом используйте фильтр в "Мой ассортимент" — это сэкономит время на обработку файла.

5. Получение артикула через API Ozon (для продавцов)

Для автоматизации работы с большими каталогами продавцы часто используют Ozon API. Через него можно получить артикулы в структурированном виде, например, для синхронизации с CRM или складской системой.

Основные методы API для работы с артикулами:

Метод API Описание Пример ответа
/v2/product/info Получение информации о товаре по offer_id или sku. 
{

"offer_id": "123456789",


"sku": "IPHONE15-128-BLACK",


"name": "iPhone 15 128GB Black"


}
/v1/analytics/data Экспорт данных по продажам с указанием offer_id. 
{

"items": [{


"offer_id": "987654321",


"sales": 15


}]


}
/v1/product/list Список всех товаров продавца с артикулами. 
{

"products": [{


"sku": "AIRPODS-PRO-2",


"offer_id": "1122334455"


}]


}

Чтобы начать работать с API:

  1. Получите Client-ID и API-Key в личном кабинете ("Настройки" → "API").
  2. Используйте документацию Ozon API для формирования запросов.
  3. Для тестирования используйте Postman или cURL.
⚠️ Внимание: При работе с API учитывайте лимиты запросов (обычно 1000 запросов в минуту). Превышение лимита может привести к временной блокировке доступа.
Пример запроса для получения артикулов через API

Чтобы получить список всех ваших товаров с артикулами, отправьте GET-запрос на https://api-seller.ozon.ru/v1/product/list с заголовками: 

Host: api-seller.ozon.ru

Client-ID: ваш_client_id


Api-Key: ваш_api_key

Ответ будет в формате JSON с массивом товаров, где для каждого указаны sku и offer_id.

6. Артикул в мобильном приложении Ozon: где искать?

Если вы пользуетесь мобильным приложением Ozon (как покупатель или продавец), найти артикул можно, но не так очевидно, как на десктопе. Разберём оба случая.

Для покупателей:

  • 📱 Откройте карточку товара в приложении.
  • 🔍 Прокрутите вниз до блока "О товаре".
  • 📋 Если артикул есть, он будет в разделе "Основные характеристики" (иногда нужно нажать "Показать все").
  • 🔗 Если артикула нет, скопируйте ссылку на товар и извлеките offer_id из URL (как описано в разделе 3).

Для продавцов (приложение Ozon Seller):

  • 📊 Перейдите в раздел "Товары".
  • 🔎 Найдите нужный товар через поиск.
  • 📌 Откройте его карточку — offer_id и sku будут указаны вверху под названием.
  • 📤 Для экспорта данных нажмите "Ещё" → "Экспорт".

В мобильной версии некоторые функции ограничены. Например, нельзя массово экспортировать артикулы — для этого лучше использовать десктопную версию или API.

7. Альтернативные способы: чат с поддержкой и парсинг

Если стандартные методы не помогли, остаются менее очевидные, но рабочие варианты. Их стоит использовать только в крайних случаях, так как они требуют дополнительных усилий.

Обращение в поддержку Ozon:

  • 💬 Напишите в чат поддержки ("Помощь" → "Чат с оператором").
  • 📝 Укажите:
    • Название товара.
    • Ссылку на карточку.
    • Причину, по которой нужен артикул (например, "для оформления возврата").
  • ⏳ Обычно ответ приходит в течение 1–2 часов.

Парсинг страницы (для продвинутых пользователей):

  • 🕵️ Откройте карточку товара в браузере.
  • 🖱️ Нажмите F12 (или Ctrl+Shift+I), чтобы открыть Инструменты разработчика.
  • 🔍 Перейдите на вкладку "Elements" и найдите блок с данными товара (обычно содержит data-offer-id или data-sku).
  • 📋 Скопируйте нужный идентификатор.
⚠️ Внимание: Парсинг может нарушать правила использования Ozon. Используйте этот метод только для личных целей и не автоматизируйте массовый сбор данных.
💡

Самый надёжный способ получить артикул — экспорт через личный кабинет или API. Обращение в поддержку и парсинг следует использовать только если другие методы не сработали.

Частые ошибки при работе с артикулами на Ozon

Даже опытные продавцы и покупатели иногда путают типы артикулов или неправильно их используют. Вот самые распространённые ошибки и как их избежать:

1. Путаница между sku и offer_id:

  • 🔄 sku — ваш внутренний идентификатор (может повторяться у разных продавцов).
  • 🔢 offer_id — уникальный номер Ozon (никогда не повторяется).
  • ❌ Ошибка: Указывать sku при обращении в поддержку вместо offer_id.

2. Использование устаревших артикулов:

  • 🗑️ Если товар был удалён и заново загружен, его offer_id изменится.
  • ⚠️ Перед использованием артикула проверьте, актуален ли товар в каталоге.

3. Копирование артикула с ошибками:

  • 📋 При ручном копировании из URL легко пропустить символ или взять лишний.
  • ✅ Всегда проверяйте артикул через поиск в личном кабинете.

4. Игнорирование артикула производителя:

  • 🏭 Для некоторых категорий (электроника, запчасти) Ozon требует указывать заводской артикул.
  • ❌ Его отсутствие может привести к отказу в модерации товара.

FAQ: Ответы на частые вопросы об артикулах на Ozon

Можно ли изменить артикул товара после загрузки на Ozon?

Да, но с ограничениями:

  • 🔄 sku (ваш артикул) можно изменить в любой момент через личный кабинет или API.
  • 🔢 offer_id (системный идентификатор) изменить нельзя — он присваивается автоматически и остаётся постоянным.

Если вам нужно сменить offer_id, придётся удалить товар и загрузить его заново (при этом отзывы и статистика продаж сбросятся).

Что делать, если артикул товара не отображается в карточке?

Попробуйте следующие шаги:

  1. Обновите страницу (F5) — иногда данные подгружаются с задержкой.
  2. Проверьте URL — возможно, offer_id спрятан там.
  3. Напишите продавцу через чат на Ozon с просьбой предоставить артикул.
  4. Если вы продавец, проверьте, заполнено ли поле Артикул производителя в личном кабинете.
Как найти артикул для возврата товара?

Для оформления возврата на Ozon обычно требуется offer_id. Его можно получить:

  • 📄 В чеке (если покупали офлайн в Ozon ПВЗ).
  • 📧 В письме с подтверждением заказа (ищите строку Номер заказа или ID товара).
  • 🔍 В истории заказов в личном кабинете ("Мои заказы" → выберите заказ → "Детали").

Если не можете найти, обратитесь в поддержку Ozon с номером заказа — они предоставят offer_id.

Можно ли использовать один и тот же артикул для разных товаров?

Нет, это нарушает правила Ozon:

  • 🔖 sku должен быть уникальным внутри вашего ассортимента (но может совпадать с sku других продавцов).
  • 🔢 offer_id всегда уникален для всего маркетплейса.

Если загрузить два товара с одинаковым sku, система либо откажет в модерации, либо объединит карточки в одну (что приведёт к путанице в заказах).

Как связаны артикул и штрихкод товара?

Это разные идентификаторы:

  • 📦 Штрихкод (EAN, UPC) — международный стандарт для сканирования на кассе.
  • 🏷️ Артикул (sku, offer_id) — внутренний идентификатор Ozon.

На Ozon штрихкод используется для:

  • 📦 Приёмки товара на складе FBS.
  • 🔍 Быстрого поиска в мобильном приложении сканером.

Артикул же нужен для работы с заказами, возвратами и API. В идеале оба идентификатора должны быть привязаны к товару.

Где в приложении Озон посмотреть процент выкупа: 5 способов для продавцов FBS и FBO

Процент выкупа на Ozon — это ключевой показатель эффективности вашего бизнеса на маркетплейсе. Он отражает, сколько заказов из оформленных покупателями были реально оплачены и не отменены. Для продавцов на схеме FBS (склад маркетплейса) и FBO (самостоятельная доставка) этот метрик помогает оценивать спрос, корректировать цены и выявлять проблемные товары. Но где именно в мобильном приложении или личном кабинете Ozon Seller посмотреть этот параметр? Многие новички теряют часы на поиски, прокликая все разделы подряд.

В этой статье мы разберём 5 проверенных способов найти процент выкупа — от очевидных (статистика в Аналитике) до скрытых (экспорт отчётов). Вы также узнаете, как отличаются данные по FBS и FBO, почему показатель может резко падать, и что делать, если цифры в приложении не совпадают с реальными продажами. А в конце — ответы на частые вопросы, которые продавцы задают поддержке Ozon.

Сразу отметим: процент выкупа не показывается в публичном приложении Ozon для покупателей. Эти данные доступны только в личном кабинете продавца (Ozon Seller) или через API. Если вы ещё не зарегистрированы как продавец — сначала пройдите верификацию на сайте seller.ozon.ru.

1. Основной способ: раздел «Аналитика» в Ozon Seller

Самый быстрый путь к проценту выкупа лежит через вкладку Аналитика в мобильном приложении или веб-версии Ozon Seller. Вот пошаговая инструкция:

1. Откройте приложение Ozon Seller (иконка с зелёным логотипом Ozon и надписью «Для продавцов»).

2. В нижнем меню тапните на иконку графика (раздел Аналитика).

3. В верхней части экрана выберите период (по умолчанию стоит «Сегодня», но для анализа лучше выбрать «Неделя» или «Месяц»).

4. Пролистайте вниз до блока «Конверсия» — здесь отображается процент выкупа (иногда подписан как «% выкупа заказов»).

⚠️ Внимание: В мобильной версии приложения данные могут обновляться с задержкой до 24 часов. Если вам нужны актуальные цифры — используйте веб-версию Ozon Seller на компьютере.

В этом же разделе вы увидите сопутствующие метрики:

  • 📊 Количество заказов — сколько раз покупатели добавили ваш товар в корзину.
  • 🛒 Выкупленные заказы — сколько из них были оплачены.
  • 🚫 Отменённые заказы — сколько покупателей передумали или не оплатили вовремя.

2. Детализация по товарам: отчёт «Эффективность карточек»

Общий процент выкупа по всему магазину — это хорошо, но для оптимизации ассортимента нужны данные по каждому товару отдельно. Их можно найти в отчёте «Эффективность карточек»:

1. В Ozon Seller перейдите в Аналитика → Эффективность карточек.

2. В фильтрах выберите период (рекомендуем «30 дней» для объективной картины).

3. В таблице найдите колонку «% выкупа» — здесь отображается конверсия для каждой позиции.

4. Отсортируйте товары по этому столбцу (по убыванию), чтобы увидеть лидеров и аутсайдеров.

💡 Полезный совет: Если у товара процент выкупа ниже 20%, проверьте:

  • 🔍 Цену — возможно, она завышена по сравнению с конкурентами.
  • 📸 Фото — некачественные изображения отталкивают покупателей.
  • ⭐ Отзывы — низкий рейтинг (ниже 4.5) снижает доверие.
  • 📦 Описание — отсутствие ключевых характеристик или ошибки в тексте.

Для глубокого анализа экспортируйте отчёт в Excel (кнопка «Экспорт» в правом верхнем углу). В файле будут дополнительные метрики, например, «Доля возвратов» и «Средний чек», которые косвенно влияют на выкуп.

3. Разница между FBS и FBO: где искать данные

Процент выкупа может отличаться в зависимости от схемы работы:

  • 🏭 FBS (Fulfillment by Ozon) — товар хранится на складе Ozon, доставка и возврат контролируются маркетплейсом. Здесь выкуп обычно выше за счёт доверия к бренду Ozon и быстрой доставки.
  • 🚚 FBO (Fulfillment by Operator) — вы сами храните товар и организуете доставку. Выкуп может быть ниже из-за длительных сроков доставки или проблем с логистикой.

Чтобы увидеть разделённые данные:

  1. В Аналитике перейдите во вкладку «Продажи».
  2. В фильтрах выберите «Тип логистики» — FBS или FBO.
  3. Обратите внимание на колонку «% выкупа» в отчёте.

⚠️ Внимание: Если вы работаете и по FBS, и по FBO, сравните процент выкупа для одних и тех же товаров на разных схемах. Разница более 15% сигнализирует о проблемах с логистикой в FBO (например, долгая доставка или частые потери посылок).

4. Скрытые данные: API и экспорт отчётов

Если вам нужны сырые данные для собственной аналитики (например, для интеграции с Excel или Google Sheets), используйте API Ozon или экспорт отчётов:

1. Экспорт через Ozon Seller:

- Перейдите в Аналитика → Эффективность карточек.

- Нажмите «Экспорт» (значок стрелочки вниз).

- Выберите формат CSV или Excel.

- В полученном файле будет колонка conversion_rate — это и есть процент выкупа.

2. API Ozon:

- Для доступа к API зарегистрируйте приложение в личном кабинете разработчика.

- Используйте метод GET /v1/analytics/data с параметром metrics=conversion_rate.

- Пример запроса:

curl -X GET "https://api-seller.ozon.ru/v1/analytics/data?metrics=conversion_rate&dimensions=sku&date_from=2026-01-01&date_to=2026-01-31" -H "Client-Id: YOUR_CLIENT_ID" -H "Api-Key: YOUR_API_KEY"

🔹 Ключевой вывод: API даёт самую точную и актуальную информацию, но требует технических навыков. Экспорт отчётов проще, но данные обновляются раз в сутки.

Как автоматизировать сбор данных через API?

Для автоматизации можно использовать скрипт на Python с библиотекой requests. Пример кода для получения процента выкупа по всем товарам:

import requests


url = "https://api-seller.ozon.ru/v1/analytics/data"


params = {


"metrics": "conversion_rate",


"dimensions": "sku",


"date_from": "2026-01-01",


"date_to": "2026-01-31"


}


headers = {


"Client-Id": "YOUR_CLIENT_ID",


"Api-Key": "YOUR_API_KEY"


}



response = requests.get(url, headers=headers, params=params)


data = response.json()


print(data)

Этот скрипт вернёт JSON с процентом выкупа (conversion_rate) для каждого sku. Для визуализации данные можно загрузить в Google Data Studio или Power BI.

5. Почему процент выкупа падает: 7 причин и решения

Если вы заметили резкое снижение процента выкупа, проверьте следующие факторы:

  • 💰 Цена выше рынка — используйте инструмент «Ценообразование» в Ozon Seller, чтобы сравниться с конкурентами.
  • ⏳ Долгая доставка (для FBO) — оптимизируйте логистику или переведите товар на FBS.
  • 📉 Сезонный спад спроса — анализируйте тренды через Аналитика → Спрос.
  • 🚨 Технические ошибки — например, неверно указанный остаток товара (покупатель оформляет заказ, но товар отсутствует).
  • 📢 Недостоверное описание — если товар не соответствует фото или характеристикам, покупатели отменяют заказ.
  • 🔄 Частые возвраты — проверьте качество товара и упаковки.
  • 🎁 Акции конкурентов — ваш товар может проигрывать по условиям (например, у них бесплатная доставка).

Критическая информация: Если процент выкупа упал ниже 15% и держится на этом уровне более 7 дней, Ozon может понизить позицию товара в выдаче или даже заблокировать карточку. В этом случае срочно:

  1. Проверьте отзывы на предмет жалоб.
  2. Сравните цены с топ-10 конкурентов.
  3. Обратитесь в поддержку Ozon через Помощь → Написать в поддержку.

6. Альтернативные способы: сторонние сервисы и парсинг

Если встроенной аналитики Ozon недостаточно, можно использовать сторонние инструменты:

  • 📈 Sellerboard — сервис для аналитики продаж на маркетплейсах. Подключается к Ozon Seller и показывает процент выкупа в реальном времени.
  • 🤖 Parser.Ozon — парсинг данных конкурентов, включая их процент выкупа (оценка по косвенным метрикам).
  • 📊 Google Data Studio — для визуализации данных, экспортированных из Ozon.

⚠️ Внимание: Сторонние сервисы могут показывать данные с погрешностью до 5–10% по сравнению с официальной статистикой Ozon. Используйте их только для дополнительного анализа, а не как основной источник.

🔹 Полезный совет: Настройте автоматическое обновление данных в Google Sheets через Ozon API:

  1. Создайте новый лист в Google Sheets.
  2. Используйте функцию =IMPORTDATA("URL_API_запроса").
  3. Настройте триггер для ежедневного обновления (через Apps Script).

FAQ: Частые вопросы о проценте выкупа в Ozon

🔍 Как часто обновляется процент выкупа в приложении?

В мобильном приложении Ozon Seller данные обновляются 1 раз в сутки (обычно утром по московскому времени). В веб-версии и через API информация актуальнее — с задержкой 2–4 часа.

📉 Почему процент выкупа в отчёте и в API отличается?

Разница может возникать из-за:

  • Разных периодов агрегации (например, в отчёте данные за вчера, а в API — за позавчера).
  • Фильтров (в отчёте может быть применён фильтр по региону или типу логистики).
  • Технических задержек (API иногда отдаёт кешированные данные).

Для точности сверяйтесь с веб-версией Ozon Seller в разделе Аналитика.

🛒 Можно ли увидеть процент выкупа по конкретному покупателю?

Нет, Ozon не предоставляет данные о выкупе в разрезе отдельных покупателей. Вы можете увидеть только общую статистику по товарам или заказам. Если нужно проанализировать поведение клиентов, используйте метрики «Частота покупок» и «Средний чек» в разделе Аналитика → Покупатели.

📊 Как повысить процент выкупа для товаров на FBO?

Для FBO критически важны:

  • ⚡ Скорость доставки — стремитесь к 1–3 дням (как у FBS).
  • 💳 Бесплатная доставка — даже если вы заложите её в цену товара.
  • 📦 Надёжная упаковка — чтобы товар не повредился в пути.
  • 📞 Оперативная связь — отвечайте на вопросы покупателей в течение часа.

Также проверьте, не попадает ли ваш товар под условия программы «Защита покупателя» — это увеличивает доверие.

🔄 Что делать, если процент выкупа резко упал без видимых причин?

Сначала исключите технические проблемы:

  1. Проверьте, не было ли сбоев в работе Ozon (например, проблемы с оплатой).
  2. Убедитесь, что остатки товара актуальны (нет ситуации, когда покупатель оформляет заказ, а товара нет).
  3. Посмотрите, не изменились ли условия доставки (например, Ozon временно приостановил работу с вашим регионом).

Если проблема не в этом — обратитесь в поддержку с указанием:

  • Названия товара и его SKU.
  • Периода, когда произошёл спад.
  • Скриншотов из Аналитики.

Где в Озоне посмотреть отчет по продажам: 5 способов для продавцов

Продажи на Ozon требуют постоянного мониторинга: нужно отслеживать динамику, анализировать спрос и корректировать стратегию. Но где именно в личном кабинете продавца скрываются эти данные? Многие новички теряют часы на поиски отчетов, не подозревая, что платформа предлагает сразу несколько способов их получения — от базовой аналитики до глубокой выгрузки через API.

В этой статье мы разберем все актуальные на 2026 год методы доступа к отчетам о продажах: от стандартных разделов личного кабинета до скрытых функций для продвинутых пользователей. Вы узнаете, как скачать данные в Excel, настроить автоматические отчеты и даже интегрировать их с внешними сервисами. А еще — раскроем лайфхак, как получить расширенную статистику по FBS-заказам, которую Озон не показывает в базовом интерфейсе.

1. Стандартный раздел «Аналитика» в личном кабинете

Самый очевидный способ — использовать встроенный раздел «Аналитика» в личном кабинете продавца. Он доступен всем пользователям без дополнительных настроек. Чтобы туда попасть:

  1. Авторизуйтесь на сайте seller.ozon.ru.
  2. В верхнем меню выберите вкладку Аналитика.
  3. В выпадающем списке кликните на Продажи.

Здесь вы увидите графики по:

  • 📈 Динамике продаж (по дням/неделям/месяцам)
  • 💰 Выручке и прибыли (с учетом комиссий Озона)
  • 📦 Количеству заказов (включая отмененные)
  • 🔄 Возвратам и отказам (с разбивкой по причинам)
💡

Используйте фильтр по датам в правом верхнем углу, чтобы сравнить продажи за разные периоды. Например, «этот месяц vs прошлый месяц» покажет сезонные колебания.

Ограничение этого метода: данные обновляются с задержкой до 24 часов, а детализация по конкретным товарам требует дополнительных кликов. Для экспорта в Excel нажмите кнопку Экспорт в правом верхнем углу — но учтите, что в файле не будет информации о FBS-заказах (их нужно выгружать отдельно).

2. Раздел «Отчеты» — скрытые возможности

Мало кто знает, но в личном кабинете есть отдельный раздел Отчеты, где собраны более детализированные данные. Чтобы его найти:

  1. Перейдите в Аналитика → Отчеты.
  2. Выберите тип отчета: По продажам, По товарам или По финансам.
  3. Настройте период и формат выгрузки (Excel или CSV).

Вот что можно выгрузить здесь:

Тип отчета Что содержит Период доступности
Продажи по дням Количество заказов, выручка, средний чек Последние 3 года
Товарная аналитика Продажи по каждому SKU, остатки, возвратность Последние 12 месяцев
Финансовые отчеты Комиссии Озона, вознаграждения, удержания Последние 24 месяца
FBS-отчеты Детализация по логистике, штрафам, хранению Последние 6 месяцев

Главное преимущество этого раздела — возможность настроить автоматическую отправку отчетов на email. Для этого:

  1. Создайте шаблон отчета с нужными параметрами.
  2. Нажмите Настроить рассылку.
  3. Укажите частоту (ежедневно/недельно/ежемесячно) и адреса получателей.

3. Выгрузка через API: для продвинутых пользователей

Если вам нужны данные в реальном времени или их автоматическая обработка, стоит подключиться к Ozon Seller API. Это официальный интерфейс для программистов, позволяющий:

  • 🤖 Получать данные о заказах каждые 5 минут (вместо суточной задержки в личном кабинете).
  • 📊 Интегрировать статистику с 1С, Google Sheets или Power BI.
  • 🔄 Автоматически обновлять остатки и цены на основе продаж.

Чтобы начать работать с API:

  1. Зарегистрируйте приложение в разделе API личного кабинета.
  2. Получите Client-ID и API-Key (сохраните их в безопасном месте!).
  3. Используйте документацию Озона для отправки запросов. Пример запроса на получение отчета по продажам: 
    GET https://api-seller.ozon.ru/v2/analytics/sales

    Headers:
    

    Client-Id: YOUR_CLIENT_ID
    

    Api-Key: YOUR_API_KEY
    

    Content-Type: application/json
    


    Body:
    

    {
    

    "date_from": "2026-01-01T00:00:00Z",
    

    "date_to": "2026-01-31T23:59:59Z"
    

    }

Как ускорить обработку API-запросов?

Используйте пагинацию (параметры limit и offset) для больших объемов данных. Например, ?limit=1000&offset=0 вернет первую тысячу записей.

Предупреждение: API имеет лимиты на количество запросов (обычно 1000 в час). При превышении Озон блокирует доступ на 1 час. Чтобы избежать этого, кэшируйте данные или используйте вебхуки для получения обновлений.

💡

API — единственный способ получить данные о продажах в реальном времени. Но для его использования нужны хотя бы базовые знания программирования или помощь разработчика.

4. Мобильное приложение Ozon Seller: удобство или ограничения?

Официальное приложение Ozon Seller (доступно для iOS и Android) позволяет просматривать основные метрики продаж прямо с телефона. Однако функционал здесь урезан по сравнению с десктопной версией.

Что можно сделать в приложении:

  • 📱 Просмотреть текущую выручку и количество заказов.
  • 📉 Увидеть график продаж за последние 30 дней.
  • 🔔 Получить push-уведомления о новых заказах или проблемах.

Чего нельзя сделать:

  • ❌ Выгрузить отчеты в Excel.
  • ❌ Настроить детализированные фильтры (например, по регионам или категориям товаров).
  • ❌ Просмотреть историю продаж старше 1 месяца.

Открыть раздел "Продажи"|Просмотреть новые заказы|Отследить динамику выручки|Проверить уведомления о штрафах-->

Приложение удобно для оперативного контроля, но для глубокого анализа лучше использовать десктопную версию или API. Кстати, в мобильной версии есть баг: иногда графики продаж показывают данные с задержкой до 48 часов — об этом стоит помнить при принятии решений.

5. Скрытые отчеты: FBS, штрафы и логистика

Озон скрывает часть данных о FBS-заказах (fulfillment by Ozon) в стандартных отчетах. Чтобы получить полную картину, нужно:

  1. Перейти в Логистика → FBS-отчеты.
  2. Выбрать Отчет по штрафам и удержаниям.
  3. Экспортировать данные в Excel для детального анализа.

В этом разделе вы найдете:

  • 🚚 Штрафы за несвоевременную отгрузку (если вы работаете по модели FBO).
  • 📦 Стоимость хранения на складах Озона (актуально для FBS).
  • 🔙 Возвраты и их причины (с разбивкой по категориям: брак, несоответствие описанию и т.д.).

Лайфхак: чтобы увидеть реальную прибыль по FBS, скачайте отчет «Финансовые операции» и отфильтруйте его по строке «Удержание за логистику». Эти данные не отображаются в стандартной аналитике!

💡

Сравнивайте штрафы за логистику с вашей средней маржой. Если они превышают 10% от выручки, стоит пересмотреть схему работы с FBS.

Еще один полезный отчет — Акт сверки. Он доступен в разделе Финансы → Документы и показывает все финансовые операции между вами и Озоном, включая:

  • Выплаты за продажи.
  • Удержания за комиссии и штрафы.
  • Корректировки по возвратам.

Сервис Функции Стоимость

ZonGuru Мониторинг позиций, анализ ключевых слов, отчеты по продажам От 29$/мес

Преимущества внешних сервисов:

Недостатки:

  • ⚠️ Дополнительные расходы (от 500 руб./мес.).

💡

Сторонние сервисы полезны для масштабирования бизнеса, но начинающим продавцам достаточно встроенных инструментов Озона.

7. Частые ошибки при работе с отчетами

Даже опытные продавцы допускают ошибки при анализе продаж. Вот самые распространенные:

⚠️ Внимание: Не путайте выручку и прибыль. В отчетах Озона по умолчанию показывается выручка (без вычета комиссий, штрафов и стоимости товара). Чтобы увидеть реальную прибыль, скачивайте Финансовый отчет и учитывайте все удержания.

Другие типичные промахи:

  • 📅 Игнорирование сезонности. Сравнивать продажи в январе и декабре бессмысленно — используйте фильтр «год к году».
  • 📦 Неучет возвратов. В графике продаж не видно, сколько заказов было возвращено. Эта информация есть в отдельном отчете Возвраты.
  • 🚫 Пропуск FBS-штрафов. Многие продавцы не проверяют раздел Логистика → Штрафы, хотя там могут быть удержания за просроченные отгрузки.

Еще одна ловушка — задержка данных. Например:

  • В разделе Аналитика продажи обновляются раз в сутки (в 3:00 по МСК).
  • Финансовые отчеты могут запаздывать на до 3 дней из-за обработки возвратов.
  • API дает данные с задержкой в 15-30 минут (даже despite обещанных 5 минут).
Как проверить актуальность данных?

Сравните количество заказов в отчете и в разделе Заказы → Новые. Если цифры расходятся более чем на 5%, данные еще не обновлены.

Чтобы избежать ошибок, используйте этот чеклист:

FAQ: Ответы на частые вопросы

Можно ли получить отчет по продажам за прошлый год?

Да, но с ограничениями. В разделе Отчеты доступны данные за последние 3 года, но детализация по товарам сохраняется только за 12 месяцев. Для более старых периодов можно запросить архив у поддержки Озона (через тикет в личном кабинете).

Почему в отчете не совпадают суммы с реальными выплатами?

Это происходит из-за:

  1. Удержаний (комиссии Озона, штрафы, стоимость логистики).
  2. Задержки выплат (деньги за продажи поступают на счет с lagом в 1-2 дня).
  3. Возвратов (сумма за возвращенный товар списывается позже).

Чтобы увидеть реальную картину, скачивайте Финансовый отчет и сверяйте его с Актом сверки.

Как экспортировать отчет по продажам в Google Sheets?

Есть два способа:

  1. Ручной экспорт: скачайте файл Excel из личного кабинета и загрузите его в Google Sheets через Файл → Импорт.
  2. Автоматизация через API: используйте скрипт на Google Apps Script для прямой загрузки данных. Пример кода: 
    function importOzonSales() {

    const url = "https://api-seller.ozon.ru/v2/analytics/sales";
    

    const options = {

Где посмотреть продажи по конкретному товару?

Перейдите в Аналитика → Товарная аналитика и:

  1. Введите артикул или название товара в поисковой строке.
  2. Выберите период (максимум — 12 месяцев).
  3. Экспортируйте данные в Excel, если нужна детализация по дням.

В отчете будут:

  • Количество продаж.
  • Выручка (без учета комиссий).
  • Коэффициент возвратов.
  • Остатки на складе.
Можно ли получить отчет по продажам в разрезе регионов?

В стандартных отчетах Озона нет разбивки по регионам, но есть обходные пути:

  1. Через API: в ответе на запрос /v2/analytics/sales есть поле region с кодом региона (например, 77 для Москвы).
  2. Через поддержку: напишите в тикет с просьбой предоставить данные по географии продаж (иногда идут навстречу).
  3. Сторонние сервисы: Sellerboard или ZonGuru умеют строить тепловые карты продаж по РФ.

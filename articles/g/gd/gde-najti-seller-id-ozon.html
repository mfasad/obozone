Как найти Seller ID на Ozon: все способы для продавцов в 2026 году

  • 3. Поиск Seller ID в мобильном приложении Ozon Seller

Идентификатор продавца (Seller ID) на Ozon — это уникальный цифровой код, который присваивается каждому партнеру маркетплейса. Он нужен для интеграции с API, настройки рекламных кампаний, работы с сервисами аналитики и даже для общения с поддержкой. Без этого кода невозможно подтвердить ваш статус продавца в большинстве внешних сервисов, подключенных к платформе.

Проблема в том, что Ozon не отображает Seller ID на главной странице личного кабинета, и многие новички тратят часы на его поиски. В этой статье мы собрали все актуальные способы получения идентификатора — от официальных (через настройки кабинета) до альтернативных (парсинг писем или API-запросы). Также разберем, почему некоторые методы перестали работать в 2026 году и как обойти типичные ошибки.

1. Где искать Seller ID в личном кабинете Ozon Seller

Самый надежный способ — найти идентификатор непосредственно в интерфейсе Ozon Seller. Однако путь к нему неочевиден: разработчики платформы регулярно обновляют расположение элементов, и старые инструкции (например, из блогов 2022 года) уже неактуальны.

Актуальный маршрут на 2026 год:

  1. Авторизуйтесь в личном кабинете Ozon Seller.
  2. Перейдите в раздел Профиль → Настройки компании.
  3. В блоке "Основная информация" прокрутите страницу до поля Идентификатор продавца (или Seller ID).
  4. Скопируйте числовой код — это и есть ваш уникальный идентификатор.
💡

Если поле с Seller ID не отображается, попробуйте обновить страницу в режиме инкогнито или с другого браузера. Иногда проблема связана с кешированием старых версий интерфейса.

Важно: в мобильной версии кабинета этот раздел может отсутствовать. Используйте десктопную версию сайта или приложение Ozon Seller для Android/iOS (но там путь другой — см. раздел 3).

2. Как узнать Seller ID через API Ozon

Если вы уже работаете с API маркетплейса, идентификатор можно получить программно. Этот метод подходит для продавцов, которые используют собственные интеграции или подключены к сторонним сервисам (например, МойСклад, 1С, RetailCRM).

Инструкция для получения Seller ID через API:

  1. Сгенерируйте Client-ID и API-Key в разделе API → Ключи доступа личного кабинета.
  2. Отправьте GET-запрос на эндпоинт: 
    https://api-seller.ozon.ru/v1/info/client

    с заголовком авторизации:

    Client-ID: {ваш_client_id}

    API-Key: {ваш_api_key}

  3. В ответе JSON найдите поле "seller_id" — это и есть ваш идентификатор.

Пример ответа API:

{

"seller_id": 12345678,


"name": "ООО Ромашка",


"status": "active"


}

Что делать, если API возвращает ошибку 403?

Ошибка 403 ("Forbidden") при запросе к API Ozon обычно означает, что у вашего ключа недостаточно прав. Проверьте:

1. Актуальность API-Key (срок действия — 1 год).

2. Наличие прав на эндпоинт /v1/info/client в настройках ключа.

3. Корректность заголовков авторизации (регистр имеет значение!).

Если проблема остается, создайте новый ключ в личном кабинете и повторите запрос.

⚠️ Внимание: Не передавайте API-Key третьим лицам. Компрометация ключа может привести к блокировке аккаунта. Для внешних интеграций используйте ограниченные ключи с минимальными правами.

3. Поиск Seller ID в мобильном приложении Ozon Seller

Мобильное приложение Ozon Seller (доступно для Android и iOS) также содержит идентификатор продавца, но найти его сложнее, чем в веб-версии. Алгоритм действий:

  • 📱 Откройте приложение и авторизуйтесь.
  • 🔍 Тапните по иконке профиля (правый верхний угол).
  • 📄 Выберите раздел "О компании" (на iOS может называться "Информация о продавце").
  • 🔢 Прокрутите до блока "Данные для интеграции" — там будет строка ID продавца.

Если раздела нет, обновите приложение до последней версии. В старых версиях (до 2023 года) Seller ID мог отображаться в разделе API и интеграции, но сейчас этот путь устарел.

4. Альтернативные способы: письма от Ozon и документы

Если доступ к личному кабинету временно ограничен (например, из-за технических работ), Seller ID можно найти в официальных письмах от Ozon. Вот где искать:

  • 📧 Письмо с подтверждением регистрации (тема: "Добро пожаловать в Ozon Seller!"). Идентификатор указан в разделе "Ваши данные для входа".
  • 📑 Договор оферты (приложение к письму с темой "Подписан договор"). Seller ID прописан в шапке документа рядом с названием вашей компании.
  • 💰 Выписки по выплатам (ежемесячные отчеты). Ищите строку SellerID: XXXXXXXX внизу PDF-файла.

🔎 Совет: Используйте поиск по почте с запросом "Seller ID" OR "Идентификатор продавца" site:ozon.ru. Это поможет быстро найти все письма, где упоминается ваш ID.

⚠️ Внимание: Не путайте Seller ID с Client-ID (для API) или Номером партнера (для программы лояльности). Эти коды выглядят похоже, но выполняют разные функции. Например, Client-ID начинается с букв (например, client123...), а Seller ID — всегда числовой.

5. Парсинг URL: скрытый способ для опытных пользователей

Этот метод подходит тем, кто разбирается в структуре веб-страниц. Ozon иногда подгружает Seller ID в URL или скрытых полях формы. Вот как его извлечь:

  1. Откройте личный кабинет и перейдите в раздел Товары → Список товаров.
  2. Нажмите F12 (или Ctrl+Shift+I), чтобы открыть Инструменты разработчика.
  3. Перейдите на вкладку "Network" и обновите страницу (F5).
  4. В фильтре введите api-seller и найдите запрос к эндпоинту /v2/product/list.
  5. Откройте запрос → вкладка "Headers" → скопируйте значение x-seller-id из заголовков.

🛠️ Техническая заметка: Этот способ работает не всегда, так как Ozon может шифровать заголовки или использовать временные токены. Если x-seller-id отсутствует, попробуйте другой эндпоинт, например, /v1/finance/transaction/list.

6. Типичные ошибки и как их избежать

Даже опытные продавцы иногда путают Seller ID с другими идентификаторами. Вот самые распространенные ошибки и их решения:

Ошибка Причина Как исправить
Неверный формат ID (буквы или символы) Скопирован Client-ID вместо Seller ID Проверьте раздел Профиль → Настройки компании
ID не найден в личном кабинете Устаревшая версия интерфейса или мобильное приложение Обновите страницу (Ctrl+F5) или используйте десктопную версию
API возвращает пустой seller_id Недостаточно прав у API-Key Создайте новый ключ с доступом к /v1/info/client
ID в письме и личном кабинете не совпадают У вас несколько аккаунтов (например, тестовый и основной) Проверьте, из какого аккаунта отправлялся запрос

⚠️ Внимание: Если вы работаете с несколькими магазинами на Ozon (например, под разными юридическими лицами), у каждого будет свой Seller ID. Не используйте идентификатор одного магазина для другого — это приведет к ошибкам в API и возможной блокировке.

7. Зачем нужен Seller ID: практическое применение

Многие продавцы ошибочно считают, что Seller ID нужен только для API. На самом деле он требуется в десятках сценариев:

  • 📊 Подключение аналитики: Google Analytics, Яндекс.Метрика, Roistat запрашивают Seller ID для сегментации трафика с Ozon.
  • 💸 Рекламные кампании: При настройке таргетированной рекламы через Ozon Advertising или MyTarget.
  • 📦 Логистика: Сервисы СДЭК, Boxberry или ПЭК используют ID для автоматизации FBS-заказов.
  • 🔄 Обмен данными: Синхронизация остатков, цен и заказов с 1С, МойСклад или RetailCRM.
  • 🛡️ Безопасность: Подтверждение права собственности на аккаунт при обращении в поддержку.

💡 Пример: При подключении Ozon к 1С через модуль обмена вам потребуется ввести Seller ID в настройках интеграции. Без него система не сможет идентифицировать ваш магазин и загрузить товары.

💡

Seller ID — это не просто технический идентификатор, а ключ ко всем внешним интеграциям. Без него невозможно автоматизировать продажи, рекламу или логистику на Ozon.

FAQ: Частые вопросы о Seller ID на Ozon

Можно ли изменить Seller ID?

Нет, Seller ID присваивается один раз при регистрации и не подлежит изменению. Если вы сменили юридическое лицо или название компании, старый ID останется прежним, но к нему привяжут новые данные.

Что делать, если Seller ID потерян, а доступ к почте утерян?

Обратитесь в поддержку Ozon Seller через форму обратной связи (раздел Помощь → Написать в поддержку). Приложите скан документа, подтверждающего ваше право на аккаунт (например, выписку из ЕГРЮЛ). Восстановление может занять до 3 рабочих дней.

Нужно ли скрывать свой Seller ID?

Сам по себе Seller ID не является конфиденциальной информацией — его можно публиковать в открытых источниках (например, в инструкциях для партнеров). Однако не рекомендуется разглашать его вместе с API-Key или данными для входа в кабинет.

Как узнать Seller ID конкурента?

Получить Seller ID другого продавца легально невозможно — эта информация закрыта. Однако вы можете анализировать его товары через Ozon API (эндпоинт /v2/product/info), если знаете product_id (идентификатор товара).

Почему у меня два разных Seller ID?

Это означает, что у вас зарегистрировано два отдельных аккаунта (например, для разных юридических лиц или типов деятельности). Проверьте, в какой кабинет вы авторизованы, и используйте соответствующий ID. Объединить аккаунты нельзя — они работают независимо.

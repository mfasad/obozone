Где смотреть заказы на Ozon: все способы для продавцов в 2026 году

  • 1. Личный кабинет Ozon Seller: базовый способ просмотра заказов
  • 2. Мобильное приложение Ozon Seller: управление заказами с телефона

  • FAQ: Частые вопросы о просмотре заказов на Ozon

Работа на маркетплейсе Ozon требует постоянного контроля заказов — от момента оформления покупателем до окончательной доставки. Но где именно продавцы могут отслеживать свои заказы, анализировать их статус и управлять логистикой? Эта статья раскроет все актуальные способы просмотра заказов на Ozon в 2026 году, включая скрытые функции личного кабинета, возможности мобильного приложения и автоматизированные инструменты для масштабных продаж.

Мы разберём не только базовые методы (вроде раздела «Заказы» в Личном кабинете), но и продвинутые — например, как настроить уведомления о новых заказах через API или экспортировать данные в Excel для анализа. Особое внимание уделим различиям между схемами FBS (склад Ozon) и FBO (самостоятельная доставка), так как интерфейс и доступные опции в них существенно отличаются.

Если вы только начинаете продавать на Ozon или хотите оптимизировать рутинные процессы, эта инструкция поможет избежать ошибок при работе с заказами — от пропущенных уведомлений до проблем с логистикой.

1. Личный кабинет Ozon Seller: базовый способ просмотра заказов

Основной инструмент для работы с заказами — Личный кабинет продавца (Ozon Seller). Здесь отображаются все заказы независимо от схемы работы (FBS или FBO), но с разными опциями управления. Чтобы открыть список заказов:

  1. Авторизуйтесь на сайте Ozon Seller.
  2. В верхнем меню выберите раздел «Заказы».
  3. По умолчанию откроется вкладка «Все», где заказы сортируются по дате создания (новые сверху).

В этом разделе доступны фильтры по статусу (Новый, В обработке, Отменён и др.), дате, артикулу товара и даже региону доставки. Для удобства можно сохранять часто используемые фильтры как «Избранные».

💡

Используйте горячие клавиши в Личном кабинете: Ctrl+F для поиска по артикулу или номеру заказа, Ctrl+Shift+N — быстрое создание этикетки для FBS-заказа.

Важный нюанс: в FBS продавец видит заказы только после того, как Ozon резервирует товар на своём складе. В FBO заказ попадает в список сразу после оплаты покупателем, но здесь продавец сам отвечает за упаковку и передачу заказа в логистическую компанию.

2. Мобильное приложение Ozon Seller: управление заказами с телефона

Для тех, кто ведёт бизнес в пути, у Ozon есть официальное мобильное приложение Ozon Seller (доступно для Android и iOS). Его функционал дублирует основные возможности веб-версии, но с адаптацией под сенсорный экран. Преимущества приложения:

  • 📲 Push-уведомления о новых заказах, отменах и изменениях статуса (настраиваются в профиле).
  • 🔄 Быстрое обновление списка заказов свайпом вниз.
  • 📦 Сканер штрихкодов для проверки товаров при сборке FBS-заказов.
  • 📊 Виджет на главный экран с количеством новых заказов (на Android).

Чтобы настроить уведомления:

  1. Откройте приложение и перейдите в Профиль → Настройки → Уведомления.
  2. Включите опции «Новые заказы», «Отмены» и «Проблемы с доставкой».
  3. Укажите период активности (например, с 9:00 до 21:00), чтобы не получать уведомления ночью.

3. Экспорт заказов в Excel: аналитика и учёт

Для глубокого анализа продаж или интеграции с 1С/МойСклад удобно экспортировать заказы в формат .xlsx или .csv. В Личном кабинете это делается так:

  1. Перейдите в раздел «Заказы».
  2. Примените нужные фильтры (например, заказы за последний месяц).
  3. Нажмите кнопку «Экспорт» в правом верхнем углу.
  4. Выберите формат (Excel или CSV) и нажмите «Скачать».

В экспортированном файле будут колонки с данными:

Поле Описание Пример значения
order_id Уникальный номер заказа 2103456789
status Текущий статус (processing, delivered и др.) awaiting_packaging
created_at Дата и время создания заказа 2026-05-15 14:30:22
items[].sku Артикул товара в заказе AB12345CD
financial_data.price Сумма заказа (с учётом скидок) 1299.00

Экспорт полезен для:

  • 📈 Аналитики продаж (например, какие товары чаще заказывают в определённом регионе).
  • 📑 Бухгалтерского учёта (сверка с выписками из Ozon Банка).
  • 🔄 Интеграции с CRM или системами автоматизации.
Как автоматизировать экспорт заказов?

Можно использовать Ozon API или сервисы вроде Zapier, чтобы ежедневно сохранять новые заказы в Google Sheets или Excel Online. Для этого потребуется получить API-key в Личном кабинете и настроить вебхуки. Подробнее — в документации Ozon API.

4. Отслеживание заказов через Ozon API: для автоматизации

Если у вас сотни заказов в день, ручной просмотр каждого становится неэффективным. В этом случае поможет Ozon API — интерфейс программирования, который позволяет:

  • 🤖 Получать списки заказов в реальном времени.
  • 📡 Отслеживать изменения статусов (например, когда заказ передан курьеру).
  • 🔄 Автоматически обновлять данные в вашей ERP-системе.

Пример запроса для получения списка заказов (на языке Python):

import requests


url = "https://api-seller.ozon.ru/v2/posting/fbs/list"


headers = {


"Client-Id": "Ваш_Client_ID",


"Api-Key": "Ваш_API_Key",


"Content-Type": "application/json"


}


payload = {


"dir": "ASC",


"filter": {


"since": "2026-05-01T00:00:00Z",


"to": "2026-05-15T23:59:59Z"


},


"limit": 1000


}



response = requests.post(url, headers=headers, json=payload)


print(response.json())

Чтобы начать работать с API:

  1. Получите Client-ID и API-Key в Личном кабинете (Настройки → API).
  2. Изучите документацию (раздел «Posting API» для FBS или «Orders API» для FBO).
  3. Используйте Postman или напишите скрипт на Python/PHP для интеграции.
💡

API позволяет получать данные о заказах с задержкой не более 5 минут, тогда как экспорт в Excel обновляется раз в час. Это критично для бизнес-процессов, где важна оперативность (например, сборка заказов на складе).

Важно: с 1 июня 2026 года Ozon ввёл ограничение на количество запросов к API — 1000 в минуту для одного аккаунта. Превышение лимита блокирует доступ на 5 минут.

5. Уведомления о заказах: email, SMS и Telegram-боты

Пропустить новый заказ — значит рисковать штрафами за несвоевременную обработку. Чтобы этого избежать, настройте уведомления через:

  • 📧 Email: письма приходят на почту, указанную при регистрации в Ozon Seller. Тема письма содержит номер заказа, а в теле — детали (товары, сумма, статус).
  • 📱 SMS: отправляются на телефон, привязанный к аккаунту. Подходит для срочных уведомлений (например, об отмене заказа).
  • 🤖 Telegram-боты: сторонние сервисы вроде @OzonSellerBot или Zapier могут пересылать уведомления в чат.

Чтобы включить email/SMS-уведомления:

  1. В Личном кабинете перейдите в Профиль → Настройки → Уведомления.
  2. Отметьте галочками события: «Новый заказ», «Отмена заказа», «Возврат».
  3. Укажите email и телефон для уведомлений (можно добавить несколько адресов через запятую).

6. Отслеживание заказов в FBO: особенности и инструменты

Продавцы, работающие по схеме FBO (самостоятельная доставка), сталкиваются с дополнительными задачами: нужно не только видеть заказы, но и организовывать их сборку, упаковку и передачу курьеру. Для этого в Личном кабинете есть специальные инструменты:

  • 📦 Распечатка этикеток: для каждого заказа генерируется штрихкод, который крепится на упаковку. Этикетки можно печатать по одной или пачками (до 100 штук за раз).
  • 🚚 Выбор логистического партнёра: Ozon интегрирован с СДЭК, Boxberry, ПЭК и др. В заказе указывается рекомендуемая служба доставки.
  • 📅 План сборки: инструмент для распределения заказов по датам и приоритетам (например, сначала собираются заказы с экспресс-доставкой).

Алгоритм работы с FBO-заказом:

  1. Получаете уведомление о новом заказе (email/SMS).
  2. Проверяете детали в Личном кабинете: товары, адрес покупателя, сроки доставки.
  3. Готовите товар к отправке: проверяете комплектацию, упаковываете, клеите этикетку.
  4. Передаёте заказ курьеру или везёте в пункт выдачи партнёра.
  5. В Личном кабинете подтверждаете отправку (кнопка «Отметить как переданный»).
💡

Для FBO-заказов используйте упаковку с логотипом вашего бренда — это повышает лояльность покупателей. Но убедитесь, что на коробке также есть этикетка Ozon со штрихкодом!

7. Аналитика заказов: отчёты и метрики в Ozon Seller

Просмотр заказов — это не только оперативная работа, но и аналитика для развития бизнеса. В Личном кабинете есть раздел «Аналитика», где можно изучить:

  • 📊 Динамику продаж: графики по дням/неделям с разбивкой по товарам.
  • 🗺️ Географию заказов: какие регионы наиболее активны (полезно для таргетированной рекламы).
  • 🔄 Возвраты и отмены: причины (недостача, брак, «передумал») и частота по каждому товару.
  • ⏳ Время обработки: сколько в среднем занимает сборка и отправка заказа (актуально для FBO).

Пример: если вы видите, что 20% заказов отменяются из-за «долгой доставки», стоит пересмотреть логистику (например, подключить FBS или выбрать более быстрого партнёра в FBO).

Для глубокого анализа экспортируйте отчёты в Excel и стройте сводные таблицы. Например, можно выявить:

  • Товары с самой высокой конверсией (отношение заказов к просмотрам).
  • Зависимость продаж от времени суток (когда покупатели активнее).
  • Эффективность акций и скидок (рост заказов после снижения цены).
💡

Отчёт «Топ товаров по возвратам» помогает выявить проблемные позиции. Если один и тот же товар возвращают чаще других, проверьте его описание на сайте (возможно, оно вводит покупателей в заблуждение).

FAQ: Частые вопросы о просмотре заказов на Ozon

Можно ли увидеть заказы, которые покупатель ещё не оплатил?

Нет, в Личном кабинете отображаются только оплаченные заказы. Исключение — предзаказы (если вы участвуете в акции с отложенной оплатой), но они помечены специальным статусом pending.

Как найти заказ по номеру телефона покупателя?

В Личном кабинете нет прямого поиска по телефону, но можно:

  1. Экспортировать заказы в Excel и отфильтровать по столбцу recipient_phone.
  2. Использовать API-запрос с фильтром по телефону (если у вас есть доступ к Posting API).

Обратите внимание: по правилам Ozon, продавцы не имеют права использовать телефоны покупателей для маркетинга (например, рассылки SMS с предложениями).

Что делать, если заказ исчез из списка?

Вероятные причины:

  • Заказ был отменён покупателем или Ozon (проверьте вкладку «Отменённые»).
  • Вы применили фильтр, который скрывает этот заказ (например, по дате).
  • Для FBS: товар не был резервирован на складе (например, из-за нехватки остатков).

Если заказ пропал без объяснений, обратитесь в поддержку Ozon с указанием номера заказа.

Как отследить заказ, отправленный через FBS?

Для FBS-заказов Ozon берёт на себя логистику, поэтому отследить его можно:

  • В Личном кабинете: статус обновляется автоматически (на складе → в доставке → доставлен).
  • На сайте Ozon для покупателей: по номеру заказа (покупатель может поделиться им с вами).
  • Через API: запрос GET /v2/posting/fbs/get вернёт текущий статус и историю движений.

Если заказ застрял на статусе на складе более 2 дней, уточните причину в поддержке — возможно, проблема с упаковкой или маркировкой.

Можно ли скрыть старые заказы из списка?

Да, в Личном кабинете есть фильтр по дате. Чтобы убрать старые заказы:

  1. В разделе «Заказы» нажмите «Фильтр».
  2. Укажите диапазон дат (например, последние 30 дней).
  3. Сохраните фильтр как «Избранный» для быстрого доступа.

Также можно архивировать заказы старше 6 месяцев — они останутся в системе, но не будут отображаться по умолчанию.

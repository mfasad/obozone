Как скачать отчеты из кабинета Ozon: все способы для продавцов и партнеров

Работа с маркетплейсом Ozon требует регулярного анализа данных — будь то продажи, остатки на складах, финансовые операции или отзывы клиентов. Без актуальных отчетов невозможно принимать взвешенные бизнес-решения, оптимизировать ассортимент или контролировать логистику. Однако многие продавцы сталкиваются с трудностями при попытке экспортировать данные из личного кабинета: где искать нужные разделы, какие форматы доступны и как автоматизировать процесс?

В этой статье мы разберем все официальные и альтернативные способы скачивания отчетов из Ozon Seller и Ozon Partner, включая скрытые функции интерфейса, API-методы и сторонние сервисы. Особое внимание уделим форматам файлов (Excel, CSV, PDF), ограничениям на количество запросов и нюансам работы с большими объемами данных. Если вы ищете способ получить отчеты за прошлые периоды, настроить автоматическую выгрузку или интегрировать данные с 1С — здесь найдете пошаговые инструкции и советы от практиков.

Важно: Ozon регулярно обновляет интерфейс личного кабинета, поэтому некоторые пути в меню могут незначительно отличаться. Мы актуализировали информацию по состоянию на июнь 2026 года и указали альтернативные маршруты для старых версий панели.

Почему нельзя просто скопировать данные из кабинета вручную

Многие новички пытаются выгружать отчеты путем ручного копирования таблиц из веб-интерфейса в Excel. Этот метод работает только для минимальных объемов данных и чреват ошибками:

  • 📉 Ограничение на количество строк: в браузере отображается не более 50–100 записей за раз (даже при прокрутке). Например, отчет по продажам за месяц может содержать тысячи строк — вручную их не скопируешь.
  • ⏳ Потеря времени: на копирование 100 позиций уходит 5–10 минут. Для полноценного анализа требуются данные за квартал или год.
  • 🔍 Отсутствие детализации: в интерфейсе часто скрыты дополнительные поля (например, идентификаторы заказов или складские адреса), которые доступны только в экспортируемых файлах.
  • 📊 Невозможность сводных расчетов: без структурированных данных в Excel нельзя построить графики, рассчитать динамику или применить фильтры.

Еще одна ловушка: при копировании через Ctrl+C теряется форматирование (даты превращаются в текст, числа — в строки), чтоlater приводит к ошибкам в формулах. Ozon специально блокирует такие действия для защиты данных, поэтому единственный надежный способ — использовать встроенные инструменты экспорта.

💡

Если вам срочно нужен отчет за вчерашний день, а экспорт занимает слишком много времени, попробуйте отфильтровать данные по дате прямо в интерфейсе кабинета. Например, в разделе Аналитика → Продажи установите диапазон "Последние 24 часа" — так система покажет только актуальные записи, которые проще скопировать вручную.

Способ 1: Стандартный экспорт из раздела «Аналитика»

Самый очевидный и доступный метод — скачивание отчетов прямо из меню Аналитика в личном кабинете. Здесь собраны все ключевые данные по продажам, трафику, остаткам и финансам. Рассмотрим пошаговую инструкцию:

  1. Авторизуйтесь в Ozon Seller или Ozon Partner (для партнерских программ).
  2. Перейдите в раздел Аналитика (в верхнем меню).
  3. Выберите нужный тип отчета:
    • 📈 Продажи — данные по заказам, выручке, возвратам.
    • 📦 Остатки — актуальный складской запас по товарам.
    • 💰 Финансы — выплаты, комиссии, удержания.
    • 📊 Трафик — переходы, просмотры, конверсии.
  • Настройте фильтры: укажите период (день/неделя/месяц/произвольный диапазон), выберите склады, бренды или категории товаров.
  • Нажмите кнопку Экспорт (обычно расположена в правом верхнем углу таблицы).
  • Выберите формат:
    • 📑 Excel (XLSX) — удобен для дальнейшей обработки (формулы, графики).
    • 📄 CSV — легкий формат для загрузки в базы данных.
    • 📄 PDF — только для печатных версий (не подходит для анализа).

    Скачанный файл будет содержать все поля, отображаемые в таблице на экране, плюс дополнительные технические данные (например, order_id, sku). Внимание: отчеты по продажам за период более 3 месяцев экспортируются только в формате CSV — Excel не поддерживает такое количество строк.

    Способ 2: Выгрузка через API Ozon (для автоматизации)

    Для продавцов с большими объемами данных или нуждающихся в регулярных отчетах ручной экспорт не подходит. В этом случае поможет API Ozon — программный интерфейс, позволяющий автоматически запрашивать данные в структурированном виде. Преимущества метода:

    • ⚡ Скорость: выгрузка тысяч записей за секунды.
    • 🤖 Автоматизация: можно настроить ежедневную отправку отчетов на email или в базу данных.
    • 📊 Гибкость: доступны поля, скрытые в веб-интерфейсе (например, warehouse_id или cancel_reason).
    • 🔄 Интеграция: данные легко стыкуются с 1С, Google Sheets или Power BI.

    Чтобы начать работу с API:

    1. Получите Client-ID и API-ключ:
      • Перейдите в Настройки → Интеграции → API.
      • Нажмите Создать ключ и сохраните его в безопасном месте (ключ показывается только один раз!).
  • Изучите документацию API для продавцов или

    headers = {

    "Client-Id": "ваш_client_id",

    "Api-Key": "ваш_api_ключ",

    "Content-Type": "application/json"

    }

    payload = {

    "date_from": "2026-06-01",

    "date_to": "2026-06-07",

    "metrics": ["revenue", "orders"],

    "dimensions": ["sku", "day"]

    }

    response = requests.post(url, headers=headers, json=payload)

    data = response.json()

    • Ответ придет в формате JSON, который можно конвертировать в Excel с помощью библиотек вроде pandas или онлайн-конвертеров. Обратите внимание: API имеет лимиты на количество запросов (обычно 1000 в час). При превышении вы получите ошибку 429 Too Many Requests.

    Способ 3: Отчеты через Ozon Business (для корпоративных клиентов)

    Если вы работаете от имени юридического лица или ваш оборот на Ozon превышает 1 млн ₽/месяц, у вас есть доступ к расширенной панели Ozon Business. Здесь доступны:

    • 📂 Архивные отчеты за 5+ лет (в стандартном кабинете — только 3 года).
    • 📈 Кастомизированные дашборды с возможностью сохранения шаблонов.
    • 🤝 Данные по партнерским программам (если вы участвуете в Ozon Affiliate).
    • 🔄 Автоматическая рассылка отчетов на email по расписанию.

    Чтобы перейти в Ozon Business:

    1. В обычном личном кабинете нажмите на аватар в правом верхнем углу.
    2. Выберите Переключиться на Ozon Business (если опция отсутствует, обратитесь в поддержку для подключения).
    3. В новом интерфейсе перейдите в раздел Отчеты → Мои отчеты.
    4. Создайте новый отчет или выберите из шаблонов (например, "Анализ продаж по регионам").
    5. Настройте параметры и нажмите Сформировать, затем Экспорт.

    Особенность Ozon Business: здесь можно объединять данные по нескольким аккаунтам (если у вас несколько магазинов на платформе). Также доступны отчеты по логистике FBS/FBO с детализацией по складским операциям (приемка, отгрузка, возвраты).

    • 📊 Визуализация: графики, тепловые карты, сравнительные таблицы.
    • 🔔 Уведомления: оповещения о падении продаж, низких остатках или негативных отзывах.

    Недостатки:

    • 🔒 Риски безопасности: передача данных третьим лицам требует доверия к сервису.
    • ⚙️ Сложность настройки (может потребоваться помощь специалиста).
    ⚠️ Внимание: Перед подключением стороннего сервиса проверьте, есть ли у него официальное партнерство с Ozon. Несанкционированный доступ к API может привести к блокировке аккаунта за нарушение пользовательского соглашения.

    Способ 5: Ручная выгрузка через поддержку Ozon

    В редких случаях стандартные методы не работают — например, если:

    • Нужны данные за период старше 3 лет (недоступны в интерфейсе).
    • Требуется детализация по конкретному заказу, который не отображается в отчетах.
    • Произошел сбой при экспорте (файл не формируется или содержит ошибки).

    В таких ситуациях можно обратиться в поддержку Ozon с запросом на предоставление данных. Алгоритм действий:

    1. Перейдите в раздел Помощь → Обратиться в поддержку.
    2. Выберите тему: Отчеты и аналитика → Проблемы с экспортом данных.
    3. В поле сообщения укажите:
      • 📅 Точный период, за который нужны данные.
      • 📋 Тип отчета (продажи, финансы, остатки и т. д.).
      • 🔍 Причину запроса (например, "нужно для налоговой проверки").
      • 📧 Email для отправки файла (желательно корпоративный).
  • Прикрепите скриншоты ошибок (если экспорт не работает).
  • Отправьте запрос и дождитесь ответа (обычно в течение 1–3 рабочих дней).

    • Важно: поддержка Ozon не предоставляет данные, которые:

    • 🚫 Относятся к другим аккаунтам (даже если они вашей компании).
    • 🚫 Содержат персональную информацию покупателей (ФИО, телефоны, адреса).
    • 🚫 Были удалены из системы (например, архивные заказы старше 5 лет).

    Если вам срочно нужны данные, а поддержка задерживает ответ, попробуйте:

    • 📞 Позвонить по телефону горячей линии: 8 800 333-70-00 (для продавцов).
    • 💬 Написать в Telegram-чат поддержки (ответ приходит быстрее, чем по email).
    💡

    Сupport Ozon может отказать в предоставлении данных, если запрос не обоснован или нарушает политику конфиденциальности. Всегда указывайте четкую причину (например, "аудит бухгалтерии" или "судебный запрос").

    Частые ошибки при скачивании отчетов и как их избежать

    Даже опытные продавцы сталкиваются с проблемами при экспорте данных. Рассмотрим типичные ошибки и способы их решения:

    • 🔄 Файл не формируется или висит на стадии "Обработка":
      • Причина: слишком большой период или много фильтров.
      • Решение: разбейте запрос на меньшие интервалы (например, по неделям).
    • 📉 В отчете отсутствуют данные:
      • Причина: неверно указаны даты или выбран неправильный склад (FBS вместо FBO).
      • Решение: проверьте фильтры и сравните с данными в интерфейсе.
    • 🔒 Доступ запрещен:
      • Причина: у вашей роли в аккаунте нет прав на экспорт (актуально для командной работы).
      • Решение: обратитесь к администратору аккаунта за доступом.
    • 📄 Файл поврежден или не открывается:
      • Причина: сбой при скачивании или несовместимость форматов.
      • Решение: попробуйте другой браузер или конвертируйте файл через Zamzar.
    • ⏱️ Экспорт занимает слишком много времени:
      • Причина: пиковая нагрузка на серверы Ozon (обычно по утрам).
      • Решение: запускайте выгрузку в непиковые часы (например, после 20:00).

    Если проблема повторяется, проверьте:

    • 🌐 Стабильность интернет-соединения (особенно при работе с большими файлами).
    • 🍪 Кэш браузера: очистите его или попробуйте режим инкогнито.
    • 🔄 Обновления браузера: Ozon Seller лучше работает в последних версиях Chrome или Firefox.
    💡

    Если отчет формируется больше 10 минут, не закрывайте страницу и не обновляйте ее — это прервет процесс. Дождитесь уведомления в кабинете или письма на email с ссылкой на скачивание.

    Как автоматизировать выгрузку отчетов (без API)

    Не все продавцы готовы разбираться с API, но ручной экспорт отнимает слишком много времени. В этом случае поможет автоматизация через макросы или скрипты. Рассмотрим два простых метода:

    Метод 1: Макросы в Excel + Power Query

    Если вы регулярно скачиваете отчеты в одном формате, можно настроить автоматическое обновление данных в Excel:

    1. Скачайте отчет в формате CSV или XLSX и откройте его в Excel.
    2. Перейдите на вкладку Данные → Получить данные → Из файла → Из папки.
    3. Выберите папку, куда сохраняются отчеты (например, C:\Ozon\Reports).
    4. Нажмите Преобразовать данные и настройте формат (разделители, типы данных).
    5. Сохраните запрос и настройте Обновить все при открытии файла.

    Теперь при добавлении нового файла в папку достаточно обновить данные в Excel (Данные → Обновить все), и все отчеты объединятся в одну таблицу.

    Метод 2: Скрипт на Python для скачивания файлов

    Если вы готовы к небольшому программированию, можно написать скрипт для автоматической выгрузки отчетов через веб-интерфейс. Пример на Python с использованием библиотеки selenium:

    from selenium import webdriver

    from selenium.webdriver.common.by import By
    

    import time
    


    Настройка браузера
    

    driver = webdriver.Chrome()
    

    driver.get("https://seller.ozon.ru/login")
    

    time.sleep(5)  # Ждем загрузки страницы
    


    Авторизация (замените на свои данные)
    

    driver.find_element(By.NAME, "email").send_keys("ваш_email")
    

    driver.find_element(By.NAME, "password").send_keys("ваш_пароль")
    

    driver.find_element(By.CSS_SELECTOR, "button[type='submit']").click()
    

    time.sleep(10)
    


    Переход в раздел отчетов
    

    driver.get("https://seller.ozon.ru/performance/sales")
    

    time.sleep(5)
    


    Настройка фильтров (пример: последняя неделя)
    

    driver.find_element(By.CSS_SELECTOR, ".filter-date-input").click()
    

    driver.find_element(By.CSS_SELECTOR, ".ranges li:nth-child(2)").click()  # "Последние 7 дней"
    

    time.sleep(3)
    


    Экспорт в CSV
    

    driver.find_element(By.CSS_SELECTOR, "button.export-button").click()
    

    driver.find_element(By.CSS_SELECTOR, "li.export-format-csv").click()
    

    time.sleep(10)  # Ждем формирования файла
    


    driver.quit()

    Этот скрипт:

    1. Открывает браузер и авторизуется в кабинете.
    2. Переходит в раздел продаж.
    3. Устанавливает фильтр "Последние 7 дней".
    4. Скачивает отчет в формате CSV.

    Для автоматического запуска можно добавить скрипт в Планировщик заданий Windows или использовать cron на Linux.

    ⚠️ Внимание: Автоматизация через веб-интерфейс нарушает пользовательское соглашение Ozon, если приводит к чрезмерной нагрузке на серверы. Не запускайте скрипты чаще 1 раза в час и ограничьте количество одновременно открытых вкладок.

    FAQ: Ответы на частые вопросы

    Можно ли скачать отчеты по конкретному товару (SKU)?

    Да, для этого в разделе Аналитика → Продажи:

    1. Добавьте фильтр по SKU (поле "Артикул").
    2. Укажите нужный период.
    3. Экспортируйте отчет в Excel или CSV.

    В результате вы получите данные только по выбранному товару: количество продаж, выручку, возвраты и т. д.

    Как скачать отчеты за прошлый год, если в интерфейсе доступны только 3 месяца?

    В стандартном кабинете Ozon Seller доступны данные за последние 3 года, но с ограничением: за один раз можно экспортировать не более 12 месяцев. Чтобы обойти это:

    1. Разбейте год на кварталы (январь-март, апрель-июнь и т. д.).
    2. Экспортируйте каждый квартал отдельно.
    3. Объедините файлы в Excel с помощью Power Query.

    Для данных старше 3 лет обратитесь в поддержку с обоснованным запросом.

    Почему в отчете по финансам не совпадают суммы с выплатами?

    Расхождения могут возникать по нескольким причинам:

    • 💰 Удержания Ozon: комиссии, штрафы или резервы за возвраты.
    • ⏳ Задержки выплат: деньги за заказы могут поступать с lagом в 1–3 дня.
    • 🔄 Корректировки: если заказ был отменен или возвращен после выплаты.

    Чтобы разобраться, скачайте детализированный отчет по транзакциям в разделе Финансы → Выплаты и сверьте каждую операцию.

Как найти SKU ID на Ozon: все способы для продавцов и покупателей

Идентификатор SKU ID на Ozon — это уникальный цифробуквенный код, который присваивается каждому товару в каталоге маркетплейса. Без него невозможно отследить остатки на складе, обновить цену, связать товары в FBS или FBO, а иногда даже оформить возврат. Проблема в том, что Ozon не всегда показывает SKU на видном месте: его может не быть ни в карточке товара для покупателя, ни в первых вкладках личного кабинета продавца.

В этой статье мы разберём все актуальные способы, как узнать SKU ID на Ozon в 2026 году — от ручного поиска в интерфейсе до автоматического экспорта через API или Excel-отчёты. Отдельно остановимся на нюансах для схем FBS ( когда товар хранится на складе Ozon) и FBO ( когда вы отправляете заказы сами). Если вы продавец, вам пригодятся лайфхаки для массового получения SKU по сотням позиций. Если покупатель — узнаете, как найти этот код в чеке или письме с заказом.

1. Что такое SKU ID на Ozon и зачем он нужен

Аббревиатура SKU (Stock Keeping Unit) переводится как «единица учёта запасов». На Ozon это внутренний идентификатор товара, который:

  • 🔹 Присваивается автоматически при создании карточки (даже если вы не указали свой артикул).
  • 🔹 Состоит из 10–12 символов (цифры и латинские буквы, пример: 123456789AB).
  • 🔹 Отличается от артикула продавца (vendor code) и штрихкода (barcode).

Без SKU ID вы не сможете:

  • 📦 Обновить остатки или цены через API или Excel-загрузку.
  • 🔄 Связать товары в FBS с вашей системой учёта (1С, MoySklad).
  • 📊 Получить данные по продажам в аналитических отчётах.
  • 🔙 Оформить возврат или замену товара (иногда требуется для поддержки).
⚠️ Внимание: На Ozon один и тот же товар может иметь несколько SKU ID — например, если он продаётся в разных цветах или размерах. Каждая модификация (= оферта) получает свой уникальный код.

2. Где посмотреть SKU ID в личном кабинете продавца (пошаговая инструкция)

Самый надёжный способ — найти SKU через панель продавца. Вот как это сделать:

1. Авторизуйтесь в личном кабинете Ozon Seller и перейдите в раздел Товары → Мои товары.

2. В списке найдите нужный товар и кликните по его названию, чтобы открыть карточку редактирования.

3. В верхней части страницы, под названием товара, будет блок Основная информация. Искомый SKU ID указан в строке Идентификатор товара на Ozon (см. скриншот ниже).

📌 Альтернативный путь: если товаров много, используйте фильтр. В строке поиска введите название или артикул, затем нажмите на строку с товаром — SKU отобразится в выпадающей панели справа.

Если товар продаётся в нескольких модификациях (например, футболки разных размеров), то:

  • 👕 Перейдите во вкладку Оферты в карточке товара.
  • 🔍 Для каждой модификации будет свой SKU ID (он указан в колонке Идентификатор).

3. Как узнать SKU ID по карточке товара (для покупателей)

Покупателям Ozon не показывает SKU ID напрямую в публичной карточке товара. Однако есть обходные пути:

🔹 Способ 1. Чек или email с заказом

В письме с подтверждением заказа (тема: «Ваш заказ №XXX на Ozon») или в чеке (PDF) ищите строку Артикул или Код товара. Иногда там указан именно SKU ID (но не всегда — может быть артикул продавца).

🔹 Способ 2. URL карточки товара

Откройте страницу товара в браузере и посмотрите на адресную строку. SKU ID может быть частью URL после символов /product/ или ?sku=. Пример:

https://www.ozon.ru/product/название-товара-123456789AB/

⚠️ Но: этот метод работает не для всех товаров — иногда в URL используется артикул продавца или внутренний код Ozon.

🔹 Способ 3. Обратиться в поддержку

Если вам срочно нужен SKU ID (например, для возврата), напишите в службу поддержки Ozon с просьбой предоставить его. Приложите:

  • 📄 Номер заказа.
  • 📦 Ссылку на товар.
  • 💳 Данные для идентификации (ФИО, email или телефон).

4. Массовый экспорт SKU ID: отчёты и API

Если у вас сотни или тысячи товаров, искать SKU ID вручную нереально. В этом случае поможет:

📊 Способ 1. Отчёт «Список товаров» в Excel

1. В личном кабинете перейдите в Аналитика → Отчёты → Список товаров.

2. Настройте фильтры (например, по статусу Активен) и нажмите Сформировать отчёт.

3. В полученном Excel-файле будет колонка SKU с нужными идентификаторами.

🤖 Способ 2. API Ozon

Для автоматизации можно использовать API Ozon Seller. Пример запроса для получения SKU по артикулу продавца:

GET https://api-seller.ozon.ru/v2/product/info?offer_id=ВАШ_АРТИКУЛ&product_id=ИД_ТОВАРА

Headers: Client-Id: ВАШ_CLIENT_ID


Api-Key: ВАШ_API_КЛЮЧ

В ответе ищите поле "sku". Подробнее — в документации API.

📌 Способ 3. Интеграция с 1С или MoySklad

Если вы синхронизируете товары через 1С или MoySklad, SKU ID автоматически подтягивается в поля Код Ozon или Внешний идентификатор. Проверьте настройки интеграции!

💡

Для массового экспорта SKU ID оптимально использовать отчёт «Список товаров» или API. Ручной поиск подходит только для единичных позиций.

5. Особенности SKU ID для FBS и FBO

Схема логистики (FBS или FBO) влияет на то, где и как используется SKU ID:

🚚 FBS (Fulfillment by Ozon)

  • 📦 SKU ID нужен для приёмки товара на склад Ozon. Его указывают в WS-форме (warehouse sheet) при отправке партии.
  • 🔄 Используется для связки ваших остатков с системой Ozon (чтобы маркетплейс «знал», сколько товара лежит на его складе).
  • 📊 В отчётах по FBS (Остатки на складах, Движение товаров) SKU — основной идентификатор.

🏠 FBO (Fulfillment by Merchant)

  • 📦 SKU ID не нужен для логистики (вы отправляете заказы сами), но требуется для обновления цен/остатков через API.
  • 🔙 При возвратах Ozon может запросить SKU для идентификации товара.
  • 📊 В аналитике FBO SKU используется реже — чаще оперируют артикулами продавца.
⚠️ Внимание: Если вы переходите с FBO на FBS, все товары должны иметь уникальные SKU ID в системе Ozon. Дубликаты приведут к ошибкам при приёмке на склад!

6. Частые ошибки и как их избежать

🔥 Ошибка 1. Путают SKU ID с артикулом продавца

SKU ID — это код Ozon, а артикул продавца (vendor code) вы указываете сами. Они могут совпадать, но чаще всего различаются. Всегда проверяйте, какой именно код вам нужен!

🔥 Ошибка 2. Копируют SKU из URL, не проверяя

Как мы писали выше, в адресной строке может быть не SKU ID, а артикул или другой код. Всегда сверяйте с данными из личного кабинета.

🔥 Ошибка 3. Не учитывают модификации

Если у товара есть варианты (цвет, размер), у каждого будет свой SKU ID. Нельзя использовать один код для всех модификаций!

🔥 Ошибка 4. Игнорируют регистр

SKU ID чувствителен к регистру! Коды 123456789AB и 123456789ab — это два разных товара для системы Ozon.

💡

Сохраните все SKU ID ваших товаров в отдельный Excel-файл с колонками: «Название», «Артикул продавца», «SKU Ozon», «Ссылка». Это сэкономит часы при работе с поддержкой или API.

7. Альтернативные способы (если ничего не работает)

Если стандартные методы не помогли, попробуйте:

🔍 Способ 1. Просмотр исходного кода страницы

1. Откройте карточку товара на Ozon.

2. Нажмите Ctrl+U (или правая кнопка мыши → Просмотр кода страницы).

3. В коде найдите строку "sku" (используйте поиск по странице, Ctrl+F). Часто SKU ID скрыт в JSON-данных.

📧 Способ 2. Запрос в поддержку Ozon Seller

Напишите в чат поддержки для продавцов с просьбой предоставить SKU ID для конкретных товаров. Укажите:

  • 🔗 Ссылки на товары.
  • 📌 Артикулы продавца (если есть).
  • 📝 Причину запроса (например, «нужно для интеграции с 1С»).

🛠 Способ 3. Парсинг через сервисы

Сервисы вроде Ozon Parser или DataCol могут извлекать SKU ID из карточек товаров автоматически. Это актуально для арбитражников или тех, кто работает с большими объёмами данных.

⚠️ Но: парсинг может нарушать правила Ozon. Используйте только для легальных задач (например, анализ конкурентов).

Как найти SKU ID для удалённого товара?

Если товар удалён из продажи, его SKU ID всё равно сохраняется в системе Ozon. Чтобы его узнать:

1. Перейдите в раздел «Товары → Архив».

2. Найдите товар и откройте его карточку.

3. SKU ID будет в блоке «Основная информация» (как и для активных товаров).

Если товар удалён безвозвратно, запросите SKU у поддержки, приложив доказательства (скриншоты, ссылки).

FAQ: Частые вопросы о SKU ID на Ozon

Можно ли изменить SKU ID на Ozon?

Нет, SKU ID присваивается автоматически при создании карточки и не подлежит редактированию. Если вам нужно поменять идентификатор, создайте новый товар и перенесите на него все данные.

Чем отличается SKU ID от артикула продавца (vendor code)?

SKU ID — это внутренний код Ozon, а артикул продавца вы указываете сами (например, FK-2026-BLUE). В системе маркетплейса они независимы, но могут совпадать, если вы вручную синхронизировали их.

Где взять SKU ID для товара, которого нет в моём списке?

Если товар добавлен не вами (например, через Ozon Express или партнёрскую программу), запросите SKU ID у менеджера Ozon или в поддержке. Самостоятельно его найти не получится.

Можно ли использовать один SKU ID для разных товаров?

Нет! Каждый товар (и каждая его модификация) должен иметь уникальный SKU ID. Дублирование приведёт к ошибкам в остатках, ценах и заказах.

Как найти SKU ID в мобильном приложении Ozon?

В мобильном приложении Ozon Seller SKU ID отображается так же, как и в веб-версии: откройте карточку товара и найдите поле Идентификатор товара на Ozon. В покупательском приложении SKU не показывается.
